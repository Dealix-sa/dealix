# Dealix Delivery Skill

## Command Sprint deliverables
Market Intelligence Lite · Revenue Map · Proof Register · Executive Command Brief ·
Approval Register · Next Action Board · Delivery Lite · Upsell Recommendation.

## Delivery rhythm
Source Passport → DQ score → Account scoring → Draft pack → Governance review →
Proof Pack assembly → Capital asset registration → Retainer eligibility check.
Every step records to the appropriate ledger (`docs/ledgers/`). See the `dealix-delivery` agent.

## Hard rules
- No external send. Every customer-facing action is a **draft queued for founder approval**.
- No proof without evidence; no upsell without proof; no customer name without written approval.
- Module status labels on everything claimed as delivered.

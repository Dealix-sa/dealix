# Dealix Website Skill

## Real surfaces
- `frontend/` — Next.js 15 + `next-intl` (RTL) + Tailwind + Radix. Build: `cd frontend && npm run build`.
- `landing/` — static HTML + `styles.css`.
There is **no** root `npm run build`. Improve incrementally; never rewrite from scratch.

## Page intents
Home · Arabic Home · Platform · Command Sprint · Business OS · Pricing · Industries · Security · Start.

## Reusable components
Hero · Pain Section · OS Layer Cards · Command Sprint Offer · Proof Pack Preview ·
Pricing Ladder · Governance Section · Industry Cards · CTA Band · FAQ.

## Every page
- exactly one primary CTA (Get Business OS Score / Book Diagnostic / Start Command Sprint)
- Arabic-first, RTL-ready, dark premium executive look
- no fake proof, no guaranteed revenue, no generic SaaS, no overpromising
- a module status label where it claims a capability

## After changes
`cd frontend && npm run build && npm run lint`. Report exact failures; fix if safe.

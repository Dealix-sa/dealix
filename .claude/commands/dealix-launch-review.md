# /dealix-launch-review

Produce the daily Dealix launch review. This is the founder's morning command.

## Steps
1. Read `docs/05_founder/DEALIX_LAUNCH_GO_NO_GO.md` (Launch Control System) and
   `docs/05_founder/DEALIX_EXECUTION_BOARD.md`.
2. Report the Launch Control snapshot: status, ICP, offer, price, pages ready,
   growth assets ready, proof count, paid sprint count, blockers.
3. Move the board: what's Today, This Week, Blocked, Waiting-for-founder, Ready-for-PR, Ready-for-launch.
4. Compute the Go/No-Go verdict against the gates in the Go/No-Go doc.

## Output
- One-screen launch snapshot.
- Today's 3 highest-leverage actions.
- Any item Waiting-for-founder-approval (so the founder can unblock in one pass).
- Go/No-Go: GO / HOLD / NO-GO with the binding reason.

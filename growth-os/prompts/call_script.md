# Prompt: Call Script
**Used by:** founder (manual calls only — no robo-calling)
**Output:** outputs/founder_review/ (call preparation package)

IMPORTANT: Calls require prior relationship or warm referral.
No cold robo-calls. Founder executes all calls manually.
Calling hours: 09:00-17:30 local time, Sunday-Thursday only.

---

## Call Script Generator

**Arabic opener:**

```
اكتب سكريبت مكالمة قصير لمؤسس ديليكس ليتصل بـ {buyer_title} في {company_name}.

السياق:
- الشركة: {company_name}, {sector}, {country}
- نقاط الألم المحتملة: {top_pains}
- الاتصال الأول أم المتابعة: {call_type}

هيكل السكريبت:
1. المقدمة (15 ثانية): من أنا ولماذا أتصل
2. الافتتاح (30 ثانية): ملاحظة مرتبطة بقطاعهم
3. السؤال الرئيسي (15 ثانية): هل هذا التحدي موجود لديكم؟
4. إذا نعم → اعرض التشخيص المجاني
5. إذا لا → "أقدّر وقتك، هل يمكنني إرسال ملخص قصير؟"
6. الإغلاق: شكر، الخطوة التالية

الطول الإجمالي: 90 ثانية كحد أقصى
```

**English script:**

```
Write a cold call script for the Dealix founder calling a {buyer_title} at {company_name}.

Context:
- Company: {company_name}, {sector}, {country}
- Likely pains: {top_pains}
- Call type: {first_call|follow_up_after_email}

Script structure:
1. Introduction (15 sec): Name + Dealix + why calling
2. Opener (30 sec): One specific sector observation — not "I researched your company"
3. Key question (15 sec): "Is [specific pain] something your team deals with?"
4. If YES → "We map that in a free 48-hour diagnostic — worth 10 minutes to hear more?"
5. If NO → "Fair enough — would it be okay to send a 2-page overview just in case?"
6. Close: Thank + confirm next step

Total script: 90 seconds maximum
No scripts longer than 90 seconds — if the call runs longer, it is a real conversation

Forbidden:
- "I just wanted to touch base"
- "I was wondering if I could take a moment of your time"
- Any pressure close
- Any guaranteed outcome claim
```

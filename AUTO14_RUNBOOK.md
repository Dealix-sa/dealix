# Auto14 Execution Max

## Purpose

Auto14 makes Dealix execute the maximum practical work on behalf of the founder and the client while keeping sensitive actions behind review gates.

## Automatically prepared

- client workspace
- intake questions
- workflow map template
- owner map template
- action queue
- proposal folder
- discovery script
- proof report template
- next week plan
- internal KPI summary

## Approval required

- external email send
- WhatsApp send
- calendar invite send
- final price commitment
- legal terms acceptance
- contract signature

## Run

```bash
python auto14.py
python -m pytest -q test_auto14.py
```

## Operating rule

Automate preparation, routing, reports, queues, and proof. Keep external actions and final commitments reviewed.

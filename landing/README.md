> static HTML marketing site (79 pages) with its own forbidden-claims test
> suite (`tests/test_landing_forbidden_claims.py`). It is not built or
> served by the live Next.js app that actually deploys to `dealix.me` —
> that app is `apps/web/`. This directory uses its own brand voice and
> taglines that were not reconciled against `apps/web`'s content during
> the 2026-07-06 brand/pricing consolidation (see
> `docs/ops/TASTE_SKILL_DESIGN_AUTOMATION_PLAN.md`, Finding 0). Kept for
> reference only; do not deploy or link to these pages from `apps/web`
> without first confirming with the founder which content is current.

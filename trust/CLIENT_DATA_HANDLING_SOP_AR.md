# SOP معالجة بيانات العملاء

## 1. جمع البيانات
- نجمع فقط ما هو ضروري.
- نسجل مصدر كل بيانة (source_url).
- لا نشتري قوائم بريدية غير موثقة.

## 2. التخزين
- Postgres داخل Railway.
- قاعدة بيانات منفصلة/معزولة حسب العميل عند الحاجة.
- نسخ احتياطي يومي.

## 3. الاستخدام
- البيانات تُستخدم فقط لخدمة العميل.
- لا مشاركة مع طرف ثالث.
- لا استخدام لتدريب نماذج عامة.

## 4. الاحتفاظ والحذف
- نحذف البيانات عند طلب العميل.
- نحذف البيانات غير النشطة بعد فترة محددة في العقد.

## 5. الوصول
- فريق Dealix فقط.
- سجل دخول للمراجعات.

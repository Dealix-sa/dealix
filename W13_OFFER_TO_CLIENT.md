# Dealix Growth Command Sprint

## Promise

We help the company see and control follow ups, owners, approvals, and proof in one daily command report.

## Scope

- diagnose current lead flow
- map channels and owners
- build action queue
- prepare reviewed drafts
- produce proof report
- recommend next week actions

## Inputs

channels, sample leads, owners, approval rules.

## Output

A practical operating report and queue the client can use immediately.

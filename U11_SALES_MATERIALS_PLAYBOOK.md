# U11 Sales Materials Factory

## Purpose

U11 turns the acquisition pack into reusable sales materials.

## Materials produced

- one page offer per sector
- proposal markdown per sector
- email sequence per sector
- WhatsApp draft sequence per sector
- manual LinkedIn post set per sector
- landing page copy per sector
- proof pack template per sector
- objection battlecards
- pricing menu

## Sectors

- clinics
- agencies
- training
- logistics

## Daily use

1. Run T10.
2. Run U11.
3. Pick one sector.
4. Review the one page offer.
5. Review the proposal.
6. Use only draft sequences.
7. Capture proof after every sales motion.

## Review rule

All materials are draft only and require owner review before external use.

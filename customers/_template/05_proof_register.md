**الغرض / Purpose:** ربط كل ادّعاء أو رقم بدليل مصدره، وتسمية ما هو مفقود صراحةً. / Tie every claim or number to its source evidence, and name what is missing explicitly.
**اليوم / Day:** Day 4.
**يُعِدّه / Owner:** Proof lead.

> «القيمة التقديرية ليست قيمة مُتحقَّقة» — *Estimated value is not Verified value.*
> لا proof مزيّف ولا شهادات مُختلَقة. كل بند: مصدر مُتحقَّق أو وسم «مفقود / missing».
> No fake proof, no fabricated testimonials. Each row: a verified source or marked "missing".

# Delivery Logs

This directory holds **canonical delivery logs** for the Delivery & Client
Success Operating System v1.

- `qa_score_log.csv` — every Revenue Sprint QA score, one row per delivery.

For per-client artifacts (intake, report, QA checklist, feedback, etc.) use the
private workspace folder `clients/<client_name>_private/` populated from
`clients/_template/`. Do not put client-identifying data into this directory.

# Dealix Brand Assets — Repo-Root Public

Repo-root `public/brand/` mirrors the assets served by static landing
pages and any non-Next surface (e.g. `landing/`, `dashboard/`).

See `assets/brand/source/README.md` for the source of truth and
`docs/brand/DEALIX_BRAND_SYSTEM.md` for the doctrine.

Until exports are placed here, all surfaces use the inline SVG fallback
in the React `DealixLogo` component or the textual wordmark form
"DEALIX" with the canonical tagline "Intelligent Deals. Real Growth."

# أسئلة الأمن المؤسسي

## هل Dealix يرسل رسائل تلقائيًا؟
لا. النظام يولد drafts وnext actions للمراجعة البشرية.

## هل تحفظون بيانات العملاء؟
حسب نطاق العمل. نبدأ بأقل بيانات ممكنة ونوثق نوع البيانات ومصدرها.

## هل تستخدمون AI خارجي؟
حسب الإعداد. يمكن تشغيل نماذج خارجية أو محلية/محدودة، مع توثيق ما يدخل النموذج.

## هل النتائج مضمونة؟
لا. نعرض تحسينات تشغيلية ومؤشرات متابعة، ولا نضمن إيرادًا محددًا.

## كيف تتعاملون مع المخرجات الخاطئة؟
كل مخرج حساس يحتاج مراجعة بشرية، ويتم تسجيل incidents وتحسين prompt contracts/evals.

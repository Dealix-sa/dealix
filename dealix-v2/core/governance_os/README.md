# Governance OS

## Purpose

Runtime governance for AI workflows.

## Runtime Checks

1. Source exists
2. PII checked
3. Permission mirrored
4. Claim safety checked
5. External action checked
6. Approval required when needed
7. Audit event written
8. Proof event written

## Decisions

- Allow
- Review
- Approve
- Redact
- Block
- Escalate

# Daily Agent Runbook

## صباحًا
1. شغل control tower.
2. راجع tasks queued.
3. شغل CRM/lead agents على أفضل prospects.
4. راجع outputs التي تحتاج approval.

## منتصف اليوم
1. اعتمد 5–10 drafts فقط.
2. سجل interactions.
3. حدث pipeline.

## نهاية اليوم
1. شغل CEO briefing.
2. سجل KPI.
3. افتح incident إذا ظهر output خطر.

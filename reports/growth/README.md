# reports/growth

Output directory for growth-pass reports produced by `/dealix-growth-os` and the
`growth-strategist` / `conversion-specialist` / `seo-geo-agent` agents.

Each report should capture: growth loops touched · experiments (hypothesis → metric → guardrail →
result) · funnel metrics · what routed to Business OS Score / Diagnostic / Command Sprint · next actions.

No vanity metrics. No spam. No external publishing without founder approval.

## تعريف أنواع الطلبات

| النوع | التعريف | السياسة |
|---|---|---|
| `bug` | مخرج لا يعمل كما وصفه النطاق الموقَّع | مُعالَج ضمن الدعم المجاني (14 يوماً) |
| `new_feature` | وظيفة أو مخرج لم يكونا في النطاق الأصلي | يتطلب Change Request |
| `clarification` | سؤال عن كيفية استخدام مخرج قائم | مُجاب مجاناً ضمن الدعم |
| `data_update` | إضافة بيانات جديدة أو تحديث بيانات قائمة | يتطلب Change Request إذا تجاوز النطاق |

## تعريف الأولويات

| الأولوية | التعريف | وقت الاستجابة المستهدف |
|---|---|---|
| `critical` | مخرج أساسي متوقف كلياً يؤثر على العمليات | 2 ساعة عمل |
| `high` | مشكلة تؤثر على نتائج يومية | 4 ساعات عمل |
| `normal` | طلب اعتيادي أو سؤال توضيحي | 24 ساعة عمل |
| `low` | طلب غير عاجل أو ملاحظة مستقبلية | 48 ساعة عمل |

## تعريف حالات الطلب

| رمز الحالة | المعنى |
|---|---|
| `open` | مستلَم — لم تبدأ المعالجة |
| `in_review` | قيد تقييم النوع والنطاق |
| `in_progress` | المعالجة جارية |
| `awaiting_client` | المعالجة مكتملة — في انتظار تأكيد العميل |
| `closed_resolved` | مُغلَق — حُلَّ الطلب |
| `closed_cr` | مُغلَق — حُوِّل لـ Change Request |
| `closed_out_of_support` | مُغلَق — خارج فترة الدعم المجاني |

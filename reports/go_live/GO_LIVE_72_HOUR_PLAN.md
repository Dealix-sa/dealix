# Dealix 72-Hour Commercial Go-Live Plan

## First 24 hours

- Confirm PR #788 status.
- Review SaaS beta offer.
- Pick 20 targets.
- Prepare up to 3 founder-reviewed outreach drafts.
- Book first diagnostic conversation.

## Hours 24–48

- Run two diagnostics.
- Prepare one-page proposal.
- Offer 7-Day Revenue Command Room Sprint.
- Document objections.

## Hours 48–72

- Close one paid or proof pilot.
- Build client workspace from template.
- Produce first proof pack.
- Convert proof into retainer path.

## Stop conditions

- No source URL.
- No clear pain.
- No manual review.
- Any client data risk.
- Any request for uncontrolled live external automation.

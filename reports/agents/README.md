# Dealix Agents Reports

This directory is used for generated, reviewable agent-stack and external-repo radar reports.

Default command:

```bash
make external-agent-radar
```

Generated files:

```text
reports/agents/external_agent_stack_radar.md
reports/agents/external_agent_stack_radar.json
```

These reports are advisory. They do not install dependencies, vendor repositories, enable MCP, or enable outbound actions.

# قائمة مهام الفاوندر اليومية — Dealix

> **المبدأ:** 90 دقيقة يومياً تبني شركة. ركّز على المهام التي تُنتج إيراداً أو تبني دليلاً.

---

## ☀️ الصباح (9:30 - 11:00 ص)

### 1. تقرير الفاوندر اليومي (5 دقائق)
- [ ] افتح: `docs/daily-briefs/YYYY-MM-DD.md` أو `https://api.dealix.me/api/v1/commercial/daily-brief`
- [ ] راجع: عدد التشخيصات، البرامج التجريبية النشطة، الرسائل المعلّقة
- [ ] حدّد: المهمة الأولى لليوم

### 2. متابعة الردود (15 دقيقة)
- [ ] راجع ردود الواتساب/الإيميل الواردة من أمس
- [ ] سجّل كل رد في النظام (API أو ملف التتبع)
- [ ] حدّد الردود الإيجابية كـ "حدث إثبات" إن لزم

### 3. إرسال الرسائل المعتمدة (20 دقيقة)
- [ ] افتح `data/outbox/warm_intros.jsonl` أو API `/api/v1/commercial/warm-intro/draft`
- [ ] راجع 2-3 مسودات رسائل
- [ ] أرسل يدوياً من الجهاز الشخصي (لا إرسال تلقائي)
- [ ] سجّل في ملف المتابعة: اسم الشركة، وقت الإرسال، القالب المستخدم

### 4. مكالمات warm intro (30 دقيقة)
- [ ] اتصل بـ 2-3 شركات من قائمة الأولويات
- [ ] هدف المكالمة: حجز تشخيص مجاني (15 دقيقة) أو مكالمة استكشافية
- [ ] سجّل نتيجة كل مكالمة فوراً

---

## 🌞 منتصف النهار (2:00 - 3:30 م)

### 5. إدارة البرامج التجريبية النشطة (30 دقيقة)
للبرنامج التجريبي النشط:
- [ ] ما هو اليوم الحالي في البرنامج؟
- [ ] افتح مهام اليوم: `GET /api/v1/commercial/pilot/{id}/brief?day={day}`
- [ ] أنجز مهام اليوم المحددة
- [ ] إذا كان يوم 3-5: أعدّ مسودة الرسائل وأرسلها للعميل للموافقة

### 6. تشخيص جديد (30 دقيقة — إذا وُجد طلب)
- [ ] افتح نموذج التشخيص: `POST /api/v1/commercial/diagnostic/generate`
- [ ] أرسل بيانات الشركة المستهدفة
- [ ] راجع التقرير المُولَّد وعدّله
- [ ] احفظ كمسودة في `data/proofs/diagnostics.jsonl`
- [ ] اتصل بالعميل لتسليم التشخيص

---

## 🌆 المساء (5:00 - 6:00 م)

### 7. تحديث النظام (15 دقيقة)
- [ ] سجّل نتائج اليوم في ملف التتبع
- [ ] إذا أُتمّ برنامج تجريبي: أنشئ طقم الإثبات (API `/api/v1/commercial/proof/build`)
- [ ] راجع فرص التطوير (API `/api/v1/commercial/upsell/evaluate`)

### 8. تقرير اليوم (5 دقائق)
اكتب في ملف التتبع:
```
التاريخ: YYYY-MM-DD
رسائل أُرسلت: X
ردود واردة: X
مكالمات أُجريت: X
تشخيصات مُنجزة: X
إيراد اليوم: X ريال
```

## 🚫 ما لا تفعله أبداً

- لا ترسل رسائل مجمّعة (رسالة واحدة لكثيرين)
- لا ترسل دون مراجعة المسودة
- لا تدّعي نتيجة لم تحدث
- لا تُشهر باسم عميل دون موافقته الخطية
- لا تبدأ تسليم برنامج تجريبي قبل تأكيد الدفع

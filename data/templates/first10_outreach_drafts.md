## قبل الإرسال — تذكير الفاوندر / Before sending — founder reminder

1. **راجع كل مسودة شخصياً** واستبدل كل العناصر النائبة. لا إرسال آلي. / **Review each draft personally** and replace every placeholder. No automation.
2. **أرسل عبر القناة التي تستخدمها العلاقة أصلاً** (واتساب أو بريد) — لا قناة جديدة لأول رسالة. / **Send via the channel the relationship already uses** — no new channel for a first message.
3. **سجّل أساس العلاقة** (متى/أين التقيتما) كسجل موافقة في `engagement_ledger`. / **Log the relationship basis** as a consent record.
4. **إن طلب أحدهم سحباً / سبام / ضماناً** → REJECT + بديل آمن، ولا تتابع. / **If anyone asks for scraping / spam / a guarantee** → REJECT + safe alternative, and do not follow up.
5. **بعد أي رد بـ "أخبرني أكثر"** → شغّل `qualify(...)` قبل أي عرض. / **After any "tell me more" reply** → run `qualify(...)` before any proposal.

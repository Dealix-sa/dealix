# RCMax Practical Revenue Command

## Purpose

RCMax turns the full Dealix commercial stack into a practical daily command for selling and serving companies.

## What it connects

- GTM launch kit
- revenue machine
- client acquisition pack
- sales materials factory

## What it outputs

- top account
- matched offer
- daily goal
- proof target
- operating day plan
- service blueprint
- proposal folder
- pricing guidance
- risk gates
- first 10 companies

## Run

```bash
python rcmax.py
python -m pytest -q test_rcmax.py
```

## Client service

The client gets workflow map, owner map, action queue, draft routes, proof report, and next plan.

## Rule

RCMax prepares the work. External actions and final commitments remain reviewed first.

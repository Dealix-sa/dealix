# Dealix Launch Package V12 — Scale & Operating Cadence Layer

V12 turns Dealix from first-revenue sprint into a repeatable operating company. It adds weekly cadence, portfolio review, sales capacity, hiring cadence, partner quotas, margin controls, automation backlog, scale scorecards, and a management dashboard.

Run:

```bash
python scripts/dealix_v12_readiness_check.py
python scripts/dealix_weekly_operating_rhythm.py
python scripts/dealix_sales_capacity_planner.py
python scripts/dealix_client_portfolio_review.py
python scripts/dealix_scale_dashboard.py
```

Core principle: scale only what is proven, measured, profitable, and safe.

# مراجعة محفظة العملاء

## تصنيف العملاء
- Green: يدفع، يستخدم، يوافق على proof، مرشح توسع.
- Yellow: قيمة غير واضحة أو تواصل ضعيف أو قبول متأخر.
- Red: scope creep، تأخير دفع، توقعات غير واقعية.

## قرارات المحفظة
- Expand: زيادة retainer أو إضافة module.
- Stabilize: تثبيت scope وتقليل الغموض.
- Rescue: تدخل CEO / proof report / reset expectations.
- Exit: إغلاق منظم إذا العميل يستهلك أكثر من قيمته.

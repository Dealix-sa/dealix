# Z7 Routing Layer

Z7 turns M6 decisions into prepared draft routes.

## Channels

- crm_task
- whatsapp_draft
- email_draft
- calendar_draft
- proposal_note
- proof_update

## Run

```bash
python z7_router.py
python -m pytest -q z7_check.py
```

## Output

```text
reports/leadership/z7/latest.json
reports/leadership/z7/latest.md
```

All routes stay in draft mode and require review.

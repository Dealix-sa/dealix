# Checklist جاهزية التمويل والشراكات

## قبل أي محادثة تمويل
- [ ] عرض مختصر من صفحة واحدة.
- [ ] traction ledger محدث.
- [ ] proof vault فيه أدلة منظمة.
- [ ] weekly metrics آخر 4 أسابيع.
- [ ] enterprise trust center واضح.
- [ ] demo narrative جاهز.
- [ ] pricing ladder واضح.
- [ ] استخدام الأموال المقترح واضح.

## لا تبدأ تمويل جدي قبل
- وجود دليل طلب حقيقي.
- وجود عميل/تجربة مدفوعة أو LOI.
- وضوح تكلفة التسليم.
- وضوح أول قطاع مستهدف.

## مخرجات مطلوبة
- Investor memo
- One-page company brief
- Metrics snapshot
- Proof packet
- Data room index

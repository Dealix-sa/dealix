## 2. Purpose Limitation — قيد الغرض

**2.1** The Processor shall process personal data only on documented instructions from the Controller.

يعالج المعالج البيانات الشخصية فقط وفق التعليمات الموثَّقة الصادرة من المتحكم.

**2.2** The Processor shall not process personal data for any purpose other than the Project purpose, including but not limited to: training AI or machine learning models, marketing, analytics, or sharing with third parties.

لا يعالج المعالج البيانات الشخصية لأي غرض غير غرض المشروع، بما في ذلك على سبيل المثال لا الحصر: تدريب النماذج، التسويق، التحليلات، أو المشاركة مع أطراف ثالثة.

## 4. Technical and Organizational Measures — التدابير التقنية والتنظيمية

The Processor commits to the following measures for protecting personal data:

يلتزم المعالج بالتدابير التالية لحماية البيانات الشخصية:

**4.1 Access control:** Access to personal data restricted to personnel with documented need-to-know. Individual credentials only — no shared accounts.

**4.2 Encryption:** Personal data encrypted in transit (TLS 1.2 minimum) and at rest (AES-256 or equivalent).

**4.3 Storage:** Personal data stored only in [APPROVED_STORAGE_LOCATION] within [GEOGRAPHIC_REGION]. No personal data stored in LLM prompts, AI conversation logs, or model training pipelines.

**4.4 Deletion:** Personal data deleted or returned to Controller within 90 days of project close (or earlier as agreed). Written confirmation of deletion provided to Controller.

**4.5 Breach response:** Processor notifies Controller within 24 hours of discovering any personal data breach. Full incident report within 72 hours.

# Dealix — Lost Deal Taxonomy
# تصنيف الصفقات الخاسرة

**Version:** 1.0 | Document every lost deal with a reason. Patterns reveal strategy fixes.

---

## Lost Reason Codes

| Code | Reason | What to Learn |
|------|--------|---------------|
| L01 | Budget — no budget available | Qualify budget earlier in discovery |
| L02 | Budget — competitor cheaper | Review value communication; don't reduce floor price |
| L03 | Timing — not right now | Add to nurture; follow up in 90 days |
| L04 | Timing — internal project freeze | Follow up when situation changes |
| L05 | Decision — wrong contact, can't reach decision-maker | Improve contact targeting |
| L06 | Decision — committee decision, no champion | Build internal champion earlier |
| L07 | Competitor — chose a different vendor | Which vendor? What was their pitch? |
| L08 | Competitor — chose to build in-house | Common at large enterprises; accept and move on |
| L09 | Fit — wrong sector or use case | Review routing; was this Tier C/D from start? |
| L10 | Fit — company too small | Floor price is the floor; don't discount |
| L11 | Trust — data concerns unresolved | Improve governance communication |
| L12 | Trust — AI skepticism, not ready | Re-engage in 6 months with proof points |
| L13 | Proposal — scope too large, overwhelming | Better scoping and phasing in proposal |
| L14 | Proposal — price too high | Review floor price applicability; was scope appropriate? |
| L15 | Process — no response after proposal | Improve follow-up cadence |
| L16 | Process — ghosted after call | Improve post-call follow-up |
| L99 | Unknown | Investigate; unknown is a failure of process |

---

## Lost Deal Review Ritual

Every week, review lost deals from the past 7 days:
- What was the reason?
- Could we have prevented it? At which stage?
- Does the pattern suggest a process or messaging fix?
- Should we re-engage in 30/60/90 days?

Document in the opportunity record and update the pipeline.

---

## Re-Engagement Rules

| Lost Reason | Re-Engage After |
|-------------|-----------------|
| L03 / L04 (timing) | 60-90 days |
| L11 / L12 (trust) | 6 months |
| L05 / L06 (wrong contact) | 30 days with different contact |
| L01 / L10 (no budget) | Only if signals change |
| L07 (chose competitor) | 12 months |
| L08 (build in-house) | 18 months |
| L09 (poor fit) | Do not re-engage |

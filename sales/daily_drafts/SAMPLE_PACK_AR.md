# حزمة مسوّدات جاهزة — Ready Sample Pack (Warm Intros + Follow-ups)

> **مسوّدات فقط — لا شيء يُرسَل من هنا.** انسخ، خصّص، أرسِل **يدويًا** من حسابك الشخصي.
> استبدل العناصر النائبة: `[الاسم]` / `[الشركة]` / `[القطاع]` / `[المدينة]` / `[سبب التواصل]`.
> Placeholders to replace: `[Name]` / `[Company]` / `[Sector]` / `[City]` / `[Why-now]`.
> القاعدة: تواصل دافئ فقط (تعرفك الجهة أو هناك إحالة). لا واتساب بارد. لا scraping.

**ربط بالسلّم — Ladder mapping:** كل مسوّدة تقود إلى درجة واحدة واضحة:
- **D** = التشخيص المجاني (٠ ر.س، ٢٤ ساعة) — Free Mini Diagnostic.
- **S** = Sprint إثبات الإيرادات (٤٩٩ ر.س، ٧ أيام) — Revenue Proof Sprint.
- **C** = Command Sprint (٧٬٥٠٠–١٥٬٠٠٠ ر.س، لمرة واحدة، بقيادة المؤسس) — أعلى من السلّم الذاتي.

## أ) مسوّدات التعريف الدافئ — Warm-Intro Drafts (5)

### مسوّدة ١ — تعريف عبر شبكة مباشرة → التشخيص المجاني (D)

**AR**
```
السلام عليكم [الاسم]،
جمعتنا [سبب التواصل] — وأتابع شغل [الشركة] في [القطاع].
أبني خدمة Revenue Intelligence لشركات B2B سعودية: تشخيص قدرة تشغيلية،
بيانات نظيفة، فرص مرتبة، ومسودات عربية جاهزة — كله بموافقة بشرية، بدون
scraping ولا واتساب بارد ولا وعود.
أقدّم تشخيصًا مجانيًا (صفحة واحدة، خلال ٢٤ ساعة): ٣ فرص مرتبة + مسودة رسالة
+ خطر يُتجنّب. تحب أرسله لك أو لأحد في فريق [الشركة]؟
```
**EN**
```
Hi [Name],
We connected through [Why-now] — and I follow [Company]'s work in [Sector].
I build Revenue Intelligence for Saudi B2B companies: operating-capability
diagnostic, clean data, ranked opportunities, ready Arabic drafts — all under
human approval, no scraping, no cold WhatsApp, no guarantees.
I offer a free one-page diagnostic (within 24h): 3 ranked opportunities + one
message draft + one risk to avoid. Want it for you or someone on [Company]'s team?
```

### مسوّدة ٢ — إحالة من طرف ثالث → التشخيص المجاني (D)

**AR**
```
مرحبًا [الاسم]،
[سبب التواصل] اقترح أتواصل معك. لا أبيع شيئًا الآن — أبحث عن ٢–٣ شركات B2B
تجرّب تشخيصًا مجانيًا يكشف أين تتسرّب الفرص في خط مبيعاتها.
المخرج صفحة واحدة خلال ٢٤ ساعة، بلا التزام. تناسبك [الشركة]؟
```
**EN**
```
Hello [Name],
[Why-now] suggested I reach out. I'm not selling anything yet — I'm looking for
2–3 B2B companies to try a free diagnostic that shows where opportunities leak
in their sales pipeline.
One page, within 24h, no commitment. Would [Company] be a fit?
```

### مسوّدة ٣ — معرفة مهنية سابقة → Sprint ٤٩٩ (S)

**AR**
```
أهلًا [الاسم]،
من فترة وأنا متابع [الشركة] في [القطاع]. عندي عرض عملي قد يفيدك:
Sprint إثبات إيرادات خلال ٧ أيام بـ٤٩٩ ر.س — Company Brain، أفضل ١٠ فرص مرتبة،
٥ مسودات عربية، وProof Pack. التزامنا: إن لم نُظهر ≥١٠ فرص، نواصل العمل بلا مقابل.
استرداد كامل خلال ١٤ يومًا. نبدأ بمكالمة ٢٠ دقيقة؟
```
**EN**
```
Hi [Name],
I've been following [Company] in [Sector] for a while. I have a concrete offer
that may help: a 7-day Revenue Proof Sprint for 499 SAR — Company Brain, top 10
ranked opportunities, 5 Arabic drafts, and a Proof Pack. Our commitment: if we
don't surface ≥10 opportunities, we keep working at no charge. Full refund within
14 days. Shall we start with a 20-minute call?
```

### مسوّدة ٤ — لقاء فعالية/مؤتمر → التشخيص ثم Sprint (D→S)

**AR**
```
[الاسم]، سعدت بلقائك في [سبب التواصل].
كما ذكرت، أشتغل على Revenue Intelligence لشركات B2B سعودية. أقترح نبدأ صغيرًا:
تشخيص مجاني (٢٤ ساعة) — وإن أعجبك العائد، Sprint ٤٩٩ ر.س يثبت القيمة قبل أي التزام.
أرسل لك نموذج التشخيص؟
```
**EN**
```
[Name], good to meet you at [Why-now].
As I mentioned, I work on Revenue Intelligence for Saudi B2B companies. Let's
start small: a free diagnostic (24h) — and if you like the signal, a 499 SAR
Sprint proves value before any commitment. Shall I send the diagnostic form?
```

### مسوّدة ٥ — صانع قرار تنفيذي → Command Sprint (C)

**AR**
```
السلام عليكم [الاسم]،
بحكم دورك في [الشركة]، أعرض مسارًا للقيادة مباشرة: Command Sprint بقيادة المؤسس،
لمرة واحدة (٧٬٥٠٠–١٥٬٠٠٠ ر.س)، نحوّل عملية إيرادات واحدة إلى تدفق مدعوم بالذكاء
الاصطناعي بنقطة موافقة بشرية، مع خريطة عملية، SOP، وProof Pack. ليس خدمة مُدارة
كاملة — بل طبقة تأسيس تنفيذية فوق السلّم الذاتي. نبدأ بمكالمة تحديد نطاق ٢٠ دقيقة؟
```
**EN**
```
Hello [Name],
Given your role at [Company], I'm offering a direct executive path: a founder-led,
one-time Command Sprint (7,500–15,000 SAR) that turns one revenue process into an
AI-assisted flow with a human approval point — plus a process map, SOP, and Proof
Pack. Not a full managed service — a premium setup layer above the self-serve ladder.
Start with a 20-minute scoping call?
```

## ب) مسوّدات المتابعة — Follow-up Drafts (3)

### متابعة +٢ — تذكير لطيف (Gentle bump)

**AR**
```
[الاسم]، أعيد الرسالة لأعلى تحسّبًا أنها مرّت سريعًا. لا ضغط — فقط أحببت
أتأكد إن وصلتك فكرة التشخيص المجاني. أكمل أو أتركها لك؟
```
**EN**
```
[Name], floating this back up in case it slipped by. No pressure — just making
sure the free-diagnostic idea reached you. Shall I continue or leave it with you?
```

### متابعة +٥ — قيمة مضافة (Value-add, case-safe)

**AR**
```
[الاسم]، بدون متابعة بيع — مثال case-safe (قالب افتراضي): شركة B2B في [القطاع]
كان عندها فرص مبعثرة في جداول؛ بعد ترتيبها ظهرت ٣ فرص ذات أولوية واضحة خلال أسبوع.
نتيجة تقديرية لا مضمونة. تحب أطبّق نفس العدسة على [الشركة] عبر التشخيص المجاني؟
```
**EN**
```
[Name], no sales chase — a case-safe example (hypothetical template): a B2B firm
in [Sector] had opportunities scattered across spreadsheets; once organized, 3
clear-priority opportunities surfaced within a week. Estimated, not guaranteed.
Want me to apply the same lens to [Company] via the free diagnostic?
```

### متابعة +١٠ — إغلاق مهذّب (Polite close)

**AR**
```
[الاسم]، سأتركها لك هنا حتى لا أثقل عليك. الباب مفتوح متى ناسبك التوقيت —
ابدأ بالتشخيص المجاني وقتما تجهز. شكرًا على وقتك، وأتمنى التوفيق لـ[الشركة].
```
**EN**
```
[Name], I'll leave it here so I'm not crowding your inbox. The door stays open
whenever timing suits — start with the free diagnostic when you're ready. Thanks
for your time, and best of luck to [Company].
```

## قائمة تحقق قبل الإرسال — Pre-send checklist

- [ ] العلاقة دافئة فعلًا (تعرفني الجهة أو هناك إحالة) — relationship is genuinely warm.
- [ ] العناصر النائبة مستبدلة بالكامل — all placeholders replaced.
- [ ] درجة السلّم المقصودة واضحة (D / S / C) — intended rung is clear.
- [ ] لا وعود مبيعات؛ "فرص مُثبتة بأدلة" — no sales promises.
- [ ] سأرسل يدويًا من حسابي وأحدّث `pipeline_tracker.csv` — manual send + log.

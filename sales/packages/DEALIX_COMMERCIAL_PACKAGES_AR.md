# حزم Dealix التجارية

كل حزمة هي تطبيق مركّز لسلم العروض القانوني (انظر `sales/PRICING_AND_OFFER_LADDER_AR.md`).
الدخول لأي حزمة يبدأ دائماً بالتشخيص المجاني (30 دقيقة).

## 1. Review Intelligence OS — نظام ذكاء التقييمات

نظام لإدارة وتحليل التقييمات والردود وتحويل صوت العميل إلى قرارات ومتابعات وفرص مبيعات.

**المخرجات:**
- مراقبة تقييمات Google/Maps
- مسودات ردود عربي/إنجليزي (بموافقة بشرية)
- تقرير سمعة شهري باتجاهات رضا العملاء

**مسار الدخول:** Micro Sprint (499 ريال) لتدقيق أولي → Managed AI Operations
(2,999–4,999 ريال/شهر) للتشغيل المستمر.

## 2. Command Center OS — غرفة القيادة

لوحة قيادة تنفيذية تربط المبيعات والمتابعة والعمليات والقرارات اليومية في مكان واحد.

**المخرجات:**
- صفحة قرار واحدة بخمسة مؤشرات KPI
- مالك معيّن لكل مؤشر
- قوالب مراجعة يومية وأسبوعية

**مسار الدخول:** Transformation Diagnostic Sprint (7,500–25,000 ريال) لتحديد
المؤشرات الصحيحة → Managed AI Operations للتشغيل.

## 3. WhatsApp Revenue OS — نظام إيراد الواتساب

نظام يحول واتساب من محادثات عشوائية إلى pipeline قابل للقياس: التقاط lead،
تأهيل، متابعة، حجز، اعتراضات — كل الإرسال بموافقة بشرية.

**المخرجات:**
- قواعد توجيه وتأهيل الاستفسارات
- تسلسلات متابعة جاهزة (مسودات)
- طابور موافقات — لا إرسال تلقائي أبداً

**مسار الدخول:** Data Intelligence Pack (1,500 ريال) لبناء القاعدة →
Transformation Diagnostic Sprint لبناء النظام الكامل.

## 4. Proposal & Follow-up OS — نظام العروض والمتابعة

نظام عروض ومتابعات يحول الاهتمام إلى إغلاق عبر قوالب، تذكيرات، اعتراضات، وتقارير.

**المخرجات:**
- قوالب عروض جاهزة حسب القطاع
- إيقاع متابعة موثق لكل عرض مُرسل
- بنك اعتراضات وردود معتمدة

**مسار الدخول:** Micro Sprint (499 ريال) لأول قالب → Managed AI Operations
للإدارة المستمرة.

## 5. Custom AI Operations System — نظام تشغيل مخصص

نظام مخصص للشركات حسب عملياتها: CRM، dashboards، automations، integrations، reporting.

**المخرجات:**
- تحليل وتصميم كامل للعمليات
- بناء وتكامل مع أدواتك الحالية
- تدريب الفريق + دعم 90 يوم + تسليم الملكية

**مسار الدخول:** Custom Enterprise System (25,000–100,000+ ريال) —
يسبقه دائماً Transformation Diagnostic Sprint.

## التسعير حسب سلم العروض

| الغرض | العرض | السعر |
|--------|-------|--------|
| نقطة البداية لكل حزمة | التشخيص المجاني | مجاني (30 دقيقة) |
| إثبات سريع | Micro Sprint | 499 ريال |
| أصل بيانات جاهز | Data Intelligence Pack | 1,500 ريال |
| تشغيل شهري لأي حزمة | Managed AI Operations | 2,999–4,999 ريال/شهر |
| تشخيص وبناء خارطة الحل | Transformation Diagnostic Sprint | 7,500–25,000 ريال |
| نظام مؤسسي كامل | Custom Enterprise System | 25,000–100,000+ ريال |

الأسعار غير شاملة VAT. لا ضمانات نتائج — نستخدم لغة الفرضيات: نتوقع،
الهدف هو، سنقيس.

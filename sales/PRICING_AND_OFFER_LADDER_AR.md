# سلم العروض والأسعار — Dealix

المصدر القانوني الوحيد للأسعار. أي مستند بيعي أو صفحة موقع يجب أن يطابق هذا السلم.

## السلم الأساسي (6 درجات)

## 1. التشخيص المجاني — مجاني (30 دقيقة)

- محادثة سريعة نحدد فيها أين تضيع الفرص
- تحديد 3 نقاط تسرب رئيسية
- توصية بأول نظام مناسب
- وضوح هل التشخيص التحولي المدفوع مناسب أم لا

## 2. Micro Sprint — 499 ريال (1–2 يوم)

- حل تشغيلي واحد سريع وملموس
- توثيق الحل وطريقة الاستخدام
- تقرير مبدئي بفرص إضافية

## 3. Data Intelligence Pack — 1,500 ريال (2–3 أيام)

- قاعدة بيانات عملاء محتملين (100+ سجل)
- تصنيف حسب القطاع والأولوية
- تقرير جودة البيانات
- خريطة التواصل الأولى

## 4. Managed AI Operations — 2,999–4,999 ريال/شهر

- تشغيل يومي للنظام
- تقرير أسبوعي بالأداء والتوصيات
- دعم واتساب مباشر
- مراجعة شهرية مع المؤسس
- مسار موسع حتى 15,000 ريال/شهر حسب النطاق (retainer)

## 5. Transformation Diagnostic Sprint — 7,500–25,000 ريال (3–7 أيام)

**نقطة الدخول المدفوعة الرئيسية.**

- Workflow Map — خريطة العمليات الحالية
- Leakage Map — أين يتسرب الإيراد وكم يكلف
- KPI Model — الأرقام التي يجب متابعتها
- First System Recommendation — أول نظام يستحق البناء
- Implementation Quote — السعر والجدول الزمني
- 14-Day Pilot Plan — كيف تبدأ خلال أسبوعين

**عرض تجريبي (Beta):** سبرينت غرفة قيادة الإيراد — 7 أيام بسعر 5,000–12,000 ريال.

## 6. Custom Enterprise System — 25,000–100,000+ ريال (4–12 أسبوع)

- تحليل العمليات الكامل
- تصميم وبناء نظام مخصص متكامل
- تدريب الفريق
- دعم 90 يوم بعد الإطلاق
- توثيق كامل وتسليم الملكية

## ملاحظات

- الأسعار غير شاملة VAT.
- الدفع: 50% مقدماً و50% عند التسليم؛ الاشتراك الشهري يُدفع مقدماً.
- لا نضمن نتائج محددة — نستخدم لغة الفرضيات: نتوقع، الهدف هو، سنقيس.
- لا يبدأ أي تنفيذ أو إرسال خارجي بدون موافقة صريحة من العميل والمؤسس.

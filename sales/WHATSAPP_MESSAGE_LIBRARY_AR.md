# Dealix — مكتبة رسائل واتساب (عربي)

> **قاعدة ذهبية**: كل الرسائل مسودات. المؤسس يراجع قبل الإرسال. لا إرسال تلقائي.
> كل رسالة تتضمن خيار الإيقاف الفوري.

---

## 1. رسائل التعريف الأول

### 1A — تعريف قصير (للشركات الكبيرة)
```
السلام عليكم،

أنا سامي من Dealix — نساعد شركات B2B على تنظيم متابعة العملاء وزيادة الإيراد من خلال أنظمة ذكاء اصطناعي عملية.

عندي سؤال سريع: كيف تتابعون الاستفسارات الواردة الآن؟

إذا غير مناسب التواصل، قولي "إيقاف" وأوقف فورًا.
```

### 1B — تعريف مع قيمة (للشركات المتوسطة)
```
السلام عليكم [الاسم]،

شفت شركتكم [اسم الشركة] في [القطاع] — يبدو إنكم تخدمون عملاء جادين.

في Dealix، نساعد شركات مثلكم تعرف أين تضيع الفرص وتوقف نزيف الإيراد.

لو مناسب، أشارككم مثال من [القطاع] في 2 دقيقة.
وإذا غير مناسب، قولي "لا شكرًا" وأوقف فورًا.
```

### 1C — تعريف بعد إحالة
```
السلام عليكم [الاسم]،

زميلكم [اسم المحيل] ذكرني بكم.

أنا سامي من Dealix — نساعد شركات B2B تنظم عمليات المتابعة وتزيد الإيراد.

هل مناسب نتكلم 10 دقائق هذا الأسبوع؟
```

---

## 2. رسائل المتابعة

### 2A — متابعة بعد يومين (بلا رد)
```
السلام عليكم [الاسم]،

أتابع معكم بخصوص رسالتي السابقة.

أقدر أشارككم تشخيص سريع من صفحة واحدة يوضح أين تتسرب الفرص في عمليات مثل عملياتكم.

هل يناسبكم؟
```

### 2B — متابعة بعد أسبوع
```
السلام عليكم [الاسم]،

آخر متابعة مني.

كثير شركات في [القطاع] تخسر فرصًا لأن المتابعة اليدوية لا تكفي مع النمو.

لو مناسب نتكلم في أي وقت، أنا هنا.
وإذا مو مناسب، لن أتابع بعدها — وأتمنى لكم التوفيق.
```

---

## 3. رسائل بعد إشارة إيجابية

### 3A — رد على "يهمني"
```
شكرًا [الاسم]!

ممتاز — أقدر أرسل لكم نموذج تشخيص أولي من 6 أسئلة يساعدنا نفهم وضعكم ونحدد الخطوة التالية.

هل تفضلون نموذج واتساب أو جوجل فورم؟
```

### 3B — رد على "أخبرني أكثر"
```
بكل سرور [الاسم].

Dealix يساعدكم في 3 أشياء:

1️⃣ تشخيص أين تضيع الفرص (3-7 أيام)
2️⃣ بناء نظام متابعة يومي واضح
3️⃣ تقارير أسبوعية تعرفون منها الأولويات

البداية تكون بتشخيص مجاني أو مدفوع (1,500 SAR).

ما هو تحديكم الأكبر الآن؟
```

### 3C — تأكيد موعد الاجتماع
```
شكرًا [الاسم]!

تأكيد موعدنا:
📅 [التاريخ]
⏰ [الوقت]
📞 [رابط زووم أو رقم الاتصال]

المدة: 30 دقيقة.

نناقش:
• تحدياتكم الحالية
• مثال نظام مشابه بنيناه
• الخطوة التالية إذا كان فيه تناسب

نراكم قريبًا!
```

## 4. رسائل إثبات القيمة

### 4A — مثال من القطاع
```
السلام عليكم [الاسم]،

في آخر تشخيص عملناه لشركة في [القطاع]، اكتشفنا:

📌 17 عميل محتمل لم يُتابَع معهم منذ 10 أيام+
📌 فريق المبيعات يقضي 3 ساعات يوميًا في إرسال رسائل يدوية
📌 لا يوجد تقرير يومي واضح بالأولويات

بنينا لهم نظام في 7 أيام — الآن يتابعون تلقائيًا ويعرفون أين كل فرصة.

تبون أشاركم الملخص؟
```

### 4B — سؤال تشخيصي سريع
```
سؤال سريع [الاسم]:

لو جاءكم 50 استفسار هذا الأسبوع — كيف تعرفون:
✅ من يحتاج متابعة عاجلة؟
✅ من رد ومن لم يرد؟
✅ كم فرصة ضاعت الأسبوع الماضي؟

لو الإجابة "نستخدم واتساب وإكسل" — عندنا حل يستحق 30 دقيقة.
```

## 5. رسائل إدارة الاعتراضات

### 5A — رد على "السعر غالي"
```
أفهمكم [الاسم].

السعر يبدأ من 1,500 SAR للتشخيص — وهو مخصص لتحديد أين الخسارة الحقيقية.

لو التشخيص ما ظهّر قيمة واضحة، لا نكمل. ما فيه التزام.

هل الميزانية هي العائق الوحيد؟
```

### 5B — رد على "عندنا نظام"
```
ممتاز [الاسم] — هذا يسهّل الأمور.

التشخيص ما يغيّر أنظمتكم الحالية — يكشف فقط أين تتسرب الفرص منها.

كثير شركات وجدت فجوات في أنظمة كانوا يعتقدون إنها تعمل مثالية.

هل تفضلون نحدد 30 دقيقة نشوف أين الفجوات؟
```

### 5C — رد على "مو الوقت المناسب"
```
أفهم تمامًا [الاسم].

متى يكون الوقت مناسبًا؟ [الربع القادم / بعد رمضان / بعد X]

أضعكم في التقويم وأتواصل وقتها.

شكرًا على وقتكم.
```

## 6. رسائل الإغلاق والعرض

### 6A — إرسال العرض
```
السلام عليكم [الاسم]،

كما وعدت، إليكم العرض المبدئي:

[رابط أو ملف PDF]

يتضمن:
✅ خريطة تسرب الإيراد لشركتكم
✅ التوصية الأولى
✅ خطة Pilot 7 أيام
✅ السعر والجدول الزمني

أنا متاح لأي سؤال أو تعديل.
```

### 6B — متابعة بعد إرسال العرض (3 أيام)
```
السلام عليكم [الاسم]،

أتابع بخصوص العرض الذي أرسلته.

هل راجعتموه؟ هل في أسئلة؟

لو احتجتم تعديل في الأسعار أو النطاق — نتكلم.
```

### 6C — إغلاق الصفقة
```
السلام عليكم [الاسم]،

ممتاز — يسعدنا نبدأ معكم.

الخطوات التالية:
1. نرسل لكم عقد التشخيص (صفحة واحدة)
2. تأكيد الدفع الأول
3. نبدأ التشخيص خلال 48 ساعة

هل هذا يناسبكم؟
```

## 7. رسائل الخدمة والتسليم

### 7A — بداية التشخيص
```
السلام عليكم [الاسم]،

نبدأ تشخيصكم اليوم!

سيستغرق 3-7 أيام.
خلال هذه الفترة قد نطلب منكم:
- وصول قراءة فقط لأدواتكم
- لقاء 30 دقيقة للتوضيح

المخرج: وثيقة تشخيص + 3 توصيات أولوية + خطة Pilot.

أي سؤال؟
```

### 7B — تقرير تقدم أسبوعي
```
السلام عليكم [الاسم]،

تحديث أسبوعي على مشروعكم:

✅ ما تم: [الإنجازات]
🔄 جارٍ: [المهام الحالية]
📅 الأسبوع القادم: [الخطوات التالية]

أي تعليقات أو تغييرات؟
```

### 7C — تسليم المخرجات
```
السلام عليكم [الاسم]،

يسعدنا إرسال مخرجات تشخيصكم:

📄 [رابط التقرير]

يتضمن:
• خريطة تسرب الإيراد
• الأولويات الثلاث
• توصية النظام المناسب
• خطة Pilot 7 أيام

نحدد اجتماع 30 دقيقة لمراجعته معًا؟
```

## 8. رسالة الإيقاف الفوري

**إذا طلب أي شخص الإيقاف:**

```
شكرًا على ردكم.
تم إيقاف التواصل فورًا.
لن نراسلكم مرة أخرى.
```

**إجراء فوري**: تسجيل `do_not_contact` في prospects.csv وsuppression_list.csv.

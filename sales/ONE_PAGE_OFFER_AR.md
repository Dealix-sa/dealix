# صفحة عرض واحدة — Dealix

## ما هو Dealix؟

Dealix شركة سعودية تبني أنظمة تشغيل بالذكاء الاصطناعي لشركات B2B. لا نبيع أدوات. نبني أنظمة تشغيل تعمل يومياً في شركتك — مع موافقة بشرية قبل أي إرسال.

## المشكلة

فريقك يبيع وينجز كل يوم، لكنك لا ترى الصورة كاملة. المحادثات مبعثرة. المتابعات تضيع. العروض تجلس بلا رد. وأنت كمؤسس تتخذ قرارات بناءً على حدسك، لا بيانات.

## الحل

8 أنظمة تشغيل، كل واحد يحل مشكلة محددة:

1. **Revenue Command Room OS** — غرفة قيادة ترى كل صفقة، كل متابعة، كل عرض، وكل إشارة خطر
2. **Company Brain OS** — عقل خاص بشركتك، يجيب أي موظف من وثائقك أنت
3. **WhatsApp/Inbox Follow-up OS** — متابعة منضبطة، صياغة بذكاء اصطناعي، موافقة بشرية
4. **Email Outreach Review OS** — إيميل يصل للإينبوكس، توثيق كامل، إلغاء اشتراك
5. **SMS Notification/Follow-up OS** — SMS بموافقة، STOP/opt-out، تتبع كل رسالة
6. **AI Trust & Compliance OS** — حوكمة الذكاء الاصطناعي، امتثال PDPL، سجل تدقيق
7. **Client Delivery OS** — تسليم منضبط، إثبات عمل، handoff احترافي
8. **Controlled Live Outbound OS** — outbound منضبط، هدف موثّق، موافقة بشرية، لا سبام

## ما نضمنه

- نظام يعمل
- تقارير شفافة
- موافقة بشرية قبل أي إرسال
- توثيق كامل (proof pack)
- امتثال PDPL

## ما لا نضمنه

- لا نضمن إيرادات أو ROI أو معدلات تحويل محددة
- لا نقدم شهادات أو عملاء وهميين
- لا نعد بأرقام غير واقعية

## الخطوة التالية

1. مكالمة تشخيص (20–30 دقيقة)
2. تقرير فجوات خلال 2–3 أيام
3. قرارك: تبدأ سبرنت أو لا

# كتاب أسعار Dealix (عربي)

> المصدر الوحيد للحقيقة: `os/config/pricing.yml` و`os/config/packages.yml`.
> هذا المستند نسخة بشرية للقراءة — أي تعارض، الملف اليامل هو المرجع.
>
> **جميع الأسعار غير شاملة لضريبة القيمة المضافة 15% (ZATCA).**
> **مشاركة أي سعر مع عميل تتطلب موافقة المؤسس (بوابة G03).**

Dealix لا يُباع كخدمة، بل كنظام تشغيل شهري:
**Audit → Pilot → Production → Managed OS → Usage/Add-ons → Command Center**

## 7) شروط الدفع

- **المشاريع:** 50% عند التوقيع، 30% بعد MVP/Pilot، 20% عند الإطلاق.
- **الاشتراكات:** شهري مقدم، حد أدنى 6 أشهر، Setup منفصل.

---

## 8) سياسة الخصومات

| الحالة | الخصم |
|---|---|
| دفع سنوي مقدم | 10% |
| أول 3 عملاء Case Study | 15% |
| Audit → Pilot خلال 14 يوم | 50% من قيمة Audit |
| عميل Enterprise | زيادة النطاق بدل الخصم |

**ممنوع:** Pilot مجاني، Retainer أقل من 8,000، اشتراك بلا حد استخدام، API مجاني، دعم مفتوح.
**سقف الخصم العام:** 15% (Pilot حتى 20%) — وأي تجاوز بموافقة المؤسس فقط.

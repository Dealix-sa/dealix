# Dealix Commercial Product Catalog

## 1. Revenue Command Room OS

**Problem:** الفرص، العروض، المتابعات، والردود موجودة في أماكن متفرقة ولا تتحول إلى next action يومي.

**Ideal customer:** شركة B2B عندها leads أو عروض أو واتساب/إيميل نشط لكن المتابعة غير منظمة.

**7-day delivery:**

- prospects ledger
- follow-up queue
- proposal queue
- daily CEO report
- command room dashboard
- first 25 reviewed draft messages

**Pricing range:** 5,000–12,000 SAR for a 7-day sprint. Suggested, not guaranteed.

**Acceptance criteria:** every opportunity has owner, status, next_action, and review date.

## 2. Company Brain OS

**Problem:** الإدارة ترى تقارير كثيرة لكنها لا ترى القرار الأهم اليوم.

**Delivery:**

- company brain map
- bottleneck map
- future radar with scenarios
- daily CEO decision
- weekly board memo
- 30-day action plan

**Pricing range:** 15,000–35,000 SAR for 14-day brain sprint.

## 3. WhatsApp / Inbox Follow-up OS

**Problem:** واتساب والإيميل يتحولون إلى مقبرة فرص.

**Delivery:**

- conversation categories
- follow-up queue
- draft replies
- delayed-response report
- escalation SOP

**Safety:** WhatsApp live send requires opt-in and approved template. Default is draft-only.

## 4. AI Trust & Compliance OS

**Problem:** الموظفون يستخدمون AI بدون سياسة أو مراجعة أو حدود بيانات.

**Delivery:**

- AI usage policy
- client data handling SOP
- human approval gates
- AI output review log
- incident response outline

## 5. Client Delivery OS

**Problem:** العميل يطلب “أداة”، لكن النتيجة التجارية غير محددة.

**Delivery method:** Map → Design → Build → Operate → Scale.

**Every client project must include:** intake, diagnosis, scope card, blueprint, acceptance criteria, delivery plan, proof pack.

## Sales guardrails

- No fake ROI.
- No fake clients.
- No fake testimonials.
- No guaranteed results.
- No live outbound by default.
- Every cold email example includes opt-out language.
- Every WhatsApp example is draft-only unless opt-in exists.

# Controlled Live Outbound OS

## Positioning

Dealix can operate email and WhatsApp outbound for companies only under approval, verification, opt-out, rate-limit, and logging gates.

## Promise

Automate the operating system, not spam.

## Channels

- Email: controlled live after DNS/auth/unsubscribe readiness
- WhatsApp: opt-in/template-only live send
- LinkedIn: manual task generation only
- Calls: manual task generation only

## Required controls

- Approval before send
- Verified target
- Source URL
- Opt-out
- Suppression list
- Rate limit
- Event logging
- Reply tracking
- No fake claims
- No guaranteed ROI

## Initial limits

- Email: 25/day
- WhatsApp: 10/day only with opt-in/templates

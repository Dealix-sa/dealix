# AI Output Risk Classes

## Class L0 — Informational
- Internal summaries, notes, tags
- No client-facing content
- Lowest risk; human review optional

## Class L1 — Internal Draft
- Draft proposals, emails not yet shared
- Review required before external use
- Moderate risk

## Class L2 — Client-Facing Content
- Approved proposals, reports, dashboards
- Must pass domain expert review
- Higher risk; accuracy matters

## Class L3 — Financial or Legal
- Pricing, contracts, compliance statements
- Must pass legal/founder review
- Highest risk; zero tolerance for error

## Class L4 — Automated Action
- Any trigger that affects external systems
- Prohibited unless explicitly whitelisted
- Not used in Dealix V3

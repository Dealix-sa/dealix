# Dealix Knowledge Index (operator reference)

Build the index:
```
python3 scripts/index_knowledge_sources.py --demo
```

Search:
```
python3 scripts/search_knowledge_base.py --query "Revenue OS"
```

Generate a pack for a topic:
```
python3 scripts/generate_knowledge_pack.py
```

Files indexed: see `business/_data/knowledge_index.json` after building.

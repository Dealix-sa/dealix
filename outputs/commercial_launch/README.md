# Commercial Launch OS — daily outputs

This directory holds **generated** founder-review artifacts. Dated batch folders
(`<YYYY-MM-DD>/`) are regenerated by the scripts and are **git-ignored** — they
are not committed. Only this README is tracked.

Generate today's batch:

```bash
python scripts/commercial_generate_400_drafts.py --target 400
```

Each `<date>/` folder contains: `draft_queue.jsonl`, `founder_review.{md,csv}`,
`top_50_priority.md`, `rejected_drafts.jsonl`, `needs_research.jsonl`,
`compliance_report.json`, `quality_report.json`, `safety_audit.json`,
`daily_metrics.json`, `next_actions.md`, `batch_manifest.json`,
`approved_manual_sends.example.csv`, and `media_social/` (calendar plan).

Every draft is approval-gated: `send_allowed=false`, `external_send_blocked=true`,
`requires_founder_approval=true`, `no_auto_send=true`. The system never sends.

## Thresholds
- 90–100 — Excellent: hold cadence
- 75–89  — Good: identify the dropping signal
- 60–74  — Drift: pause one initiative, restore cadence
- < 60   — Reset: emergency 1-on-1 with advisor; OS may be over-scoped

## Coaching Patterns

| Drop signal | Likely cause | First move |
|---|---|---|
| Daily Brief unread | Brief is too long / wrong format | Trim Brief sections; revisit `DAILY_COMMAND_BRIEF.md` |
| Weekly Review skipped | Sunday context-switch failed | Set alarm + dedicated 45-min block |
| Decision Log stale | Decisions happening verbally | Log in the moment, not later |
| Ledger sparse | Actions undertaken without entries | Build the habit; review at EOD |
| Focus drift | New shiny idea | Apply Strategy Filter; DEFER if necessary |
| KILL = 0 | Over-commitment | Force one kill; the discipline matters |
| A4 violation | Process gap | Add a rule + log incident |

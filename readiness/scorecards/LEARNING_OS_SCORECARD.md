## Thresholds
- 90–100 — Strong feedback loop; OS will compound
- 75–89  — Decent; identify the missing source
- 60–74  — Loop weakening; will compound failure if uncorrected
- < 60   — Reset; the company is operating blind

## Common Failures

- Experiments started, never closed → enforce 7-day close
- Win/loss not captured → block stage close until row present
- Message performance only when convenient → automate logging
- Monthly Strategy Update skipped → never skip; write 1-pager even if light

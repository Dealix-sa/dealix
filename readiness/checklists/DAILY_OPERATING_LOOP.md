# Daily Operating Loop Checklist

> Print and tick. 30-day discipline builds the muscle.

## Morning (07:30 – 09:00)

- [ ] 07:30 — Read Daily Command Brief
- [ ] 08:00 — Approve / reject overnight queue (drafts, proposals, claims)
- [ ] 08:30 — Confirm One Focus Today (write it in `decision_log.md`)
- [ ] 09:00 — Begin lead discovery / sales work

## Midday (09:00 – 14:00)

- [ ] 09:00 — Lead discovery (5/day target)
- [ ] 10:00 — Outreach drafts produced
- [ ] 11:00 — Founder approvals + send
- [ ] 12:00 — Sales follow-ups (pre-noon momentum)
- [ ] 14:00 — Begin delivery block

## Afternoon (14:00 – 18:00)

- [ ] 14:00 — Delivery work (one client at a time)
- [ ] 16:00 — QA review + approvals
- [ ] 17:00 — Content draft (Thursdays only) OR catch-up
- [ ] 17:30 — Append to `DEALIX_EXECUTION_LEDGER.md` (today's actions)
- [ ] 18:00 — End-of-day close + tomorrow's Brief generated

## Hard Rules (every day)

- The day starts in the Brief, not in inbox
- Every decision > 5K SAR gets logged the same day
- Every kill gets a written reason
- No outbound after 18:00 (founder rest discipline)
- Friday: no outbound, planning + review only

## End-Of-Week Cleanup (Friday)

- [ ] Pipeline tracker hygiene 100%
- [ ] Approval log spot-check (5 random entries)
- [ ] Suppression list updated
- [ ] Sunday Weekly Review prepared (data pulled)

## End-Of-Month Cleanup (last weekday)

- [ ] Monthly Strategy Update written
- [ ] Board Memo drafted (even if no board)
- [ ] MRR + cash collected reconciled
- [ ] Scorecards updated
- [ ] Risk register reviewed

## When To Break The Loop

Never. Even on travel days, run the abbreviated version (Brief read + approvals + ledger). Long breaks (vacation, illness): hand off via Ops Manager (when present) or pause non-essentials with documented stop.

# Weekly Operating Loop Checklist

> Run every Sunday. 45 minutes. Cannot be skipped two weeks in a row.

## Pre-Review (15 min)

- [ ] Pull cash collected (`revenue/cash_collected.csv`)
- [ ] Pull MRR snapshot (`revenue/mrr_tracker.csv`)
- [ ] Pull pipeline (`pipeline/pipeline_tracker.csv`)
- [ ] Pull approval log summary (`trust/approval_log.csv`)
- [ ] Pull delivery status (`clients/*/progress.md`)

## The Review (30 min)

Open `docs/founder/WEEKLY_CEO_REVIEW.md` template and complete each section.

- [ ] Revenue section filled (5 metrics)
- [ ] GTM section filled (5 fields)
- [ ] Delivery section filled (5 fields)
- [ ] Product section filled (5 fields)
- [ ] Trust section filled (5 fields)
- [ ] Learning section filled (4 fields)
- [ ] All 12 scorecards updated
- [ ] CEO Decision row complete (BUILD / FIX / KILL / DEFER — none can be blank)
- [ ] Next Week's One Focus written

## Scorecards (5 min)

- [ ] `readiness/scorecards/FOUNDER_OS_SCORECARD.md`
- [ ] `readiness/scorecards/REVENUE_OS_SCORECARD.md`
- [ ] `readiness/scorecards/DELIVERY_OS_SCORECARD.md`
- [ ] `readiness/scorecards/TRUST_OS_SCORECARD.md`
- [ ] `readiness/scorecards/LEARNING_OS_SCORECARD.md`

## Storage

- [ ] Save completed review to `weekly_reviews/YYYY-MM-DD.md` (private repo)
- [ ] Append decisions to `DEALIX_EXECUTION_LEDGER.md`
- [ ] Update `DEALIX_STAGE_STATUS.md` if any maturity change

## Hard Rules

- The KILL row cannot be empty
- The Next Week's One Focus must connect to last week's
- No section is allowed "n/a" two weeks in a row
- If a number is unknown, write "unknown — fix source by [date]"

## When You're Tempted To Skip

- 45 minutes < cost of operating blind
- One missed week = forgivable; two = compounding drift
- The hardest weeks are the ones the review is most needed

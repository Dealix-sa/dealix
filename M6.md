# M6 Operating Memory

M6 adds a memory layer for leadership actions.

## What it records

- action decisions
- status movement
- memory events
- replay steps
- weekly proof summary

## Run

```bash
python m6_runtime.py
python -m pytest -q m6_check.py
```

## Output

```text
reports/leadership/m6/latest.json
reports/leadership/m6/latest.md
```

## Safety

M6 records review outcomes. It does not perform external actions.

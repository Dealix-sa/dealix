# Proof Vault System

## الهدف
تجميع الأدلة التي تثبت تقدم Dealix بدون مبالغة أو كشف بيانات حساسة.

## أنواع الأدلة
- Pilot receipt
- Proposal sent
- Discovery call summary
- Before/after workflow map
- KPI screenshot
- Client testimonial
- Case study draft
- Retainer agreement

## كل دليل يجب أن يحتوي
- التاريخ
- المصدر
- نوع الدليل
- هل يحتوي بيانات حساسة؟
- هل قابل للاستخدام التسويقي؟
- رابط/مسار الملف

## قاعدة مهمة
لا تستخدم دليلًا خارجيًا أو تسويقيًا قبل تنقيحه أو أخذ موافقة مناسبة.

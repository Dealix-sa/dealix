# Dealix Transformation Pricing Model

## Pricing principles
1. Charge for business transformation, not hours.
2. Separate setup from monthly operating support.
3. Custom work starts with paid discovery.
4. No unlimited scope.
5. Every package must include measurable outcomes.

## Pricing ladder

### Diagnostic Sprint
Price: 7,500 - 25,000 SAR
Duration: 3-7 days
Output: workflow map + system blueprint + implementation quote

### Starter System
Price: 12,000 - 25,000 SAR setup
Monthly: 3,000 - 8,000 SAR
Use: one workflow, one dashboard, one report

### Growth System
Price: 25,000 - 75,000 SAR setup
Monthly: 8,000 - 25,000 SAR
Use: multiple workflows, dashboard, automations, reports

### Command Center
Price: 75,000 - 180,000 SAR setup
Monthly: 20,000 - 60,000 SAR
Use: executive visibility, teams, reports, governance

### Enterprise Custom
Price: 180,000 - 500,000+ SAR setup
Monthly: 50,000 - 150,000+ SAR
Use: integrations, custom system, SLA, governance, multi-team rollout

# Dealix Founder Business OS

This folder turns Dealix from a technical project into a founder-led business machine.

It creates:
- Daily target generation
- Prospect scoring
- Outreach drafts
- Offer selection
- Proposal stubs
- Founder daily report
- Weekly business review
- Delivery and client success operating rhythm

Rule: this system drafts and organizes. It does not auto-send messages.

# Sprint Daily Routine

## Morning
- Run daily CLI.
- Read founder/daily_brief.md.
- Review decision_queue.md.
- Pick one founder focus.

## Midday
- Execute revenue action.
- Update pipeline stage.
- Review approvals.
- Prepare sample/proposal/delivery item.

## End of Day
- Update daily_execution_log.md.
- Update sprint_scorecard.csv.
- Record blocker.
- Record next action.

## Rule
No day closes without updating:
- pipeline stage
- next action
- daily execution log

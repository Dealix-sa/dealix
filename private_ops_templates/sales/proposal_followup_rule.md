# Proposal Follow-Up Rule

## Rule
No proposal is sent without follow-up schedule.

## Follow-Up Cadence
- Day 1: confirm received
- Day 3: answer questions
- Day 7: close or disqualify
- Day 14: final polite follow-up

## Required Fields
Every proposal must have:
- proposal date
- amount
- decision maker
- follow-up date
- next action
- payment path

## Bad Signs
- no decision maker
- unclear scope
- asks for guaranteed results
- refuses start condition
- price too low for customization

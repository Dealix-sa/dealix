# S9 Revenue Machine Playbook

## Purpose

S9 turns the daily console into a commercial revenue machine.

## What it produces

- ranked account list
- ICP score
- offer match
- discovery call script
- objection notes
- follow up cadence
- commercial proof report

## Daily routine

1. Pick the top three accounts.
2. Review the matched offer.
3. Use the discovery script.
4. Prepare the first touch draft.
5. Review objections before replying.
6. Record proof after each motion.

## Offers

- 7 day command sprint
- 14 day growth operating system
- managed command room

## Best first sectors

- clinics
- agencies
- training
- logistics

## Review rule

S9 prepares sales motion. It does not run live outreach by itself.

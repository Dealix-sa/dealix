# Dealix Logo Lockups

Drop the approved Dealix logo SVG/PNGs here. Naming convention:

- `dealix-logo.svg` — primary horizontal lockup
- `dealix-logo-vertical.svg` — vertical lockup
- `dealix-logo-mono-white.svg` — monochrome white
- `dealix-logo-mono-navy.svg` — monochrome on light surfaces

Until those assets land, the UI falls back to the SVG monogram in
`apps/web/components/brand/dealix-logo.tsx`.

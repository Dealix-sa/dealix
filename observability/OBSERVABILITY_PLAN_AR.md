# Observability Plan

## ماذا نراقب؟
- form submissions.
- API failures.
- invalid leads.
- daily KPI gaps.
- workflow failures.
- security smoke failures.

## Alerting مبدئي
- إذا فشل lead API: إشعار داخلي.
- إذا لم يدخل أي lead خلال 7 أيام: راجع العرض والقناة.
- إذا drafts كثيرة ولا replies: راجع الرسالة والاستهداف.

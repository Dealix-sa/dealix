# Agent Memory & Context Policy

## ما يسمح حفظه
- اسم الشركة.
- القطاع.
- stage.
- الألم التجاري.
- مصدر lead.
- تاريخ التفاعل.
- next action.

## ما لا يحفظ إلا عند الحاجة وبصلاحيات مناسبة
- بيانات شخصية تفصيلية.
- عقود أو أسعار خاصة.
- أسرار تقنية.
- API keys.
- بيانات عملاء العميل.

## الاحتفاظ
- سجلات التشغيل: 90 يوم كبداية.
- سجلات العميل: حسب العقد وسياسة الخصوصية.
- بيانات prospects غير المؤهلين: مراجعة شهرية للحذف أو الأرشفة.

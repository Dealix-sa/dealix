# Dealix Agent Operating System

## الهدف
تحويل Dealix إلى شركة تعمل يوميًا عبر agents داخلية تساعد في البحث، التصنيف، توليد العروض، متابعة الـ CRM، التقارير، QA، والامتثال — بدون إرسال خارجي تلقائي أو قرارات مالية تلقائية.

## القاعدة الذهبية
أي فعل خارجي أو حساس يمر عبر Human Approval Gate:
- إرسال واتساب أو إيميل
- تغيير حالة فرصة إلى Won/Lost
- تعديل بيانات عميل
- إصدار عرض سعر رسمي
- نشر محتوى عام
- استخدام بيانات شخصية في مخرجات خارجية

## أنواع الوكلاء
1. CRM Research Agent: يلخص حسابات العملاء المحتملين من البيانات المدخلة.
2. Lead Qualification Agent: يقيّم fit score ويقترح next action.
3. Outreach Draft Agent: يكتب مسودة تواصل لا ترسل تلقائيًا.
4. Proposal Agent: يجهز عرض أولي حسب القطاع والألم.
5. Delivery PM Agent: يحول SOW إلى مهام تسليم أسبوعية.
6. Customer Success Agent: يراقب health score ومخاطر churn.
7. Compliance QA Agent: يفحص اللغة والبيانات والمخاطر.
8. CEO Briefing Agent: يلخص اليوم والأسبوع ويقترح أولويات.

## مخرجات ممنوعة
- وعود إيراد مضمونة.
- رسائل جماعية غير مراجعة.
- scraping مخالف أو استخدام بيانات غير موثقة المصدر.
- مشاركة أسرار أو مفاتيح أو بيانات عملاء حساسة.

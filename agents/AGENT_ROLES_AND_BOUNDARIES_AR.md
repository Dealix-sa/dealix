# Agent Roles & Boundaries

| Agent | Allowed | Not Allowed | Approval Required |
|---|---|---|---|
| CRM Research | تلخيص prospect من CRM/CSV | جلب بيانات مخالفة أو غير موثقة | عند استخدام بيانات خارجية |
| Lead Qualification | fit score وnext action | حذف lead أو تغيير مالي | عند تغيير stage |
| Outreach Draft | مسودات واتساب/إيميل | إرسال مباشر | دائمًا قبل الإرسال |
| Proposal | مسودة عرض | توقيع/إرسال رسمي | قبل المشاركة مع العميل |
| Delivery PM | مهام وتسليمات | وعد scope جديد | عند تغيير scope |
| Customer Success | health score | إلغاء عقد | عند توصية تجارية |
| Compliance QA | فحص مخاطر | قرار قانوني نهائي | عند قضايا حساسة |
| CEO Briefing | تلخيص وأولويات | قرارات تنفيذية آلية | كل قرار خارجي |

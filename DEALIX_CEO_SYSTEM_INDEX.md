# Dealix CEO System Index

## Purpose
Provide one executive index for all Dealix company systems.

## CEO Principle
Revenue drives. Trust governs. Delivery proves. Learning improves. Productization follows evidence.

## Core Company Systems

| System | Purpose | Cadence | Primary Evidence |
|---|---|---|---|
| CEO Command | Founder focus and decisions | Daily | daily_brief.md |
| Strategy | Market, ICP, moat, 90-day direction | Monthly | 90_DAY_PLAN.md |
| Revenue | Cash, pipeline, offers, pricing | Daily | revenue_action_log.csv |
| Sales | DMs, calls, proposals, objections | Daily | pipeline_tracker.csv |
| Delivery | Revenue Sprint execution and QA | Per delivery | delivery_report.md |
| Trust | Approval, claims, data boundary | Daily/Weekly | approval_log.csv |
| AI Governance | Agent risk, evals, permissions | Weekly | AI_RISK_REGISTER.md |
| Finance | Cash, burn, runway, MRR | Weekly/Monthly | finance review |
| Client Success | Feedback, renewal, retainer | Per client | health_score.md |
| Learning | Weekly improvement loop | Weekly | weekly_review.md |
| Productization | Convert repeated work into product | Monthly | productization_candidates.md |
| Engineering | CI, reliability, release discipline | Weekly | GitHub Actions |
| Brand & Proof | Content, case studies, authority | Weekly | proof_library |
| People | Delegation and hiring triggers | Monthly | role scorecards |
| Partners | Referrals and channel growth | Monthly | partner_pipeline.csv |
| Investor/Board | Executive reporting and readiness | Monthly/Quarterly | board pack |

## System Rule
Every system must have:
- owner
- inputs
- outputs
- rules
- metrics
- evidence
- verifier
- review cadence

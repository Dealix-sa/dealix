# Q13 Founder Daily Command

## Purpose

Q13 gives the founder one practical daily command for selling and serving companies.

## What it outputs

- account to focus on
- sector
- matched offer
- pain reason
- KPI for today
- proof to produce
- service flow
- first 10 company list

## Client service flow

1. collect intake
2. map current follow-up flow
3. assign owners
4. build command queue
5. prepare draft routes
6. review approvals
7. deliver proof report
8. prepare next-week plan

## Run

```bash
python founder_daily_command.py
python -m pytest -q q13_ok.py
```

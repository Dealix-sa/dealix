# Client Success Logs

This directory holds **canonical client success logs** for the Delivery &
Client Success Operating System v1.

- `retention_tracker.csv` — one row per client engagement, tracking feedback,
  health score, retainer ask, retainer status, proof status, and next action.
- `client_success_dashboard.md` — weekly view of active clients, feedback
  waiting, retainer opportunities, proof opportunities, risks, and the focus
  for the current week.

For per-client artifacts (intake, report, QA checklist, feedback, health
score, etc.) use the private workspace folder
`clients/<client_name>_private/` populated from `clients/_template/`. Do not
put client-identifying data into this directory.

# GCC Marketing OS — System Specification

## Vision

Dealix GCC Draft Factory is an always-on 24/7 outreach intelligence system that:
- Monitors the GCC market for qualified companies
- Classifies them by country, sector, language, and buyer type
- Writes high-quality Arabic and English draft packages
- Gates every draft through quality and compliance checks
- Delivers only the best opportunities to the founder for review and approval

The founder reviews, decides, and approves sends. The system never sends autonomously.

## Quality Standards

Every draft must pass:
- Quality Gate: Arabic authenticity / English non-generic / structure / length
- Compliance Gate: opt-out / no deceptive claims / sensitive sector rules / country-specific

## Sending Philosophy

High draft volume → Controlled send volume → Multi-channel distribution

Never mass-email from a single inbox. Distribute by:
- Market (Arabic local / International GCC)
- Sector (Legal / FM / Consulting / etc.)
- Channel (Email / LinkedIn / Website form)
- Inbox reputation and warming stage

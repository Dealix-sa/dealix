# Z8 Daily Operating Console

Z8 is the commercial daily console for Dealix.

## What it combines

- M6 operating memory.
- Z7 route counts.
- Commercial offer menu.
- Target sector list.
- Owner workload.
- Proof summary.
- Next sales actions.

## Run

```bash
python dealix_daily_os.py
python -m pytest -q z8_check.py
```

## Output

```text
reports/leadership/z8/latest.json
reports/leadership/z8/latest.md
apps/web/app/z8/page.tsx
z8_snapshot.ts
```

## Commercial use

Use Z8 every day to decide what to sell, who owns each motion, how many approvals are pending, and what proof exists.

Everything remains review-first. The console prepares decisions and sales motion but does not execute live actions.

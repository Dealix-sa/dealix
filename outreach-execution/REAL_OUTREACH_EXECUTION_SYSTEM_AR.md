# نظام تنفيذ التواصل الحقيقي — V16

الغرض من V16 هو تحويل أعلى العملاء المحتملين من V5/V15 إلى تنفيذ يومي مضبوط:

1. بناء batch صغير عالي الجودة.
2. تخصيص الرسالة حسب القطاع والألم والدليل.
3. إدخال الرسائل في review queue.
4. اعتماد بشري قبل أي استخدام خارجي.
5. تسجيل الردود.
6. تجهيز مكالمات discovery.
7. توليد follow-up حسب الاعتراض.

## قواعد التشغيل

- لا إرسال آلي.
- لا ادعاء نتيجة مضمونة.
- لا استخدام بيانات حساسة.
- لا استخدام رقم/إيميل دون مصدر موثق.
- لا أكثر من 10–20 رسالة مراجعة يوميًا في البداية.
- الأولوية للجودة والملاءمة لا الحجم.

## Daily cadence

| الوقت | العمل |
|---|---|
| صباحًا | بناء batch من أعلى prospects |
| قبل الظهر | تخصيص الرسائل |
| منتصف اليوم | مراجعة بشرية واعتماد |
| بعد الظهر | تسجيل الردود وتجهيز المكالمات |
| نهاية اليوم | dashboard وnext actions |

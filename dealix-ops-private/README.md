# dealix-ops-private / المنطقة الخاصة للعمليات

## Purpose / الغرض

This is the founder's private operating area. Real client data, populated
CSV trackers, sales call notes, invoices, and per-client folders live
here, NOT in `docs/`.

هذه منطقة العمل الخاصة بالمؤسس. بيانات العملاء الحقيقية وملفات التتبع
المعبأة وملاحظات المكالمات والفواتير ومجلدات العملاء تعيش هنا، وليس
داخل `docs/`.

## Rules / القواعد

- Template files (the scaffold) ARE committed; real data is gitignored.
- Never paste production credentials or PII into committed files.
- See `/.gitignore` for the list of paths excluded from git.
- See `docs/founder/CEO_OPERATING_MODEL.md` for the operating model.

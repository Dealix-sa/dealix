# استراتيجية التسعير — Dealix

> **مصدر التسعير الرسمي / Canonical pricing source.** المصدر الوحيد المعتمد
> للأسعار وسلم العروض هو [`docs/OFFER_LADDER_AND_PRICING.md`](OFFER_LADDER_AND_PRICING.md)
> (الدرجات 0–5). تمت مواءمة هذا الملف معه؛ عند أي تعارض يُعتمد ملف سلم العروض.
> The single source of truth is the offer ladder file; on conflict it wins.
>
> **ملاحظة استبدال:** التسميات القديمة (Founder Operator 299–999، Growth OS
> 2,999، Scale OS 7,999) **مُلغاة** واستُبدلت بسلم العروض القياسي أدناه.
> Old SaaS tier names (Founder Operator / Growth OS / Scale OS) are retired.

## المبدأ

التسعير يعكس **قيمة السياق + التنفيذ الآمن + إثبات العائد**، وليس عدد المستخدمين فقط. الترقية بين الدرجات لا تحدث إلا بعد إثبات حقيقي موثق من الدرجة السابقة. طبقة الأداء اختيارية و**تتطلب تعريف تأهيل + نزاعات + عقود** قبل أي فوترة حقيقية.

## سلم العروض القياسي — الدرجات 0–5

### 0) Free AI Ops Diagnostic — مجاني

| الحقل | التفاصيل |
|--------|-----------|
| **السعر** | **مجاني** (باب الدخول). |

| **المخرجات** | تقرير تشخيصي صفحة واحدة + 3 أولويات + توصية الخطوة التالية. |
| **مسار الترقية** | → 7-Day Revenue Proof Sprint (499). |

### 1) 7-Day Revenue Proof Sprint — 499 SAR

| الحقل | التفاصيل |
|--------|-----------|
| **السعر** | **499 SAR** (دفعة واحدة مسبقة). |
| **لمن؟** | مؤسس B2B أو مدير مبيعات يريد إثبات سريع خلال أسبوع. |
| **ماذا تشمل؟** | تقرير تشخيصي مفصل، 5 مسودات رسائل بموافقة، Proof Pack يوم 7، تقرير تنفيذي، خطة 30 يوم. |
| **متى يترقى العميل؟** | بعد proof موثق → Data-to-Revenue Pack أو Managed Revenue Ops. |
| **المخاطر** | توقع «إرسال تلقائي» — يُصحح بالتدريب؛ التسليم draft_only. |
| **شروط التأهيل** | ICP أو استعداد لتعريفه؛ قبول سياسات الموافقة؛ لا قوائم بارد. |

### 2) Data-to-Revenue Pack — 1,500 SAR

| الحقل | التفاصيل |
|--------|-----------|
| **السعر** | **1,500 SAR** (مشروع واحد). |
| **لمن؟** | شركة B2B لديها بيانات عملاء/مبيعات غير مستثمرة. |
| **ماذا تشمل؟** | تحليل pipeline كامل، خريطة فرص مرتّبة، 10 مسودات استهداف، تقرير ROI basis، playbook مبيعات. |
| **متى يترقى؟** | → Managed Revenue Ops. |
| **شروط التأهيل** | CRM export أو Excel، 3 شهور بيانات، وصف ICP. |

### 3) Managed Revenue Ops — 2,999–4,999 SAR/شهر

| الحقل | التفاصيل |
|--------|-----------|
| **السعر** | **2,999–4,999 SAR/شهر** (اشتراك شهري). |
| **لمن؟** | شركات B2B أتمت pilot ناجح وتريد استمرارية. |
| **ماذا تشمل؟** | تقرير أسبوعي تنفيذي، ≤20 مسودة/شهر بموافقة، Proof Pack شهري، KPIs، جلسة استراتيجية شهرية. |
| **متى يترقى؟** | → Executive Command Center. |
| **نمط التسليم** | بقيادة المؤسس / شبه-مؤتمت اليوم. |
| **المخاطر** | توسع نطاق دون امتثال — يُمنع بالسياسات والتدقيق. |
| **شروط التأهيل** | سياسات قنوات موقعة؛ مسؤول امتثال معرف؛ بيانات مصدر شرعية. |

### 4) Executive Command Center — 7,500–15,000 SAR/شهر

| الحقل | التفاصيل |
|--------|-----------|
| **السعر** | **7,500–15,000 SAR/شهر**. |
| **لمن؟** | فرق ناضجة، مؤسس/C-Suite يريد رادار قرار يومي. |
| **ماذا تشمل؟** | تقرير يومي تنفيذي، رادار سوق، مسودات كاملة، Proof Pack، استراتيجية ربعية، وصول أولوية. |
| **متى يترقى؟** | → Agency Partner OS أو SOW مؤسسي. |
| **نمط التسليم** | بقيادة المؤسس / شبه-مؤتمت اليوم. |
| **شروط التأهيل** | ≥ 3 pilots مكتملة + ≥ 1 case study موثق. |

### 5) Agency Partner OS — مخصص + rev-share

| البند | النطاق |
|--------|--------|
| **النموذج** | مخصص + **حصة إيراد 15–30%**. |
| **شروط الفتح** | بعد إتمام ≥ 3 proof packs واتفاقية شراكة موقعة. |

العمل التنفيذي المخصّص الأكبر على نطاق مؤسسي يُعرّف هنا أو ضمن SOW منفصل؛ لا يُعلن سعراً ثابتاً على الموقع العام.

## Performance Layer — طبقة الأداء (اختياري — غير مفعّلة)

| البند | النطاق |
|--------|--------|
| لكل lead مؤهل | **25–75 SAR** |
| لكل اجتماع محجوز (بعد موافقة) | **150–500 SAR** |
| عمولة نجاح على إيراد مغلق | **3–10%** حسب السوق وحجم الصفقة |

**لمن؟** عملاء يريدون ربط الدفع بالنتيجة بعد تعريف تأهيل صارم.  
**المخاطر:** نزاعات التأهيل — يجب عقد + أدلة + SLA مراجعة.  
**لا تُفعّل** في الإنتاج بدون قانوني واضح. هذه طبقة اختيارية فوق الدرجات وليست درجة مستقلة.

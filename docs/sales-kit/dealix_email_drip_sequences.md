# Dealix — Email Drip Sequences

**5-email sequence per persona.** Used when a prospect engages (opens email / clicks link) but doesn't book demo after Day 0.

All emails:
- ≤ 200 words
- PDPL opt-out at bottom
- One clear CTA per email
- Progressive value (each email teaches something, not just sells)

## Timing rules (all sequences)

| Day | Action | Why |
|-----|--------|-----|
| 0 | Initial outreach | |
| +3 | Value follow-up (not push) | Still recent, add proof |
| +7 | Value follow-up #2 | Educate, soft CTA |
| +14 | Case study / social proof | Show momentum |
| +21 | Break-up / last touch | Create urgency, extract referral |

**After +21:** stop automated emails. Move lead to "nurture" (quarterly touches only).

## Email deliverability best practices

1. **From name:** "سامي العسيري — Dealix"
2. **From email:** sami.assiri11@gmail.com or sami@dealix.me (use custom domain once SPF/DKIM setup)
3. **Subject lines:** < 50 characters, Arabic preferred for Saudi targets
4. **Opt-out in every email:** PDPL requirement
5. **No attachments** in first 3 emails (spam trigger)
6. **Send times:** Sunday-Wednesday 09:00-11:00 Saudi time (best open rates)
7. **Warm recipients only** — these sequences are for prospects who already engaged or were warm-introduced. No bulk sends, no purchased lists.
8. **Track opens but not clicks** in first emails (builds trust)

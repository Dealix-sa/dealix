## 📅 الخطة في 5 مراحل

### المرحلة 1 — تجهيز البنية (12 دقيقة، اليوم)

افتح `RAILWAY_MOYASAR_STEP_BY_STEP.md` واتبع:
- **A)** Railway: امسح Start Command + Deploy
- **B)** Railway: الصق Env Variables (عدّل 4 قيم CHANGE_ME)
- **C)** Moyasar: أضف Webhook

**تحقق:**
```bash
bash dealix_1_riyal_test.sh
```

✅ **النجاح:** يطلع لك Payment URL تقدر تدفع عليه 1 ريال

### المرحلة 2 — اختبار الدفع الفعلي (2 دقيقة، اليوم)

1. خذ Payment URL من السكريبت
2. افتحه في المتصفح
3. ادفع 1 ريال بطاقتك (هذا اختبار، موقف ضد Dealix)
4. تحقق: Moyasar Dashboard → Payments → يظهر 1 SAR paid

✅ **النجاح:** أول دفعة وصلت الحساب. Dealix يستقبل فلوس الآن.

### المرحلة 3 — إرسال 2-3 رسائل (20 دقيقة، اليوم)

افتح `dealix_personalized_messages.md`:

**اليوم (بالترتيب):**
1. **عبدالله العسيري (Lucidya)** — LinkedIn DM
2. **عمر الخميس (BRKZ)** — LinkedIn DM
3. **رائد بن جويعد (مستشار رقمنة)** — LinkedIn DM

**غداً:**
4. **أحمد الحازمي (Foodics)**
5. **فاطمة السبيعي**

سجّل كل رسالة في `dealix_14day_tracker.html` (افتحه في المتصفح، يحفظ تلقائياً).

✅ **النجاح:** 3-5 رسائل مرسلة، 1-2 رد متوقع خلال 72 ساعة. النتائج التقديرية ليست نتائج مضمونة.

### المرحلة 4 — Demos (3-7 أيام)

عند ما يرد أحدهم ويحجز Demo:

**قبل الـ Demo (15 دقيقة):**
1. افتح `dealix_demo_script_30min.md` — اقرأه كاملاً
2. افتح `dealix_objection_handler.md` — احفظ النقاط
3. جهّز tabs في المتصفح (السكريبت يشرح)
4. افتح `dealix_roi_calculator.html` — جاهز للشاشة

**أثناء الـ Demo (30 دقيقة):**
اتبع السكريبت دقيقة بدقيقة.

**بعد الـ Demo (خلال 10 دقائق):**
- إذا وافق على التشخيص المجاني → ابدأ بجمع عيّنة قائمة فرصه
- إذا قال "أفكر" → حدد موعد متابعة ملموس

✅ **النجاح:** 1-2 تشخيص مجاني خلال الأسبوع الأول

### المرحلة 5 — التشخيص المجاني → سبرنت إثبات الإيراد (أسبوع)

بعد التشخيص المجاني، وإن ناسب العميل:
1. **اليوم 1-2:** Setup + ربط مصدر بيانات الفرص
2. **اليوم 3-5:** Dealix يرتّب الفرص ويصيغ المسودات، والعميل يراجع ويعتمد
3. **اليوم 6:** مكالمة مراجعة 15 دقيقة
4. **اليوم 7:** تسليم حزمة الإثبات — "نوسّع النطاق؟"

✅ **النجاح:** أول سبرنت إثبات إيراد (499 ريال) مكتمل بحزمة إثبات

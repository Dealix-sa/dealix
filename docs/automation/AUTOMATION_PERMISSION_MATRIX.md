# Automation Permission Matrix

## Purpose
Define what Dealix can automate safely at each maturity level.

## A0 — Safe Internal Automation
Allowed:
- generate reports
- calculate metrics
- update internal scorecards
- validate schemas
- generate dashboard snapshots
- suggest next actions

Approval:
Not required if local/private.

## A1 — Draft Automation
Allowed:
- draft DMs
- draft proposals
- draft client reports
- draft public content

Approval:
Sami review before use.

## A2 — Explicit Approval
Allowed only after explicit approval:
- client-specific report sent externally
- proposal wording
- pricing exception
- public proof
- client-facing AI output

## A3 — Never Automatic
Never automate:
- refunds
- legal commitments
- contract changes
- sensitive data export
- public client claims
- guaranteed revenue claims
- outbound sending without approval

## Rule
AI and automation prepare. Sami approves critical moves.

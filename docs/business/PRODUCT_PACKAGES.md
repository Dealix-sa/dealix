# Productized Packages

Every package below is registered in
`dealix.hermes.products.PRODUCT_REGISTRY` and binds to a delivery
playbook in `dealix.hermes.delivery.PLAYBOOK_REGISTRY`.

## 1. Revenue Hunter Pilot

ICP profile, scored leads, message drafts, proposal drafts, outcome
report. Upsell: Revenue Hunter Managed retainer.

## 2. AI Trust Kit

AI use policy, agent registry, tool permission matrix, approval
workflow, evidence pack, risk register. Upsell: AI Governance OS
retainer.

## 3. Agency White-Label Kit

Brand-applied offer set, approved claims library, delivery playbooks,
revenue-share contract, quarterly performance review. Upsell:
Strategic Partner tier.

## 4. Founder OS Setup

Weekly review cadence, decision log, outcome ledger, daily kill switch,
approval queue, sovereign memory setup. Upsell: Executive PMO Lite.

## 5. Market Radar Report

Sector signal map, top 20 buying intents, channel and message quality
benchmarks, AI visibility snapshots, plays for next quarter. Upsell:
Quarterly Market Radar retainer.

## 6. Executive PMO Lite

Weekly review pack, decision log + outcome ledger, risk register +
approval queue, customer value reports, quarterly strategic memo.
Upsell: Strategic Partner retainer.

## 7. MCP Risk Review

MCP server inventory, tool risk scores, approval-gated tool registry,
descriptor drift monitor, quarterly review cadence. Upsell: MCP
Governance retainer.

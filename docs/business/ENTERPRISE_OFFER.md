# Enterprise Offer

For Saudi enterprises adopting AI under PDPL and SDAIA guidance.

## Scope

1. **Discovery** — agent inventory, tool inventory, data sensitivity
   review.
2. **Install** — AI Trust Kit deployed inside the enterprise
   workspace.
3. **Wire** — Hermes control plane wrapped around the enterprise's
   existing AI calls.
4. **Operate** — draft-only mode, weekly review, approval queue.
5. **Promote** — promotion to `LOW_RISK_AUTONOMY` only after the
   Automation Readiness Score is earned.

# Message Performance

> Tracks which outbound messages produce replies and downstream pipeline.

## Format

| ID | Channel | Variant | Sent | Replies | Reply Rate | Calls | Calls Rate | Notes |
|---|---|---|---:|---:|---:|---:|---:|---|

## Variant Definition

A variant is a message version with a unique combination of:
- Hook (first line).
- Value proposition (middle).
- Call to action (close).

A swapped emoji or punctuation is NOT a new variant.

## Rules

1. Minimum 25 sends before a variant is graded.
2. Reply rate is rolling 30-day.
3. Replies that are "remove me" or out-of-office do not count as replies.

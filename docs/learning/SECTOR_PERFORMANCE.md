# Sector Performance

Which sectors actually pay Dealix and how much.

## Metrics per sector
- Outbound volume.
- Reply rate.
- Call-booked rate.
- Win rate.
- Average deal value.
- Retention rate.

## Cadence
- Monthly review of the top five active sectors.
- Quarterly review across all sectors touched.

## Decisions this feeds
- Which sectors get the next sector report.
- Which sectors get founder time.
- Which sectors are deprioritized.

## Storage
- Monthly snapshot in `dealix-ops-private/learning/sectors/<YYYY-MM>.csv`.

## Rule
Concentration in winning sectors compounds. Spreading thin is a signal of indecision.

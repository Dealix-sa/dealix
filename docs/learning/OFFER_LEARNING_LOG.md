# Offer Learning Log — Dealix

## الدور

أداء كل offer — conversion، time-to-cash، objection patterns.

## المصدر

```
<private_ops>/learning/offer_learning.csv
```

## الحقول

`date,source,insight,evidence,decision,next_action,owner`

## أمثلة

- "Data Pack 1500 SAR يحوّل 2x Diagnostic 4999 في sector ERP".
- "Free Diagnostic لا يحوّل لـ paid — يجذب tire-kickers".
- "Sample مدفوع 499 SAR يقلل tire-kickers 80%".

## القواعد

- لا تغيير offer بدون 2 weeks evidence.
- لا "free upgrade" بدون decision موثَّق في `decisions.csv`.
- pricing change يحتاج Trust gate approval (لا يدّعي ROI ضمان).

## الملكية

- Owner: Founder.

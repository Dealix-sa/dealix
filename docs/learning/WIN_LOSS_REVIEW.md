# Win / Loss Review

Monthly review of every closed-won and closed-lost deal.

## For each deal
- Sector and size.
- Source (outbound, inbound, partner, referral).
- Pricing band.
- Length of cycle.
- Decision-maker contact.
- One-line reason for outcome.

## Synthesis questions
- Which sources produced the most wins?
- Which pricing bands closed fastest?
- What objection appeared most in losses?
- What signal in the early conversation predicted a win?
- What signal predicted a loss?

## Outputs
- A one-paragraph summary added to `docs/learning/MONTHLY_STRATEGY_UPDATE.md`.
- Any clear pattern becomes a candidate decision in `docs/founder/DECISION_LOG.md`.
- Any new objection class is added to `docs/revenue/CLOSING_PLAYBOOK.md`.

## Rule
A deal without a reason logged is a deal we cannot learn from. Always record the reason.

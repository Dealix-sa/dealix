# Weekly Intelligence Review

The single document that closes the Learning Loop every week.

## Purpose
Force Dealix to learn from its own week. Capture what happened, what worked, what failed, what bottleneck remains, and what will change next week.

## Owner
Sami (Founder, Learning OS).

## Review Cadence
Weekly. Written every Sunday evening or Monday morning before the CEO review.

## Inputs
- Daily briefs from the week.
- Pipeline tracker movements.
- Delivered artifacts and their QA outcomes.
- Approval log.
- Customer messages and feedback.

## Outputs
- A new file `dealix-ops-private/learning/yyyy-mm-dd_weekly_intelligence_review.md`.
- A list of changes to apply this coming week.
- An update to the scorecard.

## Rules
- Each of the five sections must contain at least one concrete entry tied to evidence.
- Vague entries ("learned a lot") are not accepted.
- One bottleneck per week is named explicitly.

---

## The Five Sections

### 1. What happened
Plain log of events. Leads added, DMs sent, replies received, proposals sent, payments received, sprints delivered.

### 2. What worked
Specific actions or systems that produced outcomes. Tie each one to a number or an artifact.

### 3. What failed
Specific actions that did not produce outcomes, and the most likely reason.

### 4. What bottleneck
The single biggest constraint on the next paid outcome. Only one bottleneck per week.

### 5. What will change
Two to three concrete changes for next week. Each one must name an owner and a deliverable.

---

## How To Use

1. Open the template `weekly_reviews/template.md` in the private repo.
2. Copy in real numbers from this week.
3. Save as `weekly_reviews/yyyy-mm-dd.md`.
4. Link it from the CEO review on Monday.
5. Update the scorecard accordingly.

## Metrics
- Number of weekly reviews written in a row.
- Number of "what will change" items shipped the following week.
- Number of repeated bottlenecks (target: low; a repeated bottleneck signals failure to act).

## Evidence
- Files in `dealix-ops-private/learning/` and `dealix-ops-private/weekly_reviews/`.
- Scorecard updates referencing each week's review.

## English Section

### Canonical positioning (verbatim)

Dealix is a Saudi-first AI Business Operating System. Its first commercial wedge is Revenue + Proof + Command, but its long-term architecture covers the operating core of the company: growth, customers, delivery, support, finance, governance, knowledge, and executive command.

### Launch rule

> Sell the wedge. Build the platform. Never confuse market entry with company identity.

### The 12 operating systems (mother platform = Dealix AI Business OS)

| System | One-line purpose |
|---|---|
| Command OS | Executive command: briefs, decisions, and priorities for the manager. |
| Revenue OS | Track where opportunities leak and turn them into evidenced offers. |
| Proof OS | Collect and organize the evidence that proves value and quality. |
| Client OS | Structured knowledge of every customer, relationship, and health. |
| Delivery OS | Track execution of commitments and delivery. |
| Support OS | Manage support and post-sale cases. |
| Finance OS | Connect revenue to cash, billing, and financial compliance. |
| Data OS | Data quality and flow as the basis for every decision. |
| Governance OS | Governance, approvals, and compliance (PDPL/ZATCA). |
| Academy OS | Knowledge, training, and team enablement. |
| Partner OS | Partners and expansion channels. |
| Venture OS | Build and launch new ventures (strategy only). |

### Status table (live / partial / future)

| System | Status |
|---|---|
| Command OS | Core — sellable now (wedge) |
| Revenue OS | Core — sellable now (wedge) |
| Proof OS | Core — sellable now (wedge) |
| Governance OS | Core — sellable now (wedge) |
| Data OS | Core — sellable now (wedge) |
| Client OS | Phase 2 / beta |
| Delivery OS | Phase 2 / beta |
| Finance OS | Phase 2 / beta |
| Support OS | Phase 2 / beta |
| Academy OS | Phase 3 |
| Partner OS | Phase 3 |
| Venture OS | Strategy only |

### Phasing

- **Core (sell now):** Command + Revenue + Proof + Governance + Data.
- **Phase 2:** Client + Delivery + Finance + Support.
- **Phase 3:** Academy + Partner.
- **Later:** Venture.

### Non-negotiables enforced everywhere

- No scraping, no cold WhatsApp automation, no LinkedIn automation, no external auto-send.
- No sales numbers, conversion rates, or ROI stated as fact — always "estimated" or "case-safe pattern".
- No PII in case studies.
- No external action on a customer's behalf without explicit human approval.
- Estimates are not guarantees.

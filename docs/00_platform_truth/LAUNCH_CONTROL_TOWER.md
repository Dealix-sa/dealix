## 4. معايير القرار للإطلاق العام / Go / No-Go Criteria (Public Launch)

الإطلاق العام مرفوض (**No-Go**) حتى تتحقّق **كل** المعايير التالية:

Public launch stays **No-Go** until **all** of the following hold:

- [ ] **≥ 3 Sprints مدفوعة / paid Sprints.**
- [ ] **3 Proof Packs مكتملة / completed Proof Packs.**
- [ ] **1 دراسة حالة / case study** (case-safe، بموافقة، بلا PII).
- [ ] **0 حوادث ثقة / trust incidents.**

عند تحقّق الأربعة → تتحوّل `Go-No-Go` إلى **Go**، ويُرفع السعر حسب `PRODUCT_FAMILY_MAP.md`. حتى ذلك الحين، نبقى في **Private Command Sprint**.

When all four hold → `Go-No-Go` flips to **Go**, and pricing rises per the product map. Until then, we stay in **Private Command Sprint**.

## 5. كيف تُحدَّث هذه الوثيقة يوميًّا / How to Update This File Daily

1. **صباحًا / Morning:** اقرأ كتلة الحالة + قائمة اليوم. ابدأ من أعلى القائمة. Read the status block + Today list; start at the top.
2. **خلال اليوم / During the day:** أنجز بنود "اليوم"؛ ضع `[x]` عند الإتمام. Check off Today items.
3. **مساءً / Evening:**
   - حدّث **كتلة الحالة** (القسم 1) بالأرقام الحقيقية فقط. Update the status block with real numbers only.
   - حدّث عمود **اليوم** في جداول الـ KPIs. Update the Today column in KPI tables.
   - إن وقعت أي حركة ثقة → سجّلها في "مؤشّرات الثقة" فورًا (وهي حادثة تستوقف اليوم). Log any trust movement immediately.
   - راجع معايير القرار (القسم 4): هل اقتربنا من Go؟ Check Go/No-Go criteria.
   - غيّر **آخر تحديث** أعلى الملف. Update the "Last updated" date.
4. **قاعدة الصدق / Honesty rule:** لا رقم يُرفع بلا حدث حقيقي مقابله. الكتلة هي مرآة الواقع لا التمنّي. No number rises without a real event behind it.

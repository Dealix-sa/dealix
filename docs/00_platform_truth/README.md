## The doctrine (11 non-negotiables)

1. No overpromise of full automation.
2. No cold outreach / scraping / auto-send.
3. No financial/legal/security external action without human approval + log.
4. Human approval for any external customer-facing action.
5. Honest status: LIVE / BETA / INTERNAL / DOCS_ONLY / DEPRECATED / BLOCKED / FUTURE.
6. Saudi-Arabic-first positioning.
7. No public claim without evidence in the claims register.
8. No case study without customer approval.
9. No data deletion without a logged record.
10. No pricing change without the Price Authority doc.
11. Commercially coherent **and** operationally verifiable.

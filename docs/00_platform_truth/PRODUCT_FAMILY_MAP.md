## 1. مبدأ السلّم / The Ladder Principle

نبيع **رؤية واسعة بعرض أول ضيّق**. كل درجة تُثبت قيمة قبل أن تفتح الدرجة التالية. لا قفز فوق الإثبات.

We sell a **wide vision through a narrow first offer**. Each rung proves value before the next rung opens. No skipping over proof.

```
تشخيص مجاني → Command Sprint → Business OS Setup → Managed Business OS → Executive OS → Enterprise/Partner
Free Diagnostic   499–1,500       3,000–5,000        2,999–7,500/mo        12,000–20,000/mo   Custom
```

## 3. شرط الترقية Sprint → Managed / The Upsell Condition

لا تُعرض الترقية إلى **Managed Business OS** إلا إذا تحقّقت **كل** الشروط التالية:

The upsell to **Managed Business OS** is offered **only** when **all** of the following hold:

1. **دفع الـ Sprint فعليًّا / Paid the Sprint** (لا ترقية من تجربة مجانية).
2. **استلم Command Pack / Received the Command Pack.**
3. **راجع المخرجات / Reviewed the outputs** (اجتماع أو تأكيد مكتوب).
4. **لديه فعل تالٍ واضح / Has a clear next action** خرج من الـ Pack.
5. **وافق على إيقاع أسبوعي / Agreed to a weekly rhythm.**

إن نقص أيّ شرط — **لا تُعرض الترقية**. نُعيد للعميل قيمة الـ Sprint أولًا.

If any condition is missing — **do not offer the upsell.** Return the Sprint value to the customer first.

## 4. القواعد التي تحكم السلّم / Rules Governing the Ladder

- لا قفز فوق الإثبات: كل درجة تتطلّب بوّابة خروج الدرجة السابقة. No skipping proof.
- كل عميل مدفوع → مجلد تسليم + Proof Pack. Every paid customer → delivery folder + proof pack.
- لا ادّعاء إيراد مضمون في أي درجة. No guaranteed revenue at any rung.
- التسعير الحقيقي يرتفع بعد الإثبات لا قبله (القسم 1). Pricing rises after proof, not before.

# تقارير القيمة الأسبوعية
# Weekly Value Reports

---

## المبدأ — Principle

كل أسبوع يُنتج تقريراً واحداً لكل عميل نشط. التقرير يُثبت القيمة بأرقام — لا بوصف سردي. الأرقام المذكورة تقديرية وليست مُتحقَّقة.

Every active client receives one report per week. The report proves value with numbers, not narrative descriptions. All figures mentioned are estimated, not verified.

## جدول التقارير — Report Schedule

| الحدث | التوقيت | من يُرسَل له |
|-------|---------|-------------|
| تقرير Sprint (رتبة 2–3) | عند إتمام كل Sprint | العميل المباشر |
| تقرير أسبوعي (رتبة 4–5) | كل يوم الأحد | العميل + WEEKLY_VALUE_REPORT_QUEUE |
| تقرير شهري تنفيذي (رتبة 4–5) | أول يوم من الشهر | مسؤول أول من العميل |

---

## معايير الجودة — Quality Standards

- **لا أرقام مُلفَّقة:** كل رقم مصدره بيانات تشغيلية من مدخلات العميل
- **لا مقارنة بين عملاء:** كل تقرير مستقل ومبني على بيانات عميله فقط
- **التمييز الواضح:** كل رقم يُوسَم بـ"تقدير" أو "قبل/بعد" — لا رقم يظهر كنتيجة مضمونة
- **الموافقة قبل الإرسال:** كل تقرير يحتاج موافقة المؤسس قبل إرساله للعميل

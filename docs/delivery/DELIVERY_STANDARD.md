# Dealix Delivery Standard

Every project follows:

1. Discovery
2. Scope
3. Data request
4. Build
5. Internal QA
6. Client review
7. Final report
8. Proof pack
9. Upsell recommendation

No project is complete without:

- signed scope
- delivery checklist
- QA checklist
- proof pack
- client handoff

# Revenue Sprint Handoff Checklist

What must be true before the artefact is handed off to the buyer on day 12.

## Artefact

- [ ] Final artefact stored in the buyer's private Dealix folder.
- [ ] Artefact path recorded in `pipeline_tracker.csv` for the buyer's row.
- [ ] Artefact filename includes sprint ID.

## Documentation

- [ ] One-page "how to use this artefact" note included.
- [ ] Any agent prompts used are documented in the private AI Run Ledger (not in the buyer-facing artefact).

## Trust posture

- [ ] No PII outside what the buyer originally provided.
- [ ] No third-party data without verified licensing.
- [ ] All A1 messages were human-approved before sending.

## Buyer acknowledgement

- [ ] The buyer has acknowledged receipt of the artefact in writing.
- [ ] The buyer has confirmed they can open and read it.

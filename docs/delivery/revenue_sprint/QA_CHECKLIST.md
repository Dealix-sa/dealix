# Revenue Sprint QA Checklist (Public)

This is the canonical QA checklist used to gate every Revenue Sprint delivery.
The same checklist is mirrored in the private operations area for per-client runs.

## Lead Quality
- [ ] Every lead matches ICP.
- [ ] Every A-priority lead has evidence.
- [ ] Duplicates removed.
- [ ] Bad-fit leads removed.
- [ ] Sector relevance is clear.

## Outreach Quality
- [ ] Messages are specific.
- [ ] Messages are short.
- [ ] CTA is clear.
- [ ] No fake urgency.
- [ ] No guaranteed results.
- [ ] Arabic + English versions present.

## Trust Quality
- [ ] No sensitive data exposed.
- [ ] No unsupported public claim.
- [ ] No compliance claim.
- [ ] No client name used publicly without approval.
- [ ] Approval requirement checked.

## Delivery Quality
- [ ] Executive summary included.
- [ ] Lead table included.
- [ ] Outreach pack included.
- [ ] 14-day action plan included.
- [ ] Next step included.

## Final Decision
PASS / FIX / BLOCKED

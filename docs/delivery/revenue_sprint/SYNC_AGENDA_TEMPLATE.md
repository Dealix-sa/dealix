# Revenue Sprint Sync Agenda Template

Each of the four 30-minute syncs follows the same agenda. The structure exists so we do not drift into open-ended consulting.

## Sync agenda (30 minutes)

| Minute | Item |
|---|---|
| 0-5 | Where the artefact stands. One screen share. |
| 5-15 | What the buyer needs to decide today. One decision per sync. |
| 15-25 | Open questions the buyer or Dealix is holding. |
| 25-30 | Next steps. Single owner per item. |

## After-sync rules

- A one-paragraph note is written by the Delivery Lead in the sprint workspace within 24 hours.
- Any decision is logged in the AI Run Ledger with sprint ID and decision text.
- Any scope question is converted to a Change Request the same day.

## Calendar discipline

- No sync runs longer than 30 minutes. If a longer conversation is needed, a separate session is scheduled.
- No sync starts without a pre-agreed agenda item.
- The Delivery Lead may skip a sync (with the buyer's agreement) if no decision is pending; this is logged.

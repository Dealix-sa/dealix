# SDR Scorecard — بطاقة أداء SDR

## Purpose
Performance metrics for the SDR / VA role (sales development representative or commercial-oriented virtual assistant). Scorecard is reviewed monthly and at quarter-end.

## Metrics (monthly)

| Metric | Definition | Target (monthly) |
|---|---|---|
| Qualified intros initiated | Approved outbound messages sent following voice/process rules | ≥350 |
| Reply rate | % of intros that get any reply | ≥10% |
| Positive reply rate | % of intros that get an interested reply | ≥3% |
| Discovery calls booked | Calls placed on founder's calendar | ≥8 |
| Lead qualification accuracy | % of booked calls that pass qualification at the call | ≥70% |
| Off-process incidents | Count of incidents (banned claims, off-brand language, list violations) | 0 |
| CRM hygiene | % of contacts with complete fields and source citation | ≥95% |

## Behaviors expected
- Uses approved messaging templates only; deviations sent for review.
- Records every outreach action with date, channel, version.
- Flags any unusual request from a prospect (legal, regulatory, complaint).
- Does not use scraping, bulk-list import, or automated mass DMs.
- Respects PDPL and the data-handling rules per `docs/governance/PDPL_DATA_RULES.md`.

## Bands

| Band | Score | Action |
|---|---|---|
| Top performer | All metrics at or above target; zero incidents | Continue; consider scope expansion |
| Meeting bar | All metrics within 20% of target | Continue; coach on lowest metric |
| Below bar | One or more metrics >20% below target | 30-day plan with weekly check-ins |
| Off-process | Any incidents; or below-bar two months running | Performance review per `REVIEW_CADENCE.md` |

## Rules
- A single off-process incident triggers a written remediation note. Two in a quarter triggers role review.
- Scorecard is reviewed in a 30-minute monthly call between founder and SDR.
- Targets reset if the underlying ICP changes; documented when reset.

## Operations
- Monthly: SDR submits the numbers; founder reviews and signs off within 5 days.
- Quarterly: scorecard summary contributes to compensation review per `COMPENSATION.md`.

## Evidence
- Each metric value links to its source (CRM export, message log).

## Owner & cadence
- Owner: Founder (manager); SDR submits.
- Cadence: monthly review; quarterly summary.

## القسم العربي

**المقاييس الشهرية:** تعريفات مؤهلة مُبتدأة ≥350، نسبة الردود ≥10%، نسبة الردود الإيجابية ≥3%، مكالمات اكتشاف محجوزة ≥8، دقة تأهيل ≥70%، حوادث خارج العملية = 0، نظافة CRM ≥95%.

**السلوكيات:** قوالب معتمدة فقط، تسجيل كل تواصل، الإبلاغ عن أي طلب غير عادي، لا scraping أو bulk import أو DMs آلية جماعية، احترام PDPL.

**الفئات:** ممتاز (جميع الأهداف، صفر حوادث). يفي بالمعيار (ضمن 20% من الهدف). تحت المعيار (مقياس واحد >20% أسفل). خارج العملية (أي حادثة أو شهرين تحت المعيار).

**القواعد:** حادثة خارج العملية تستدعي ملاحظة معالجة مكتوبة. اثنتان في الربع تستدعي مراجعة دور.

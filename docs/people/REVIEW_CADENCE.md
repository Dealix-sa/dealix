# Review Cadence

How performance is reviewed at Dealix. Short, factual, written down.

## Cadence

| Cadence | Format | Output |
|---|---|---|
| Weekly | 1:1 with Founder, 20 minutes | One paragraph in a shared doc |
| Monthly | Written self-review, 1 page | Filed in the personnel folder |
| Quarterly | Founder review, 1 hour | Score + agreed actions |
| Annual | Comp and equity review | Updated agreement |

## Review structure

- What was the named outcome this period?
- Did it ship?
- What got in the way?
- What changes next period?

## Confidentiality

- Performance notes live in the personnel folder, private.
- Comp decisions are never discussed across roles.

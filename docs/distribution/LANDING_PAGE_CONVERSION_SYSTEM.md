# Landing Page Conversion System

## Purpose
Turn Dealix website traffic into sample requests, calls, and Revenue Sprint leads.

## Page Sections
1. **Hero** — Saudi B2B revenue intelligence and execution systems.
2. **Problem** — B2B teams waste time on unqualified leads and unclear outreach.
3. **Offer** — Revenue Sprint: qualified opportunity targets + outreach angles + 14-day action plan.
4. **Proof** — Use approved proof only.
5. **Process** — ICP → opportunity map → outreach pack → action plan.
6. **CTA** — Request a sample.
7. **Trust Note** — No guaranteed revenue, no spam automation, human-approved workflows.

## Forms
- Sample request form.
- Contact form.
- Sector interest form.

## Required Fields
- Name.
- Company.
- Website.
- Email.
- Sector.
- What they sell.
- Best customer.
- Permission checkbox if needed.

## Rule
Website leads go to `inbound_leads.csv` or CRM.

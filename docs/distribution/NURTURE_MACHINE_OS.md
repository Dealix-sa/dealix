# Nurture Machine OS

## Purpose
Keep relevant leads warm without spam.

## Nurture Triggers
- Not now.
- Interested later.
- No budget now.
- Asked to reconnect.
- Weak timing.

## Nurture Assets
- Sector insight.
- Checklist.
- Anonymized lesson.
- Mini report.
- Founder update.

## Cadence
- 30 days.
- 60 days.
- 90 days.

## Rules
- Respect opt-outs.
- No repeated annoying follow-ups.
- Send useful updates only.
- Keep status in `nurture_queue.csv`.

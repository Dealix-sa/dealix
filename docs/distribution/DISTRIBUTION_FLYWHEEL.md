# Distribution Flywheel

## Purpose
Make Dealix growth compound instead of restarting every week.

## Flywheel
1. Build lead intelligence.
2. Send relevant approved outreach.
3. Get replies.
4. Send samples.
5. Convert proposals.
6. Collect payment.
7. Deliver with QA.
8. Capture feedback.
9. Get proof or anonymized learning.
10. Publish content.
11. Create inbound.
12. Ask referrals.
13. Improve sector machine.

## Compounding Assets
- lead intelligence base
- message performance
- objection library
- sector scorecard
- sample library
- proposal library
- proof library
- partner network
- content library
- data model
- automation scripts

## Rule
Every paid sprint must produce at least one compounding asset.

# Distribution Flywheel / دولاب التوزيع

## Purpose / الغرض

Make Dealix growth compound instead of restarting every week.

جعل نمو Dealix يتراكم بدلاً من إعادة البدء كل أسبوع.

## Flywheel / الدولاب

1. Build lead intelligence. / بناء ذكاء العملاء.
2. Send relevant approved outreach. / إرسال تواصل معتمد ذي صلة.
3. Get replies. / الحصول على ردود.
4. Send samples. / إرسال عينات.
5. Convert proposals. / تحويل العروض.
6. Collect payment. / تحصيل الدفع.
7. Deliver with QA. / التسليم مع ضمان الجودة.
8. Capture feedback. / التقاط التغذية الراجعة.
9. Get proof or anonymized learning. / الحصول على إثبات أو تعلم مجهول الهوية.
10. Publish content. / نشر محتوى.
11. Create inbound. / خلق طلبات واردة.
12. Ask referrals. / طلب إحالات.
13. Improve sector machine. / تحسين ماكينة القطاع.

## Compounding Assets / الأصول المُتراكِمة

- lead intelligence base / قاعدة ذكاء العملاء
- message performance / أداء الرسائل
- objection library / مكتبة الاعتراضات
- sector scorecard / بطاقة درجات القطاع
- sample library / مكتبة العينات
- proposal library / مكتبة العروض
- proof library / مكتبة الإثباتات
- partner network / شبكة الشركاء
- content library / مكتبة المحتوى
- data model / نموذج البيانات
- automation scripts / سكريبتات الأتمتة

## Rule / القاعدة

Every paid sprint must produce at least one compounding asset.

كل سبرنت مدفوع يجب أن ينتج أصلاً تراكمياً واحدًا على الأقل.

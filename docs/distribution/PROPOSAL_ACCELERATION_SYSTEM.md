# Proposal Acceleration System

## Purpose
Move qualified interest into proposal quickly.

## Proposal Trigger
- Positive reply.
- Sample requested.
- Call booked.
- Buyer asks for scope / pricing.
- Clear pain exists.

## Proposal SLA
- Draft within 24 hours of qualified trigger.
- Follow-up within 48 hours.
- Close / recycle within 14 days.

## Proposal Variants
- Starter Sprint.
- Standard Sprint.
- Premium Sprint.
- Retainer / Revenue Desk.

## Rules
- No proposal without scope.
- No delivery without start condition.
- No guarantee claims.
- Follow-up date required.

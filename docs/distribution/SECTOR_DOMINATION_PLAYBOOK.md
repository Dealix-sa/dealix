# Sector Domination Playbook / دليل السيطرة القطاعية

## Purpose / الغرض

Build repeatable market expertise in one sector at a time.

بناء خبرة سوقية قابلة للتكرار في قطاع واحد في كل مرة.

## Per Sector Target / هدف كل قطاع

- 100 accounts discovered / 100 حساب مكتشف
- 50 accounts enriched / 50 حساب مُثرى
- 25 A/B leads approved / 25 عميل A/B معتمد
- 25 outreach messages / 25 رسالة تواصل
- 5 replies / 5 ردود
- 3 samples / 3 عينات
- 1 proposal / 1 عرض
- 1 payment path / 1 مسار دفع

## Sector Assets / أصول القطاع

- sector ICP / ملف العميل المثالي للقطاع
- buyer titles / مسميات المشترين
- common pain / الألم المشترك
- message angles / زوايا الرسائل
- objection patterns / أنماط الاعتراضات
- sample template / قالب العينات
- proposal variant / متغير العرض
- proof angle / زاوية الإثبات
- partner list / قائمة الشركاء

## Sector Decision / قرار القطاع

**Double Down / مضاعفة:**
if positive replies and sample interest.

إذا كانت الردود إيجابية والاهتمام بالعينات قائم.

**Fix / إصلاح:**
if replies exist but no samples.

إذا كانت الردود موجودة لكن لا توجد عينات.

**Kill / إيقاف:**
if no replies after quality outreach.

إذا لم تكن هناك ردود بعد تواصل جيد.

**Defer / تأجيل:**
if market is relevant but timing weak.

إذا كان السوق ذو صلة لكن التوقيت ضعيف.

## Rule / القاعدة

Do not scale a sector until it produces conversion signal.

لا توسّع قطاعًا قبل أن ينتج إشارة تحويل.

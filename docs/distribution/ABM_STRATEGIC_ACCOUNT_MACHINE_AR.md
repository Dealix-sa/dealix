# ماكينة الحسابات الاستراتيجية (ABM)

> النسخة الإنجليزية القانونية: [`ABM_STRATEGIC_ACCOUNT_MACHINE.md`](./ABM_STRATEGIC_ACCOUNT_MACHINE.md).

## مرجع الدوكترين
- الالتزامات: #1، #2، #5.
- القرارات المثبّتة: الموافقة-أولًا.

## الغرض

استهداف حسابات B2B سعودية عالية القيمة ببحث مخصّص، قيمة مخصّصة قبل أي عرض، وملكية حساب مسمّاة. ABM هو المسار البطيء في محفظة التصريف: حسابات أقل، تحضير أعمق، محادثة يقودها المؤسس.

## ملخّص الحساب (إلزامي لكل حساب)

| الحقل | الوصف |
|------|------|
| ملف الشركة | القطاع، الحجم، الهيكل، التحركات الأخيرة |
| المشترون المحتملون | عناوين الوحدة المشترية وأسماء أحيانًا |
| فرضية الألم | الألم المحدد اللي يقدر Dealix يعالجه |
| إشارات القطاع | أخبار، توظيف، RFPs، شراكات |
| حدث الإطلاق | ما الذي تغيّر يجعل الآن وقتًا مناسبًا |
| فكرة عينة مخصّصة | أصل محدد لهذا الحساب |
| مسار العلاقة | من يعرف من؛ مقدمة دافئة أو باردة |
| إمكانية تقديم شريك | هل في شريك يقدر يدفّي هذا الحساب |

## أهداف التشغيل (إحماء)

- 10 حسابات استراتيجية تُسمّى/شهر.
- 10 ملخّصات مخصّصة/شهر.
- 10 drafts تواصل مستوى مؤسس/شهر.
- 3 عينات مخصّصة/شهر (واحدة لكل فئة ألم مشترٍ).

أهداف نشاط لمرحلة الإحماء، مو وعود نتائج.

## القواعد الجوهرية

- الحسابات الاستراتيجية تستقبل قيمة مخصّصة قبل أي عرض.
- لا تواصل عام يُرسَل لحساب استراتيجي. كل لمسة مخصّصة.
- حساب استراتيجي ما رد على لمستين مخصّصتين يُراجَع للملاءمة قبل اللمسة الثالثة.
- الرد الإيجابي يقفز فورًا لدورة العينة/العرض بأولوية.
- التواصل البارد لحساب استراتيجي محكوم بالموافقة مثل أي تواصل.

## الإيقاع

- شهري: تسمية 10 حسابات جديدة + ملخّصات.
- أسبوعي: مراجعة الحسابات النشطة للّمسة التالية.
- لكل لمسة: مخصّصة، معتمدة، مرسلة.
- لكل رد: موجّه لطابور العينة/العرض بأولوية.

# قائمة التحقق — PDPL + الثقة في التواصل والذكاء الاصطناعي

## مبادئ عامة

1. لا نجمع بيانات شخصية حساسة إلا بموافقة صريحة.
2. كل بيانات العملاء تُخزن بأمان وتُنقل مشفرة.
3. كل مخرج AI يُراجع قبل تسليمه للعميل.
4. التواصل الخارجي يدوي أو مسودة بشرية، لا إرسال تلقائي.
5. نحتفظ بسجل audit لكل إجراء.

## PDPL Outreach Checklist

- [ ] الهدف: شركات B2B عامة فقط
- [ ] لكل شركة: source_url و verification_status و confidence
- [ ] لا بيانات شخصية حساسة (هوية، عنوان منزل، تاريخ ميلاد)
- [ ] كل رسالة تحتوي على opt-out واضح
- [ ] لا scraping لبيانات خاصة
- [ ] لا مواضيع misleading
- [ ] لا وعود مضمونة

## AI Trust Checklist

- [ ] مخرجات AI مُعلَّمة كمسودات
- [ ] مراجعة بشرية قبل الإرسال
- [ ] لا قرارات مالية/قانونية تلقائية
- [ ] لا نتائج طبية/مالية بدون expert review
- [ ] سجل التدقيق متاح للمراجعة

## Audit Log

- `ledgers/outreach_log.csv` — كل رسالة مُرسلة
- `ledgers/reply_log.csv` — الردود والتصنيف
- `reports/revenue/YYYY-MM-DD/` — التقارير اليومية

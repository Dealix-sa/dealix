# Client Onboarding — إعداد العميل

The first two weeks after a deal closes. Set scope, access, review steps, and the value baseline.

## Week 1 — الأسبوع الأول

**English.**
- Kickoff call: confirm scope and the workflow to pilot.
- Identify the client reviewer and sender.
- Collect example documents (no PII beyond what is needed; redact where possible).
- Set the review and approval steps.

**عربي.**
- مكالمة الانطلاق: تأكيد النطاق وسير العمل للتجربة.
- تحديد المراجع والمُرسِل لدى العميل.
- جمع مستندات نموذجية (دون بيانات شخصية زائدة؛ نُخفي حيث أمكن).
- ضبط خطوات المراجعة والموافقة.

## Week 2 — الأسبوع الثاني

**English.**
- Configure AI drafts for the chosen workflow.
- Run a test batch; reviewer edits.
- Record the value baseline (time per document today).
- Confirm the measurement window.

**عربي.**
- تهيئة مسودات الذكاء الاصطناعي لسير العمل المختار.
- تشغيل دفعة اختبارية؛ المراجع يعدّل.
- تسجيل خط الأساس للقيمة (الوقت لكل مستند اليوم).
- تأكيد نافذة القياس.

## What we confirm in writing — ما نؤكده كتابةً

- The system never sends externally; the client sends manually.
- No PII leaves the client's environment unnecessarily.
- Value figures are estimated until verified.

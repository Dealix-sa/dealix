# 30-Day Go-To-Market Plan

A founder-led, review-only motion. The system drafts; the founder sends. The
goal of the first 30 days is **proof + first paid diagnostics**, not volume sending.

## Week 0 — Setup (before day 1)

- [ ] Run `make` checks and the commercial acceptance tests (all green).
- [ ] Lock the 5 verticals (already in `config/commercial_verticals.json`).
- [ ] Build a real lead list → `data/commercial_seed_leads.jsonl` (business
      contacts / inbound / opt-in only; never scrape personal emails).
- [ ] Complete the **email domain readiness checklist** (SPF/DKIM/DMARC,
      Postmaster, bounce, unsubscribe, warm-up ramp, suppression owner).
- [ ] Decide the manual sending process (your inbox + CRM logging).

## Week 1 — Prove the loop

- [ ] Generate the daily queue, review the top 50, send **10–20 personalised
      drafts/day manually**.
- [ ] Lead with your two strongest verticals.
- [ ] Track replies in the CRM. Goal: book the first 3–5 diagnostics.

## Week 2 — First paid diagnostics

- [ ] Deliver the first **Entry Diagnostics / AI Workflow Audits** (499–2,500 SAR).
- [ ] Produce the **proof asset** per vertical (leakage map / governance map).
- [ ] Ask every diagnostic buyer for a referral.

## Week 3 — Pilot conversations

- [ ] Convert proven diagnostics into **Paid Pilots** (5,000–25,000 SAR).
- [ ] Tighten the messaging that's converting; retire what isn't.
- [ ] Expand the lead list in the two best verticals.

## Week 4 — Repeatability

- [ ] One pilot live and measured.
- [ ] First **monthly retainer** conversation (3,000–25,000 SAR/month).
- [ ] Review `daily_metrics.json` trend: drafts, reply rate, diagnostics booked.

## North-star metrics (first 30 days)

| Metric | Target |
|--------|--------|
| Daily drafts generated | ≥ 400 (automatic) |
| Personalised manual sends/day | 10–20 |
| Diagnostics booked | 8–12 |
| Paid diagnostics delivered | 4–6 |
| Pilots in conversation | 2–3 |

## Guardrails (do not break)

- No automated sending. No WhatsApp cold outreach. No LinkedIn automation.
- No guaranteed-ROI claims. No data-access claims.
- Every send is manual, personalised, and opt-out-compliant.

# Pilot Delivery Checklist — قائمة تسليم التجربة

The quality gate for a Paid Pilot. Pass every item before the pilot is called complete.

## Setup — الإعداد

- [ ] One workflow selected and scoped.
- [ ] Value baseline recorded.
- [ ] Reviewer and sender named.
- [ ] Measurement window set (2–4 weeks).

الإعداد: سير عمل واحد محدد؛ خط أساس مسجّل؛ مراجع ومُرسِل محددان؛ نافذة قياس مضبوطة (2–4 أسابيع).

## During delivery — أثناء التنفيذ

- [ ] AI drafts produced for each item.
- [ ] Every draft reviewed before use.
- [ ] Sending done manually by the client.
- [ ] No forbidden phrases in any output.
- [ ] Opt-out line present where outreach drafts are used.

أثناء التنفيذ: مسودات منتَجة؛ كل مسودة مُراجَعة؛ الإرسال يدوي من العميل؛ لا عبارات محظورة؛ سطر إلغاء الاشتراك موجود.

## Measurement — القياس

- [ ] Observed time per document recorded.
- [ ] Time saved vs baseline computed.
- [ ] Figures labeled estimated/observed, not guaranteed.

القياس: الوقت المُلاحَظ لكل مستند مسجّل؛ الوقت المُوفَّر مقابل خط الأساس محسوب؛ الأرقام مُوسومة تقديرية/مُلاحَظة لا مضمونة.

## Close — الإغلاق

- [ ] Handover report prepared. See [16_HANDOVER_SUCCESS_REPORT.md](16_HANDOVER_SUCCESS_REPORT.md).
- [ ] Retention/expansion conversation scheduled. See [17_RETENTION_EXPANSION.md](17_RETENTION_EXPANSION.md).
- [ ] No PII in any case-safe summary.

الإغلاق: تقرير التسليم مُعدّ؛ محادثة الاحتفاظ/التوسع مجدولة؛ لا بيانات شخصية في أي ملخص آمن.

# Objection Handling (AR / EN)

| Objection | Response |
|-----------|----------|
| **السعر عالي / Price is high** | We land with a low-risk Audit (from 499 SAR). You only scale after you see the bottleneck and a measurable pilot. |
| **عندنا CRM / We have a CRM** | A CRM stores contacts; we operate the *workflow* — intake, follow-up, reporting — and you approve every external step. |
| **عندنا IT / We have IT** | IT runs infrastructure. We install the operating workflow on top; we don't replace your team. |
| **ما نحتاج AI / We don't need AI** | This isn't AI for its own sake — it's fixing where work leaks today. AI drafts; you decide. |
| **نخاف على البيانات / Data concerns** | Privacy-by-design: data minimization, your consent, you approve every step, NDA/DPA before any data. |
| **أرسلوا معلومات / Send info** | Happy to — and a 15-min call lets me tailor it to your worst workflow. |
| **ليس الآن / Not now** | Understood. May I send one relevant before/after and check back next quarter? |
| **من أنتم / Who are you** | Dealix — a Saudi/GCC AI Revenue & Operations OS. Founder-led, approval-first. |
| **هل عندكم case studies / Case studies?** | We share anonymized process examples; early-stage results are framed as targets, not guarantees. |
| **هل يرسل تلقائيًا / Does it auto-send?** | No. The system never sends. AI drafts and ranks; you approve and send manually. |
| **هل البيانات آمنة / Is data safe?** | Minimization, human-in-the-loop, audit trail, and no external sending from automation. |

## Rules
- Never invent familiarity or social proof.
- Never promise ROI.
- Always reaffirm: approval-first, no blind automation.

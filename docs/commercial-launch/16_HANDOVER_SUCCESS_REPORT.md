# Handover Success Report — تقرير نجاح التسليم

The evidence document delivered at the end of a pilot or engagement. Honest, labeled, no inflated claims.

## Report structure — هيكل التقرير

**English.**
1. Scope delivered.
2. Baseline vs observed (time per document).
3. Estimated time saved across the window.
4. Reviewer notes on quality.
5. Recommended next step.

**عربي.**
1. النطاق المُسلَّم.
2. خط الأساس مقابل المُلاحَظ (الوقت لكل مستند).
3. الوقت المُوفَّر التقديري عبر النافذة.
4. ملاحظات المراجع على الجودة.
5. الخطوة التالية المُوصى بها.

## Claim discipline — انضباط الادعاءات

**English.** Figures are observed within this engagement, not guaranteed for the future. We do not present any percentage as a guarantee. Verified value is recorded only when independently confirmed.

**عربي.** الأرقام مُلاحَظة ضمن هذا التعاقد، وليست مضمونة للمستقبل. لا نقدّم أي نسبة كضمان. تُسجَّل القيمة المُتحقَّقة فقط عند تأكيدها بشكل مستقل.

## Case-safe note — ملاحظة الحالة الآمنة

**English.** If reused as a case study, this report is anonymized with no PII and labeled "Hypothetical / case-safe template" unless the client consents to be named.

**عربي.** عند إعادة استخدامه كحالة دراسية، يُجهَّل التقرير دون بيانات شخصية ويُوسم "نموذج افتراضي/آمن" ما لم يوافق العميل على ذكر اسمه.

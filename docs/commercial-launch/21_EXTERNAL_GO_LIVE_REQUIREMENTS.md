# External Go-Live Requirements / متطلبات الإطلاق الخارجي

These are OUTSIDE this repo and MUST be completed manually before any real sending.
هذه خارج المستودع ويجب تنفيذها يدويًا قبل أي إرسال حقيقي.

- Domain DNS configured
- SPF, DKIM, DMARC records
- Google Postmaster Tools (monitor spam rate < 0.3%)
- Bounce tracking + suppression process
- Manual CRM sheet or CRM pipeline
- Calendly (or manual scheduling)
- Payment/checkout (if applicable; ZATCA-compliant invoicing)
- Privacy policy, Terms, DPA (for business clients)
- Email warm-up / ramp plan
- Sender reputation monitoring
- Named manual-approval owner
- Complaint handling + incident process
- Legal/privacy (PDPL) review

> WhatsApp: contact only with prior opt-in; honor opt-out. LinkedIn: no bots/automation. These are platform requirements, enforced manually.

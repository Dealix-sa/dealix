# Handover & Success Report

## Handover pack
- Workflow documentation (before/after)
- Access & ownership map
- Approval-gate configuration
- Runbook for the workflow owner
- Known limitations and risks

## Success report
- Baseline vs. outcome on the agreed metric (framed as target vs. result)
- What changed in the workflow
- Lessons learned
- Recommended next step on the offer ladder

## Rules
- Results are reported honestly; no inflated or guaranteed claims.
- The success report is shared with the client before any external use, and any
  case study requires explicit client consent (see existing `claim_policy.yaml`).

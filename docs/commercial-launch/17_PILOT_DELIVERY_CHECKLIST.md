# Pilot Delivery Checklist

- [ ] Acceptance criteria signed off (measurable, agreed)
- [ ] Limited dataset scoped (data minimization)
- [ ] Single workflow owner identified
- [ ] Approval gates defined (who approves what)
- [ ] Working pilot built on the limited dataset
- [ ] Human-in-the-loop verified — no external auto-send anywhere
- [ ] Baseline metric captured
- [ ] Pilot metric captured vs. baseline (as target)
- [ ] Risks and edge cases documented
- [ ] Handover pack drafted
- [ ] Go/No-Go decision for Department OS recorded

## Definition of done
The client agrees the pilot produced the agreed, measurable workflow output, and
no step bypassed human approval.

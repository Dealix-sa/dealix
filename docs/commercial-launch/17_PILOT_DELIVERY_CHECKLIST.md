# Dealix — Pilot Delivery Checklist — قائمة تحقق تسليم التجربة

> **Golden rule / القاعدة الذهبية:**
> AI recommends and drafts. Deterministic workflows verify. Founder approves. Nothing is sent automatically.
> الذكاء يقترح ويصيغ، مسارات حتمية تتحقق، المؤسس يعتمد، لا شيء يُرسل تلقائيًا.

Related: [16_CLIENT_ONBOARDING.md](16_CLIENT_ONBOARDING.md) · [15_DELIVERY_OPERATING_SYSTEM.md](15_DELIVERY_OPERATING_SYSTEM.md) · [18_HANDOVER_AND_SUCCESS_REPORT.md](18_HANDOVER_AND_SUCCESS_REPORT.md)

**EN —** Use this checklist for every paid pilot (5,000–25,000 SAR, 14–30 days). The pilot proves value on one workflow with a limited dataset and a measurable result — under a human approval gate.

**AR —** استخدم هذه القائمة لكل تجربة مدفوعة (5,000–25,000 ريال، 14–30 يومًا). التجربة تثبت القيمة على مسار واحد ببيانات محدودة ونتيجة قابلة للقياس — تحت بوابة اعتماد بشري.

---

## Before start / قبل البدء
- [ ] Scope and approver confirmed in writing. / تأكيد النطاق والمعتمِد كتابيًا.
- [ ] Limited dataset received; data boundary documented. / استلام البيانات المحدودة وتوثيق الحدود.
- [ ] Baseline metric recorded. / تسجيل مؤشر الأساس.
- [ ] Approval gate agreed (who approves, how). / الاتفاق على بوابة الاعتماد.

## During the pilot / أثناء التجربة
- [ ] One workflow built; draft outputs generated. / بناء مسار واحد وتوليد المخرجات.
- [ ] Every draft includes an opt-out line where it is an email/WhatsApp message. / كل مسودة بريد/واتساب تحتوي سطر إلغاء اشتراك.
- [ ] No banned claims (guaranteed ROI, "we have your data," "as discussed"). / لا ادعاءات ممنوعة.
- [ ] Sensitive-sector drafts privacy-first, human-in-the-loop. / مسودات القطاع الحساس قائمة على الخصوصية وبتدخل الإنسان.
- [ ] Outputs stay drafts; nothing sent from any system. / المخرجات تبقى مسودات؛ لا إرسال من أي نظام.
- [ ] Client reviews drafts on the agreed cadence. / مراجعة العميل بالإيقاع المتفق.

## Verification / التحقق
- [ ] Deterministic checks passed on all drafts. / اجتياز الفحوص الحتمية.
- [ ] Result measured against baseline (described as estimated). / قياس النتيجة مقابل الأساس (موصوفة كتقدير).
- [ ] No scraping, no extraction, no auto-send occurred. / لا استخراج ولا إرسال تلقائي.

## Closeout / الإغلاق
- [ ] Success report drafted ([18](18_HANDOVER_AND_SUCCESS_REPORT.md)). / صياغة تقرير النجاح.
- [ ] Handover artifacts assembled. / تجميع مخرجات التسليم.
- [ ] Data retained or deleted per the agreed boundary. / الاحتفاظ أو الحذف حسب الحدود المتفقة.
- [ ] Upsell to Department OS framed honestly. / تأطير الترقية بصدق.
- [ ] Metrics dashboard updated ([20](20_COMMERCIAL_METRICS_DASHBOARD.md)). / تحديث لوحة المؤشرات.

---

## Stop conditions / شروط التوقف

**EN —** Stop the pilot if the client requests scraping, mass outreach, auto-sending, removal of the approval gate, or access to data outside the agreed boundary. These are non-negotiable.

**AR —** أوقف التجربة إن طلب العميل استخراجًا أو تواصلًا جماعيًا أو إرسالًا تلقائيًا أو إزالة بوابة الاعتماد أو وصولًا لبيانات خارج الحدود المتفقة. هذه غير قابلة للتفاوض.

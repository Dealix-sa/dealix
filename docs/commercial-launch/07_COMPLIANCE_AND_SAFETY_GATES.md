# Compliance and Safety Gates — بوابات الامتثال والسلامة

Hard stops that must pass before any message, proposal, or report leaves Dealix.

## Governing rule — القاعدة الحاكمة

**AI drafts, ranks, and recommends. Founder reviews, approves, and sends manually. The system never sends externally.**

الذكاء الاصطناعي يكتب المسودات ويرتّب الأولويات ويوصي. المؤسس يراجع ويوافق ويرسل يدويًا. النظام لا يرسل إلى أي طرف خارجي إطلاقًا.

## Gate 1 — Manual send — الإرسال اليدوي

Every external message is sent by the founder by hand. No automation path exists. See [05_CHANNEL_POLICY.md](05_CHANNEL_POLICY.md).

## Gate 2 — Forbidden phrases — العبارات المحظورة

Block any artifact containing: "guaranteed ROI", "100%", "replace your team", "replace lawyers", "automate everything", "no human needed", "from our database", "as discussed" (without prior contact), or fabricated urgency.

احجب أي مادة تحتوي على: "عائد مضمون"، "100%"، "نستبدل فريقك"، "نستبدل المحامين"، "أتمتة كل شيء"، "بدون تدخل بشري"، "من قاعدة بياناتنا"، "كما اتفقنا" (دون تواصل سابق)، أو استعجال مُفتعَل.

## Gate 3 — Outreach hygiene — نظافة التواصل

- Opt-out line present.
- No faked familiarity.
- WhatsApp only after opt-in.
- One recipient per message.

سطر إلغاء الاشتراك موجود. لا افتعال للألفة. واتساب بعد موافقة فقط. مستلم واحد لكل رسالة.

## Gate 4 — Claim safety — سلامة الادعاءات

All value figures are labeled estimated until verified. No percentages presented as guarantees. See [CLAIM_SAFETY.md](../05_governance_os/CLAIM_SAFETY.md).

كل أرقام القيمة تُوسم تقديرية حتى التحقّق. لا نسب تُقدَّم كضمانات.

## Gate 5 — Privacy — الخصوصية

No PII in case studies or reports: no emails, phones, national IDs, or real names. Use anonymized labels. PDPL-aware language throughout. See [PII_CLASSIFICATION.md](../04_data_os/PII_CLASSIFICATION.md) and [PDPL_AWARE_LANGUAGE.md](../02_saudi_positioning/PDPL_AWARE_LANGUAGE.md).

## Gate checklist — قائمة البوابات

| Gate | Pass condition |
|---|---|
| Manual send | Founder sends by hand |
| Forbidden phrases | None present |
| Outreach hygiene | Opt-out + no faked familiarity |
| Claim safety | Estimated labeled |
| Privacy | No PII |

# Proposal Factory v3 / مصنع العروض v3

## Purpose / الغرض

Convert qualified interest into approved proposal drafts quickly.

تحويل الاهتمام المؤهل إلى مسودات عروض معتمدة بسرعة.

## Proposal Inputs / مدخلات العرض

- company / الشركة
- sector / القطاع
- trigger / المحفّز
- sample result / نتيجة العينة
- pain hypothesis / فرضية الألم
- recommended offer / العرض الموصى به
- price band / نطاق السعر
- start condition / شرط البدء
- trust notes / ملاحظات الثقة

## Proposal Types / أنواع العروض

- Starter Revenue Sprint / سبرنت إيرادات مبتدئ
- Standard Revenue Sprint / سبرنت إيرادات قياسي
- Premium Revenue Sprint / سبرنت إيرادات متميز
- Managed Pilot / تجربة مُدارة
- Revenue Desk Retainer / اشتراك مكتب الإيرادات
- Enterprise Command Center / مركز قيادة المؤسسة

## SLA / اتفاقية الخدمة

- Draft within 24h after qualified trigger. / مسودة خلال 24 ساعة بعد المحفّز المؤهل.
- Follow-up within 48h. / متابعة خلال 48 ساعة.
- Decision push within 7 days. / دفع نحو قرار خلال 7 أيام.
- Close/recycle within 14 days. / إغلاق/إعادة تدوير خلال 14 يومًا.

## Rules / القواعد

- No proposal without clear next action. / لا عرض بدون إجراء تالٍ واضح.
- No guaranteed revenue. / لا ضمانات إيراد.
- No hidden scope. / لا نطاق مخفي.
- No delivery without start condition. / لا تسليم بدون شرط بدء.

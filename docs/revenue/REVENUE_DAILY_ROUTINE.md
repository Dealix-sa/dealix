# Revenue Daily Routine
## Morning
- Run mission-control.
- Run revenue-ops.
- Check top CEO action.
- Pick today's revenue move.
## Execution
If pipeline < 25:
- add leads.
If DMs < 25:
- send founder-led DMs.
If replies exist:
- prepare sample or book call.
If sample sent:
- follow up for proposal.
If proposal sent:
- pursue payment / PO / written approval.
If paid:
- run delivery kickoff.
## End of Day
- update pipeline.
- update revenue_action_log.csv.
- update evidence ledger.
- run close-day.
## Rule
No day closes without one revenue action.

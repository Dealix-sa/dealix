# Proposal Conversion System
## Purpose
Turn qualified interest into paid Revenue Sprint or PO / written approval.
## Owner
Sami / Founder CEO.
## Proposal Required Fields
- client
- objective
- scope
- deliverables
- timeline
- price
- start condition
- follow-up date
- trust note
## Start Condition
Work starts after:
- payment confirmation
- purchase order
- written approval from authorized decision maker
## Proposal Rules
- No proposal without clear scope.
- No proposal without follow-up date.
- No proposal without decision maker or next step.
- No guaranteed revenue.
- No full delivery before start condition.
## Follow-Up Cadence
Day 1: confirm received.
Day 3: answer questions.
Day 7: close / disqualify.
Day 14: final polite follow-up.
## Metrics
- proposals sent.
- proposal-to-payment rate.
- average deal size.
- days to close.
- objections.

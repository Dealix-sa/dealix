# Offer Ladder v3 / سلم العروض v3

## Purpose / الغرض

Move buyers from low-friction sample to paid sprint to retainer.

نقل المشترين من عينة منخفضة الاحتكاك إلى سبرنت مدفوع إلى اشتراك شهري.

## Offer Ladder / سلم العروض

### 1. Free / Low-Friction Sample / عينة مجانية أو منخفضة الاحتكاك
**Goal / الهدف:**
start conversation.

بدء محادثة.

**Output / المخرج:**
5 opportunity targets + suggested angles.

5 أهداف فرصة + زوايا مقترحة.

### 2. Revenue Sprint / سبرنت الإيرادات
**Price / السعر:**
2,500–7,500 SAR.

**Output / المخرج:**
30–75 qualified targets + outreach pack + 14-day action plan.

30–75 هدفًا مؤهلًا + حزمة تواصل + خطة عمل 14 يومًا.

### 3. Managed Pilot / تجربة مُدارة
**Price / السعر:**
9,500–25,000 SAR.

**Output / المخرج:**
deeper opportunity engine + outreach support + weekly review.

محرك فرص أعمق + دعم تواصل + مراجعة أسبوعية.

### 4. Revenue Desk Retainer / اشتراك مكتب الإيرادات
**Price / السعر:**
5,000–20,000 SAR/month.

**Output / المخرج:**
weekly opportunity flow + messaging + pipeline review + sector learning.

تدفق فرص أسبوعي + رسائل + مراجعة الأنبوب + تعلم قطاعي.

### 5. Enterprise Command Center / مركز قيادة المؤسسة
**Price / السعر:**
25,000–100,000 SAR/month.

**Output / المخرج:**
custom revenue intelligence, executive reporting, trust controls, integrations.

ذكاء إيرادات مخصص، تقارير تنفيذية، ضوابط ثقة، تكاملات.

## Rule / القاعدة

Do not jump to enterprise before proof.

لا تقفز إلى المؤسسة قبل الإثبات.

# Sales Funnel

The funnel that turns market attention into recurring revenue.

```
Attention → Lead → Reply → Sample → Call → Proposal → Paid → Delivered → Retainer
```

## Metrics per layer
| Layer | Metric | Target |
|---|---|---|
| Attention | Impressions, profile visits | Tracked, not optimised in early stage |
| Lead | Qualified leads / week | 25+ |
| Reply | Reply rate | 10–20% |
| Sample | Sample acceptance rate | 40%+ |
| Call | Call-to-book rate | 30%+ |
| Proposal | Proposal-to-close rate | 25%+ |
| Paid | Average deal value | Tracked, growing |
| Delivered | On-time delivery | 95%+ |
| Retainer | Conversion to retainer | 20%+ of paid clients |

## Sources of attention
- Founder content (LinkedIn, X, written reports).
- Partner referrals.
- Direct outbound to ICP.
- Inbound from public case studies and sector reports.

## Rule
A funnel without numbers is not a funnel. Each layer has a measured rate, reviewed weekly.

# Sample Factory

> Produces free diagnostic samples and trial artifacts for ICP-fit prospects.

## 1. Sample types

| Type | Purpose | Time-box |
|---|---|---|
| Sector diagnostic | One-page diagnostic for a target sector | 24 h |
| Account brief | Per-account intelligence brief | 48 h |
| Outbound sample | 3 personalised drafts for a customer's pipeline | 48 h |
| Distribution audit | Read of customer's current motion | 5 days |
| Revenue Sprint kickoff | Sprint-shaped diagnostic | 7 days |

## 2. Production rules

- Built from real, citable signal.
- No invented numbers.
- Bilingual AR + EN.
- Stored in `clients/<customer>/samples/` (or fallback in `templates/samples/`).
- Brand check passes before delivery.

## 3. Delivery

- Sample is delivered through the manual operator channel (founder email, partner intro).
- Delivery is recorded in audit.
- Recipient is added to nurture if not in suppression.

## 4. KPI

| KPI | Target |
|---|---|
| Time to sample | ≤ time-box for type |
| Brand-pass rate | ≥ 95 % |
| Sample → next-rung conversion | tracked weekly |

## 5. Trust

Samples never include another customer's data. Public signal only, unless the customer has signed an explicit data-use clause.

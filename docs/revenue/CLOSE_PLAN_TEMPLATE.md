# Close Plan Template

A mutual plan shared with the buyer that names the steps, owners, and dates
between "interested" and "signed". Used for any deal above the deal-desk
threshold. See [DEAL_DESK_SYSTEM](DEAL_DESK_SYSTEM.md).

## Template

```
Close Plan — <Customer> × Dealix

Goal: <one sentence outcome the buyer wants>
Target sign date: <YYYY-MM-DD>

Decision-makers:
  Economic: <name, title>
  Champion: <name, title>
  User: <name, title>

Validation:
  - [ ] <step>  Owner: <name>  Due: <YYYY-MM-DD>
  - [ ] <step>  Owner: <name>  Due: <YYYY-MM-DD>

Commercial:
  - [ ] Pricing aligned to <package> in PRICING_AND_PACKAGING.md
  - [ ] Payment terms within INVOICE_FLOW.md
  - [ ] Procurement requirements (see docs/enterprise/PROCUREMENT_READINESS.md)

Implementation:
  - [ ] Kickoff date
  - [ ] First milestone date

Risks:
  - <risk> → <mitigation>
```

## Use

- One close plan per deal above the deal-desk threshold
- Shared with the buyer for explicit agreement
- Updated weekly during [PIPELINE_REVIEW_RHYTHM](PIPELINE_REVIEW_RHYTHM.md)
- Closed (won/lost) within 48 hours of resolution and reviewed in [WIN_LOSS_REVIEW](WIN_LOSS_REVIEW.md)

## Non-negotiables

The close plan is shared with the buyer; the contents must not contain any
commitment that has not been internally approved through [DEAL_DESK_SYSTEM](DEAL_DESK_SYSTEM.md).
See [`docs/founder/DO_NOT_SAY.md`](../founder/DO_NOT_SAY.md).

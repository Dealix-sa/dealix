# Dealix Command Sprint — العرض الرئيسي — The Flagship Offer

## ما هو الـ Command Sprint

الـ **Command Sprint** هو أول عرض تجاري لـ Dealix: تشخيص تشغيلي مكثّف مدته **7 أيام** يحوّل شركة واحدة من "عندنا بيانات وفوضى" إلى **Command Pack** واحدة — حزمة قرار تنفيذية يقرأها صاحب القرار في 20 دقيقة ويتصرّف بناءً عليها.

هذا ليس وعدًا بالإيرادات. هذا وعد بـ **وضوح القرار + أدلة + خطوة تالية واحدة**.

> Dealix ليست أداة. Dealix هي نظام تشغيل أعمال. الـ Command Sprint هو أصغر باب لأكبر رؤية.

## ما الذي يتسلّمه العميل (The Command Pack)

| Operating System | المُخرَج — Deliverable | المحتوى |
|---|---|---|
| Market Intelligence OS | Company Intelligence Brief | من هي الشركة، السوق، المنافسون، نقاط الضعف التشغيلية المُوثَّقة بمصدر |
| Revenue OS | Revenue Map | أين يتسرّب الإيراد، أين الفرص المُثبتة بأدلة، ترتيب الأولويات |
| Proof OS | Proof Register | كل ادعاء مربوط بمصدر/دليل — لا أرقام بلا مرجع |
| Command OS | Executive Command Brief | صفحة قرار واحدة لصاحب القرار: الوضع، الخيارات، التوصية |
| Governance OS | Approval Register | كل إجراء خارجي يحتاج موافقة بشرية مُسجَّلة — مَن وافق ومتى |
| Delivery OS Lite | Next Action Board | 3 إجراءات تالية مرتّبة بالأثر، مالك ومُهلة لكل إجراء |

**الـ Command Pack الكاملة** = Company Intelligence Brief + Revenue Map + Proof Register + Approval Register + Executive Command Brief + Next Action Board + **Upsell Recommendation** (توصية واحدة موثّقة لما بعد الـ Sprint).

## سلّم السعر — The Price Ladder

| المرحلة — Stage | السعر — Price (SAR) | الشرط — Gate |
|---|---|---|
| أول 3 عملاء — First 3 customers | 499 – 1,500 | Proof Deposit — مقابل التزام بحالة دراسية / مراجعة |
| بعد 3 Proof Packs | 3,000 – 5,000 | أثبتنا التسليم 3 مرّات بأدلة |
| بعد حالة دراسية واحدة معتمدة | 5,000 – 7,500 | Case study موافَق عليها من العميل |

### إطار الـ Proof Deposit

مبلغ **499 ليس السعر الحقيقي للعمل**. هو **Proof Deposit** — التزام إثبات لأول 3 عملاء فقط. نحن نتحمّل فجوة القيمة عمدًا مقابل: حالة دراسية، أو إحالة، أو شهادة موثّقة. بعد أن نملك 3 Proof Packs، يختفي هذا السعر نهائيًا. السعر يرتفع مع الدليل، لا مع الطلب.

> الخصم عندنا ليس لأن العميل قال "غالي". الخصم مقابل قيمة مقابلة موثّقة فقط.

## التموضع — Positioning (ما ليس عليه هذا العرض)

- **ليس CRM.** لا ندير جهات اتصال نيابة عنك.
- **ليس Chatbot.** لا نبيع محادثة آلية.
- **ليست وكالة تسويق.** لا نشغّل حملات ولا نرسل رسائل جماعية.

الـ Command Sprint هو **طبقة قرار** فوق بياناتك: ذكاء سوق + خريطة إيراد + أدلة + حوكمة، مغلّفة في حزمة تنفيذية واحدة.

## ندرة مقصودة — Founder Scarcity

> "نفتح 3 Command Sprints فقط هذا الشهر لأن التسليم يدوي ومبني على مراجعة مباشرة من المؤسس."

هذه ليست حيلة تسويقية. التسليم فعلاً يدوي في هذه المرحلة، وكل Pack تُراجَع شخصيًا. الجودة مربوطة بالسعة، والسعة محدودة بصدق.

## ما يجب ألا يتوقّعه العميل — What NOT to Expect

- **لا ضمان إيرادات.** نعطي فرصًا مُثبتة بأدلة، لا أرقام مبيعات مضمونة.
- **لا إرسال تلقائي.** لا رسالة تغادر باسم العميل دون موافقته الصريحة.
- **لا واتساب جماعي، لا scraping خلف تسجيل دخول، لا تجاوز لشروط المصادر أو robots.txt.**
- **لا استخدام لبيانات العميل في تدريب أي نموذج.**

## اتفاقية مستوى الخدمة — 7-Day SLA

| اليوم | المرحلة |
|---|---|
| 1 | الاستلام + Market Intelligence + جمع الأدلة |
| 2–3 | بناء Company Intelligence Brief + Revenue Map |
| 4 | Proof Register + التحقق من كل ادعاء بمصدر |
| 5 | Executive Command Brief + Next Action Board |
| 6 | مراجعة المؤسس + Approval Register |
| 7 | تسليم الـ Command Pack + Upsell Recommendation |

# Founder Shortlist Rules — قواعد قائمة المؤسس

> العتبات: [`data/targeting/scoring_weights.yml`](../../data/targeting/scoring_weights.yml) (`thresholds`)

## من يدخل القائمة

شركة تدخل founder shortlist فقط إذا:
- درجتها **A أو A+** (`targeting_score ≥ 80`)، و
- حالة الامتثال **approved** (وليست `needs_review`/`reject`)، و
- لديها **دليلان مستقلان على الأقل**.

الهدف اليومي: **10–20 شركة** في القائمة، تتقلّص إلى **5–10 درافتات**، ثم **3–5
إرسالات يدوية** فقط.

## من target إلى client (Gates)

| المرحلة | Gate |
|---|---|
| Target | score ≥ 80 |
| Draft | ≥ 2 مصادر أدلة |
| Send | موافقة المؤسس |
| Diagnostic | تأكيد الألم |
| Offer | قيمة عمل واضحة |
| Sprint | مدفوع |
| Managed | proof مُسلّم |

```
Target → founder review → manual message → diagnostic
→ command sprint → command pack → managed business OS
```

## Stop Doing

- لا تُدرج شركة بمصدر دليل واحد ضعيف.
- لا تُرسل بدون موافقة المؤسس.
- لا تستهدف قطاعًا حساسًا دون مراجعة حوكمة + زاوية Governance OS.

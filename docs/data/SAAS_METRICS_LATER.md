# SaaS Metrics Later
## Purpose
Define SaaS metrics to use only when Dealix has SaaS-like revenue.
## Not Primary Yet
Before SaaS, Dealix should focus on:
- cash collected
- pipeline value
- proposal-to-payment rate
- delivery QA
- retainer attempts
- MRR if retainers exist
## SaaS Metrics Later
- MRR
- ARR
- CAC
- LTV
- churn
- net revenue retention
- gross margin
- CAC payback
- Rule of 40
## Rule
Do not optimize SaaS metrics before there is repeatable recurring revenue.

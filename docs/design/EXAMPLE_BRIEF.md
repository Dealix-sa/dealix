# Dealix Design Artifact Brief Template

Use this template before generating a dashboard, deck, landing page, proof pack, or prototype.

```text
Artifact name:
Artifact type:
Business goal:
Primary user:
Source context:
Target audience:
Sections/screens needed:
Must include:
Must avoid:
Data sources:
Claims/proof available:
Approval state:
Safety status:

```

## Default artifact rule

If any field is missing, generate a draft and label assumptions. Do not mark the artifact as approved for client or production use.

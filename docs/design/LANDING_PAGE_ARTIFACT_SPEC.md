# Dealix Landing Page Artifact Spec

## Goal

Create a landing page artifact that explains Dealix's value clearly and converts qualified Saudi B2B prospects into a discovery/pilot conversation.

## Required sections

1. Hero: clear outcome
2. Pain: scattered follow-up, unclear decisions, slow delivery proof
3. Offer: Dealix operating system layer
4. Product surfaces: Revenue Command Room, Company Brain, Client Growth Operator, Client Delivery OS
5. Trust: draft-first, reviewed communications, proof packs
6. Pilot: 7-14 day focused workflow
7. CTA: book/reply/request pilot

## Safety rule

No fake logos, fake clients, fake ROI, guaranteed revenue, or hidden automation claims.

## Output recommendation

Start as markdown or isolated HTML preview before touching production routes.

# Founder War Room Artifact Spec

## Goal

Create a Dealix founder-facing artifact that shows the most important operating decisions of the day.

## Primary user

Founder / CEO.

## Required sections

1. What changed today
2. Highest-value decision
3. Revenue pressure
4. Delivery pressure
5. Trust/safety risk
6. Product/engineering blocker
7. Recommended action
8. Review date

## Required information architecture

```text
Signal -> Decision -> Action -> Proof
```

## Output recommendation

Start as markdown in `reports/design/founder-war-room-v0.md`, then promote to a dashboard route only after review.

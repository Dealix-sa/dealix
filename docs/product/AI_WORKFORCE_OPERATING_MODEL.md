# AI Workforce Operating Model

AI agents in Dealix are not autonomous employees. They are governed workers inside controlled workflows.

## Agent Types

1. **Analyst Agents**  
   Analyze data, classify, summarize, score.

2. **Drafting Agents**  
   Create drafts, reports, scripts, recommendations.

3. **Retrieval Agents**  
   Search knowledge and answer with sources.

4. **Workflow Agents**  
   Move tasks through defined workflows.

5. **Guardrail Agents**  
   Check safety, claims, PII, and policy.

6. **Reporting Agents**  
   Generate reports and proof packs.

## Rule

Every agent has:

- role  
- allowed tools  
- forbidden actions  
- input schema  
- output schema  
- QA requirement  
- approval requirement  
- audit log

# Product Metrics

The metrics that say whether product is helping the business.

## Reliability
- Uptime of any user-facing surface.
- Error rate per critical workflow.
- Time to detect, time to recover for incidents.

## Adoption
- Active users of any internal tool that should reduce founder time.
- Volume of leads / messages / reports produced by automation vs by hand.

## Quality
- QA pass rate (first pass) on automation-produced outputs.
- Claim Guard violations caught vs missed.

## Cost
- Compute cost per active client.
- Token cost per agent run.

## Cadence
- Reviewed monthly with the founder.
- Snapshots stored in `dealix-ops-private/product/metrics/`.

## Rule
A metric that no one reviews is a wallpaper metric. Either review it or remove it.

# Data Sources Policy (Dealix Data OS)

## Allowed sources

```text
customer_upload
client CRM export
inbound forms
referrals
manual research (non-invasive)
public / open data (license-respecting)
licensed data provider
```

## Restricted

```text
Public business information may support research and benchmarks.
Personal contact data requires lawful basis, relationship context, or consent.
```

## Blocked

```text
scraped personal data
unknown-source contact lists
cold WhatsApp automation
LinkedIn automation
```

## Open data (Saudi)

Open government datasets can support **market intelligence** and benchmarks — not a blanket license for **personal outreach**. Always separate “market facts” from “contacting people.”

See also: [`SOURCE_REGISTRY.md`](../ledgers/SOURCE_REGISTRY.md), [`AI_INFORMATION_GOVERNANCE.md`](../governance/AI_INFORMATION_GOVERNANCE.md).

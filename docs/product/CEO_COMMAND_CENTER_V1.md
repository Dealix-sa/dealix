# CEO Command Center v1 / مركز قيادة الرئيس التنفيذي v1

## Purpose / الغرض

Give Sami one screen to control the whole company.

إعطاء سامي شاشة واحدة للتحكم في الشركة بأكملها.

## Panels / اللوحات

### 1. Top CEO Action / الإجراء الأعلى للرئيس التنفيذي
The one action that matters now.

الإجراء الوحيد الذي يهم الآن.

### 2. Revenue Funnel / قُمع الإيرادات
Market accounts, leads, approved outreach, sent, replies, samples, proposals, payment.

حسابات السوق، العملاء، التواصل المعتمد، المُرسَل، الردود، العينات، العروض، الدفع.

### 3. Approval Inbox / صندوق الموافقات
Outreach, proposals, proof, pricing, trust escalations.

التواصل، العروض، الإثباتات، التسعير، تصعيدات الثقة.

### 4. Follow-Up Due / المتابعات المستحقة
All follow-ups due today.

كل المتابعات المستحقة اليوم.

### 5. Sample Factory / مصنع العينات
Positive replies needing sample.

الردود الإيجابية التي تحتاج عينة.

### 6. Proposal Factory / مصنع العروض
Qualified opportunities needing proposal.

الفرص المؤهلة التي تحتاج عرضًا.

### 7. Payment Capture / تحصيل الدفع
Proposals needing payment / PO / written approval.

العروض التي تحتاج دفع / أمر شراء / موافقة كتابية.

### 8. Delivery Queue / طابور التسليم
Paid or approved client work.

عمل العميل المدفوع أو المعتمد.

### 9. Retention Queue / طابور الاحتفاظ
Feedback, retainer, proof, referral.

تغذية راجعة، اشتراك شهري، إثبات، إحالة.

### 10. Trust Flags / إشارات الثقة
Overclaim, data, suppression, AI risk, approval delay.

مبالغة، بيانات، استبعاد، مخاطر الذكاء، تأخير الموافقة.

### 11. Finance / المالية
Cash, MRR, pipeline, weighted pipeline, expenses, margin.

النقد، الإيراد الشهري، الأنبوب، الأنبوب المرجَّح، المصاريف، الهامش.

### 12. Workers / العاملون
Last run, failure count, queue backlog.

آخر تشغيل، عدد الفشل، تراكم الطابور.

## CEO Actions / إجراءات الرئيس التنفيذي

Approve / Reject / Edit / Escalate / Defer / Push Payment / Start Delivery.

موافقة / رفض / تعديل / تصعيد / تأجيل / دفع الدفع / بدء التسليم.

## Rule / القاعدة

CEO should approve decisions, not manually operate the machine.

الرئيس التنفيذي يجب أن يوافق على القرارات، لا أن يشغل الماكينة يدوياً.

# Conversion Diagnostics

> Where deals die and what to do about it.

## 1. Funnel stages

```
Signal → Score → Engage → Reply → Sample → Proposal → Commit → Pay → Deliver → Retain → Refer
```

## 2. Per-stage diagnostic questions

| Stage | Question |
|---|---|
| Signal | Are we missing a sector? |
| Score | Are scores stable across weeks? |
| Engage | Is the right persona being reached? |
| Reply | Is reply quality matching cadence? |
| Sample | Is the sample being opened and consumed? |
| Proposal | Are we losing on price, scope, or trust? |
| Commit | What objections recur? |
| Pay | What is the days-to-cash distribution? |
| Deliver | Are QA gates passing? |
| Retain | Are health bands trending green? |
| Refer | Are referrals coming from green customers? |

## 3. Diagnostic outputs

`performance/conversion_snapshot.csv` columns:

```
week, stage, volume_in, volume_out, conversion, top_blocker, recommended_action, owner, source
```

## 4. Cadence

- Weekly snapshot.
- Monthly review with the founder.
- Quarterly funnel re-architecture.

## 5. Trust

Diagnostics never auto-act. They recommend. Founder + owner act.

# Learning Loop

> Every week, the operating system gets smaller, sharper, faster.

## 1. Inputs

- Experiment results
- Win / loss notes
- Reply router classifications
- Brand check failures
- Trust gate hits
- Customer NPS

## 2. Outputs

- Updated objection library
- Updated message performance library
- Updated sector performance library
- Updated next-best-action recommendations

## 3. Cadence

| Day | Activity |
|---|---|
| Friday | Loop runs |
| Sunday | Updates feed Sector Ranking and Account Scoring |
| Monday | Updated recommendations surface in Sales Cockpit |

## 4. Files

- `performance/objection_library.csv`
- `performance/message_performance.csv`
- `performance/sector_performance.csv`
- `performance/next_best_action.csv`

## 5. Trust

Learning loop updates never silently change customer-facing language; Brand Guardian re-validates any messaging change before propagation.

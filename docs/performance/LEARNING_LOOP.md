Every closed experiment writes a learning note here. Notes are append-only.

## 1. Schema

```

- **Hypothesis:** …
- **Result:** supported | not supported | partial
- **Numbers:** before / after
- **What surprised us:** …
- **What we change now:** …
- **Doctrine update (if any):** …
- **Linked artefacts:** …
```

## 3. Doctrine

- Every closed experiment writes a learning note, including failed ones.
- Notes are append-only; corrections are made via a new note that references the original.
- The founder reviews accumulated learning monthly.
- A change to doctrine cited in a learning note must propagate to the relevant doc and the verifier.

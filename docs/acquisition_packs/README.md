# Dealix Client Acquisition Packs — حزم اكتساب العملاء

نظام تشغيلي كامل لتحويل Dealix من فكرة ومنصة إلى **ماكينة فرص يومية**: مصدر شركات → تقييم → تحديد الفجوة → مسودة مخصّصة بموافقة → اجتماع → دليل (Proof). كل ذلك ضمن حوكمة Dealix غير القابلة للمساومة.

A complete operating system that turns Dealix from an idea and a platform into a **daily opportunity machine**: source companies → score → identify the gap → tailored draft (with approval) → meeting → proof. All inside Dealix's non-negotiable governance.

---

## القواعد غير القابلة للمساومة — Non-negotiables

هذه الحزم تحترم دستور Dealix. اقرأها قبل أي تشغيل:

- **لا scraping ولا قوائم مشتراة.** مصادر الشركات = سجلات عامة + إعلانات عامة + إشارات سوق من المؤسس فقط.
- **لا إرسال بارد ولا أتمتة واتساب/LinkedIn ولا إرسال جماعي.** كل تواصل = **مسودة** يراجعها إنسان ويرسلها بعد موافقة (NO_LIVE_SEND).
- **لا ضمان مبيعات أو نسب تحويل.** القيمة دائماً «تقديرية» حتى تُثبَت بدليل.
- **لا بيانات شخصية حساسة في التسويق.** جهة اتصال أعمال فقط، مع أساس معالجة (consent_basis) مُسجَّل.
- **بوابة موافقة قبل أي إرسال خارجي.** عبر `approval_center`.

These packs honor the Dealix doctrine — read it before operating: no scraping or purchased lists; no cold/bulk/automated outreach (every message is a human-approved draft); no guaranteed sales; no sensitive personal data in marketing; an approval gate before any external send.

## أول شيء تستخدمه غداً — Start here tomorrow

ابدأ بهذه الثلاثة فقط:

1. افتح: [02_daily_engine/DAILY_COMPANY_NUMBERS_ENGINE.md](02_daily_engine/DAILY_COMPANY_NUMBERS_ENGINE.md)

Start with just these three: open the daily engine, fill the company scoring CSV daily, and hand the marketer the field manual.

**الإيقاع الصحيح:** كل يوم قطاع واحد — ~50 شركة مُحلَّلة، ~15 مؤهلة، ~10 مسودات مخصّصة، تقرير نهاية اليوم.
**The right rhythm:** one sector per day — ~50 companies analyzed, ~15 qualified, ~10 tailored drafts, an end-of-day report.

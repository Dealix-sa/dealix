# Partner Ecosystem OS v3 / نظام نظام الشركاء v3

## Purpose / الغرض

Build a referral and co-sell network around Dealix.

بناء شبكة إحالة وبيع مشترك حول Dealix.

## Partner Categories / فئات الشركاء

- ERP implementers / منفذو ERP
- CRM consultants / مستشارو CRM
- cybersecurity advisors / مستشارو الأمن السيبراني
- agencies / الوكالات
- accountants / المحاسبون
- legal/compliance consultants / مستشارو القانون/الامتثال
- startup ecosystem operators / مشغلو نظام الشركات الناشئة
- chamber/event organizers / الغرف/منظمو الفعاليات
- SaaS founders / مؤسسو SaaS

## Partner Motions / حركات الشركاء

- referral ask / طلب إحالة
- co-sell opportunity / فرصة بيع مشترك
- sector report collaboration / تعاون تقرير قطاعي
- white-label later / علامة بيضاء لاحقًا
- enterprise intro / تعريف مؤسسي

## Weekly Target / الهدف الأسبوعي

- 10 referral asks / 10 طلبات إحالة
- 3 partner conversations / 3 محادثات شركاء
- 1 qualified intro / 1 تعريف مؤهل

## Rules / القواعد

- partner cannot promise outcomes / لا يستطيع الشريك وعد النتائج
- partner cannot change price/scope / لا يستطيع الشريك تغيير السعر/النطاق
- commission only on collected cash if agreed / العمولة فقط على النقد المُحصَّل إذا اتُّفِق
- all partner leads logged / كل عملاء الشركاء مسجلون

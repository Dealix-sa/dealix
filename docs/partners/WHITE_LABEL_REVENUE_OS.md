# White-Label Revenue OS (for Partners)

> The bounded, governance-reviewed white-label option for proven
> partners. Not a default offer.

## What's white-labelable

| Component                 | White-label?           |
| ------------------------- | ---------------------- |
| Diagnostic                | yes, with attribution  |
| Sector reports            | yes, with attribution  |
| AI Sales Assistant SaaS   | governance review only |
| Decision Passport         | no                     |
| Proof Curation            | no                     |
| Executive Command Center  | governance review only |

## Eligibility

A partner can apply for white-label after:

1. ≥ 6 months of `pilot_partner` status.
2. ≥ 3 referred Managed Pilots successfully delivered.
3. Executed data-processing agreement.
4. Governance review row in
   `<PRIVATE_OPS>/partners/white_label_reviews.csv`.

## Pricing posture

- All white-label deals reference `docs/company/PRICING.md`.
- Revenue share % is fixed by the partner program; off-program rates
  require board-level approval.

## Termination

- A white-label arrangement can be terminated if the partner publishes
  any claim that violates the proof-safe asset policy. Termination is
  recorded in the partner pipeline as `terminated`.

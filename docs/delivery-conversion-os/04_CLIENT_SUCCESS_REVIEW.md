# Client Success Review — مراجعة نجاح العميل

## الغرض (AR)
يحدد هذا المستند إيقاع مراجعة نجاح العميل بعد بدء العقد الشهري. الهدف هو ضمان تحقّق القيمة باستمرار، وبناء الثقة، وكشف محفزات التوسّع مبكرًا. كل المتابعات يدوية ومعتمدة من المؤسس.

## Purpose (EN)
This document defines the client success review cadence after a retainer begins. The goal is to ensure value is continuously realized, build trust, and surface expansion triggers early. All follow-ups are manual and founder-approved.

## الإيقاع / Cadence
- مراجعة قصيرة شهرية مع العميل (30–45 دقيقة).
- مراجعة ربع سنوية أعمق للاتجاه والقيمة التراكمية.

A short monthly review (30–45 min) plus a deeper quarterly review of trajectory and cumulative value.

## أجندة المراجعة / Review Agenda
1. **التقدّم مقابل الأهداف** / Progress vs goals: ما تحقق هذا الشهر.
2. **العوائق** / Blockers: ما الذي يعيق القيمة.
3. **التغذية الراجعة** / Feedback: رضا العميل واحتياجاته.
4. **الخطوة التالية** / Next steps: التزامات الطرفين.
5. **إشارات التوسّع** / Expansion signals: احتياجات جديدة محتملة.

## مقاييس النجاح / Success Metrics

| المقياس / Metric | السؤال / Question |
|---|---|
| تحقق القيمة / Value realization | هل يرى العميل قيمة ملموسة؟ |
| الرضا / Satisfaction | ما مستوى رضا العميل؟ |
| الاستمرارية / Retention health | ما احتمال التجديد؟ |
| إشارات التوسّع / Expansion signals | هل ظهرت احتياجات جديدة؟ |

## التوثيق / Documentation
- لخّص كل مراجعة بملاحظات موجزة.
- إذا ظهرت فرصة، سجّل `expansion_identified` يدويًا.
- لا تسجّل نتائج لم تتحقق فعلًا.

Summarize each review briefly. If an opportunity appears, record `expansion_identified` manually. Never record outcomes that did not occur.

## السلامة / Safety
- لا ادعاءات غير مثبتة عن النتائج.
- لا مشاركة بيانات العميل خارجيًا.
- كل تواصل ومتابعة يدوي ومعتمد؛ لا إرسال تلقائي.

No unproven outcome claims. No external sharing of client data. All contact and follow-up are manual and approved; no auto-sending.

> ملفّان قراران: **GO** للتجربة المدفوعة الخاصة، و**NO-GO (بوابة مانعة)** للإطلاق العام.
> يكمّل (لا يستبدل) [`DEALIX_LAUNCH_CLOSURE_VERDICT.md`](DEALIX_LAUNCH_CLOSURE_VERDICT.md).

## 1. GO — تجربة مدفوعة خاصة (Private Paid Pilot)

كل البنود يجب أن تكون **PASS** قبل بدء أول pilot مدفوع:

| # | البند | الحالة المطلوبة | EN |
|---|-------|-----------------|----|
| 1 | env-check | PASS | environment check passes |
| 2 | security smoke | PASS | security smoke passes |
| 3 | api import | OK | api import OK |
| 4 | توحيد الأسعار (هذه المجموعة) | موحّد | pricing unified via this doc set |
| 5 | أول 30 مرشَّحاً مدرَجين | جاهز | first 30 prospects listed |
| 6 | سكربت التشخيص | جاهز | diagnostic script ready |
| 7 | اتفاقية pilot | جاهزة | pilot agreement ready |
| 8 | مسار دفع يدوي | جاهز | manual payment path ready |
| 9 | قالب proof pack | جاهز | proof pack template ready |
| 10 | الخصوصية/الشروط (مراجعة داخلية) | تمّت | privacy/terms reviewed internally |

**النتيجة:** إذا كل البنود PASS → **GO للتجربة المدفوعة الخاصة فقط.**

## 2. NO-GO — الإطلاق العام (Public Launch) — بوابة مانعة

أي بند مُفعَّل أدناه = **NO-GO، يمنع الإطلاق العام**:

| # | بوابة مانعة | EN gate |
|---|-------------|---------|
| 1 | 0 عملاء مدفوعين | 0 paid customers |
| 2 | لا يوجد proof pack | no proof pack |
| 3 | security smoke فاشل | security smoke failing |
| 4 | يوجد تعارض أسعار | pricing conflict exists |
| 5 | الخصوصية/الشروط غير مُراجَعة | privacy/terms not reviewed |
| 6 | أتمتة واتساب غير مضبوطة | WhatsApp automation not controlled |
| 7 | الادعاءات تتضمن لغة ضمان | claims include guarantee language |
| 8 | قدرة المؤسس غير واضحة | founder capacity unclear |

---

## 3. البوابة الصريحة — Explicit Blocking Gate

> **الإطلاق العام محظور (BLOCKED) حتى وجود ≥3 تجارب مدفوعة AND ≥3 proof packs AND ≥1 case study مُجهَّلة.**
>
> **Public launch is BLOCKED until ≥3 paid pilots AND ≥3 proof packs AND ≥1 anonymized case study exist.**

لا استثناء. لا يُلغى إلا بتحقّق الشروط الثلاثة معاً، موثّقة في [`FIRST_3_PAID_PILOTS_BOARD.md`](FIRST_3_PAID_PILOTS_BOARD.md).

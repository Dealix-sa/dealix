# Content Operating Routine
## Daily
- Capture one market observation.
- Capture one customer/lead learning if available.
- Do not publish without evidence.
## Weekly
- Review weekly intelligence review.
- Choose 2 founder posts.
- Choose 1 operating lesson.
- Review proof library.
- Run claim review before publishing.
## Monthly
- Review brand positioning.
- Review content-to-pipeline influence.
- Update proof library.
- Create one sector insight if evidence exists.
## Before Publishing
Ask:
1. Is this based on evidence?
2. Is any client/lead identifiable?
3. Is approval needed?
4. Is there any banned claim?
5. Is the language realistic?
6. Does this build trust?
## Rule
If content creates more risk than trust, do not publish.

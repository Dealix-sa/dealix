# Proof Library

The single, curated library of every approved proof point Dealix can reference.

## Sections
- Case studies (linked).
- Sector reports (linked).
- Client quotes (with permission).
- Numbers we can cite (with method).
- Independent mentions (press, partners).

## Entry template

```
## Title
Short.

## Type
Case study / Quote / Number / Mention.

## Source
Link or reference.

## Approval
Approval level, date, approver.

## Use cases
Where this proof can be cited (proposals, posts, decks).
```

## Maintenance
- Reviewed monthly. Outdated proofs are archived.
- Every new proof requires Claim Guard pass before entry.

## Rule
A claim that is not in the Proof Library is not yet a public claim.

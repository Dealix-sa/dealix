## The format rules

- One page, two sides max.
- A4 / Letter, no fancy graphics.
- One CTA per version.
- QR code (optional) for WhatsApp consent flow.
- "founder-approved" stamp on every CTA.

## The trust preflight

Every version goes through the preflight. Banned:

- "Guaranteed", "100%", "best", "transformative".
- Cold WhatsApp mention.
- Pricing number.
- Fake proof.

## When to update

- After 10 deals: which version led to the close?
- After 20 deals: which version got the most engagement?
- After 50 deals: which version is rarely used? Retire it.

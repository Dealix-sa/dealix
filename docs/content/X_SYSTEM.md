# X / Twitter System

A lighter, faster channel for short signals and founder presence.

## Post types
- **Signal** — one observation, one data point.
- **Thread** — short founder lesson, 4–8 posts.
- **Quote** — sharp line from a Dealix doc or report.
- **Reply** — adding one specific thing to a relevant ongoing thread.

## Cadence
- Daily lightweight presence.
- One thread per week.
- One reply to a target-sector account per day.

## Style
- Tighter than LinkedIn.
- Bilingual where natural.
- No hashtags farms.

## Rule
X is a conversation channel, not a broadcast channel. The reply matters as much as the post.

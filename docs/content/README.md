## غير قابل للتفاوض — Non-negotiables in content

- النشر يدوي دائماً. Dealix يصيغ ولا ينشر ولا يرسل بلا موافقة المؤسس.
- لا scraping، لا إرسال بارد، لا أتمتة لينكدإن — لا كخدمة ولا كوصف.
- لا أرقام بيع أو نسب تحويل أو عائد كحقيقة — كله تقديري أو نمط آمن. النتائج «فرص مُثبتة بأدلة».
- لا PII ولا اسم عميل بلا إذن نشر موقّع. الحالات الافتراضية تُوسم «Hypothetical / case-safe template».
- لا ادعاء بلا مصدر؛ كل رقم يحمل `source_ref` ومستوى إثبات L0–L5.
- الصوت تنفيذي سعودي. ممنوع: «نحوّل»، «نضاعف عشر مرات»، «ثوري»، «نضمن». بلا رموز تعبيرية، بلا اسم نموذج.

# Content to Pipeline System
## Purpose
Track whether Dealix content creates conversations, leads, trust, or pipeline.
## Influence Types
- inbound DM
- reply
- referral
- meeting
- proposal influence
- trust proof
- partner interest
## Rules
- Do not over-attribute.
- Log only clear influence.
- Content is useful if it creates trust, learning, or pipeline.
- If content gets attention but no business value, adjust.

# Dealix Company OS Index

The Dealix Company Operating System is a tiered set of documents. Each tier has a named command center. Each command center has one named owner. This index is the entry point.

## Tier 0 — Trust

- [docs/trust/](../trust/) — Approval matrix, autonomy policy, no-overclaim policy, safe language library, audit policy, human oversight, incident response, public/private boundary, data trust architecture, control plane. Command center: [docs/trust/TRUST_COMMAND_CENTER.md](../trust/TRUST_COMMAND_CENTER.md).

## Tier 1 — Revenue

- [docs/revenue/](../revenue/) — Offer ladder, pipeline stages, pricing, invoicing, payments, cash rules, revenue readiness. Command center: [docs/revenue/REVENUE_COMMAND_CENTER.md](../revenue/REVENUE_COMMAND_CENTER.md).

# فهرس كل الأنظمة | All Systems Index

## الغرض | Purpose

**عربي:** هذا الفهرس يجمع كل طبقات أنظمة التشغيل في Dealix في جدول واحد: اسم الطبقة، مسار توثيقها، وغرضها. يستخدمه المؤسس كخريطة مرجعية واحدة للوصول السريع لأي نظام. كل الأنظمة تتبع نفس مبدأ السلامة: الذكاء الاصطناعي يُجهّز، المؤسس يعتمد، التنفيذ يدوي فقط، لا إرسال خارجي تلقائي.

**English:** This index gathers all of Dealix's operating-system layers in one table: layer name, docs path, and purpose. The founder uses it as a single reference map for quick access to any system. All systems follow the same safety principle: AI prepares, Founder approves, manual action only, no automated external sending.

---

## المبدأ الحاكم | Governing Principle

> الذكاء الاصطناعي يُجهّز، المؤسس يوافق، الإجراء يدوي فقط، لا إرسال خارجي تلقائي.
> AI prepares, Founder approves, Manual action only, No automated external sending.

## كيف تُستخدم الخريطة | How to Use This Map

1. ابدأ من **مركز القيادة Master Command Center** للتشغيل اليومي.
2. انتقل للطبقة المتخصصة عند الحاجة (إيراد، تسليم، إثبات...).
3. عند أي خطر، اذهب مباشرة إلى **Crisis OS**.
4. سجّل كل قرار مهم في **Operating Memory OS**.

## ملاحظات الحالة | Status Notes

- بعض الطبقات تحتوي ملف نظرة عامة `00_*` وملفات تفصيلية مرقّمة.
- الطبقات حديثة الإنشاء قد تحتوي ملف نظرة عامة فقط حاليًا.

---

## حدود السلامة المشتركة | Shared Safety Boundaries

- لا إرسال آلي، لا كشط، لا إطلاق إعلانات حية، لا إرسال/نشر تلقائي.
- لا أرقام وهمية، لا ضمان عائد، لا ادعاءات غير مثبتة.
- لا أسرار أو مفاتيح API في أي مخرَج أو توثيق.

> هذا الفهرس هو نقطة المرجع الواحدة لكل أنظمة Dealix.

# Strategic Opportunity OS

## Purpose
This pack helps the founder turn Dealix into a compounding strategic asset, not only a working product.

It combines opportunity discovery, product bets, GTM expansion, AI/data leverage, observability, data quality, security posture, partnerships, and founder decision cadence.

## What this pack adds

1. Strategic opportunity map
2. Opportunity scoring model
3. Tooling adoption matrix
4. AI and data capability roadmap
5. Product bet portfolio
6. GTM expansion playbook
7. Founder 100-day execution plan
8. Verification script for repository readiness
9. Optional Python dependency extras for reviewed tooling

## Important rule
New tools are not automatically production dependencies. They are optional extras and must pass owner, data, cost, and security review before use.

## Founder operating question

Every opportunity must answer:

> Does this improve speed, trust, revenue, retention, data advantage, or strategic defensibility?

If the answer is unclear, do not install, build, hire, or launch it yet.

# Opportunity Scoring Model

## Purpose
Choose the best founder opportunities before spending time, engineering, or budget.

## Scoring dimensions

Score each item from 0 to 3.

| Dimension | Score |
|---|---:|
| ICP fit | 0-3 |
| Revenue impact | 0-3 |
| Customer trust impact | 0-3 |
| Product reuse | 0-3 |
| Data advantage | 0-3 |
| Speed to test | 0-3 |
| Cost control | 0-3 |
| Risk control | 0-3 |
| Founder leverage | 0-3 |
| Strategic defensibility | 0-3 |

## Decision bands

| Total | Decision |
|---:|---|
| 24-30 | Test now |
| 18-23 | Good, schedule soon |
| 12-17 | Needs more evidence |
| 0-11 | Do not start |

## Stop conditions

Stop an opportunity when:

- No metric moves after the test window.
- Customer evidence is weak.
- Risk grows faster than learning.
- It distracts from the current wedge ICP.
- It requires custom work that does not become a reusable platform capability.

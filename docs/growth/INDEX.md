# Growth System — Surface Index

Read-only index over the canonical growth modules. This file exists so
`scripts/verify_growth_system.py` can confirm the surface is wired.

## Canonical modules (source of truth)

- `autonomous_growth/` — sector intel, content, distribution, enrichment,
  competitor analysis, market research.
- `auto_client_acquisition/self_growth_os/` — Dealix's self-growth loops.
- `auto_client_acquisition/growth_beast/` — growth experimentation engine.

## Governance

All growth modules honor the 11 non-negotiables:

- **NO_LIVE_SEND** — outreach drafts queue for approval; never auto-send.
- **NO_COLD_WHATSAPP / NO_LINKEDIN_AUTOMATION / NO_SCRAPING** — banned.
- **NO_FAKE_PROOF** — every growth claim cites source or carries
  `is_estimate=true`.

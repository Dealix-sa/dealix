# DEALIX Brand System

**Wordmark:** DEALIX
**Tagline:** INTELLIGENT DEALS. REAL GROWTH.
**Owner:** Founder + Brand Lead
**Source of truth:** `docs/brand/brand-tokens.json` and `docs/brand/`

## What this system is

The Dealix Brand System defines how Dealix shows up across every surface — product UI, proposals, decks, social, reports, partner collateral. It is not decoration. It is a contract. Every visual, every line of copy, every interaction is built to signal one position: Dealix is a Saudi B2B Revenue Operating System for intelligent deal flow, founder-approved growth, and trust-gated AI execution.

## Five brand pillars

| Pillar | What it means in copy and design |
|---|---|
| Built on Trust | Plain language, evidence-first, explicit limits, no hype |
| Driven by Growth | Concrete numbers, dated proof, clear next step |
| Closing Deals | Decisive verbs, founder-readable, no jargon walls |
| Focused on Results | Outputs over outputs-of-outputs; deliverables named |
| Global Mindset, Local Impact | Bilingual when needed, Saudi context, global craft level |

## What the brand is NOT

- Not a scraping tool. Never depict bulk data harvesting visuals.
- Not a spam machine. Never imply auto-sent outbound at scale.
- Not "just a CRM" or "just an agency".
- Not AI-as-magic. Always show the human approval gate.

## Brand governance

Any new external-facing asset must:

1. Use tokens from `docs/brand/brand-tokens.json`. No off-palette colors.
2. Pass the voice checklist in `DEALIX_BRAND_VOICE.md`.
3. Include the standard disclaimer: "Estimated value is not Verified value / القيمة التقديرية ليست قيمة مُتحقَّقة" on any document that cites projected impact.
4. Be reviewed by Brand Lead before publication if it carries the wordmark.

## Trust gate

| Class | Example surface | Approver |
|---|---|---|
| A0 | Internal Slack, working drafts | Self |
| A1 | Proposal templates, deck templates | Brand Lead |
| A2 | Customer-facing proposals, sector reports | Founder + Brand Lead |
| A3 | Press, partner co-branding, paid placement | Founder only |

## Failure mode

- Off-palette gradient in a customer deck.
- "AI-powered" or "guaranteed" appearing in copy.
- Wordmark placed on a busy photo without the contrast plate.
- Arabic body text rendered in a Latin-only fallback font.

## Recovery path

1. Pull the asset from circulation.
2. Re-export using tokens in `docs/brand/brand-tokens.json`.
3. Re-run the voice checklist in `DEALIX_BRAND_VOICE.md`.
4. Re-issue with a new version stamp and notify the original recipient.

## Disclaimer

Dealix does not guarantee revenue, meetings, or conversion outcomes. All forecasts in branded materials are estimated, founder-approved, and bound by the trust gates above.

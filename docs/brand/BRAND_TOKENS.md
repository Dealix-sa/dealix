# Dealix Brand Tokens

Source of truth for design tokens used across artefacts.
The verifier `scripts/verify_launch_readiness.py` checks this file's presence.

## Typography

- Primary (AR): IBM Plex Sans Arabic.
- Primary (EN): Inter.
- Fallback: system-ui, -apple-system, sans-serif.

## Voice

- Direct. No marketing fluff.
- Bilingual when customer-facing — Arabic first, English second.
- No guaranteed-outcome language. No "ضمان دخل" / "guaranteed ROI".
- Evidence > opinion.

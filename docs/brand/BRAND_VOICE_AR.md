## 1. الصوت الأساسي — Core Voice

نبرة Dealix هي نبرة **تنفيذي سعودي يخاطب صاحب قرار**: موجزة، مبنية على الدليل، عربية أولاً، إنجليزية جاهزة. نكتب كما نتحدّث في غرفة قرار: جملة واحدة لكل فكرة، رقم له مصدر، ووعد بحجم ما نستطيع إثباته فقط.

Executive, Saudi-first, concise, evidence-forward, Arabic-first / English-ready. We write like we speak in a decision room.

**أربعة مبادئ:**
- **الوضوح قبل الذكاء** — لو احتاجت الجملة شرحاً، أعد كتابتها.
- **الدليل قبل الصفة** — «حسّنّا الترتيب» تُستبدل بـ «أعدنا ترتيب 1,240 فرصة ونزعنا 18% مكرّرة».
- **التواضع الواثق** — نقول ما نجهله بصدق؛ الصدق نفسه إثبات.
- **الموافقة في اللغة** — نكتب «بانتظار موافقتك» لا «أرسلنا نيابةً عنك».

## 2. الكلمات الممنوعة — Banned Words

ممنوعة في كل أصل، عربياً وإنجليزياً:

| ممنوع | لماذا |
|-------|-------|
| transform / نُحوّل جذرياً | وعد مبالغ بلا دليل |
| supercharge | hype تسويقي فارغ |
| 10x | رقم أداء بلا مصدر |
| revolutionary / ثوري | ادعاء فئة بلا إثبات |
| disrupting / نُربك السوق | لغة استعراضية |
| «نضمن» / guaranteed | وعد نتيجة محظور |
| AI-powered (كشعار) | صفة عامة بلا معنى تشغيلي |

ممنوع أيضاً: أي معدّل تحويل أو ROI أو رقم مبيعات يُذكر كحقيقة، وأي إيموجي، وأي اسم نموذج.

## 3. الكلمات المفضّلة — Preferred Words

| مفضّل | الاستخدام |
|-------|-----------|
| **ملتزم / committed** | نُظهر التزاماً، لا ضماناً |
| **موثّق / evidenced** | كل ادعاء معه دليل |
| **founding partner** | للعميل الأول، علاقة لا صفقة |
| **approval-first** | الموافقة قبل الإرسال |
| **فرص مُثبتة بأدلة / evidenced opportunities** | بديل «مبيعات مضمونة» |
| **proof / دليل** | الوحدة الأساسية للثقة |
| **محكوم / governed** | كل إجراء داخل بوابة |

## 4. قبل / بعد — Before / After Rewrites

**(أ) وعد النتيجة**
- قبل: «Dealix تضمن لك مبيعات أكثر بـ 10x.»
- بعد: «Dealix تُخرج لك أعلى 50 فرصة مُسجَّلة بأدلة، وخطة 10 إجراءات بانتظار موافقتك.»

**(ب) الصفة الفارغة**
- قبل: "Our revolutionary AI transforms your sales."
- بعد: "Our governed agents draft, score, and prove opportunities; you approve before anything sends."

**(ج) الإرسال نيابةً**
- قبل: «نرسل رسائلك للعملاء تلقائياً.»
- بعد: «نجهّز مسودات شخصية؛ لا تُرسَل أي رسالة خارجية بلا موافقتك الصريحة.»

**(د) المرحلة الحرجة**
- قبل: «مئات العملاء يثقون بنا.» (غير صحيح)
- بعد: «نحن في مرحلة الشريك المؤسِّس؛ صفر عملاء مدفوعين حتى الآن — وهذا نقولُه بصدق.»

**(هـ) الـ outreach**
- قبل: «نجمع آلاف جهات الاتصال ونراسلهم.»
- بعد: «نبحث الشركات المناسبة بمصادر مشروعة، ونجهّز مسودة لكل واحدة بانتظار موافقتك.»

## 5. السجلّ حسب القناة — Register by Channel

| القناة | السجلّ | ملاحظة |
|--------|--------|--------|
| **الموقع / One-pager** | تنفيذي موجز، عربي أولاً | الموقف في سطر + ثلاث ركائز |
| **الديك / Deck** | رسالة واحدة لكل شريحة | إثبات لا hype ([دليل العرض](./DEALIX_PRESENTATION_STYLE_GUIDE_AR.md)) |
| **البريد البارد / Cold email** | مهذّب، شخصي، CTA واحد | لا إلحاح مزيف، إفصاح حاضر |
| **LinkedIn** | قيادة فكرية، صوت المؤسس | نشر يدوي فقط ([نظام السوشال](./BRAND_SOCIAL_SYSTEM_AR.md)) |
| **واتساب / WhatsApp** | بعد الردّ فقط، ودّي مباشر | لا بدء بارد إطلاقاً |
| **التقارير / Proof reports** | جافّ، رقمي، مصدري | كل رقم بمصدره |

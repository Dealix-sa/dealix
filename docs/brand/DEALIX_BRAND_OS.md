# Dealix Brand OS

The Brand Operating System defines how Dealix shows up to the market.
Every customer-facing artifact (deck, page, email draft, proposal,
case-safe summary) is built against the rules below.

## Positioning

> **Governed AI Operations for Saudi B2B.**
> Operating capability + auditable proof. Not AI tools. Not spam.

We sell **outcomes** wrapped in **governance**, NOT generic AI features.
Every promise we make is paired with a Proof Pack and a Capital Asset.

Differentiators (must appear in every long-form asset):
1. Governance-by-default (approval_center, audit trail, eval gates).
2. Source passport on every claim (no scraping, founder-submitted sources).
3. Proof Pack score >= 70 on every paid engagement.
4. Capital Asset registry — every project leaves a reusable artifact.
5. Bilingual operating model (English + Arabic).

## Voice

| Trait | Rule |
|---|---|
| Tone | Direct, technical, calm. No hype. |
| Tense | Present + active. ("Dealix ships ... " not "Dealix will ship ... ") |
| Numbers | Always cite source. Use "estimated" when not yet verified. |
| Arabic | Modern Standard Arabic for business. RTL paragraphs flow `dir="rtl"`. |
| Forbidden | "guaranteed", "best-in-class", "10x faster" without source. |

## Visual

| Element | Spec |
|---|---|
| Primary palette | Deep navy `#0E1F3A`, electric teal `#1FB6A8`, sand `#E7D8A1`. |
| Type | Inter (Latin), Cairo / IBM Plex Sans Arabic (Arabic). |
| Logo lockup | Mark + wordmark; clear-space = 1x cap-height. |
| Icons | Lucide / Phosphor outline, 1.5 stroke. |
| Photography | Documentary, no stock-photo "AI brain" cliches. |

## Bilingual disclaimer (mandatory footer)

Every customer-facing markdown file MUST end with:

> *Estimated value is not Verified value / القيمة التقديرية ليست قيمة مُتحقَّقة*

## Non-negotiable cross-references

This Brand OS enforces the following non-negotiable items:
- NN4 (no unsourced claims) — copy review checklist below.
- NN5 (no guaranteed outcomes) — banned phrase list above.
- NN10 (proof pack) — every case study must link a Proof Pack ID.

## Brand review checklist

Run before publishing any external asset:

- [ ] Positioning sentence intact.
- [ ] No banned phrases.
- [ ] Every metric has a `source_ref`.
- [ ] Arabic version exists (or AR disclaimer note).
- [ ] Bilingual disclaimer present at footer.

## نيّة الخطوط (Typography Intent)

- **العربية أولاً، الإنجليزية جاهزة.** خط عربي تنفيذي واضح للعناوين، ووزن أخف للنص.
- **تسلسل هرمي صارم:** عنوان واحد كبير لكل شريحة، رقم أو ادّعاء واحد بارز.
- **أرقام كبيرة للأدلة:** الرقم المُثبت يُعرض كبيراً، مع مستوى الدليل بجانبه (L0–L5).
- لا فقرات كثيفة على الشرائح؛ المحتوى الكثيف يبقى في الوثائق.

---

## اللغة البصرية (Visual Language)

نستخدم استعارات نظام التشغيل، لا استعارات الروبوت:

- **بطاقات (Cards):** كل فرصة/قرار/دليل بطاقة مستقلة.
- **خطوط أنابيب (Pipelines):** تدفّق الفرص من إشارة إلى متابعة إلى إغلاق.
- **حالات الموافقة (Approval States):** Draft → Approved → Sent، بلون لكل حالة.
- **مستويات الإثبات (Proof Levels):** L0–L5 كشارات على البطاقات.
- **غرفة قيادة المؤسس (Founder Command Center):** لوحة تجمع الإيراد والمخاطر والقرار الأسبوعي.

## افعل / لا تفعل (Do / Don't)

| افعل | لا تفعل |
|---|---|
| خلفيات داكنة تنفيذية بـ Midnight Navy | خلفيات بيضاء عامة بلا هوية |
| تمييز بـ Cyan وإثبات بـ Emerald | استخدام أحمر/أخضر خارج القائمة |
| Amber للتنبيه فقط | Amber كلون رئيسي للعلامة |
| بطاقات وpipelines وحالات موافقة | صور روبوت/دماغ AI عامة |
| رقم واحد بارز بمستوى دليله | لوحات مزدحمة بلا تسلسل |
| ادّعاء كبير مع Proof Pack | ادّعاء كبير بلا دليل |

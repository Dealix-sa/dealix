# Revenue Operating Model — نموذج تشغيل الإيرادات

Customers enter low and expand as value is proven. No tier promises guaranteed results; each delivers a defined artifact.

| Tier | Scope | Price (SAR) |
|---|---|---|
| AI Workflow Audit | Map one department's revenue workflow, identify gaps | 499 – 2,500 |
| Paid Pilot | Build and run one workflow, measure a case-safe pattern | 5,000 – 25,000 |
| Department OS | Full operating system for one department | 25,000 – 150,000 |
| Monthly Retainer | Ongoing operation, drafts, and reviews | 3,000 – 25,000 / month |
| Enterprise Custom OS | Multi-department, tailored build | 150,000+ |

## English — The Customer Journey

**Diagnostic → Pilot → Retainer.**

1. **Entry.** A customer requests an AI Workflow Audit, usually for one department.
2. **Diagnostic.** We map the existing workflow and identify evidenced opportunities.
3. **Pilot.** A paid pilot builds one workflow and measures a case-safe pattern over a fixed window.
4. **Retainer / Department OS.** On proven value, the customer converts to ongoing operation.
5. **Expansion.** Additional departments or an Enterprise Custom OS.

At every step, AI drafts and ranks; the customer's owner approves and acts. The system never sends externally.

## العربية — رحلة العميل

**تشخيص ← تجربة ← اشتراك.**

1. **الدخول.** يطلب العميل تدقيق سير العمل، غالبًا لقسم واحد.
2. **التشخيص.** نرسم سير العمل الحالي ونحدد الفرص المُثبتة بأدلة.
3. **التجربة.** تبني التجربة المدفوعة سير عمل واحدًا وتقيس نمطًا آمنًا للحالة خلال نافذة محددة.
4. **الاشتراك / نظام القسم.** عند إثبات القيمة، يتحول العميل إلى تشغيل مستمر.
5. **التوسّع.** أقسام إضافية أو نظام مؤسسة مخصص.

في كل خطوة، يصيغ الذكاء الاصطناعي ويرتّب؛ ويعتمد مسؤول العميل ويتصرف. النظام لا يرسل خارجيًا أبدًا.

## English — Revenue Guardrails

- We quote estimated value, never guaranteed ROI.
- We use case-safe patterns, not named customer claims.
- We never describe outreach automation as a service.

## العربية — ضوابط الإيرادات

- نقدّر القيمة ولا نَعِد بعائد مضمون.
- نستخدم أنماطًا آمنة للحالة لا ادعاءات بأسماء عملاء.
- لا نصف أتمتة التواصل كخدمة.

# Specialized Sprint Library — 50 Sprints

## الاستخدام — Usage

كل Sprint هنا قابل للبناء كوحدة مستقلة. اختر Sprint واحداً بناءً على الحاجة التشغيلية المحددة في الفحص المجاني.

## Sprint 1: Follow-up Recovery Sprint

**الحاجة التي يحلها:** غياب نظام متابعة المهتمين

**الإجراءات الرئيسية:**
1. استيراد قائمة المهتمين الحاليين وتصنيفها (Hot/Warm/Cold)
2. بناء لوحة متابعة بـ 3 مراحل
3. إنشاء نموذج رسالة متابعة لكل مرحلة (مسودة)
4. توثيق Workflow المتابعة الأسبوعية
5. تسليم Proof Pack مع قبل/بعد

**المخرجات:** لوحة متابعة + نماذج رسائل مسودة + Workflow موثق

**السعر:** 499 ريال

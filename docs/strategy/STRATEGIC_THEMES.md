# Strategic Themes

## Purpose
Keep Dealix focused on the few strategic themes that matter now.

## Current Strategic Themes

### Theme 1 — First Revenue Proof
Goal:
Close first paid Revenue Sprint or PO / written approval.
Metrics:
- 25 DMs
- 3 samples
- 1 proposal
- 1 payment follow-up

### Theme 2 — Delivery Proof
Goal:
Deliver the first sprint with QA and feedback.
Metrics:
- intake complete
- report complete
- QA pass
- feedback request

### Theme 3 — Trust-First AI
Goal:
Use AI only where it prepares, drafts, scores, or analyzes safely.
Metrics:
- no A3 automation
- approvals logged
- no banned claims

### Theme 4 — Founder Leverage
Goal:
Reduce repeated manual work through templates, checklists, and safe automation.
Metrics:
- one system update/week
- repeated work documented
- productization candidates logged

## Not Strategic Now
- full SaaS platform
- investor deck
- enterprise dashboard
- autonomous outbound agent
- hiring before repeatability

# Dealix Moat System

The moat is not the code. The moat is the operating system around
Saudi B2B revenue.

## Moats

1. Saudi-native B2B revenue positioning.
2. Trust-first and approval-first operations.
3. Sector-specific sales playbooks.
4. Delivery templates refined by real clients.
5. Revenue and win/loss learning loops.
6. Private company memory.
7. Proof library and case studies.
8. Partner distribution network.

## Weak Moats
- generic AI agents
- generic CRM features
- dashboards without workflow
- unpaid users
- unverified claims

## Rule
Every month, strengthen at least one real moat.

## Moat Health Check

Each monthly **Board Pack** must report, for one moat:

- which artefact got stronger this month?
- which artefact decayed (stale doc, lost case study, broken playbook)?
- which moat is now the weakest, and what action follows?

# Market Experiment System
## Purpose
Test market, ICP, offer, and message assumptions with evidence.
## Experiment Types
- sector
- buyer title
- message angle
- offer scope
- price
- sample type
- follow-up cadence
## Rules
- One experiment has one hypothesis.
- Every experiment has a metric.
- Every experiment has a decision.
- Failed experiments are useful if they change behavior.
## Decisions
Build / Fix / Kill / Defer / Continue
## Evidence
- experiments/market_experiments.csv
- pipeline tracker
- revenue action log
- weekly review

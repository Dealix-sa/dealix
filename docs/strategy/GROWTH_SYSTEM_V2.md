# Growth System v2
## Purpose
Create consistent growth from repeatable revenue operations.
## Growth Loop
Target ICP
→ Find leads
→ Send founder DM
→ Learn from replies
→ Send samples
→ Convert proposals
→ Collect payment
→ Deliver
→ Capture proof
→ Ask retainer
→ Improve ICP/message/offer
## Growth Levers
1. ICP quality.
2. Lead source quality.
3. Message clarity.
4. Sample value.
5. Proposal trust.
6. Payment path.
7. Delivery quality.
8. Retention offer.
9. Proof.
10. Founder cadence.
## Weekly Growth Review
- Which sector worked?
- Which source worked?
- Which message worked?
- Which objection repeated?
- Which offer converted?
- What should be killed?
## Growth Rule
Do not scale weak signal. Improve or kill it.

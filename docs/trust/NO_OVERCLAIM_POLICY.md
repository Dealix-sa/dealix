# No-Overclaim Policy — سياسة عدم المبالغة

## Purpose

Protect Dealix and its clients from claims that cannot survive a regulator, an auditor, or an honest client. Every word that leaves Dealix to the outside world passes this filter.

## Banned claims

| Banned | Why |
|---|---|
| "Guaranteed revenue" / "نضمن الإيرادات" | We cannot guarantee a buyer's market. |
| "100% compliant" / "متوافق 100%" | Compliance is a process, not a state. |
| "Fully automated" / "آلي بالكامل" | Humans approve A1+ moves. |
| "AI-powered sales" | Vague, vendor-flavored. |
| "Saudi Arabia's #1" / "الأول في السعودية" | Unverifiable superlative. |
| "Instant ROI" / "عائد فوري" | ROI is observed over time. |
| "Replaces your sales team" | We augment, not replace. |
| "Unhackable", "bulletproof", "zero risk" | No system is. |
| "Conversion guaranteed" | Statistically unsound. |
| Named partnerships (SDAIA, Monsha'at, NEOM) without a signed letter | Implies sponsorship. |

## Approved alternatives

| Instead of | Say |
|---|---|
| Guaranteed sales | Evidenced opportunities |
| Fully automated | Agent-prepared, human-approved |
| 100% compliant | PDPL-aware practice |
| Instant ROI | Estimated value, reviewed at handover |
| Saudi #1 | Saudi-native, KSA-first |
| AI-powered | Governed AI operations |
| We will get you X clients | We will surface X qualified opportunities for your review |

See [SAFE_LANGUAGE_LIBRARY.md](./SAFE_LANGUAGE_LIBRARY.md) for the full phrase bank.

## Operational steps

1. Every public-facing draft runs through the claim filter (C-07).
2. Banned phrases block publish; drafter rewrites using the alternatives.
3. Any new banned phrase added here triggers a sweep of existing public docs.
4. Compliance and revenue claims require A3 approval regardless of phrasing.

## Evidence

- `claim_filter.log` records every block and the rewrite that passed.
- Quarterly review compares blocked phrases against current marketing trends to add new entries.

## AR — ملخّص

سياسة عدم المبالغة تحظر العبارات التي لا يمكن إثباتها أمام مراجع أو منظّم: ضمان الإيرادات، التوافق 100%، الأتمتة الكاملة، الأولوية الوطنية. تستبدلها عبارات قابلة للإثبات: فرص مُثبتة بأدلة، ممارسة مدركة لنظام حماية البيانات، إعداد بالوكيل وموافقة بشرية. القيمة التقديرية ليست قيمة مُتحقَّقة.

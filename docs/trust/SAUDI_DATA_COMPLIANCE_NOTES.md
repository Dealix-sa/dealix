# Saudi Data Compliance Notes
## Purpose
Create practical privacy and data protection discipline for Dealix operations.
## Rules
- Minimize personal data collected.
- Store real leads and clients only in private ops.
- Do not publish client names without approval.
- Do not export sensitive data without review.
- Redact private data before public use.
- Record approvals for public proof.
## Data Categories
- business public info
- contact info
- client confidential info
- payment data
- delivery reports
- AI-generated summaries
## Controls
- public/private boundary
- approval log
- data quality audit
- data boundary verifier
- no real dashboard JSON in public repo

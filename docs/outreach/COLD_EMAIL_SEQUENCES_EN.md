## Template rules

- No misleading subjects. No fake `Re:`/`Fwd:`. The subject describes the content honestly.
- Exactly one specific personalization signal per message in `[specific signal]` (see docs/outreach/PERSONALIZATION_RULES_AR.md).
- One low-commitment CTA per message.
- `[unsubscribe link]` is mandatory in the footer of every message before any send.
- Offer from the approved ladder, with no promised outcome ("evidenced opportunities", not "guaranteed").
- Square brackets are fields filled with real content before approval.

## 1) First-touch

**Subject:** [Sector] — a quick idea for [Company]

```
Hello [Name/Role],

I noticed [specific signal from a public source]. That often comes with [pain hypothesis we address].

At Dealix we help [sector] organize existing revenue opportunities before chasing more.

If useful, I start with a short free diagnostic [one step] — want me to send details?

Best,
[Name] · Dealix

—
To stop these emails: [unsubscribe link]
```

## 2) Follow-up 1

**Subject:** Following up — [Company]

```
Hello [Name],

Following my earlier note with a clearer angle: [a new value point tied to [specific signal]].

If the timing is off, let me know and I'll reach out again later with your permission.

Would a short step [one CTA] work this week?

[Name] · Dealix

—
To stop these emails: [unsubscribe link]
```

## 3) Follow-up 2

**Subject:** An example that may help [Company]

```
Hello [Name],

Sharing a case-safe pattern (hypothetical template, no customer names): [brief description of a common opportunity leak in [sector] and how we address it].

Results are estimated and shown as evidenced opportunities — no promises.

If you'd like, [one CTA: free diagnostic / 15-minute call].

[Name] · Dealix

—
To stop these emails: [unsubscribe link]
```

## 4) Proposal-intro

**Subject:** A suggested step for [Company]

```
Hello [Name],

Based on your interest, I suggest an offer card sized for you:
[offer from ladder: Sprint 499 / Data-to-Revenue Pack 1,500 / Managed Revenue Ops] — scoped, with no revenue commitment.

I'll attach a one-page summary — send it now?

[Name] · Dealix

—
To stop these emails: [unsubscribe link]
```

## 5) Breakup

**Subject:** Closing the loop for now — [Company]

```
Hello [Name],

I respect that you're busy, so I'll close this loop for now and stop emailing.

If priorities change later, the door is open — a one-word reply is enough.

Thanks for your time.

[Name] · Dealix

—
To stop these emails: [unsubscribe link]
```

## قواعد القوالب

- لا مواضيع مضلِّلة. لا `Re:`/`Fwd:` زائفة. الموضوع يصف المحتوى بصدق.
- إشارة تخصيص محدّدة واحدة لكل رسالة في `[إشارة محدّدة]` (انظر docs/outreach/PERSONALIZATION_RULES_AR.md).
- دعوة واحدة (CTA) منخفضة الالتزام.
- `[رابط الانسحاب]` إلزامي في تذييل كل رسالة قبل أي إرسال.
- العرض من السلّم المعتمد، بلا وعد بنتيجة («فرص مُثبتة بأدلة» لا «نضمن»).
- الأقواس المربعة حقول تُملأ بمحتوى حقيقي قبل الموافقة.

## 1) التواصل الأول (first-touch)

**الموضوع:** [القطاع] — فكرة سريعة لـ [اسم الشركة]

```
السلام عليكم [الاسم/الدور]،

لاحظت [إشارة محدّدة من مصدر عام]. غالباً يصاحب ذلك [فرضية ألم نحلّها].

نحن في Dealix نساعد [القطاع] على ترتيب فرص الإيراد القائمة قبل البحث عن المزيد.

لو مفيد، أبدأ بتشخيص مجاني قصير [خطوة واحدة] — هل أرسل التفاصيل؟

مع التحية،
[الاسم] · Dealix

—
لإيقاف هذه الرسائل: [رابط الانسحاب]
```

## 2) المتابعة الأولى (follow-up 1)

**الموضوع:** متابعة — [اسم الشركة]

```
السلام عليكم [الاسم]،

أتابع رسالتي السابقة بزاوية أوضح: [نقطة قيمة جديدة مرتبطة بـ [إشارة محدّدة]].

إن كان توقيت غير مناسب، أخبرني وأعيد التواصل لاحقاً بإذنك.

هل خطوة قصيرة [دعوة واحدة] مناسبة هذا الأسبوع؟

[الاسم] · Dealix

—
لإيقاف هذه الرسائل: [رابط الانسحاب]
```

## 3) المتابعة الثانية (follow-up 2)

**الموضوع:** مثال قد يفيد [اسم الشركة]

```
السلام عليكم [الاسم]،

أشارك نمطاً آمناً (قالب افتراضي، بلا أسماء عملاء): [وصف موجز لتسريب فرص شائع في [القطاع] وكيف نعالجه].

النتائج تقديرية وتُعرض كفرص مُثبتة بأدلة — لا وعود.

إن رغبت، [دعوة واحدة: تشخيص مجاني/مكالمة 15 دقيقة].

[الاسم] · Dealix

—
لإيقاف هذه الرسائل: [رابط الانسحاب]
```

## 4) تمهيد العرض (proposal-intro)

**الموضوع:** خطوة مقترحة لـ [اسم الشركة]

```
السلام عليكم [الاسم]،

بناءً على اهتمامكم، أقترح بطاقة عرض مناسبة لحجمكم:
[عرض من السلّم: Sprint 499 / Data-to-Revenue Pack 1,500 / Managed Revenue Ops] — نطاق محدّد، بلا التزام إيراد.

أرفق التفاصيل في صفحة واحدة — أرسلها الآن؟

[الاسم] · Dealix

—
لإيقاف هذه الرسائل: [رابط الانسحاب]
```

## 5) إغلاق الحلقة (breakup)

**الموضوع:** أُغلق الملف مؤقتاً — [اسم الشركة]

```
السلام عليكم [الاسم]،

أحترم انشغالكم وأُغلق هذا الملف مؤقتاً حتى لا أزعجكم.

إن تغيّرت الأولوية لاحقاً، الباب مفتوح — يكفي رد بكلمة.

شكراً لوقتكم.

[الاسم] · Dealix

—
لإيقاف هذه الرسائل: [رابط الانسحاب]
```

# قواعد توسّع الإعلانات | Ads Scale Rules

> **AR:** يحدّد هذا المستند قواعد التعامل مع الإعلانات ضمن نظام التوسّع. **هذا النظام للتخطيط فقط ولا يُطلق أي حملة مدفوعة حيّة.** كل ما يُنتَج هنا مسودّات وخطط ومعايير قياس يراجعها المؤسس، ويبقى الإطلاق الفعلي يدويًا وخارج نطاق الأتمتة.
>
> **EN:** This document defines how ads are handled within the scale OS. **This OS is planning-only and launches no live paid campaign.** Everything produced here is drafts, plans, and measurement criteria for founder review; the actual launch stays manual and outside automation.

## المسموح (تخطيط) | Allowed (Planning)

| النشاط Activity | الناتج Output |
|---|---|
| صياغة فرضية الحملة / draft campaign hypothesis | وثيقة فرضية / hypothesis doc |
| تقدير الميزانية / budget estimate | نموذج تكلفة / cost model |
| تصميم معايير القياس / define metrics | إطار قياس / measurement frame |
| إنتاج مسودّات الإبداع / draft creatives | مسودّات للمراجعة / drafts for review |
| تحليل ما بعد الحملة / post-campaign analysis | تقرير تشخيصي / diagnostic report |

## الممنوع | Forbidden

- **إطلاق حملة مدفوعة حيّة** من النظام. / *Launching a live paid campaign* from the OS.
- ربط حسابات إعلانية للصرف الآلي. / Connecting ad accounts for auto-spend.
- أي صرف مالي دون موافقة المؤسس. / Any spend without founder approval.

## عتبات الجاهزية للإعلان | Ad-Readiness Thresholds

| الشرط Condition | العتبة الإرشادية Indicative Threshold |
|---|---|
| رسالة مثبتة / proven message | تحوّل عضوي مثبت أولًا / organic conversion proven first |
| اقتصاد الوحدة / unit economics | يحتمل تكلفة الاكتساب / supports acquisition cost |
| قدرة التسليم / delivery capacity | تستوعب الطلب الإضافي / absorbs added demand |

## تدفّق الإطلاق (يدوي) | Launch Flow (Manual)

1. تُجهَّز خطة الحملة كمسودّة. / Campaign plan prepared as a draft.
2. يراجع المؤسس ويعتمد الميزانية. / Founder reviews and approves budget.
3. **يطلق المؤسس الحملة يدويًا** خارج النظام. / *Founder launches manually* outside the OS.
4. تُجمَع النتائج لاحقًا للتحليل. / Results collected later for analysis.

## قاعدة الأمان | Safety Rule

> **No paid ads live launch.** تخطيط وتحليل فقط، والإطلاق يدوي وبموافقة. / Planning and analysis only; launch is manual and approved.

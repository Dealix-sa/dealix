# متى نتوسّع | When to Scale

> **AR:** يحدّد هذا المستند المحفّزات الكمية التي يجب استيفاؤها قبل أي قرار توسّع. التوسّع يبدأ فقط حين تتحقق مؤشرات قابلة للقياس، لا حين يبدو السوق "واعدًا". كل عتبة هنا إرشادية للمراجعة ويعتمدها المؤسس.
>
> **EN:** This document defines the quantitative triggers that must be met before any scale decision. Scaling begins only when measurable indicators are hit, not when a market merely "feels promising." Each threshold here is an indicative guide for review, approved by the founder.

## المحفّزات الكمية | Quantitative Triggers

| المؤشر Metric | العتبة الإرشادية Indicative Threshold | لماذا Why |
|---|---|---|
| طلب متكرر / Repeat demand | ≥ 3 جهات بنفس الحاجة / 3+ parties, same need | إشارة سوق لا صدفة / market signal, not chance |
| معدّل التحوّل / Conversion | مستقر عبر دورتين / stable across 2 cycles | عملية قابلة للتكرار / repeatable process |
| اقتصاد الوحدة / Unit economics | موجب ومستقر / positive & stable | التوسّع يضاعف الربح لا الخسارة / scaling multiplies profit not loss |
| استغلال القدرة / Capacity utilization | ≥ 70% بثبات / sustained ≥ 70% | الحاجة حقيقية للتوسّع / real need to expand |
| رضا التسليم / Delivery quality | ضمن المعايير / within standards | لا نتوسّع على جودة هشّة / no scaling on fragile quality |

## شروط مسبقة | Preconditions

1. المؤشرات أعلاه مستوفاة ومثبتة بأدلّة. / The metrics above are met and evidenced.
2. لا أرقام مُختلقة ولا جذب وهمي. / No fabricated numbers, no fake traction.
3. القدرة الحالية لن تنهار بالتوسّع (انظر `05`). / Current capacity won't collapse (see `05`).

## ما ليس محفّزًا | What Is *Not* a Trigger

- حماس لحظي أو طلب فردي غير متكرر. / Momentary excitement or a single non-repeat request.
- ضغط تنافسي بلا دليل طلب. / Competitive pressure without demand evidence.
- توقّع عائد مضمون. / Expectation of guaranteed ROI.

## عملية القرار | Decision Process

1. **Measure** — تُجمَع المؤشرات. / Metrics gathered.
2. **Check** — تُقارَن بالعتبات. / Compared to thresholds.
3. **Draft** — تُولّد توصية توسّع كمسودّة. / Scale recommendation drafted.
4. **Approve** — يعتمد المؤسس أو يؤجّل. / Founder approves or defers.

## قاعدة الأمان | Safety Rule

> لا توسّع بلا محفّز مُستوفى ومُعتمَد. التنفيذ الخارجي يدوي. / No scaling without a met, approved trigger. External execution is manual.

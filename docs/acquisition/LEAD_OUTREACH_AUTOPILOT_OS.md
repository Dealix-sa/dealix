# Lead & Outreach Autopilot OS

## Purpose
Create a semi-autonomous system that sources leads, qualifies them, drafts
outreach, prepares follow-ups, and asks Sami for approval before external
sending.

## Owner
Sami / Founder CEO.

## Principle
Automate preparation. Human-approve external action.

## Flow
Source leads
→ enrich company profile
→ score fit
→ classify A/B/C
→ draft outreach
→ prepare approval batch
→ Sami approves
→ send manually or through approved email workflow
→ log action
→ follow up
→ convert reply to sample/proposal

## Autopilot Can Do
- find public companies
- extract public company info
- classify sector
- draft messages
- prepare follow-up calendar
- create sample opportunities
- generate daily lead report

## Requires Approval
- sending outbound messages
- using personal contact data
- sending proposals
- making claims
- publishing proof
- pricing exceptions

## Never Automatic
- spam sending
- scraping restricted platforms against terms
- guaranteed sales/revenue claims
- sending sensitive data
- contract/refund/legal commitments

## Minimum Batch
25 leads per batch.

## Batch Output
- lead list
- score
- reason
- suggested message
- risk notes
- approval status

## Evidence
- acquisition/lead_batches/
- pipeline/pipeline_tracker.csv
- revenue/revenue_action_log.csv
- founder/outreach_approval_queue.md

## Compliance Notes
- SDAIA / PDPL: protect personal data, only use public business information
  unless the individual has consented.
- CST anti-spam regulations: no unsolicited mass messaging; outbound is
  founder-approved, batched, and traceable.
- Lead sources used by this OS are public business directories and search
  (Riyadh Chamber, Monsha'at ecosystem, Blue Pages, Invest Saudi
  directory, company websites, Google).

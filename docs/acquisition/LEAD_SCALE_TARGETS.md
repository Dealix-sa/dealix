# Lead Scale Targets

## Purpose

Scale Dealix lead generation from proof batches to a multi-sector
opportunity database.

## Levels

### L1 — Proof
25 leads / sector.
Goal: test message and sample offer.

### L2 — Sector Base
100 leads / sector.
Goal: identify reply patterns and sample conversion.

### L3 — Multi-Sector Base
500 leads across 5 sectors.
Goal: compare sectors and focus on best conversion.

### L4 — Market Intelligence Base
1,000–5,000 leads.
Goal: build Saudi B2B opportunity intelligence.

## Current Sectors

- ERP / CRM
- Cybersecurity
- B2B agencies
- Logistics / industrial services
- Consulting / implementation firms

## Weekly Target

- Add 100 leads.
- Approve 25–50 outreach actions.
- Send only approved high-fit messages.
- Generate 3–5 samples.
- Push 1–3 proposals.

## Rule

Scale the database faster than outreach.
Do not scale sending faster than reply quality, trust, and follow-up
capacity.

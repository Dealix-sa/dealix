# Outreach Throughput Rules

## Purpose

Scale outreach safely without spam, quality collapse, or follow-up overload.

## Throughput Levels

### Warmup
10–25 approved messages/day.
Use when message is unproven.

### Controlled Scale
25–50 approved messages/day.
Use after positive replies appear.

### Sector Scale
50–100 approved messages/day across channels.
Use only if follow-up capacity exists.

### No-Go
Do not send 200+ cold messages/day without:
- clear relevance
- suppression list
- reply handling
- follow-up capacity
- opt-out respect
- trust review

## Rule

Scale research faster than sending.
Scale sending only after reply and sample quality.

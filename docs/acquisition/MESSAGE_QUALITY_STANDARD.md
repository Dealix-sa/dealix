# Message Quality Standard

Every message that leaves Dealix passes this bar before send.

## Required
- **Specific** — mentions something true about the prospect's business.
- **Short** — under 120 words.
- **Human** — sounds like a person, not a template.
- **Relevant** — connects to something the prospect cares about.
- **Clear CTA** — one ask, one next step.

## Forbidden
- Hype words ("revolutionary", "game-changing", "10x guaranteed").
- Fake personalisation ("I love what you're doing!" with no detail).
- Guaranteed outcomes.
- Generic openers ("Hope you're doing well!").
- Emoji walls.

## Pre-send checklist
- [ ] Names spelled correctly.
- [ ] Company name spelled correctly.
- [ ] Specific detail verified from a public source.
- [ ] Claim Guard passed (no over-claim).
- [ ] Suppression-list check passed.
- [ ] CTA is concrete and unambiguous.

## Sample structure
1. One line of context.
2. One line of value.
3. One CTA.

## Rule
If the same message could be sent to ten different companies without changes, it is not ready.

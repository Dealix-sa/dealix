# Outbound Cadence System
## Purpose
Define how Dealix contacts leads without spam, overclaiming, or losing follow-up discipline.
## Cadence
Day 0: Founder DM.
Day 2: polite follow-up.
Day 5: sample/value angle.
Day 7: proposal/call if engaged.
Day 14: close or recycle.
## Rules
- No fake urgency.
- No guaranteed sales.
- No unsupported claims.
- No mass personalization lies.
- Stop if prospect declines.
- Track every action.
## Message Types
- founder intro.
- sector-specific value.
- sample offer.
- follow-up.
- payment/proposal follow-up.
## Metrics
- DMs sent.
- follow-ups sent.
- replies.
- positive replies.
- negative replies.
- samples requested.
- calls booked.

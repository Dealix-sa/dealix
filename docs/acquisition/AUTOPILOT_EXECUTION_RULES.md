# Autopilot Execution Rules

## Purpose
Allow Dealix to operate with minimal founder input while protecting trust
and compliance.

## Autopilot Can Execute Without Approval
- create lead batch templates
- score leads
- generate draft messages
- generate approval queue
- generate follow-up queue
- generate samples drafts
- generate internal reports

## Requires Sami Approval
- sending external messages
- sending proposals
- sending client reports
- publishing proof
- discounting
- committing to delivery scope

## Never Automatic
- spam sending
- refunds
- legal/contract commitments
- sensitive data exports
- guaranteed revenue/sales claims
- public client references

## Founder Input Target
Less than 15 minutes per batch:
- approve sector
- approve message
- approve send

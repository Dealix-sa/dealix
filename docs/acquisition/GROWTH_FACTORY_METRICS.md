# Growth Factory Metrics

## Input Metrics

- leads discovered
- leads enriched
- leads scored
- A leads
- B leads
- rejected leads

## Output Metrics

- messages drafted
- messages approved
- messages sent
- follow-ups scheduled
- replies
- positive replies
- samples created
- proposals created
- payment follow-ups
- cash collected

## Quality Metrics

- reply rate
- positive reply rate
- sample-to-proposal rate
- proposal-to-payment rate
- bad-fit rate
- complaint/opt-out rate
- trust incidents

## CEO Metrics

- founder approval time
- actions approved
- bottlenecks removed
- revenue actions completed

## Rule

If opt-out/complaint risk rises, reduce throughput and improve targeting.

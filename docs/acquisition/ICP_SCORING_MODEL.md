# ICP Scoring Model

Score every prospect 0–100 before any outreach.

## Dimensions (each 0–20)
1. **Sector fit** — does the prospect match a target sector this quarter?
2. **Revenue motion** — is there a clear B2B sales or account-led revenue motion?
3. **Budget signal** — public spend, headcount, prior vendors, hiring.
4. **Reachability** — is there a public decision contact (founder, head of sales, CRO)?
5. **Pain signal** — is there a visible problem Dealix addresses (slow growth, missed quota, manual ops)?

## Banding
- **80–100** — A-tier. Personal founder outreach.
- **60–79** — B-tier. Standard outreach cadence.
- **40–59** — C-tier. Nurture via content only.
- **0–39** — Park. Do not contact.

## Reviewing the model
- Re-score the top 20 prospects monthly.
- Update weights in the Monthly Strategy Review based on win-rate by band.

## Rule
A score is recorded with reasoning. "Felt right" is not a score.

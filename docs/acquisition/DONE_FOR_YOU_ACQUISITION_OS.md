# Done-For-You Acquisition OS

## Purpose
Make Dealix capable of finding leads, preparing outreach, creating approval queues,
drafting messages, tracking follow-ups, and moving opportunities toward samples,
proposals, and payment with minimal founder input.

## Owner
Sami / Founder CEO.

## Operating Principle
The system does the work.
Sami approves external commitments.

## Autopilot Flow
Source public leads
→ qualify
→ score
→ classify A/B/C
→ draft outreach
→ create approval queue
→ create Gmail drafts or manual send queue
→ log outbound
→ schedule follow-up
→ detect reply
→ trigger sample
→ trigger proposal
→ trigger payment follow-up

## Sami Input Target
Less than 15 minutes per batch:
1. Approve sector.
2. Approve message.
3. Approve send/draft batch.
4. Approve proposal/pricing if needed.

## Autopilot Can Do
- research public companies
- prepare lead batches
- score fit
- draft messages
- prepare follow-ups
- create Gmail drafts
- generate samples
- generate proposals for review
- generate CEO daily report
- update trackers

## Requires Sami Approval
- sending external messages
- sending proposals
- pricing/discount decisions
- client reports
- public proof
- any claim about results
- any A2/A3 action

## Never Automatic
- spam
- scraping restricted platforms against terms
- refunds
- contract changes
- legal commitments
- sensitive data exports
- guaranteed revenue/sales claims

## First Target Sectors
1. Saudi ERP / CRM vendors
2. Saudi cybersecurity firms
3. B2B agencies in Riyadh
4. Logistics / industrial services
5. Consulting / implementation firms

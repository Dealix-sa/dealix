# System Scorecard

The System Scorecard converts the state of each operating system into a
0–100 number and a status band.

## Bands

| Score | Status |
|---:|---|
| 90–100 | PASS |
| 75–89 | READY INTERNAL |
| 50–74 | FIX |
| 0–49 | BLOCKED |

## Aggregate

`Company OS Aggregate = average of system scores`.

The aggregate is the headline number for the Weekly CEO Review.

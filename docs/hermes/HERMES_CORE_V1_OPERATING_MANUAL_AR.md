# 🧠 دليل تشغيل نظام هيرمس المتقدم — إصدار المؤسس (Hermes Operating Manual)

دليل تشغيل الطبقة الاستراتيجية والعقلية والتحليلية المتقدمة لشركة ديلكس.

---

## 1. فلسفة تشغيل هيرمس (Sovereign Founder Brain)
يمثل نظام هيرمس العقل التشغيلي والاستراتيجي المساعد للمؤسس، وهو يقوم بتحويل إشارات السوق والاعتراضات الفردية إلى قرارات موثقة وأصول تسويقية ودراسات حالة وأدلة إثبات ROI.

## 2. التدفقات والأوامر المتقدمة المتاحة

### 1. تشغيل التدفق الاستراتيجي بالكامل (Daily Executive Run):
```powershell
.\scripts\hermes-core-run.ps1
```
ينشئ موجز سامي اليومي، ورادار الفرص، وحزم الامتثال، وتقارير المراجعة الأسبوعية.

### 2. تسجيل وتحويل إشارات السوق الذكية للوح المبيعات:
```powershell
py -3 scripts/hermes_capture_signal.py --source "market" --sector "Logistics" --text "إلزام العنوان الوطني للطرود من هيئة النقل" --why "فرصة TGA" --offer "Delivery Accuracy" --first-action "send outreach"
```

### 3. إعداد غرفة صفقات تفصيلية لحوكمة المفاوضات:
```powershell
py -3 scripts/hermes_deal_room.py --client "Mena Ads" --pain "الخوف من تسريب البيانات" --offer "AI Trust Diagnostic" --target-price "5000" --floor-price "3500" --strategic-value "proof pack"
```

### 4. معالجة وتوثيق نتائج المبيعات والتعلم والدروس المستفادة:
```powershell
py -3 scripts/hermes_log_outcome.py --action-type "outreach" --target "Mena Ads" --expected "reply" --actual "replied interested with price question" --won "false" --revenue "0" --learning "price question requires standard objection handler and immediate call booking"
```

### 5. تقييم أولويات وجدوى الفرص برمجياً:
```powershell
py -3 scripts/hermes_score.py "{\"verified_revenue_potential\":85,\"speed_to_cash\":80}"
```

### 6. تحويل نتائج تسليم العملاء إلى دراسة حالة تجارية:
```powershell
py -3 scripts/hermes_case_study.py --client "Al-Majd Group" --offer "AI Trust Diagnostic" --problem "تسريب PII عبر وكلاء مبيعات AI" --baseline "لا توجد حوكمة" --work "بناء مصفوفة الموافقات وتصميم Risk Map" --result "تأمين 100% من مخرجات العملاء" --metric "AI compliance secured" --anonymous "true"
```

### 7. بوابة تدقيق إمكانية الانتقال لـ SaaS ومنع البناء الزائد:
```powershell
py -3 scripts/hermes_productization_gate.py --offer "AI Trust Diagnostic" --paid-runs 3 --proof-packs 2 --retainers 1 --delivery-burden 45
```

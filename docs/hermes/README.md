## English Summary

- This directory holds the full Dealix Max OS specification rewritten as 18 Arabic-first specification documents plus this index.
- Coverage maps directly to spec sections §26–§50 — philosophy, event model, orchestrator, the seven workspaces (Sovereign, Internal, Customer, Partner, Trust, Venture), and the daily operating rhythm.
- Recommended reading order: Philosophy → Event Model → Orchestrator → Workspaces Overview → individual workspace docs → Risk/Quality/Evidence/Scale-Kill → Daily Rhythm → Final Blueprint.
- All financial figures use existing ranges already documented under `docs/commercial/`; no new revenue claims are introduced.
- Every doc closes with a short English summary so a non-Arabic reader can navigate without machine translation.

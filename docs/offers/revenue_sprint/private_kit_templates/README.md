# Private Kit Templates — Reference Copies

These files are **reference copies** of the operational templates that
live in the private operations repository:

    dealix-ops-private/offers/revenue_sprint/

They are kept here only so the kit structure can be reviewed in one place.
**The live, working versions of these files belong in the private repo.**

When you start operating the Revenue Sprint Kit:

1. Copy each file into `dealix-ops-private/offers/revenue_sprint/`.
2. Drop the `private_kit_templates/` folder if you do not want the
   reference copies in the public repo, OR keep them as the canonical
   text of the templates while the private repo holds filled-in
   per-client copies.

# Agent Governance v3

## Purpose
Govern all Dealix agents by risk, approval, evidence, and reversibility.

## Agent Classes
- L0 Prompt Assistant
- L1 Draft Assistant
- L2 Internal Analyst
- L3 Workflow Router
- L4 Controlled Tool User
- L5 Governed Automation

## Risk Dimensions
- approval class A0–A3
- reversibility R0–R3
- sensitivity S0–S3
- external impact
- financial impact
- trust impact

## Required For Every Agent
- purpose
- owner
- inputs
- outputs
- tools
- approval class
- eval suite
- logs
- disable path
- incident path

## Rule
No L4/L5 agent without eval, logs, and approval gates.

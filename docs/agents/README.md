# docs/agents

Operating folder for the **agents** system inside Dealix Company OS.

This folder is reserved for the agents operating manuals, playbooks, schemas, and decision records. Add new files here using the Dealix Document Standard (see `docs/ops/DOCUMENT_STANDARD.md`).

Required sections in every file: Purpose, Owner, Review Cadence, Inputs, Outputs, Rules, Metrics, Evidence, Last Reviewed.

`README.md` files are exempt from the document quality verifier.

# Board Discipline

## Purpose
Create investor-grade operating discipline before fundraising.

## Owner
Sami / Founder CEO.

## Review Cadence
Monthly.

## Board Pack Sections
1. Executive summary
2. Revenue
3. Pipeline
4. Cash and runway
5. Delivery
6. Trust and risk
7. Learning
8. Productization
9. CEO decisions needed

## Rule
No investor deck before:
- first paid sprint
- delivery proof
- customer feedback
- pipeline metrics
- financial model v1

## Metrics
- cash collected
- MRR
- pipeline value
- paid sprints
- retention attempts
- runway

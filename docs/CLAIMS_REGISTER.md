# Dealix — Claims Register — سجل الادعاءات
<!-- Owner: Founder | Effective: 2026-06-05 | العربية أولاً -->

> **القاعدة:** كل ما يُقال للعميل يجب أن يقع في عمود "مسموح". أي شيء في عمود "ممنوع"
> لا يُكتب في أي قناة (صفحات، عروض، رسائل، محتوى).

---

## 1. مسموح مقابل ممنوع — Allowed vs Forbidden (AR)

| ✅ مسموح (Allowed) | ❌ ممنوع (Forbidden) |
|--------------------|----------------------|
| بيان قدرة مع مصدر + درجة ثقة | "نضمن" / "guarantee" أي نتيجة أو نسبة |
| إثبات مؤطَّر بمستوى الأدلة (Proof Level) | "زادت المبيعات" بدون رقم مذكور ومصدره |
| "فرص مُثبتة بأدلة" بدلاً من وعد بيع | تسمية عميل بدون موافقة خطية |
| "قيمة تقديرية" مع الإفصاح | شهادات/توصيات وهمية أو ضمنية |
| توصيف ما سُلّم فعلياً | "autonomous" / "15 agents" / "V14/V17" / "beast" في نسخة العميل |
| "بيانات ناقصة = insufficient_data" | إيحاء بإرسال خارجي نيابة عن العميل بدون موافقة صريحة |

## 1. Allowed vs Forbidden (EN)

| ✅ Allowed | ❌ Forbidden |
|-----------|-------------|
| Capability statements with **source + confidence** | "نضمن" / "guarantee" of any result or rate |
| Proof framed by **evidence level** | "sales increased" without a **cited number + source** |
| "Evidenced opportunities" instead of a sale promise | Naming a customer **without written consent** |
| "Estimated value" with the disclosure line | Fake or implied testimonials |
| Description of what was actually delivered | "autonomous" / "15 agents" / "V14/V17" / "beast" in customer-facing copy |
| "Missing data = insufficient_data" | Implying Dealix sends external messages without explicit approval |

## 2. سلم مستويات الإثبات — Proof Levels Ladder

| المستوى | المعنى | EN |
|---------|--------|----|
| **L1** | سُلِّم المخرج | Delivered |
| **L2** | وافق العميل على المخرج | Customer approved |
| **L3** | استخدم العميل المخرج | Customer used |
| **L4** | لوحظ أثر | Effect observed |
| **L5** | وُثِّق الأثر بأرقام | Effect documented with numbers |

> **أول 30 يوماً:** يكفي **L2 / L3** فقط. لا يُطالَب بـ L4/L5 قبل توفر بيانات حقيقية.
> First 30 days only need **L2/L3** — do not claim L4/L5 before real data exists.

---

## 3. قاعدة الاستبدال — Substitution Rule

| بدلاً من | استخدم |
|----------|--------|
| "نضمن مبيعات" / "guaranteed sales" | "فرص مُثبتة بأدلة" / "evidenced opportunities" |
| "زادت المبيعات X%" | رقم مذكور بمصدره ومستوى إثباته، أو احذفه |

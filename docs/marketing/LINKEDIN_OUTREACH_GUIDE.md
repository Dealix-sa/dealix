# LinkedIn Outreach Guide

## 1. Posture

LinkedIn is a relationship channel. Volume kills it. Signal wins it.

## 2. Pre-flight

- [ ] Profile is real and reachable.
- [ ] Profile is in ICP.
- [ ] Suppression honoured.
- [ ] Persona matched.
- [ ] Brand check passed.

## 3. Connect notes

- ≤ 270 characters.
- Reference one shared signal (event, sector, mutual).
- No pitch in the connect note.
- Bilingual when appropriate.

## 4. First message (after accept)

- 60-120 words.
- One observation, one relevance line, an invitation to talk.
- No deck attached.

## 5. InMail

- 90-160 words.
- Open with the observation, not the ask.
- Subject specific: e.g., "Riyadh expansion — one peer pattern".
- No "Quick question?" subjects.

## 6. Comments and engagement

- Comment on relevant posts before any DM.
- Add value, not "Great post!".
- One comment per session per profile.

## 7. Posting cadence

- Founder: 3 × per week.
- Dealix page: 1 × per week.
- Format mix: insight, learning, sector pattern, proof excerpt.

## 8. Compliance

- No scraping.
- No automation tools against LinkedIn's ToS.
- No purchased connections.

## 9. Bilingual

Match the language of the most recent post the recipient made, with the other language available on request.

# Content Calendar OS

The Content Calendar OS is the single source of truth for what Dealix says, where, and when.

## Surfaces

- LinkedIn (founder voice, weekly cadence)
- Newsletter (bi-weekly)
- Blog / SEO (monthly long-form)
- Sector micro-letters (monthly, by ICP segment)
- Webinar / event (quarterly)

## Pipeline

`idea → draft → review → approved → scheduled → published → measured`

All entries land in `marketing/content_calendar.csv` under `$PRIVATE_OPS`. The Content Strategist agent may stage drafts; only founder-approved entries reach the `scheduled` state.

## Cadence rules

- Never two product-focused posts in a row on LinkedIn.
- Every public claim must reference a stored proof item id.
- Arabic and English run in parallel — never auto-translate one from the other.

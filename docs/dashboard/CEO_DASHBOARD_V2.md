# CEO Dashboard v2

## Purpose
Turn Dealix operating data into CEO decisions.

## Owner
Sami / Founder CEO.

## Review Cadence
Daily for operating metrics, weekly for targets.

## Inputs
- pipeline_tracker.csv
- mrr_tracker.csv
- approval_log.csv
- generated bottlenecks
- targets
- decision queue

## Outputs
- CEO dashboard
- operating alerts
- target score
- founder focus
- weekly learning input

## Rules
- Dashboard must show action, not only numbers.
- Real private data must not be committed to the public repo.
- Demo data must be clearly fake.
- Alerts must be prioritized as red, yellow, or green.
- Every recommendation must trace back to metrics or evidence.

## Metrics
- Target completion rate.
- Bottlenecks detected.
- Decisions pending.
- Revenue movement.
- Trust risks.
- Founder focus clarity.

## Evidence
- dashboard JSON
- Daily CEO Brief
- Decision Queue
- Weekly Intelligence Review
- GitHub Actions checks

# Decision Rights

## Purpose
Define which decisions require founder approval, trust approval, or system-level rules.

## CEO-Only Decisions
- pricing exceptions
- refunds
- contract changes
- public case studies
- strategic pivots
- hiring
- SaaS expansion
- agent external action release

## Trust Approval Required
- external claims
- client data use
- public proof
- A2 actions
- unusual proposal terms

## System-Automated Decisions
Only allowed for:
- internal scoring
- internal dashboard generation
- internal brief generation
- duplicate checks
- checklist verification

## Never Automated
- refunds
- legal commitments
- sensitive data exports
- guaranteed claims
- public client references

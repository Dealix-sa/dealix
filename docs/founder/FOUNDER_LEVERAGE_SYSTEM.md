# Founder Leverage System

## Purpose
Increase Sami's leverage by reducing repeated manual work and focusing founder time on highest-value decisions.

## Owner
Sami / Founder CEO.

## Review Cadence
Weekly.

## Leverage Principles
- Founder time goes first to revenue, trust decisions, strategy, and closing.
- Repeated work becomes checklist.
- Repeated checklist becomes template.
- Repeated template becomes delegated or automated.
- Founder should not solve the same operational issue three times manually.

## Founder Work Categories

| Category | Keep / Delegate / Automate |
|---|---|
| Closing calls | Keep |
| Strategic decisions | Keep |
| Trust approvals | Keep initially |
| Lead formatting | Automate later |
| Delivery table cleanup | Template/delegate |
| Weekly reporting | Automate |
| Dashboard generation | Automate |
| Outreach sending | Keep founder-led until proof |

## Leverage Score

| Signal | Points |
|---|---:|
| Revenue work done daily | +20 |
| Repeated work documented | +20 |
| One system update weekly | +20 |
| Founder bottleneck reduced | +20 |
| Work delegated or automated safely | +20 |

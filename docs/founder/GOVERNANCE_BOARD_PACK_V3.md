# Governance Board Pack v3 / حُزمة حوكمة المجلس v3

## Purpose / الغرض

Create a monthly executive board-grade pack for Dealix.

إنشاء حزمة تنفيذية شهرية بمستوى مجلس إدارة لـ Dealix.

## Sections / الأقسام

1. **CEO Summary** / ملخص الرئيس التنفيذي
2. **Revenue Funnel** / قُمع الإيرادات
3. **Cash / MRR / Runway** / النقد / الإيراد الشهري / المدى المالي
4. **Sector Performance** / أداء القطاع
5. **Channel Performance** / أداء القناة
6. **Delivery / QA** / التسليم / ضمان الجودة
7. **Client Success / Retention** / نجاح العميل / الاحتفاظ
8. **Trust / AI Risk** / الثقة / مخاطر الذكاء
9. **Productization** / المنتجة
10. **People / Partners** / الناس / الشركاء
11. **Next Month Decisions** / قرارات الشهر القادم

## Required Decisions / قرارات مطلوبة

- double down sector / مضاعفة قطاع
- kill weak channel / إيقاف قناة ضعيفة
- pricing change / تغيير تسعير
- productization move / حركة منتجة
- hiring/delegation move / حركة توظيف/تفويض
- trust policy update / تحديث سياسة الثقة

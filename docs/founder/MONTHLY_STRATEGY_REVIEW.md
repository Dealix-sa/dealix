# Monthly Strategy Review

Run on the last working day of the month.

## Agenda
1. Strategy
   - Is the current offer ladder still the right one?
   - Is the current ICP still the right one?
   - Is the current positioning still the right one?
2. Pricing
   - Win rate by price band.
   - Refunds, discounts, scope creep.
   - One pricing experiment to run next month.
3. Market
   - New entrants, new sectors, new buying signals.
   - One sector to focus on next month.
4. Product Roadmap
   - Features built, features killed, features deferred.
   - Customer evidence behind each "build now" item.
5. Financial
   - Cash collected, MRR, AR, refunds.
   - Gross margin estimate.
   - Runway estimate.
6. Hiring / Delegation
   - Trigger checks from `docs/people/HIRING_TRIGGERS.md`.
   - What can move from founder to a role next month?

## Outputs
- Updated `docs/strategy/12_MONTH_ROADMAP.md` (or equivalent).
- New entries in `DECISION_LOG.md` for any strategic shift.
- One refreshed page in `docs/learning/MONTHLY_STRATEGY_UPDATE.md`.

## Rule
The monthly review is the only place strategy can change. Daily noise does not change strategy.

# Founder Operating Rhythm — Dealix

## الدور — Role

إيقاع المؤسس الشخصي اليومي/الأسبوعي. يربط بين Daily Brief و Weekly War Room و القرارات اليومية.

## اليومي — Daily

| الوقت | النشاط |
| --- | --- |
| 08:30 | تشغيل `make ceo-daily-brief` + قراءة الـ brief |
| 08:45 | تشغيل `make revenue-forecast` + مراجعة الـ pipeline |
| 09:00 | اختيار **action واحد فقط** لليوم وكتابته في `<private_ops>/founder/today.md` |
| 09:15–12:30 | تنفيذ Action اليوم (لا اجتماعات إلا إذا كانت Action نفسها) |
| 13:30–16:00 | مكالمات + sample / proposal work |
| 16:00–17:00 | مراجعة approvals queue + توقيع ما هو جاهز |
| 17:00–17:30 | تحديث `conversation_log.csv` يدويًا |

## الأسبوعي — Weekly

| اليوم | النشاط |
| --- | --- |
| الأحد 17:00 | تخطيط الأسبوع — مراجعة `<private_ops>/launch/blockers.csv` |
| الاثنين 09:00 | Weekly Growth War Room |
| الخميس 14:00 | Trust risk review |
| الجمعة 11:00 | Learning loop sync + تحديث `learning/*.csv` |

## القواعد — Rules

- لا أكثر من **3 actions جدية** في اليوم.
- لا قرار strategic قبل قراءة Daily Brief.
- لا اجتماعات صباحًا قبل 09:30 (وقت تنفيذ خالص).
- لا "تجاوز" برنامج اليوم بسبب رسالة WhatsApp — تُحوَّل لـ `tomorrow.md`.

## الملكية — Ownership

- Owner: Founder.

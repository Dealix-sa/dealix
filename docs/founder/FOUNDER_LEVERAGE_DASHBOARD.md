# Founder Leverage Dashboard

The CEO's single read on **where their hours land** and **what they cost the
company by *not* delegating**.

Generated by
[`scripts/founder_leverage_dashboard.py`](../../scripts/founder_leverage_dashboard.py)
into `data/founder_briefs/founder_leverage_<week_end>.md`.

## Inputs

- PRIVATE_OPS `ceo/leverage_time_audit.csv` — weekly hours bucketed Make / Manage / Move
- PRIVATE_OPS `ceo/decisions.jsonl` — count of `delegate` decisions per week
- [`docs/people/HIRING_TRIGGER_SYSTEM.md`](../people/HIRING_TRIGGER_SYSTEM.md) — current trigger thresholds

## Definitions

| Bucket | What counts | Goal |
|---|---|---|
| **Make** | Personally producing the artefact (writing code, drafting proposals, doing the demo) | Decreasing toward 0 outside core differentiated work |
| **Manage** | Reviewing, approving, unblocking, hiring | Stable — this is the CEO job |
| **Move** | Strategy, learning, decisions, customer conversations | Increasing — this is leverage |

## The leverage ratio

```
leverage_ratio = move_hours / (make_hours + manage_hours + move_hours)
```

Target: ≥ 0.40 by week 12, ≥ 0.55 by week 24.

## Dashboard sections

1. **This week's bucket mix** — table + sparkline (markdown text-only)
2. **4-week trend** — make %, manage %, move %
3. **Top delegation candidates** — items aged > 3 days from the daily brief's Q4
4. **Bottlenecks tied to me** — top 3 from `scripts/dealix_bottleneck_radar.py` where `owner == "ceo"`
5. **Hire / automate triggers crossed** — from [../people/HIRING_TRIGGER_SYSTEM](../people/HIRING_TRIGGER_SYSTEM.md)

## API surface

- `GET /api/v1/founder/leverage/dashboard` — current ratio + bucket mix + bottlenecks
- `GET /api/v1/founder/leverage/time-audit/recent` — last 4 weeks aggregate

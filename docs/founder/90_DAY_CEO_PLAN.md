# 90-Day CEO Plan

> Three windows. Each with a target. Each with evidence.

## Days 1–7 — Foundation

### Outputs

- 25 ICP-qualified leads loaded into pipeline
- 25 personalised DMs sent
- 3 Signal Samples delivered
- 1 proposal sent
- Payment / PO follow-up running daily
- Weekly CEO review run on Day 7
- One playbook updated based on what we learned

### Discipline

- `make daily` every morning
- `make close-day` every evening
- `make weekly-close` on Day 7
- `make audit` on Day 7

### Anti-targets (what we do not do)

- Build new features
- Add a new ICP
- Talk to investors

## Days 8–30 — Repeatability

### Outputs

- 100 total DMs sent (cumulative)
- 10–15 replies
- 6–9 samples sent
- 2–4 proposals sent
- 1 paid Sprint target
- Delivery readiness confirmed (templates, QA, handoff)

### Discipline

- Weekly review with Build / Fix / Kill / Defer / Continue
- Monthly strategy review at Day 30
- Kill List updated; ≥ 2 items added
- Friction Log has ≥ 5 entries reviewed

### Anti-targets

- Hiring
- Public case study without consent
- Discount > 20%

## Days 31–60 — First Proof

### Outputs

- 3 paid Sprints delivered
- 2 completed deliveries with positive feedback captured
- 1 approved public proof asset (sample anonymised; or named case study
  with consent)
- First retainer attempt made (with named price and scope)

### Discipline

- Capital allocation review
- Moat maturity scored
- 1 workflow promoted to Level 2 (templated)

### Anti-targets

- SaaS prototyping
- Marketplace fantasies
- Outsourcing sales

## Days 61–90 — Compounding

### Outputs

- 1–3 retainers signed
- Templates promoted (any workflow at Level 3 readiness)
- Productization candidates documented (with the evidence trail)
- Partner list drafted (only partners who pass `PARTNER_STRATEGY.md`)
- Delegation trigger review (any triggers fired?)
- Engineering health review for the quarter
- Board pack written (even if board = self)

### Discipline

- Quarterly decision audit
- Quarterly threat-model review
- Quarterly content-pillar review

### Anti-targets

- Hiring without a sustained trigger
- Expanding ICP beyond 3 sectors
- Public claims that lack evidence

## Daily Operating Loop

```
make daily        # open daily brief
< execute >
make close-day    # write evening evidence
```

## End Of Each Day

- Was the One Revenue Action moved?
- What evidence was captured?
- What goes on tomorrow's Kill List discipline?

## End Of Each Week

```
make weekly-close
make audit
```

## End Of Each Month

- `MONTHLY_STRATEGY_REVIEW.md`
- `MONTHLY_MOAT_REVIEW.md`
- `ENGINEERING_HEALTH_REVIEW.md`
- `BOARD_PACK_TEMPLATE.md` → write the pack

## What to start today

```
make audit
make stage
make daily
make kit
```

Then enter 25 real leads. The architecture only becomes alive after
that.

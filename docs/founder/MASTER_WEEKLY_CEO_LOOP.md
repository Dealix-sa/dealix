# Master Weekly CEO Loop

## Purpose
Define the weekly founder operating rhythm.

## Weekly Commands
```bash
make ceo-weekly
make weekly-close
make finance
make business-score
make audit
```

## Weekly Review Sections
1. Revenue reality.
2. Financial reality.
3. Delivery reality.
4. Trust reality.
5. Learning.
6. Productization signals.
7. Kill/defer list.
8. Next week focus.

## Weekly Required Outputs
- one CEO decision
- one learning decision
- one system update
- one kill/defer decision
- one next-week focus

## Weekly Rule
If no system was updated, explain why.

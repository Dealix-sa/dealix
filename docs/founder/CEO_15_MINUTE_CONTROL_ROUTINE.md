# CEO 15-Minute Control Routine

## Purpose

Let Sami control the 24/7 growth factory in less than 15 minutes per day.

## Daily Review

1. Open mission_control.md.
2. Open strategic_stoplight.md.
3. Open daily_acquisition_report.md.
4. Approve/reject outreach batch.
5. Approve proposal if ready.
6. Review trust red flags.
7. Execute or approve one revenue action.

## CEO Does Not Do Daily

- manual lead searching
- manual scoring
- manual CSV cleanup
- manual report compiling
- repetitive follow-up planning

## CEO Must Still Approve

- external sending
- proposals
- pricing
- proof
- A2/A3 actions

## Rule

Sami controls the machine. Sami should not be the machine.

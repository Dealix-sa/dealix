# Founder Operating Scorecard v1

Generated by `scripts/generate_operating_scorecard.py` and exposed via
`GET /api/v1/internal/control/scorecard`.

Sections:

- Revenue score (derived from cash collected)
- Trust score (derived from open trust flags)
- Runtime score (derived from worker failures)
- Founder leverage score (derived from outreach drafts queue)
- Productization score (count of productization candidates)
- Top bottleneck (one line)
- Next best action (one line)

This is a founder-first heuristic — not an accounting source of truth.
For accounting, use the existing finance OS endpoints.

# Daily Command Brief

The first page the founder reads every morning. One screen. Five blocks. No fluff.

## Purpose
Give the founder a single-screen operating brief every morning so the first hour of the day is spent making decisions, not searching for context.

## Owner
Sami (Founder).

## Review Cadence
Daily, between 07:00 and 09:00 KSA.

## Inputs
- Pipeline tracker (private).
- Revenue and cash collected (private).
- Delivery board state.
- Approval queue.
- Open decisions from yesterday.

## Outputs
- A signed-off daily brief in `dealix-ops-private/founder/daily_brief.md`.
- A list of approved actions for the day.
- An updated decision log entry.

## Rules
- Brief must fit on one screen.
- Brief must contain the five blocks below in order.
- Every action in the brief must have an owner and a deadline.

---

## Block 1 — Money

- Cash on hand (KSA bank): _____
- MRR: _____
- Revenue collected this week: _____
- Outstanding invoices / POs: _____
- Pipeline value (weighted): _____

## Block 2 — Sales

- Leads in `New`: _____
- Leads in `Contacted`: _____
- Replies waiting on me: _____
- Proposals out: _____
- Next 3 outbound touches I will approve today.

## Block 3 — Delivery

- Active sprints: _____
- Customers at risk: _____
- Sample / report due today: _____
- Customer-facing deadline this week: _____

## Block 4 — Trust

- Approvals waiting on me: _____
- Autonomy violations flagged: _____
- Risk log items added in last 24h: _____

## Block 5 — Decisions Required

Three or fewer decisions the founder must make today. Each line is:

`[Decision] | [Owner-of-question] | [Deadline] | [Default if I say nothing]`

---

## How To Use

1. Open this file at 07:00.
2. Pull numbers from the private ops repo (or AI prepares them).
3. Sign the brief by writing today's date at the bottom.
4. Take the three decisions before opening anything else.

## Metrics
- Number of consecutive days the brief was signed.
- Number of decisions made by 10:00 KSA.
- Number of "Decisions Required" still open by end of day.

## Evidence
- Signed daily briefs accumulating in `dealix-ops-private/founder/daily_brief.md`.
- Approval log entries timestamped against decisions in this brief.

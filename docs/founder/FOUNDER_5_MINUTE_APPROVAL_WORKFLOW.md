# Founder 5-Minute Approval Workflow

## Purpose
Let Sami control the entire distribution machine with minimal daily input.

## Daily Steps
1. Open `founder/distribution_command_center.md`.
2. Open `founder/outreach_approval_queue.md`.
3. Approve, edit, or reject batch.
4. Review proposal queue if any.
5. Review trust red flags.

## Approval Words
- **Approved** — move to send / draft queue.
- **Needs Edit** — revise message or lead selection.
- **Rejected** — do not send.

## Rule
Sami approves leverage points, not repetitive work.

## Escalation
If a queued item touches:
- A guarantee claim.
- Personal data of an individual without business consent context.
- A name or proof not previously approved.

Then route to `founder/trust_red_flags.md` for explicit decision before any external action.

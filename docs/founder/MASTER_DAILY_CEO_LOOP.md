# Master Daily CEO Loop

## Purpose
Define the exact daily operating loop for Sami as Founder CEO.

## Morning
1. Run `make control-tower`.
2. Run `make stage`.
3. Run `make daily`.
4. Read top CEO action.
5. Pick one revenue-moving action.

## Execution Block
Focus only on the highest-priority action:
- add leads
- send DMs
- prepare samples
- send proposal
- pursue payment / PO
- complete delivery
- request feedback / retainer

## Trust Block
Review:
- approvals waiting
- A2 items
- A3 blocked actions
- risky claims
- public/private boundary

## End of Day
1. Update pipeline.
2. Update revenue_action_log.csv.
3. Update daily_execution_log.md.
4. Run `make advance`.
5. Run `make close-day`.

## Daily Non-Negotiable
No day closes without:
- one revenue action
- pipeline update
- trust review
- next action for tomorrow

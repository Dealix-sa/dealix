# RTL QA — عروض ومواد دعائية

**قبل إرسال أي asset عربي:** أكمل القائمة. الهدف: **مرآة تخطيط كاملة** وليس محاذاة يمين فقط.

## تخطيط

- [ ] `dir="rtl"` على الجذر (HTML) أو RTL في إعدادات الشرائح
- [ ] العناوين والفقرات `text-align: right`
- [ ] القوائم النقطية: indent من اليمين
- [ ] الجداول: عمود الأول = الأهم (يمين)
- [ ] المخططات: تدفق من اليمين لليسار (سهمات معكوسة)
- [ ] شريط تقدم / timeline يبدأ من اليمين
- [ ] أزرار CTA في الموضع المتوقع ثقافياً (يمين أو وسط)

## خطوط ونص

- [ ] IBM Plex Sans Arabic أو Noto — لا خطوط غير معتمدة
- [ ] `line-height` ≥ 1.5 للنص العربي
- [ ] لا تمطيط حروف عربية
- [ ] lam-alef (لا) تظهر متصلة
- [ ] أرقام السرد: Arabic-Indic اختياري؛ الجداول: Western

## ثنائي اللغة

- [ ] EN blocks مع `dir="ltr"` منفصلة
- [ ] أسماء علامات تجارية وURLs لا تنكسر الترتيب (`bidi`)
- [ ] زوج خطوط متناسق (IBM Plex Arabic + Inter)

## وسائط

- [ ] لا رموز مسيئة أو stock «مصافحة أعمال» عامة

# Dealix — Leave-Behind (مسح ٣ دقائق)

**الاستخدام:** PDF أو رابط بعد الاجتماع — **العناوين فقط** يجب أن تروي القصة كاملة.

---

1. نحوّل بيانات الإيرادات إلى قرارات مُوثّقة خلال ٧ أيام — بدون إرسال بارد
2. فرق الإيرادات السعودية تخسر وقت القرار عندما البيانات مشتّتة والموافقات غير موثّقة
3. كل أسبوع بدون ترتيب Top accounts يُهدر ساعات مبيعات على فرص ضعيفة التأهيل
4. Chatbot عام ووكالة «AI» بدون Proof لا تمرّ لجنة المشتري السعودية
5. Dealix = Software + Service + AI Operations مربوطة بجواز قرار وإثبات
6. سبرنت ذكاء الإيرادات يُسلّم ١٠ حسابات مُرتّبة وحزمة إثبات خلال ٧ أيام
7. لا إجراء خارجي يخرج بدون موافقة بشرية وجواز قرار
8. كل مشروع يُغلق بحزمة إثبات من ١٤ قسماً — وليس بوعد إيراد
9. التشخيص يكشف ROI قبل أي التزام طويل
10. نُخفّف مخاطر التبنّي بمراحل قصيرة ومعايير قبول واضحة
11. نُجهّز مواد منفصلة للمالية والتقنية والتشغيل من نفس القصة
12. الخطوة التالية: تشخيص ٣–٥ أيام + MAP مشترك بتاريخ محدد

# دليل صوت المؤسس — LinkedIn

**Cadence:** 3× أسبوع (ثلاثاء، أربعاء، خميس صباحاً)  
**لغة:** عربي أولاً — إنجليزي للـ VC فقط

## البنية الفائزة

1. **Hook:** رفض أو رقم (سطر واحد)
2. **قصة:** 3–5 أسطر — سياق سعودي
3. **إطار:** 3 نقاط (Passport / Proof / Draft)
4. **CTA:** تشخيص — لا ضغط مبيعات

## أمثلة Hooks

- «رفضنا إرسال ١,٢٠٠ رسالة واتساب — وهذا ما بنينا بدلاً منها.»
- «٧ أيام = ١٠ حسابات مُرتّبة + Proof Pack — ليس «استشارة AI».»
- «لجنة الشراء السعودية لا تشتري chatbot — تشتري إثباتاً.»

## ممنوع

- إيموجي مبالغ
- «ثوري» / «الأفضل»
- وعود إيراد
- هجوم شخصي على منافس

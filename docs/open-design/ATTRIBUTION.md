# Open Design Attribution

This Dealix integration was inspired by public concepts reviewed from `nexu-io/open-design`.

Open Design repository: `https://github.com/nexu-io/open-design`

The upstream project is licensed under Apache-2.0 according to its package metadata and repository badges. This Dealix PR does not vendor upstream source code, images, binaries, dependencies, or media assets. It only adapts operating concepts into Dealix-native documentation and project-level agent instructions.

Concepts adapted:

- agentic design workflow
- `DESIGN.md`-style brand contract
- skills + design systems + artifacts operating model
- sandbox/review-before-production mindset
- multi-agent compatibility expectations

Do not copy Open Design assets or source files into Dealix without a separate license/compliance review.

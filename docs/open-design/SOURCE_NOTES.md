# Open Design Source Review Notes

Reviewed public source repository: `nexu-io/open-design`.

## Relevant concepts

- Local-first agentic design workspace
- Design systems as brand contracts
- Skills and plugins as reusable generation units
- Multi-agent compatibility
- Artifact generation and export
- Sandboxed preview and review-before-handoff
- Clear repository agent guidance through `AGENTS.md`

## Dealix interpretation

Dealix should use these concepts to strengthen its own commercial surfaces:

- command room dashboards
- founder decision screens
- client proof packs
- landing pages
- proposal decks
- product prototypes

## Boundary

This integration is conceptual and operational. It does not copy upstream code or assets.

# AI Threat Model — نموذج تهديدات الذكاء الاصطناعي

## Purpose

A STRIDE-style threat model applied to the Dealix AI surfaces. The dominant data flow is: client chat or input → LLM gateway → tools/database → drafted output → human approval → external channel. Each hop is a threat surface.

## STRIDE applied

| STRIDE category | Example threat in Dealix |
|---|---|
| Spoofing | impersonated user input into an agent; impersonated supplier API response |
| Tampering | prompt or retrieval store altered by an unauthorized actor |
| Repudiation | agent action without an audit trail |
| Information disclosure | client data echoed in a model log; PII to non-vetted supplier |
| Denial of service | runaway prompt cost exhausts budget |
| Elevation of privilege | agent invokes a tool outside its permission scope |

|---|---|---|
| User input → gateway | LLM01 prompt injection, spoofing | input boundary, identity check |
| Gateway → model | tampering, supplier-side disclosure | pinned versions, [AI_SUPPLIER_POLICY.md](./AI_SUPPLIER_POLICY.md), [docs/06_llm_gateway/REDACTION_POLICY.md](../06_llm_gateway/REDACTION_POLICY.md) |

| Tool invocation | elevation of privilege, excessive agency | [docs/agents/AGENT_PERMISSIONS.md](../agents/AGENT_PERMISSIONS.md), [docs/agentic_operations/AGENT_TOOL_BOUNDARY.md](../agentic_operations/AGENT_TOOL_BOUNDARY.md) |

## Trust boundaries

- Any external content (chat input, scraped web, supplier output) is **data**, never **instructions**.
- The model is treated as an untrusted code generator: outputs are validated against schemas, not executed blindly.
- The approval queue is the last trust boundary before the public internet.

## Operations

1. Threat model is re-walked on any new surface (new tool, new channel, new supplier).
2. Each surface is mapped to at least one row in [AI_RISK_REGISTER.md](./AI_RISK_REGISTER.md).
3. Quarterly tabletop: run an adversarial prompt set against the gateway; document outcomes.

## Non-negotiables

- No tool call generated by a model fires without permission and schema validation.
- No external content is concatenated into a system prompt.
- No log line carries unredacted PII.

## Evidence

- Threat model file: `dealix-ops-private/ai_management/threat_model.md`.
- Tabletop results stored per quarter.

## Owner & cadence

## AR — ملخّص

نموذج تهديدات الذكاء الاصطناعي يطبّق STRIDE على تدفّق: مدخل → بوّابة LLM → أدوات/بيانات → مخرج → موافقة → قناة. كل سطح له تهديدات وضوابط. المحتوى الخارجي بيانات لا تعليمات، والنموذج مولّد غير موثوق، وطابور الموافقة هو الحدّ الأخير قبل الإنترنت. القيمة التقديرية ليست قيمة مُتحقَّقة.

# Automation Release Decision

## Purpose
Decide whether an automation or agent capability can be released.

## Decision Checklist
- Purpose clear.
- Inputs defined.
- Outputs defined.
- Data sensitivity classified.
- Approval class assigned.
- A3 actions blocked.
- External outputs require review.
- Logging exists.
- Evaluation exists.
- Rollback/disable path exists.

## Release Decision
Go / No-Go / Limited / Defer

## Rule
No automation moves from internal draft to external action without trust approval.

# AI Operating Rules
## Purpose
Define how AI can and cannot be used inside Dealix.
## Allowed Now
- draft internal summaries
- score leads internally
- generate first drafts
- suggest bottlenecks
- prepare reports for review
- generate checklists
## Requires Approval
- outbound messages
- proposal language
- public content
- client reports
- pricing recommendations
- client-specific conclusions
## Never Automatic
- sending external messages
- signing commitments
- refunding
- changing contracts
- publishing claims
- exporting sensitive data
## Rule
AI prepares. Sami approves critical moves.

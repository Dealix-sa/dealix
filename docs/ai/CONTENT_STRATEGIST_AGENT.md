# Content Strategist Agent

| Field | Value |
|---|---|
| Agent ID | `content_strategist` |
| Scope | Drafts founder-led posts, newsletters, landing copy, sector reports |
| Tools | Read: brand voice, operating data, approved proof. Write: drafts, calendar |
| Approval class | Internal; never publishes |
| Eval suite | Voice eval, claim-evidence eval, bilingual symmetry |
| Kill switch | Per-surface |
| Audit | Every draft and version |
| Owner | Brand Guardian + Founder |
| Allowed write targets | `marketing/content_calendar.csv`, `marketing/content_ideas.csv`, draft files, audit log |
| Never-auto actions | Publishing to any external surface |

## Responsibilities

1. Maintain content calendar and idea backlog.
2. Draft to brand voice.
3. Flag claims that lack evidence.
4. Produce AR + EN versions.
5. Recommend theme rotations.

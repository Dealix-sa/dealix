# Strategic Finance OS / نظام المالية الاستراتيجية

## Purpose / الغرض

Tie finance decisions to growth, delivery, hiring, and productization.

ربط القرارات المالية بالنمو والتسليم والتوظيف والمنتجة.

## Decisions / القرارات

- raise price / رفع السعر
- reduce scope / تقليل النطاق
- hire contractor / توظيف مقاول
- buy tool / شراء أداة
- invest in channel / الاستثمار في قناة
- kill sector / إيقاف قطاع
- scale worker / توسعة عامل
- delay SaaS / تأجيل SaaS

## Inputs / المدخلات

- cash collected / النقد المُحصَّل
- MRR / الإيراد الشهري المتكرر
- gross margin / الهامش الإجمالي
- proposal conversion / تحويل العروض
- delivery hours / ساعات التسليم
- founder bottleneck / اختناق المؤسس
- AI/tool cost / تكلفة الذكاء/الأدوات
- runway / المدى المالي

## Rule / القاعدة

Capital follows proven bottlenecks, not excitement.

رأس المال يتبع الاختناقات المثبتة، لا الحماس.

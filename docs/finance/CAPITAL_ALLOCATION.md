# Capital Allocation

> Where each SAR — and each founder hour — goes.
> Reviewed monthly. Decisions logged.

## The Seven Priorities (in order)

1. **Revenue generation** — outbound, sales, closing
2. **Delivery quality** — research, QA, evidence
3. **Trust & safety** — governance, evidence, audits
4. **Repeatability** — templates, playbooks
5. **Automation** — only what is repeated and stable
6. **Brand proof** — case studies, content with evidence
7. **Strategic optionality** — partnerships, longer-arc bets

The order matters. Priority 1 starves before priority 7 is funded.

## Cash Allocation Targets (pre-PMF)

| Category | % of monthly spend |
|----------|---------------------|
| Tools / infra (revenue + delivery) | 25–35% |
| Founder compensation | 30–40% |
| Trust / counsel / compliance | 5–10% |
| Content / brand production | 5–10% |
| Reserve contribution | 10–20% |
| Strategic optionality | ≤ 5% |

Anything above these caps requires a T3 decision.

## Monthly Capital Review

In the Monthly Strategy Review:

1. Pull `expenses.csv` for the month.
2. Categorise every line.
3. Compare to targets.
4. List one expense to challenge (cut, renegotiate, replace).
5. List one priority that is under-funded relative to its rank.

## Capex vs Opex

- Capex (one-off > 5,000 SAR): T3 decision.
- Recurring opex (any): T3 decision if > 2,000 SAR/month.

## Spending Anti-Patterns

- "Strategic" subscription with no measured output.
- Duplicate tools (two CRMs, two analytics tools, two LLM providers
  without a reason).
- Tools bought because of fear of missing out.
- Hiring before workflow repeatability.

## Capital Allocation Audit

Quarterly: every recurring expense must justify itself with:

- The priority it serves
- The output it produces
- The metric it moves

Anything without all three is cancelled.

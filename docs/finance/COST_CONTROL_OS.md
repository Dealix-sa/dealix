# Cost Control OS

## Purpose
Prevent tools, AI, ads, data, and contractors from growing faster than revenue.

## Track
- LLM spend
- data provider spend
- email tools
- hosting
- contractors
- ads
- software subscriptions

## Unit Metrics
- cost per lead discovered
- cost per enriched lead
- cost per approved outreach
- cost per positive reply
- cost per sample
- cost per proposal
- cost per paid client

## Rule
If cost rises without conversion, stop or reroute.

# Cost Control OS / نظام التحكم في التكاليف

## Purpose / الغرض

Prevent tools, AI, ads, data, and contractors from growing faster than revenue.

منع الأدوات والذكاء والإعلانات والبيانات والمقاولين من النمو أسرع من الإيراد.

## Track / تتبع

- LLM spend / إنفاق LLM
- data provider spend / إنفاق مزودي البيانات
- email tools / أدوات البريد
- hosting / الاستضافة
- contractors / المقاولون
- ads / الإعلانات
- software subscriptions / اشتراكات البرمجيات

## Unit Metrics / المقاييس الوحدية

- cost per lead discovered / تكلفة كل عميل مكتشف
- cost per enriched lead / تكلفة كل عميل مُثرى
- cost per approved outreach / تكلفة كل تواصل معتمد
- cost per positive reply / تكلفة كل رد إيجابي
- cost per sample / تكلفة كل عينة
- cost per proposal / تكلفة كل عرض
- cost per paid client / تكلفة كل عميل مدفوع

## Rule / القاعدة

If cost rises without conversion, stop or reroute.

إذا ارتفعت التكلفة بدون تحويل، توقف أو أعد التوجيه.

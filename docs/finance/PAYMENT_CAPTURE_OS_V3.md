# Payment Capture OS v3 / نظام تحصيل الدفع v3

## Purpose / الغرض

Turn proposals into payment, PO, or written approval.

تحويل العروض إلى دفع أو أمر شراء أو موافقة كتابية.

## Payment Paths / مسارات الدفع

- bank transfer / تحويل بنكي
- invoice / فاتورة
- payment link / رابط دفع
- purchase order / أمر شراء
- written approval / موافقة كتابية

## Follow-Up Cadence / إيقاع المتابعة

- **Day 1 / اليوم 1:** confirm received. / تأكيد الاستلام.
- **Day 3 / اليوم 3:** answer questions. / الإجابة على الأسئلة.
- **Day 7 / اليوم 7:** decision request. / طلب القرار.
- **Day 14 / اليوم 14:** close/recycle. / إغلاق/إعادة تدوير.

## Red Flags / الإشارات الحمراء

- wants delivery before start condition / يريد التسليم قبل شرط البدء
- vague approval / موافقة غامضة
- asks for guarantee / يطلب ضمانة
- pushes scope without payment / يدفع النطاق بدون دفع
- unclear decision maker / صانع قرار غير واضح

## Rule / القاعدة

Proposal is not revenue. Cash / PO / written approval is the gate.

العرض ليس إيرادًا. النقد / أمر الشراء / الموافقة الكتابية هي البوابة.

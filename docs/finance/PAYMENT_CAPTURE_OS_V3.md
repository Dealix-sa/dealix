# Payment Capture OS v3

## Purpose
Turn proposals into payment, PO, or written approval.

## Payment Paths
- bank transfer
- invoice
- payment link
- purchase order
- written approval

## Follow-Up Cadence
- Day 1: confirm received.
- Day 3: answer questions.
- Day 7: decision request.
- Day 14: close/recycle.

## Red Flags
- wants delivery before start condition
- vague approval
- asks for guarantee
- pushes scope without payment
- unclear decision maker

## Rule
Proposal is not revenue. Cash / PO / written approval is the gate.

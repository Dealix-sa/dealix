# Financial Dashboard

The one-page financial view, refreshed weekly.

## Lines
- Cash collected (week, month-to-date, quarter-to-date).
- Cash expected (this month, next month).
- MRR (current, Δ vs last month).
- AR outstanding by age (0–30, 31–60, 61+).
- Average deal size (last 90 days).
- Gross margin estimate (last 90 days).
- Refunds (this month, year-to-date).
- Runway estimate at current burn.

## Source files
- `dealix-ops-private/revenue/cash_collected.csv`
- `dealix-ops-private/revenue/mrr_tracker.csv`
- `dealix-ops-private/revenue/pipeline_value.csv`
- `dealix-ops-private/finance/cash_forecast.csv`

## Cadence
- Refreshed every Monday morning.
- Reviewed in the Weekly CEO Review.

## Rule
A weekly financial dashboard prevents almost every founder surprise. Do not skip a week.

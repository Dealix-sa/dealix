# Client Expansion OS / نظام توسعة العميل

## Purpose / الغرض

Convert successful delivery into retention, referral, proof, and expansion.

تحويل التسليم الناجح إلى احتفاظ وإحالة وإثبات وتوسعة.

## After Delivery / بعد التسليم

1. Request feedback. / طلب التغذية الراجعة.
2. Score client health. / تقييم صحة العميل.
3. Ask retainer if value is clear. / طلب الاشتراك الشهري إذا كانت القيمة واضحة.
4. Ask referral if trust is strong. / طلب الإحالة إذا كانت الثقة قوية.
5. Ask proof approval if appropriate. / طلب موافقة الإثبات إذا كان مناسبًا.
6. Log productization signals. / تسجيل إشارات المنتجة.

## Expansion Offers / عروض التوسعة

- Revenue Desk Retainer / اشتراك مكتب الإيرادات
- Monthly Sector Report / تقرير قطاعي شهري
- Weekly Opportunity Engine / محرك فرص أسبوعي
- Executive Command Center / مركز قيادة تنفيذي
- Partner/Channel Support / دعم الشركاء/القنوات

## Rule / القاعدة

Every delivery must create a next commercial path.

كل تسليم يجب أن يخلق مسارًا تجاريًا تاليًا.

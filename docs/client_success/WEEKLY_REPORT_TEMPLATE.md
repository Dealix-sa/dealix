# Weekly Report Template — Client Engagement

The standing weekly update Dealix sends every active client.

## Format
Short. One page. Sent every week on the same day and hour.

```
Client: <name>
Week of: YYYY-MM-DD
Prepared by: Dealix

## What we did this week
- ...
- ...
- ...

## What we are doing next week
- ...
- ...

## Numbers
- Leads delivered: <n>
- Messages sent (on client's behalf, if in scope): <n>
- Replies: <n>
- Meetings booked: <n>
- Deals closed (if visible to Dealix): <n>

## Decisions needed
- <decision> by <date> from <client contact>

## Risks and blockers
- ...
```

## Rules
- One page. Always.
- Numbers are required for retainer clients.
- Decisions block work — if a decision is missing for more than 7 days, escalate.

## Rule
A weekly report missed once is a slip. Missed twice is a relationship in trouble.

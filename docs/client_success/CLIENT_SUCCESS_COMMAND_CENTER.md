# Client Success Command Center

## Purpose
Turn delivery into retention, proof, referrals, and retainer revenue.

## Owner
Sami / Client Success owner.

## Review Cadence
Weekly per active client.

## Inputs
- delivery report
- client feedback
- usage signals
- meetings created
- additional needs
- objections

## Outputs
- client health score
- renewal recommendation
- upsell opportunity
- testimonial/case study request
- retainer offer

## Rules
- Ask for feedback after delivery.
- Identify next opportunity before closing project.
- No public case study without approval.
- Retainer offer only after value is clear.

## Metrics
- client health score
- feedback received
- case study approved
- retainer probability
- renewal conversion

## Evidence
- feedback.md
- health_score.md
- renewal.md
- case study approval

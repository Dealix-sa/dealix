# Offer-Market Fit — Experiment Template

One row per fit test. Fit tests are internal experiments and never
public claims.

## Worksheet

| Test ID | Offer ID | Sector | Hypothesis | Method | Sample size | Result | Evidence ref | Decision |
|---|---|---|---|---|---|---|---|---|
| _example_ | _rung_0..rung_4_ | _sector_ | _hypothesis_en_ | _outreach / interview / paid pilot_ | _N_ | _result_en (no guarantees)_ | _proof_ref_id_ | continue / fix / kill |

## Rules

- Sample sizes < 5 must mark every numeric result with `is_estimate=true`.
- Any "result" written into this worksheet that resembles "X% growth"
  or "Y SAR revenue" must cite a Proof Ledger event with payment proof.
- A "kill" decision is preferred to a vague "fix" — keep the worksheet
  honest.

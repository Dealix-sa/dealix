# Command Sprint Delivery OS

> أول عرض تجاري: Dealix Command Sprint = Revenue + Proof + Command + Governance Lite.

The Command Sprint is the 7-day first engagement. This doc is the delivery
playbook the founder follows after a company pays and a customer folder is
created (see [../01_go_to_market/TARGETING_TO_DELIVERY_HANDOFF.md](../01_go_to_market/TARGETING_TO_DELIVERY_HANDOFF.md)).

## Governance Lite

Every artifact carries `source / evidence / assumption / confidence /
recommendation / approval_required / next_action / owner / due_date`. No claim
ships without evidence; no recommendation ships without a confidence level.

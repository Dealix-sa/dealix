# مكتبة برومبتات البحث — Search Prompts (محكوم، قنوات عامة)

برومبتات بحث عن شركات من **مصادر عامة فقط**. كل برومبت مصمَّم ليوجّه نحو قنوات خضراء/صفراء ([SAUDI_DATA_SOURCE_CATALOG.md](../../ops/SAUDI_DATA_SOURCE_CATALOG.md)) — لا يطلب scraping ولا أرقاماً شخصية.

> قاعدة كل برومبت: «من مصادر عامة منشورة رسمياً فقط؛ لا تستخرج بيانات من مواقع تمنع ذلك؛ لا أرقام شخصية.»

## ب1 — اكتشاف شركات قطاع من قناة عامة

```
ابحث عن شركات {القطاع} في {المدينة} المدرَجة في مصادر عامة رسمية فقط
(دلائل غرف التجارة، بيانات حكومية مفتوحة، Google Places).
لكل شركة أعد: الاسم، القطاع، المدينة، قناة تواصل أعمال منشورة، ورابط المصدر العام.
لا تستخرج من مواقع تمنع ذلك. لا أرقام شخصية. لا تخمين.
```

## ب2 — التقاط فجوة عامة قابلة للإثبات

```
من الصفحات العامة الرسمية لـ {اسم الشركة} فقط، حدّد فجوة واحدة قابلة للإثبات
ضمن: واتساب غير منظَّم، متابعة ضعيفة، عرض غير مقنع، CRM غير مستخدَم، تقارير لا تتحول لقرار.
أرفق اقتباساً/رابطاً للمصدر العام. إن لم تجد دليلاً عاماً، قل "لا دليل عام".
```

## ب3 — تحقّق ملاءمة ICP

```
بناءً على بيانات عامة فقط، قيّم ملاءمة {اسم الشركة} لـ ICP:
قطاع B2B، حجم صغير/متوسط، فجوة عامة واضحة، قناة عامة متاحة، لا بيانات حسّاسة.
أعد نقاطاً 0–100 مع تبرير مختصر لكل بُعد.
```

## محظورات في كل برومبت

- لا أرقام شخصية، لا scraping خلف تسجيل دخول، لا مصادر حمراء.
- لا اختلاق بيانات؛ «لا دليل عام» جواب مقبول.

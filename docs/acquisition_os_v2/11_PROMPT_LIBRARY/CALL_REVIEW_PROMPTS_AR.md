# مكتبة برومبتات مراجعة المكالمات — Call Review Prompts

برومبتات لمراجعة المكالمات وتحسين الأداء، مع احترام PDPL (لا مراجعة تسجيل بلا إذن، لا PII في المخرَج).

> شرط: لا تُمرَّر تسجيلات بلا إذن صريح من الطرفين. أزِل أي PII قبل المراجعة.

## ب1 — تقييم التزام المكالمة بالحدود

```
راجع ملاحظات هذه المكالمة (بلا PII).
تحقّق من: تعريف المُرسِل، ذكر الفجوة العامة، طلب الإذن، عدم وعد بأرقام، عرض الانسحاب.
أعد قائمة بأي بند مفقود + اقتراح تصحيح بجملة.
```

## ب2 — استخلاص الخطوة التالية

```
من ملاحظات المكالمة، استخرج الخطوة التالية الواضحة وتاريخها وصاحبها.
إن لم تكن واضحة، اقترح خطوة منخفضة الالتزام (تشخيص مصغّر).
لا تخترع التزامات لم تُذكر.
```

## ب3 — اكتشاف فجوة جديدة من المكالمة

```
من سياق العميل في المكالمة، حدّد إن ظهرت فجوة إضافية ضمن الخمس
(واتساب، متابعة، عرض، CRM، تقارير).
اربطها بالعرض المناسب. إن لم تظهر فجوة جديدة، قل ذلك صراحةً.
```

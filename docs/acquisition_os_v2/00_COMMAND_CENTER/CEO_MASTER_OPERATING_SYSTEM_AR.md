## خطة 30 / 60 / 90 يوماً

### اليوم 1–30 — إثبات أول إيراد
- الهدف: تشخيص مدفوع واحد + Proof Pack مُسلَّم خلال 7 أيام.
- قطاع واحد مركّز (انظر **رُبريك اختيار القطاع** أدناه).
- لا أكثر من عرض واحد افتراضي مفعّل (Mini Audit → Sprint).
- KR مرجعي: [CEO_90_DAY_OKR_AR.md](../../commercial/operations/CEO_90_DAY_OKR_AR.md) OKR 1.

### اليوم 31–60 — تكرار النمط
- 3 تشخيصات مدفوعة + أول Sprint مكتمل.
- توثيق أول حالة كـ **«نموذج افتراضي/آمن للحالة»** ما لم يُذكر عميل حقيقي بموافقة.
- بناء قائمة منع التواصل (do-not-contact) فعلياً — [do_not_contact_schema.json](../08_COMPLIANCE_GOVERNANCE/do_not_contact_schema.json).

### اليوم 61–90 — أول دخل متكرر
- أول عميل Managed Ops شهري.
- مراجعة معدّلات الرد والانسحاب لكل قطاع؛ إسقاط القطاعات الضعيفة.
- قرار GO/expand واحد فقط مبني على أدلة، لا على حماس.

## مؤشرات الأداء (KPIs) — تقديرية وموسومة

| KPI | تعريف | ملاحظة |
|-----|--------|--------|
| ليدز مؤهّلة/يوم | شركات اجتازت السكورينج | هدف تشغيلي، ليس وعد بيع |
| لمسات مُخصّصة/يوم | رسائل قائمة على فجوة + بموافقة | لا إرسال جماعي |
| اجتماعات/أسبوع | مكالمات تأهيل أو عرض | — |
| عروض/أسبوع | one-pager أو proposal | — |
| **أحداث إثبات/أسبوع** | `payment_received` / `proof_pack_delivered` | **المؤشر الأهم** |
| معدّل الانسحاب | opt-out ÷ المرسَل | > 5% = أوقف وراجع النسخة |

> الأرقام أعلاه أهداف تشغيلية تقديرية. أي رقم نتيجة (تحويل، عائد) يُذكر كـ **«تقديري»** أو **«نمط case-safe»** — لا كضمان. الإحصائيات الخارجية (HubSpot/McKinsey) تُستشهد كـ **ادعاءات طرف ثالث باسم مصدرها** فقط.

## رُبريك اختيار القطاع / Sector-selection rubric

اختر القطاع بنقاط 1–5 لكل بُعد؛ اعمل على الأعلى أولاً:

1. **وضوح الفجوة** — هل المشكلة (متابعة ضعيفة، CRM غير مستخدم…) ظاهرة وقابلة للإثبات؟
2. **توفّر قنوات عامة** — هل توجد دلائل غرف تجارة/بيانات مفتوحة تكفي للسورسينج المحكوم؟ (انظر [SAUDI_DATA_SOURCE_CATALOG.md](../../ops/SAUDI_DATA_SOURCE_CATALOG.md))
3. **القدرة على الدفع** — متوسط حجم الصفقة يبرّر السلّم.
4. **سرعة الإثبات** — هل يمكن تسليم Proof خلال 7 أيام؟
5. **ملاءمة PDPL** — لا قطاعات تتطلب بيانات حسّاسة (صحية/مالية) بلا ترخيص.

القطاعات الأولية المرجعية: [SAUDI_SECTOR_TAXONOMY.md](../../02_saudi_positioning/SAUDI_SECTOR_TAXONOMY.md).

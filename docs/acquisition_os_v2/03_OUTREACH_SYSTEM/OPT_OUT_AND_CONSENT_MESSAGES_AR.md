# رسائل الموافقة والانسحاب — Opt-out & Consent Messages

نصوص جاهزة للموافقة والانسحاب عبر كل القنوات. هذا الملف هو المصدر المشترك؛ يشير إليه القسم 08 أيضاً ([08_COMPLIANCE_GOVERNANCE/OPT_OUT_AND_CONSENT_MESSAGES_AR.md](../08_COMPLIANCE_GOVERNANCE/OPT_OUT_AND_CONSENT_MESSAGES_AR.md)).

> القاعدة: **كل رسالة خارجية تحمل مخرج انسحاب.** كل انسحاب يُنفَّذ فوراً ويُسجَّل في قائمة المنع ([do_not_contact_schema.json](../08_COMPLIANCE_GOVERNANCE/do_not_contact_schema.json)).

## تذييل الانسحاب القياسي

| القناة | النص |
|--------|------|
| واتساب | «لإيقاف الرسائل في أي وقت، رد بكلمة: إيقاف.» |
| بريد | «للانسحاب من أي مراسلة لاحقة، رد بكلمة: إيقاف.» |
| صوتي/مكالمة | «إن لا يناسبك التواصل، أخبرني وأوقفه فوراً.» |

## تأكيد الانسحاب (يُرسَل مرة واحدة بعد الطلب)

```
تم إيقاف التواصل بناءً على طلبك. لن تصلك رسائل أخرى منّا. شكراً لوقتك.
```

(بعد هذا التأكيد: لا أي رسالة إضافية إطلاقاً.)

## طلب الموافقة قبل التوسّع في التواصل

```
هل تأذن أن أرسل لك ملخّص صفحة واحدة عن الفجوة وكيف تُعالَج؟
بلا أي التزام، ويمكنك إيقاف التواصل في أي وقت بكلمة: إيقاف.
```

## سجل الموافقة (PDPL)

لكل جهة سُجّل: تاريخ ومصدر الموافقة، القناة، نص الموافقة، وتاريخ أي انسحاب. لا PII حسّاس في السجل؛ تُجزَّأ المعرّفات. المرجع: [SAUDI_PDPL_AND_WHATSAPP_RULES_AR.md](../08_COMPLIANCE_GOVERNANCE/SAUDI_PDPL_AND_WHATSAPP_RULES_AR.md).

## كلمات الإيقاف المقبولة

«إيقاف» · «stop» · «الغاء» · «unsubscribe» — أي منها يُفعّل الانسحاب فوراً.

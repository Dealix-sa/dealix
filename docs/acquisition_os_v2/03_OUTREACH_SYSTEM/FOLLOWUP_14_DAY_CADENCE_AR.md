# كادنس المتابعة — 14 يوماً — 14-Day Follow-up Cadence

كادنس متابعة محترم، قائم على فجوة، يتوقّف فوراً عند الانسحاب أو عدم الاهتمام. مصمَّم ليكون **خفيفاً ومتباعداً** — لا مطاردة، لا إرسال جماعي.

## الجدول

| اليوم | القناة | المحتوى | شرط الإرسال |
|------:|--------|---------|--------------|
| 0 | الأنسب (بريد أعمال/واتساب opt-in) | افتتاحي قائم على فجوة | الشروط مستوفاة |
| 3 | نفس القناة | متابعة قصيرة + عرض الانسحاب | لا رد بعد |
| 7 | قناة بديلة (إن متاحة وعامة) | زاوية فجوة مختلفة | لا رد بعد |
| 11 | الأنسب | ملاحظة صوتية أو سطر قصير | لا رد بعد |
| 14 | الأنسب | رسالة إغلاق مهذّبة | لا رد بعد |

## رسالة الإغلاق (اليوم 14)

```
مرحباً {الاسم}،
لن أثقل عليك أكثر. إن أصبح التوقيت مناسباً لاحقاً، أنا موجود.
أتمنى لكم التوفيق. للانسحاب نهائياً، رد بإيقاف.
```

## قواعد صارمة

- **أقصى 5 لمسات** خلال 14 يوماً ثم توقّف.
- أي **إيقاف/عدم اهتمام** → إضافة فورية لقائمة المنع ([do_not_contact_schema.json](../08_COMPLIANCE_GOVERNANCE/do_not_contact_schema.json)) وإيقاف كل القنوات.
- لا تكرار الكادنس لنفس الحساب قبل 90 يوماً.
- معدّل انسحاب القطاع > 5% → أوقف الكادنس وراجع النسخة (انظر [ANTI_SPAM_POLICY_AR.md](../08_COMPLIANCE_GOVERNANCE/ANTI_SPAM_POLICY_AR.md)).

# رُبريك الجودة — Quality Rubric

أداة تقييم لكل رسالة/مكالمة قبل وبعد الإرسال. أي بند «فشل» يوقف الإرسال.

## بنود حاجزة (فشل أيٍّ منها = إيقاف)

| # | البند | تحقّق |
|---|-------|-------|
| 1 | فجوة عامة مُثبتة من مصدر موثّق | ☐ |
| 2 | لا أرقام شخصية / مصدر أخضر-أصفر فقط | ☐ |
| 3 | تعريف المُرسِل واضح | ☐ |
| 4 | مخرج انسحاب موجود | ☐ |
| 5 | لا وعد مبيعات / لا «نضمن» | ☐ |
| 6 | أي إحصائية خارجية باسم مصدرها | ☐ |
| 7 | ليس في قائمة منع التواصل | ☐ |
| 8 | ضمن ساعات الهدوء + opt-in حيث يلزم | ☐ |

## بنود الجودة (تقييم 1–5)

| البند | الوصف |
|-------|--------|
| التخصيص | قائمة على فجوة فعلية لا قالب عام |
| الوضوح | لغة تنفيذية واقعية بلا مبالغة |
| الإيجاز | تصل للنقطة بسرعة |
| الخطوة التالية | واضحة ومنخفضة الالتزام |

## القاعدة

- مجموع حاجز < 8/8 → **لا إرسال**.
- جودة < 3/5 في أي بند → أعِد الصياغة.
- المراجعة الأسبوعية: عيّنة 10 رسائل ضد هذا الرُّبريك + معدّل الانسحاب ([ANTI_SPAM_POLICY_AR.md](../08_COMPLIANCE_GOVERNANCE/ANTI_SPAM_POLICY_AR.md)).

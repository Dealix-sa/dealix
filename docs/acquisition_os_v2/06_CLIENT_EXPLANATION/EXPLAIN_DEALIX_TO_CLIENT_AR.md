## ما هي Dealix بجملة

Dealix ترتّب بياناتك، تحدّد الفرص، تجهّز مسودات آمنة، وتخرج تقريراً تنفيذياً مربوطاً بقرار — **برمجيات + خدمة + تشغيل AI محكوم**، لا استشارة عابرة ولا أتمتة إرسال جماعي.

## قبل / بعد (ملموس)

| المجال | قبل | بعد (تقديري، يعتمد على تنفيذكم) |
|--------|-----|--------------------------------|
| استقبال الطلبات | واتساب غير منظَّم، ردود متأخرة | استقبال محكوم بقوالب opt-in |
| المتابعة | فرص تسقط بلا أثر | كادنس متابعة 14 يوماً منضبط |
| البيانات | جداول مبعثرة، تكرار | بيانات مرتّبة + dedupe + سكورينج |
| العرض | صفحة خدمات عامة | عرض قائم على فجوة + one-pager |
| التقارير | أرقام بلا قرار | تقرير تنفيذي يحدّد الخطوة التالية |

> «بعد» يصف نمطاً تشغيلياً، ليس وعداً برقم بيع.

## ما لا نفعله (نقولها بوضوح)

- لا scraping، لا قوائم مشتراة، لا حصاد أرقام شخصية.
- لا إرسال خارجي نيابة عنكم بلا موافقتكم الصريحة.
- لا أتمتة واتساب باردة، لا أتمتة LinkedIn.
- لا ضمان مبيعات — نقدّم **فرصاً مُثبتة بأدلة**.

# رسائل جاهزة لكل قطاع — Sector Messages (قائمة على فجوة، تحترم الموافقة)

نماذج رسائل افتتاحية لكل قطاع، **قائمة على فجوة عامة** ومع مخرج انسحاب. كل نموذج يُخصّص بفجوة فعلية مُثبتة قبل الإرسال — لا يُرسَل كقالب عام.

> القطاعات من [SAUDI_SECTOR_TAXONOMY.md](../../02_saudi_positioning/SAUDI_SECTOR_TAXONOMY.md). حزمة الأبطال المرجعية: [MARKET_INTELLIGENCE_SALES_CHAMPION_PACK_AR.md](../../commercial/MARKET_INTELLIGENCE_SALES_CHAMPION_PACK_AR.md).

## عقار

```
السلام عليكم {الاسم}، {المُرسِل} من Dealix.
لاحظت من صفحتكم العامة أن طلبات العملاء على واتساب تصل بلا تصنيف،
ما قد يؤخّر متابعة العروض. نساعد مكاتب العقار على تنظيم هذا بمخرجات ملموسة.
أرسل لك ملخّص صفحة واحدة؟ للإيقاف رد بإيقاف.
```

## مقاولات

```
السلام عليكم {الاسم}، {المُرسِل} من Dealix.
لاحظت أن استفسارات المشاريع تأتي عبر قنوات متعددة بلا تتبّع موحّد.
نرتّب ذلك ونبرز الفرص الأهم بتقرير تنفيذي. ملخّص قصير يناسبك؟
للإيقاف رد بإيقاف.
```

## تدريب

```
السلام عليكم {الاسم}، {المُرسِل} من Dealix.
لاحظت أن متابعة المسجّلين المحتملين تنقطع بعد أول رد.
نضبط كادنس متابعة منظّم لمراكز التدريب. أرسل ملخّص الفجوة؟
للإيقاف رد بإيقاف.
```

## عيادات (بلا بيانات حسّاسة)

```
السلام عليكم {الاسم}، {المُرسِل} من Dealix.
لاحظت أن حجوزات الاستفسار على القنوات العامة بلا تأكيد منظّم.
ننظّم الاستقبال دون أي بيانات صحية حسّاسة، التزاماً بـ PDPL.
ملخّص قصير يناسبك؟ للإيقاف رد بإيقاف.
```

## وكالات / خدمات

```
السلام عليكم {الاسم}، {المُرسِل} من Dealix.
لاحظت أن صفحة خدماتكم عامة بلا إبراز للفجوة التي تحلّونها.
نعيد صياغة العرض بمخرجات ملموسة. أرسل ملخّص صفحة واحدة؟
للإيقاف رد بإيقاف.
```

## قواعد الاستخدام

- خصّص `{الفجوة}` بدليل عام فعلي قبل الإرسال.
- واتساب = opt-in فقط (القسم 03).
- لا وعود بأرقام؛ «فرص مُثبتة بأدلة».
- تحقّق من قائمة المنع أولاً.

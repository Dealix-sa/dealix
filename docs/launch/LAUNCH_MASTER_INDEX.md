## EN — Known gaps

Carried honestly into the launch. None block the first paid pilot except where
the readiness scorecard marks them blocking.

- **G1 caveat (delivery import cascade).** A Python import-cascade pulls the full
  app stack into the canonical OS modules. It must be fixed — ship the
  dependencies or decouple the imports — before a delivery operator runs the
  Sprint on a clean box. Until then, pin the delivery environment. Source:
  [`DELIVERY_DRYRUN_REPORT.md`](DELIVERY_DRYRUN_REPORT.md).
- **G3 caveat (proof score is completeness only).** The Proof Pack score
  measures section completeness, not evidence strength. A 100 / `case_candidate`
  pack is not verified evidence and must never be shown to a customer as
  verified proof.
- **Legacy-narrative backlog.** 61 legacy files still carry pre-standard product
  language. They are flagged for a scheduled cleanup; launch-path assets are
  already on-doctrine, so this is non-blocking. Source:
  [`LAUNCH_READINESS_GATES.md`](LAUNCH_READINESS_GATES.md) gate G4.

# Dealix X5 Action Registry

X5 adds an action registry, approval inbox, audit log, report output, and web page foundation.

## Run

```bash
python scripts/leadership/x5_run.py
python scripts/leadership/x5_snapshot.py
python -m pytest -q tests/test_x5.py
```

## Output

```text
reports/leadership/x5/latest.json
reports/leadership/x5/latest.md
apps/web/lib/x5-snapshot.ts
apps/web/app/x5/page.tsx
```

## Safety

X5 creates review items and audit entries. Actions start as ready_for_review and are not automatic.

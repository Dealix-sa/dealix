# Founder Start Here / ابدأ هنا أيها المؤسس

> **Purpose (EN):** The first page a founder or operator should read.
>
> **الغرض (AR):** أول صفحة يجب أن يقرأها المؤسس أو المشغّل.

_Layer: V9 — Strategic Moat & Enterprise Readiness OS_

## In One Minute / في دقيقة

- Dealix prepares, analyzes, drafts, ranks, and recommends.
- You — the founder — review, approve, send manually, sell, sign, and decide.
- The system never sends anything externally on its own.

## Your First Hour / ساعتك الأولى

1. Read `docs/00_MASTER_INDEX.md` to see every system.
2. Read `docs/03_DAILY_OPERATING_GUIDE.md` for the daily routine.
3. Run the V9 verifiers to confirm the repo is healthy:
   ```bash
   python scripts/strategic_moat_verify.py
   python scripts/enterprise_readiness_verify.py
   python scripts/trust_center_verify.py
   ```
4. Review `docs/05_GO_NO_GO_MASTER.md` before doing anything customer-facing.

## What To Never Do / ما يجب ألا تفعله أبدًا

- Never let the system send email/WhatsApp/LinkedIn on its own.
- Never commit secrets or API keys.
- Never claim certifications or ROI you cannot prove.
- Never scrape or auto-submit forms.

---

## Operating Boundaries / حدود التشغيل

**AI prepares; the founder approves and sends.** Founder approval remains required.

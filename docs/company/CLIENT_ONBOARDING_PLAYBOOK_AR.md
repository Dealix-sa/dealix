# دليل استقبال العملاء

## ما قبل التوقيع

1. Discovery call.
2. Diagnostic Sprint.
3. عرض Pilot.
4. توقيع اتفاقية Pilot وDPA.

## أول أسبوع

1. Kickoff meeting مع صانع القرار.
2. تعيين مدير المشروع.
3. قائمة بالوصول المطلوب.
4. تحديد baseline.

## أول شهر

1. بناء النظام الصغير.
2. اختبار مع 1-3 مستخدمين.
3. جمع feedback.
4. تقرير نجاح Pilot.

## النجاح

- العميل يرى نتيجة واضحة.
- النظام مستقر.
- الفريق يستخدمه.

## الانتقال للاشتراك

1. مراجعة النطاق الجديد.
2. تحديث SLA.
3. توقيع اتفاقية الاشتراك.
4. بدء التسليم الشهري.

# Dealix Enterprise Company OS v3 / نظام تشغيل شركة Dealix v3

## Purpose / الغرض

Turn Dealix into a 24/7 Saudi B2B revenue, delivery, trust, and intelligence company operating system.

تحويل Dealix إلى نظام تشغيل شركة سعودية للإيرادات والتسليم والثقة والذكاء التشغيلي يعمل على مدار الساعة.

## CEO Principle / مبدأ الرئيس التنفيذي

The company must produce measurable commercial movement every day.

الشركة يجب أن تنتج حركة تجارية قابلة للقياس كل يوم.

## Company Loops / دورات الشركة

### 1. Distribution Loop / دورة التوزيع
Market → leads → scoring → approval → outreach → replies.

السوق ← العملاء المحتملون ← التقييم ← الموافقة ← التواصل ← الردود.

### 2. Conversion Loop / دورة التحويل
Replies → samples → proposals → payment / PO / written approval.

الردود ← العينات ← العروض ← الدفع / أمر الشراء / الموافقة الكتابية.

### 3. Delivery Loop / دورة التسليم
Payment → intake → delivery → QA → handoff → feedback.

الدفع ← الاستلام ← التسليم ← ضمان الجودة ← التسليم النهائي ← التغذية الراجعة.

### 4. Retention Loop / دورة الاحتفاظ
Feedback → health score → retainer ask → renewal → expansion.

التغذية الراجعة ← مؤشر صحة العميل ← طلب الاشتراك الشهري ← التجديد ← التوسع.

### 5. Proof Loop / دورة الإثبات
Delivery → approved proof → content → inbound → referrals.

التسليم ← إثبات معتمد ← محتوى ← طلبات واردة ← إحالات.

### 6. Productization Loop / دورة المنتجة
Repeated workflow → template → automation → product module.

سير عمل متكرر ← قالب ← أتمتة ← وحدة منتج.

### 7. Trust Loop / دورة الثقة
AI output → policy eval → approval → audit → incident learning.

مخرج الذكاء ← تقييم السياسة ← الموافقة ← التدقيق ← التعلم من الحوادث.

### 8. Finance Loop / دورة المال
Proposal → payment → margin → runway → pricing decision.

العرض ← الدفع ← الهامش ← المدى المالي ← قرار التسعير.

## North Star / النجم القطبي

Cash collected from qualified Saudi B2B revenue operations.

النقد المُحصَّل من عمليات إيرادات B2B السعودية المؤهلة.

## Daily CEO Output / المخرج اليومي للرئيس التنفيذي

- one top action / إجراء واحد أولوي
- one commercial movement / حركة تجارية واحدة
- one updated queue / طابور واحد مُحدَّث
- one resolved bottleneck / اختناق واحد محلول

## Weekly CEO Output / المخرج الأسبوعي للرئيس التنفيذي

- channel decision / قرار قناة
- sector decision / قرار قطاع
- pricing decision / قرار تسعير
- productization decision / قرار منتجة
- trust review / مراجعة ثقة
- cash review / مراجعة نقدية

## Rule / القاعدة

Every system must serve one of: Cash, proof, retention, trust, learning, or founder leverage.

كل نظام يجب أن يخدم واحدة من: النقد، الإثبات، الاحتفاظ، الثقة، التعلم، أو رافعة المؤسس.

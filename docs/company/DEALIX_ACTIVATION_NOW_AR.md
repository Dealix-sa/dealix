## 4) خريطة 30 / 60 / 90 + بوابات القرار

**اليوم 0–30 — التشغيل الحقيقي (Activate):**
- عبّئ `data/warm_list.csv` و`agency_accounts_seed.csv` بجهات حقيقية من شبكتك (مصدر = CRM أو مقدمة شخصية، **لا اختراع**).
- شغّل الحلقة اليومية (القسم 5) → 5 مسوّدات/يوم جاهزة للموافقة.
- أطلق التشخيص المجاني علناً (LinkedIn + الشبكة الدافئة).
- **بوابة:** ≥ 1 فاتورة مدفوعة + ≥ 1 Proof Pack مُسلّم قبل توسيع العروض.

**اليوم 31–60 — الإثبات والتكرار (Prove):**
- حوّل عملاء Sprint إلى Pack/Retainer عبر الإثبات.
- انشر ملخّصاً آمناً لحالة واحدة (case-safe، بموافقة العميل).
- **بوابة الإيقاف:** إذا الإيراد < 25,000 ريال بنهاية اليوم 60 → أوقف بناء عروض جديدة وركّز على البيع (إعادة تفعيل الشبكة الدافئة، إيقاع المحتوى، اختبار الأسعار).

**اليوم 61–90 — التوسيع (Scale):**
- ثبّت ≥ 3 عقود إدارة.
- إذا الإيراد ≥ 40,000 ريال + 3 عقود → فعّل Wave 3 (حوكمة المؤسسات: تصدير أدلة PDF، توحيد trust_os).
- **حارس الوقت:** إذا وقت المؤسس/Sprint > 5 ساعات بعد العميل الخامس → أوقف بيع Sprints الجديدة وادفع أتمتة Sprint لأعلى القائمة.

## 5) الإيقاع التشغيلي — الأوامر التي تُنتج مخرجات حقيقية الآن

هذه ليست نظرية — جرّبناها في دفعة التفعيل هذه وأنتجت ملفات فعلية:

```bash
# مرة واحدة: اضبط اسمك ليظهر في كل مسوّدة تواصل (بدل الاسم الافتراضي)
export DEALIX_FOUNDER_NAME="بسام"
export DEALIX_FOUNDER_NAME_EN="Bassam"

# 1) عبّئ جهاتك الحقيقية
cp data/warm_list.csv.template data/warm_list.csv   # ثم عبّئها (CRM/مقدمة شخصية فقط)

# 2) ولّد مسوّدات تواصل ثنائية اللغة مؤهّلة مسبقاً (لا إرسال خارجي)
python scripts/warm_list_outreach.py
#   → data/outreach/warm_list_drafts.md  (انسخ 5/يوم بعد المراجعة)

# 3) الموجز اليومي (ليدز منتظرة، احتكاك، تجديدات، موافقات، توصيات)
python scripts/dealix_pm_daily.py
#   → اطبعه أو احفظه في data/daily_brief/YYYY-MM-DD.md

# 4) (عند ربط الـAPI) اليوم التجاري الكامل
DEALIX_API_BASE=... DEALIX_ADMIN_API_KEY=... python scripts/run_dealix_daily_ops.py
```

أمثلة مخرجات مولّدة في هذه الدفعة: `data/outreach/warm_list_drafts.SAMPLE.md` (مسوّدة مؤهّلة accept score=100) · `data/daily_brief/2026-06-07.md` (موجز يوم صفر).

**الإيقاع الأسبوعي (الأحد):** `bash scripts/founder_weekly_loop.sh` — بوابات + تقرير تعلّم.

# استراتيجية توجيه مزودي الذكاء الاصطناعي

## المبدأ

لا نعتمد على مزود واحد. نوجه الطلبات حسب الجودة والتكلفة والتوفر.

## المستويات

### Tier 1 — المهام الحرجة
- النماذج: OpenAI GPT-4o / Anthropic Claude 3.5 Sonnet
- الاستخدام: عروض، محتوى للعميل، تحليل قانوني/مالي
- مراجعة بشرية: إلزامية

### Tier 2 — المهام التشغيلية
- النماذج: GPT-4o-mini / Claude Haiku
- الاستخدام: تصنيف ردود، مسودات أولية، ملخصات
- مراجعة بشرية: spot-check

### Tier 3 — المهام الداخلية
- النماذج: نماذج محلية / open-source
- الاستخدام: بحث داخلي، توليد تقارير لا تخرج للعميل

## معايير التوجيه

1. **جودة** — المهام التي تخرج للعميل → Tier 1.
2. **تكلفة** — المهام المتكررة عالية الحجم → Tier 2/3.
3. **سرعة** — الردود التفاعلية → Tier 2.
4. **خصوصية** — بيانات حساسة → Tier 3 محلي أو مزود معتمد.

## السلامة

- لا نرسل بيانات العملاء لمزود غير موثوق.
- لا نستخدم مزودًا بدون اتفاقية معالجة بيانات.
- نسجل أي إخفاق في التوجيه.

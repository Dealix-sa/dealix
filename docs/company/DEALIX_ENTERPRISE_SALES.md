# Dealix Enterprise Sales

Enterprise = SAR 25,000+ engagements, typically AI Governance Review or
multi-quarter Custom AI Service Setup contracts.

## Deal stages

| Stage | Definition | Required artifact |
|---|---|---|
| 1. Qualified | Source passport signed, financial owner identified | Source passport |
| 2. Discovery | 2 calls, sponsor map built | Sponsor map deck |
| 3. Diagnostic | Free or paid AI Ops Diagnostic delivered | Diagnostic Proof Pack |
| 4. Governance Review SOW | Scope + price + timeline signed | SOW + DPA |
| 5. Pilot | 30-day audit + recommendations | Pilot Proof Pack |
| 6. Custom Setup | Build + integration | Setup acceptance doc |
| 7. Operate | Retainer + quarterly review | Quarterly Proof Pack |

## AI Governance Review SOW (Stage 4 template)

Required sections:
- Scope (which AI systems audited)
- Source Passport (which data sources approved)
- Eval suites to run
- Risk register
- Deliverables (Proof Pack + Capital Asset list)
- Acceptance criteria
- Price (SAR 25,000 – 50,000)
- Term (typically 4–8 weeks)

## Security questionnaire response pack

Located under `docs/security/`. Includes:
- PDPL compliance statement
- Data residency (Saudi Arabia preferred)
- Encryption at rest + in transit
- Incident response playbook
- DPA template
- Subprocessor list

## MSA (Master Service Agreement)

Template: `docs/legal/MSA_TEMPLATE.md` (to be added when first enterprise
deal closes; currently fallback is a single-engagement SOW).

## Doctrine

- No promised outcomes. All commitments are work products + capability
  delivered, not revenue guarantees.
- Approval_center gates every external comms send to enterprise contacts.

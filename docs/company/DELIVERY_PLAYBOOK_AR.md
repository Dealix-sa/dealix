# دليل التسليم

## مراحل التسليم

### 1. Diagnostic Sprint (أسبوع 1)

- مقابلة Discovery.
- جمع بيانات baseline.
- تحديد ألم واحد.
- تقديم تقرير صفحة واحدة.

### 2. Pilot (أسبوع 2-3)

- بناء نظام صغير.
- ربط مصدر بيانات واحد.
- إنشاء Dashboard.
- اختبار مع المستخدم.

### 3. Production Rollout (أسبوع 4+)

- توسيع النظاق.
- تدريب الفريق.
- مراقبة الأداء.
- تقرير أسبوعي.

## ما نقيسه

- الوقت المستغرق في المهام اليدوية.
- وقت الاستجابة.
- معدل التحويل.
- SLA compliance.

## أدوات

- FastAPI backend
- PostgreSQL
- Redis
- Next.js frontend
- Workflow engine داخلي

## لا نفعل

- لا نعد بنتائج مضمونة.
- لا نجمع بيانات زائدة.
- لا نسلم بدون مراجعة بشرية.

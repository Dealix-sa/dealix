# Dealix Autonomous Enterprise OS

Dealix is positioned as an AI-native operating company that runs its
own revenue motion through:

- a founder-controlled console
- a token-gated internal API
- policy-as-code
- a registered agent set
- a release-blocking eval gate
- a private ops runtime
- a master verifier
- a CI workflow that builds the frontend

It is intentionally NOT autonomous to the point of unattended external
action. The founder approves every external move; the AI prepares,
the workers execute, the trust plane audits.

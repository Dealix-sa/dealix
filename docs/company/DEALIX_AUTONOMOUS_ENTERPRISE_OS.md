# Dealix Autonomous Enterprise OS

"Autonomous" here means: each loop in the business runs without a human
asking for it. It does NOT mean: each loop runs without a human approving
external impact.

## Loops that run autonomously

* Lead research → lead intelligence record
* Lead scoring
* Reply classification
* Trust flag detection
* Worker heartbeats + scorecard refresh
* Sovereign readiness refresh

## Loops that require founder approval

* Outreach send
* Sample send
* Proposal send
* Invoice send
* Payment follow-up
* Public proof publishing
* Pricing change
* Contract / payment terms change
* Sensitive data export

## Why this split exists

Because external impact is where Dealix earns or loses reputation. The
internal loops above are recoverable; the external ones are not.

# الأتمتة المسموح بها | Allowed Automations

> **AR:** يعدّد هذا المستند ما يُسمح للأتمتة بإنتاجه. القاسم المشترك أنها جميعًا **تنتج مصنوعات داخلية** لا أفعالًا خارجية: مسودّات وتقارير وتشخيصات وتقويمات وفحوصات يراجعها المؤسس قبل أي استخدام.
>
> **EN:** This document lists what automation may produce. Common thread: they all generate *internal artifacts*, not external actions — drafts, reports, diagnostics, calendars, and checks that the founder reviews before any use.

## القائمة المسموح بها | Allowed List

| العملية Operation | الناتج Output | ملاحظة Note |
|---|---|---|
| Generate drafts | مسودّات رسائل/عروض / message & proposal drafts | تُرسَل يدويًا فقط / sent manually only |
| Score drafts | درجات جودة / quality scores | للترتيب الداخلي / internal ranking |
| Rank drafts | ترتيب المسودّات / ranked drafts | اقتراح أولوية / priority suggestion |
| Generate reports | تقارير داخلية / internal reports | للمراجعة / for review |
| Generate proposals | مقترحات قيمة / value proposals | مسودّة للاعتماد / draft for approval |
| Generate diagnostics | تشخيصات / diagnostics | تحليل لا فعل / analysis not action |
| Generate social calendar | تقويم محتوى / content calendar | نشر يدوي / manual posting |
| Run checks | فحوصات/تحقق / checks & validations | لا أثر خارجي / no external effect |
| Produce artifacts | مصنوعات قابلة للتدقيق / auditable artifacts | تبقى داخلية / stay internal |

## لماذا هذه مسموحة | Why These Are Allowed

- لا تتجاوز حدود النظام؛ مخرجاتها مصنوعات يراجعها الإنسان. / They stay inside the system; outputs are human-reviewed artifacts.
- لا تتصل بطرف خارجي ولا تُرسِل ولا تقدّم نموذجًا. / They contact no external party, send nothing, submit no form.
- قابلة للتدقيق: كل ناتج له مصدر وطابع زمني. / Auditable: each output has a source and timestamp.

## شروط عامة | General Conditions

- كل ناتج يُوسَم كـ **مسودّة** حتى يعتمده المؤسس. / Every output is tagged *draft* until founder approval.
- لا يتضمّن الناتج أسرارًا أو مفاتيح API. / Output contains no secrets or API keys.
- لا ادعاءات غير مثبتة ولا عائد مضمون في أي مصنوع. / No unproven claims or guaranteed ROI in any artifact.

## الحدّ الفاصل | The Hard Line

> توليد المحتوى مسموح. **إرساله أو تقديمه أو إطلاقه ممنوع آليًا** — انظر `02_FORBIDDEN_AUTOMATIONS.md`.
> Generating content is allowed. *Sending, submitting, or launching it automatically is forbidden* — see `02_FORBIDDEN_AUTOMATIONS.md`.

## أمثلة عملية | Practical Examples

- مسودّة بريد متابعة لعميل + ترتيبها بالأولوية، ثم إرسال يدوي. / Draft a follow-up email + rank it, then send manually.
- تقرير تشخيص أسبوعي عن الأنبوب للمراجعة. / Weekly pipeline diagnostic report for review.
- تقويم محتوى اجتماعي شهري كمسودّة للنشر اليدوي. / Monthly social content calendar as a draft for manual posting.

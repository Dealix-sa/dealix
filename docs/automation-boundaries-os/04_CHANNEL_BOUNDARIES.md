# حدود القنوات | Channel Boundaries

> **AR:** يحدّد هذا المستند ما تستطيع الأتمتة فعله لكل قناة تواصل على حدة. القاعدة موحّدة عبر القنوات: توليد المسودّة مسموح، والإرسال أو التقديم يدوي وبموافقة المؤسس. لا قناة تُستثنى من هذا المبدأ.
>
> **EN:** This document defines what automation may do per communication channel. The rule is uniform: drafting is allowed, sending/submitting is manual and founder-approved. No channel is exempt.

## جدول القنوات | Channel Matrix

| القناة Channel | مسموح Allowed | ممنوع Forbidden |
|---|---|---|
| البريد Email | توليد مسودّة، تقييم، ترتيب / draft, score, rank | إرسال آلي / auto-send |
| واتساب WhatsApp | صياغة رسالة مقترحة / propose message text | إرسال آلي / auto-send |
| لينكدإن LinkedIn | مسودّة منشور/رسالة / draft post or message | أتمتة النشر أو التواصل / automate posting/outreach |
| الهاتف Phone | تحضير نقاط الاتصال / prepare call notes | اتصال آلي / auto-dial |
| نموذج ويب Web Form | تعبئة مسودّة للمراجعة / draft fill for review | تقديم آلي / auto-submit |

## تفصيل القنوات | Channel Details

### البريد | Email
- يُولّد مسودّة، يقيّمها، يرتّبها. النسخ واللصق والإرسال يدوي. / Drafts, scores, ranks. Copy, paste, send are manual.

### واتساب | WhatsApp
- يقترح نص رسالة مع السياق. لا تكامل إرسال آلي. / Proposes message text with context. No auto-send integration.

### لينكدإن | LinkedIn
- يصوغ مسودّات فقط. **لا أتمتة** للتواصل أو الطلبات (يخالف شروط المنصّة). / Drafts only. *No automation* of outreach or requests (violates platform terms).

### الهاتف | Phone
- يحضّر نقاط الحديث وملخّص العميل. الاتصال بشري. / Prepares talking points and client summary. Calling is human.

### نموذج ويب | Web Form
- يجهّز محتوى التعبئة كمسودّة. التقديم يدوي بعد المراجعة. / Prepares fill content as a draft. Submission is manual after review.

## قاعدة موحّدة | Unified Rule

> عبر كل القنوات: **AI prepares, Founder approves, Manual action only, No external sending.**
> Across all channels: AI prepares, Founder approves, Manual action only, No external sending.

# Daily Media Routine — الإيقاع الإعلامي اليومي

> Cross-link: [Content Calendar](03_30_DAY_CONTENT_CALENDAR.md), [Social Metrics](14_SOCIAL_METRICS.md), [Brand Voice](01_BRAND_VOICE.md).

## English — The Routine

A short, repeatable daily loop. AI drafts; the founder edits and publishes manually. No scheduling tools with stored credentials.

| Block | Time box | Action |
|---|---|---|
| Morning draft | 15 min | Pull today's calendar entry; AI drafts AR + EN; founder edits |
| Claim check | 5 min | Run forbidden-claim check from [Brand Voice](01_BRAND_VOICE.md); add disclosure if needed |
| Publish | 5 min | Founder publishes manually to the day's platform |
| Engage | 15 min | Reply to comments and DMs in voice; no automation |
| Log | 5 min | Enter actuals manually in [Social Metrics](14_SOCIAL_METRICS.md) |

## العربية — الإيقاع

حلقة يومية قصيرة قابلة للتكرار. الذكاء يصيغ؛ المؤسس يحرّر وينشر يدويًا. لا أدوات جدولة ببيانات اعتماد مخزّنة.

| المرحلة | المدة | الإجراء |
|---|---|---|
| مسودة الصباح | 15 د | سحب مدخل التقويم؛ الذكاء يصيغ عربي + إنجليزي؛ المؤسس يحرّر |
| فحص الادعاء | 5 د | تشغيل فحص الادعاء المحظور من [صوت العلامة](01_BRAND_VOICE.md)؛ إضافة الإفصاح عند اللزوم |
| النشر | 5 د | المؤسس ينشر يدويًا على منصة اليوم |
| التفاعل | 15 د | الردّ على التعليقات والرسائل بالنبرة؛ بلا أتمتة |
| التسجيل | 5 د | إدخال الأرقام الفعلية يدويًا في [المقاييس](14_SOCIAL_METRICS.md) |

## English — Weekly Add-Ons

- **Monday:** plan the week's pillar mix against [02](02_CONTENT_PILLARS.md).
- **Thursday:** batch-record short video scripts ([07](07_TIKTOK_SHORTS_OS.md), [08](08_YOUTUBE_SHORTS_OS.md)).
- **Sunday:** review metrics trend; adjust next week. No metric invented.

## العربية — إضافات أسبوعية

- **الاثنين:** خطّط مزيج ركائز الأسبوع مقابل [02](02_CONTENT_PILLARS.md).
- **الخميس:** صوّر دفعة نصوص الفيديو القصير ([07](07_TIKTOK_SHORTS_OS.md)، [08](08_YOUTUBE_SHORTS_OS.md)).
- **الأحد:** راجع اتجاه المقاييس؛ اضبط الأسبوع القادم. بلا مقياس مُختلق.

## English — Guardrails

If a post fails the claim check, it does not publish until fixed. If a topic is trust-related, the disclosure is mandatory. Engagement never includes auto-replies or bulk messaging.

## العربية — الضوابط

إن فشل المنشور في فحص الادعاء لا يُنشر حتى يُصحَّح. إن تعلّق الموضوع بالثقة فالإفصاح إلزامي. التفاعل لا يتضمّن ردودًا آلية أو رسائل جماعية.

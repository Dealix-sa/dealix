# Daily Media Routine

| When | Action |
|------|--------|
| Morning | Pick today's item from the generated calendar; adapt to your voice. |
| Morning | Publish **manually** to the chosen platform. |
| Midday | Engage: 3–5 substantive comments on relevant posts. |
| Evening | Log results into the manual social-metrics template. |

## Weekly
- Regenerate the 30-day calendar plan.
- Review top-performing pillars/verticals; double down manually.

## Rules
No schedulers, no bots, no auto-publish. The plan is generated; the human posts.

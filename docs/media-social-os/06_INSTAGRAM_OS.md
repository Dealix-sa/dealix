# Instagram OS

## Formats
- Short Reels (15–30s) — one workflow pain → one fix.
- Carousels — before/after diagrams.

## Reel script pattern
- Hook (AR + EN): the pain in one line.
- Body: the operating fix; "AI drafts, you approve".
- CTA: "Book an AI Workflow Audit."

## Visual direction
Clean, dark/professional, brand palette (navy/gold). No stocky hype imagery.

## Rules
No exaggerated claims; manual posting only; no scraping or automation.

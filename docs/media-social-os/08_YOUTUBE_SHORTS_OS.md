# YouTube Shorts OS — دليل يوتيوب شورتس

> Cross-link: [Content Calendar](03_30_DAY_CONTENT_CALENDAR.md), [TikTok / Reels OS](07_TIKTOK_SHORTS_OS.md).

## English — Operating Rules

- **Manual only.** Shorts are recorded, edited, and uploaded by the founder. No auto-posting, no YouTube Data API publishing, no stored credentials.
- Up to 60 seconds. One concept. Clear title text on screen.
- Arabic-first delivery; English subtitles. No emojis. No model name.
- Title and description carry the standard disclosure when the topic is trust-related.

## العربية — قواعد التشغيل

- **يدوي فقط.** يصوّر المؤسس المقاطع ويحرّرها ويرفعها. لا نشر تلقائي، لا واجهة نشر، لا بيانات اعتماد مخزّنة.
- حتى 60 ثانية. مفهوم واحد. نص عنوان واضح على الشاشة.
- الإلقاء عربي أولًا مع ترجمة إنجليزية. بلا رموز. بلا اسم نموذج.
- العنوان والوصف يحملان الإفصاح المعياري عند تناول الثقة.

- [40–45 ث] الدعوة: "اشترك لسلسلة الحوكمة."

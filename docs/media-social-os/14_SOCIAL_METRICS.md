# Social Metrics (Manual Tracking)

No platform API reads/writes. Track manually. Template:

```bash
python scripts/media_social_metrics_template.py
```

## Tracked per post
- date, platform
- impressions, engagement_rate
- profile_visits, saves
- qualified_dms

## Review questions
- Which pillar drives the most qualified DMs?
- Which vertical resonates?
- What converts a viewer into a booked Audit?

All numbers are entered by hand; nothing is auto-collected.

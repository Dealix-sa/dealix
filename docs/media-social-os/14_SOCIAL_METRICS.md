# Social Metrics — مقاييس التواصل الاجتماعي

> Cross-link: [Daily Media Routine](13_DAILY_MEDIA_ROUTINE.md), [Ads OS](10_ADS_OS.md).
> All values are entered manually from platform dashboards. No numbers are invented in this document.

## English — Metric Definitions

| Metric | Definition | Source |
|---|---|---|
| `posts_published` | Count of posts the founder published manually | Manual log |
| `impressions` | Times content was shown | Platform dashboard |
| `engagement_rate` | Engagements ÷ impressions | Platform dashboard |
| `saves` | Saves / bookmarks | Platform dashboard |
| `profile_visits` | Visits to the founder/company profile | Platform dashboard |
| `qualified_inbound` | Inbound the founder marks as qualified | Manual |
| `meeting_booked` | Discovery calls confirmed | Manual |
| `pillar_mix_actual` | Share of posts per pillar vs target | Manual log |

## English — Input Note

This document defines metric names only. No targets, baselines, or results are asserted here. The founder enters actuals weekly from each platform's native dashboard. Estimated figures are labeled estimated; verified figures require the platform export as evidence.

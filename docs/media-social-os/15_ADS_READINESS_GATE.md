# Ads Readiness Gate — بوابة جاهزية الإعلانات

> GO only for planning and copy. NO-GO to launch any paid ad until ALL gate conditions below are met. No ad-platform API is included in this system; any spend is set up and approved manually.
>
> المضي فقط للتخطيط والنص. توقّف لإطلاق أي إعلان مدفوع حتى تُستوفى كل شروط البوابة أدناه. لا تتضمن هذه المنظومة أي واجهة برمجية لمنصة إعلانات؛ أي إنفاق يُعَدّ ويُعتمَد يدويًا.

## EN — Gate

### Allowed now (GO)
- Planning ad campaigns.
- Writing ad copy and reviewing claims.
- Defining audiences and messaging on paper.

### Forbidden until the gate passes (NO-GO)
Do not launch any paid ad until **all** of the following are true:

| # | Gate condition | Met? |
|---|---|---|
| 1 | Landing page tracking is installed and verified | [ ] |
| 2 | UTM taxonomy is defined and documented | [ ] |
| 3 | Conversion event is defined and firing correctly | [ ] |
| 4 | Privacy review is complete (PDPL-aligned, no sensitive data) | [ ] |
| 5 | Budget is approved with a clear cap | [ ] |
| 6 | Audience is reviewed (no prohibited targeting) | [ ] |
| 7 | Claim review is complete (no guarantees, no unproven ROI) | [ ] |
| 8 | Negative keywords list is prepared | [ ] |
| 9 | A named approval owner has signed off (Founder) | [ ] |

If any box is unchecked, the answer is NO-GO. No partial launch.

### Note
No ad-platform API is part of this OS. Account setup, billing, and campaign launch are manual founder actions outside this repository's automation.

## AR — البوابة

### مسموح الآن (المضي)
- تخطيط الحملات الإعلانية.
- كتابة نص الإعلان ومراجعة الادعاءات.
- تعريف الجماهير والرسائل على الورق.

### ممنوع حتى تجتاز البوابة (التوقّف)
لا تُطلق أي إعلان مدفوع حتى تتحقق **كل** الشروط التالية:

| # | شرط البوابة | مُستوفى؟ |
|---|---|---|
| 1 | تتبع صفحة الهبوط مثبَّت ومُتحقَّق منه | [ ] |
| 2 | تصنيف UTM مُعرَّف وموثَّق | [ ] |
| 3 | حدث التحويل مُعرَّف ويُطلَق بشكل صحيح | [ ] |
| 4 | مراجعة الخصوصية مكتملة (متوافقة مع PDPL، لا بيانات حساسة) | [ ] |
| 5 | الميزانية معتمدة بسقف واضح | [ ] |
| 6 | الجمهور مُراجَع (لا استهداف محظور) | [ ] |
| 7 | مراجعة الادعاءات مكتملة (لا ضمانات، لا عائد غير مُثبت) | [ ] |
| 8 | قائمة الكلمات السلبية مُعدّة | [ ] |
| 9 | مالك اعتماد مسمّى وقّع على الموافقة (المؤسس) | [ ] |

إذا بقي أي مربع غير محدد، فالجواب توقّف. لا إطلاق جزئي.

### ملاحظة
لا تتضمن هذه المنظومة أي واجهة برمجية لمنصة إعلانات. إعداد الحساب والفوترة وإطلاق الحملة إجراءات يدوية للمؤسس خارج أتمتة هذا المستودع.

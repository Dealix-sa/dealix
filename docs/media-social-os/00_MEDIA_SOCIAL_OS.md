# Media & Social OS

> Plan and asset library only. **Nothing here auto-publishes.** There is no
> platform API, token, scheduler or auto-post path in this repo. The founder
> publishes manually.

## Purpose
A founder-led, B2B, operations-first media engine for Dealix across LinkedIn, X,
Instagram, TikTok and YouTube Shorts — plus a press kit and an ads *planning*
layer.

## Generate the plan
```bash
python scripts/media_social_calendar_generate.py
```
Outputs `content_calendar.json` + `content_calendar.md` under
`outputs/commercial_launch/<today>/media_social/` with `auto_publish: false` on
every item.

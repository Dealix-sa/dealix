# Creative Brief Library

Reusable brief template for any asset (post, reel, carousel, ad):

- **Objective:** what this asset should achieve.
- **Pillar & vertical:** which content pillar / sector.
- **Audience:** buyer persona/title.
- **Core message:** one sentence (the operating fix).
- **Hook (AR/EN).**
- **CTA:** usually "Book an AI Workflow Audit".
- **Format & length.**
- **Visual direction:** brand palette, clean/professional.
- **Compliance check:** no hype, no guarantees, no scraped data.

## Brief seeds (one per vertical)
- FM: "SLA breaches surfaced before the client complains."
- Contracting: "Catch cost variance early."
- Real estate: "Stop losing listing leads."
- Legal: "Privacy-first intake-to-billing."
- Consulting: "Your pipeline out of one person's head."

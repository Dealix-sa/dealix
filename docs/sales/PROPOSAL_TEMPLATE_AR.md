## 2. الحل المقترح من ديلكس (The Dealix Solution Blueprint)
يقوم محرك **ديلكس (Dealix Engine)** بتفعيل مسار تشغيلي مخصص مدته **14 يوماً** يرتكز على الأبعاد التالية:
* **الأتمتة الذكية**: تفعيل وكلاء الذكاء الاصطناعي التوليدي لأداء الإجراء التشغيلي بسرعة وكفاءة عالية.
* **الحوكمة التامة**: إخضاع كافة عمليات الوكيل لبوابة الحوكمة والموافقة البشرية عبر منصة ديلكس، بما يتوافق مع أنظمة PDPL وSDAIA.
* **إثبات القيمة (Proof Pack)**: إخراج مستند إثبات وتوثيق معتمد غير قابل للتعديل يوضح الأثر التشغيلي والمالي الفعلي للحل.

## 4. الاستثمار المالي وطريقة الدفع (Financial Investment)

لقد صممنا هذا العرض بشكل مقولب وواضح لإلغاء أي تعقيد في التفاوض:

```text
خيار التهيئة السريع (Sprint Setup) 
← [السعر بالريال السعودي، مثال: 5,000 SAR] دفعة واحدة 
← يشمل تفعيل الخدمة كاملة + دعم فني لمدة 14 يوماً حية + حزمة الإثبات.
```

```text
خيار الاشتراك التشغيلي (Monthly Retainer) 
← [السعر بالريال السعودي، مثال: 4,000 SAR] شهرياً 
← يشمل تحديثات الحوكمة الدورية، فحص امتثال دوري، وصيانة النماذج والوكلاء.
```

* *ملاحظة*: الأسعار خاضعة لضريبة القيمة المضافة المعمول بها في المملكة العربية السعودية.

## 5. شروط وأحكام العمل (Standard B2B Terms)
1. **خصوصية البيانات**: يلتزم الطرفان التزاماً تاماً بنظام حماية البيانات الشخصية (PDPL)، وتضمن ديلكس عدم مشاركة أو معالجة أي بيانات PII حساسة خارج الحدود المصرح بها.
2. **الملكية الفكرية**: تعود ملكية البيانات الخاصة بالعميل ونتائج التشغيل للعميل بالكامل، وتظل الخوارزميات وبنية ديلكس التشغيلية ملكاً فكرياً لشركة ديلكس.
3. **طريقة السداد**: يتم سداد 50% كدفعة مقدمة للبدء في العمل، والـ 50% المتبقية عند تسليم تقرير حزمة الإثبات (Proof Pack) بنهاية الـ 14 يوماً.

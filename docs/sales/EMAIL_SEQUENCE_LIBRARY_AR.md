# Email Sequence Library

> **Status:** Draft copy. Native Arabic review required before any send.
> **Trust preflight:** mandatory before founder review.
> **Schema:** `schemas/launch/outreach_draft.schema.json` + `templates/launch/outreach_draft.example.json`.

## The library structure

Each sequence has 4 messages:

1. **First touch** (3–6 lines, peer tone).
2. **Follow-up 1** (3–4 lines, sent at +3 days if no reply).
3. **Follow-up 2** (3–4 lines, sent at +7 days if no reply).
4. **Breakup** (2–3 lines, sent at +14 days if no reply).

The breakup is **important** — it sets up the next touch in 60 days without burning the bridge.

## Rules for all sequences

1. **Subject lines** are short, specific, and not clickbait. No "FREE", no "URGENT", no emoji.
2. **Body** is short. 3–6 lines max. The reader is busy.
3. **CTA** is one clear thing. "رد بكلمة نعم" beats "اختر من الخيارات".
4. **Tone** matches the persona. No "Dear Sir/Madam" in 2024.
5. **No price** in any message.
6. **No attachment** unless explicitly asked.
7. **No fake proof** (testimonials, case studies without permission).
8. **No "guaranteed" or "100%".**
9. **Signature** has name, role, and one link (LinkedIn or website).
10. **All four messages** go through `trust_preflight_dry_run.py` before founder review.

## What NOT to put in a sequence

- Industry jargon the buyer doesn't use.
- "Hope this email finds you well" (it never did).
- "Just checking in" (it never worked).
- A calendar link in the first message (too forward).
- A video in the first message (too forward for cold).
- A "case study" PDF in the first message (they have not earned the right yet).

## The sequence picker

`outreach_draft_factory_dry_run.py` picks the sequence based on the persona field in the target account JSON:

- Owner-Operator → Sequence A
- Head of Sales → Sequence B
- GM → Sequence C
- Skeptic → Sequence D

The founder can override per-deal.

## When to rewrite

- After 20 sent: if reply rate < 10%, rewrite Sequence A.
- After 10 sent: if reply rate < 5%, rewrite that specific sequence.
- If a sequence has 0 replies after 20 sends across multiple personas, retire it.

# Founder-Led Sales Playbook

> **Status:** Operating manual for the founder in the first 90 days.
> **Audience:** the founder. Print, tape, run.

## The principle

> The founder is the closer, the trust signal, and the brand. Not every founder action is a sale — but every sale needs the founder at some point.

## Where the founder's time goes (week 1–4)

| Day-of-week | Time block | Activity |
| --- | --- | --- |
| Sunday | 60 min | Weekly review. Re-score the active list. Plan the week. |
| Monday | 60 min | Approval queue review. Send the day's outreach. |
| Tuesday | 90 min | Discovery calls (1–2). |
| Wednesday | 60 min | Approval queue + follow-ups. |
| Thursday | 90 min | Discovery calls + demos. |
| Friday | 60 min | Approvals + content publishing. |
| Saturday | off | off |

Total: ~7 hours/week. This is the minimum to keep the funnel moving.

## The 5 things the founder does that nobody else can

1. **Send the personal outreach.** Even if a draft is approved, the founder's signature matters.
2. **Take the discovery call.** A founder call converts 2x a non-founder call.
3. **Approve the proposal.** Pricing is founder-only.
4. **Run the close.** Discounts are founder-only.
5. **Write the case study.** Founder's voice in the case study is the proof.

## The 5 things the founder delegates

1. **The research.** Anyone can do the 15-min research per account.
2. **The draft writing.** Anyone can write a draft (with the trust preflight).
3. **The follow-up cadence.** Anyone can send the day-3 / day-7 follow-ups.
4. **The audit data pull.** Anyone with access can pull the data.
5. **The audit report build.** Anyone with the template can build the report.

## The delegation sequence (when to hire)

- **Week 1–4:** founder does everything except research.
- **Week 5–8:** founder delegates research + follow-ups. Founder does discovery + close.
- **Week 9–12:** founder delegates discovery prep. Founder does close only.
- **Week 13+:** founder delegates close prep. Founder does strategic accounts only.

The rule: every delegation must have a written SOP + a measurable quality bar. If you can't measure it, you can't delegate it.

## The 5 founder-level decisions per week

1. **What to send today.** (Approval queue review.)
2. **What to pursue this week.** (Active list re-score.)
3. **What to write a case study about.** (When a client delivers a result.)
4. **What to add to the offer ladder.** (After 5 deals in a vertical, recalibrate.)
5. **What to stop doing.** (When a channel produces < 5% reply rate for 2 weeks.)

## The 5 founder-level decisions per month

1. **What vertical to double down on.** (Re-score `SAUDI_VERTICAL_SELECTION_MATRIX_AR.md`.)
2. **What offer to add or retire.** (Re-score the offer ladder.)
3. **What content pillar to focus on.** (Re-pick from the 10 in `FOUNDER_AUTHORITY_CONTENT_SYSTEM_AR.md`.)
4. **What partner to engage.** (From the partner playbooks.)
5. **What tooling to invest in.** (Re-evaluate the bundle.)

## The 5 signs the founder is doing too much

1. **> 10 hours/week on outreach.** → delegate drafts.
2. **> 5 hours/week on research.** → hire a research VA.
3. **> 3 hours/week on approvals.** → tighten the preflight.
4. **< 1 hour/week on content.** → the founder is invisible.
5. **< 1 hour/week on the weekly review.** → the funnel is unmanaged.

## The 5 signs the founder is doing too little

1. **Approval queue grows > 20 items.** → bottleneck.
2. **Discovery calls < 2/week.** → pipeline is drying.
3. **Closes < 1/month.** → the offer is broken or the lead quality is broken.
4. **No case study in 60 days.** → the offer is not delivering.
5. **No content in 30 days.** → the founder is invisible.

## The first 30 days (the founder's checklist)

- [ ] Read `BEST_FIRST_WEDGE_DECISION_AR.md`.
- [ ] Set the ICP rubric calibration (5 manual scores from your network).
- [ ] Approve the 4 email sequences (A, B, C, D).
- [ ] Send the first 10 outreach messages (manual, in your own tool).
- [ ] Take the first 3 discovery calls.
- [ ] Deliver the first audit (if applicable).
- [ ] Write the first case study draft (no client logo without permission).
- [ ] Publish the first 3 LinkedIn posts.
- [ ] Set the first pricing range (after 5 deals or earlier if obvious).
- [ ] Run the first weekly review.
- [ ] Decide: double down, pivot, or pause.

## The first 60 days

- [ ] Re-score the vertical matrix.
- [ ] Re-score the offer ladder.
- [ ] Build the second case study.
- [ ] Hire/contract the first research VA.
- [ ] Run the first 30-day content calendar.
- [ ] Identify the first partner.
- [ ] Start the first Command Center pilot.

## The first 90 days

- [ ] Define the repeatable revenue motion.
- [ ] Hire/contract the first sales setter or assistant.
- [ ] Build the founder command room (offer 3 + daily digest).
- [ ] Start the first enterprise conversation.
- [ ] Run the first partner co-selling motion.
- [ ] Run the first quarterly business review.
- [ ] Decide the next 90-day plan.

## When to update

- Every 30 days, re-read this doc.
- Every quarter, rewrite the 30/60/90 checklists based on what actually shipped.

# Sales Script (عربي مختصر)

السلام عليكم، أنا أبني **Dealix** كـ **AI Operating Partner** للشركات السعودية.

بدل ما نقدّم «أداة AI» فقط، نأخذ جزءًا واضحًا من التشغيل — **مبيعات، خدمة عملاء، تقارير، أتمتة، أو معرفة داخلية** — ونحوّله إلى **نظام قابل للقياس** مع **تقرير وإثبات أثر** و**موافقات** قبل أي إرسال خارجي.

عندنا **ثلاثة برامج بداية**:

1. **Grow Revenue Sprint** — لترتيب فرص المبيعات والبيانات.  
2. **Automate Work Sprint** — لأتمتة عملية متكررة مع موافقة بشرية.  
3. **Company Brain Sprint** — لتحويل ملفات الشركة إلى مساعد داخلي **بمصادر** (ولا إجابة بدون مصدر).

أقدر أرسل لك **مثال تقرير مختصر** يوضح شكل المخرجات.

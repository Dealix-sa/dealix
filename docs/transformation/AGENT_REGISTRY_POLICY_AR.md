# سياسة تسجيل الوكلاء (Agent Registry Policy)

## الغرض
ضبط حوكمة جميع وكلاء الذكاء الاصطناعي العاملين ضمن بيئة Dealix لضمان عدم وجود وكيل مجهول الهوية أو خارج المراقبة.

## مستويات المخاطر للوكلاء (Risk Levels)
1. **L1 (منخفض المخاطر):** قراءة البيانات العامة فقط. (موافقة تلقائية).
2. **L2 (متوسط المخاطر):** تحليل بيانات داخلية بدون مشاركة خارجية. (يتطلب إشراف أسبوعي).
3. **L3 (عالي المخاطر):** إرسال بيانات أو تنفيذ إجراء خارجي (مثل إرسال إيميل). **(يتطلب بوابة موافقة بشرية Human Approval لكل إجراء)**.
4. **L4 (حرج جداً):** الوصول لبيانات حساسة (PII). (يمنع العمل بدون موافقة مجلس الإدارة).

## متطلبات تسجيل الوكيل الجديد
يجب ألا يتم تفعيل أي وكيل إلا بوجود:
- اسم معرف (Agent ID).
- مالك مسؤول (Owner).
- مستوى مخاطر واضح.
- قائمة محددة بالأدوات المسموحة (Tools Allowlist).

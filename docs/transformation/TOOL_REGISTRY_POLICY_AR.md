# سياسة تسجيل الأدوات (Tool Registry Policy)

## الغرض
التأكد من أن جميع الأدوات التي يتصل بها الوكلاء (APIs, Scripts, MCP Tools) آمنة ولا تشكل خطراً على النظام أو بيانات العملاء.

## مصفوفة تقييم الأداة
| نوع الأداة | طبيعة العمل | الإجراء المطلوب |
|---|---|---|
| أدوات القراءة العامة | البحث في الإنترنت | مراقبة السجلات |
| أدوات الكتابة الداخلية | إنشاء ملفات، تعديل Ledgers | تسجيل تدقيق (Audit Log) |
| أدوات التواصل الخارجي | إرسال إيميل، رسالة واتساب | بوابة موافقة بشرية (Human Gate) إجبارية |
| أدوات استخراج البيانات | سحب داتا، معالجة PII | تقييم امتثال PDPL مسبق |

## الحظر التام
يمنع استخدام أي أداة تقوم بـ:
- مسح البيانات بدون Backup.
- إرسال البيانات الشخصية خارج المملكة بدون توافق نظامي.

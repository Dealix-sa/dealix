# Weekly Revenue Review — المراجعة الأسبوعية للإيرادات

## الغرض (AR)
يحدد هذا المستند إيقاع المراجعة الأسبوعية: ما المقاييس التي يجب فحصها، وما القرارات التي تُتخذ بناءً عليها. الهدف هو تعلّم أسرع وتركيز الجهد على ما يولّد دخلًا حقيقيًا، مع الالتزام بأن كل إجراء خارجي يدوي ومعتمد من المؤسس.

## Purpose (EN)
This document defines the weekly review cadence: which metrics to inspect and which decisions to make from them. The goal is faster learning and focusing effort on what generates real revenue, while keeping every external action manual and founder-approved.

## الإيقاع / Cadence
- مرة واحدة أسبوعيًا، جلسة مدتها 60–90 دقيقة.
- نفس اليوم والوقت كل أسبوع (مثلًا: الأحد صباحًا).
- المُدخل الأساسي: لوحة مؤشرات المؤسس وسجل الأحداث اليدوية.

A single 60–90 minute session weekly, same slot each week, fed by the founder dashboard and the manual events ledger.

## المقاييس التي تُفحص / Metrics to Inspect

| المقياس / Metric | السؤال / Question |
|---|---|
| drafts_generated / founder_actions | هل حجم النشاط كافٍ؟ / Is activity volume sufficient? |
| manual_sends_recorded | كم تواصلًا يدويًا تم فعلًا؟ / How many manual contacts happened? |
| positive_replies_recorded | ما معدل الرد الإيجابي؟ / Positive reply rate? |
| discovery_calls_booked | كم مكالمة اكتشاف؟ / Discovery calls booked? |
| diagnostics_sold / pilots_sold | معدلات التحويل / Conversion rates |
| retainers_started | الدخل المتكرر / Recurring revenue |
| pipeline_value_sar / realized_revenue_sar | القيمة المتوقعة مقابل المحققة / Pipeline vs realized |
| stuck_stage | أين تتعثر الصفقات؟ / Where do deals stall? |

## القرارات / Decisions
- **مضاعفة / Double down**: على القطاع والقناة الأعلى تحويلًا.
- **إيقاف / Stop**: الفرص الضعيفة التي تجاوزت حد الجمود.
- **تعديل / Adjust**: الرسائل بناءً على الاعتراض الأكثر تكرارًا (top_objection).
- **تصعيد / Escalate**: المرحلة المتعثرة (stuck_stage) عبر تغيير في النهج.

Double down on the top-converting vertical/channel, stop weak opportunities past the stall threshold, adjust messaging from the top objection, and address the stuck stage.

## مخرجات الجلسة / Session Outputs
- قائمة قرارات مكتوبة وأصحابها.
- تحديث قواعد خط الأنابيب إن لزم.
- أهداف الأسبوع القادم.

A written decision list with owners, any pipeline rule updates, and next-week targets.

## السلامة / Safety
لا تُتخذ أي قرارات تتضمن إرسالًا تلقائيًا أو ادعاءات غير مثبتة أو ضمانات عائد.
No decision may introduce automated sending, unproven claims, or guaranteed ROI.

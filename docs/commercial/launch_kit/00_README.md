3. **WhatsApp for warm contacts only.** No cold messages, no broadcast lists, no automation.
4. **No invented numbers.** Where `[insert a real figure]` appears, fill it only with evidenced data. If none exists, delete the line.
5. **Log what you sent.** Run [`../../sales-kit/L4_TRUTH_CHECK.md`](../../sales-kit/L4_TRUTH_CHECK.md) before any follow-up that assumes a prior send.

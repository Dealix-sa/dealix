# Content Factory SOP

1. Engine drafts LinkedIn/X/newsletter/video + a one-page insight into the daily
   content queue (`reports/autonomous_growth/content/`).
2. Founder reviews for accuracy and tone.
3. Founder publishes **manually**.

Rules: hypothesis language only ("we expect / the goal is / we will measure").
No guaranteed claims, no fake proof, no auto-post. Every piece carries a soft CTA
to request a custom Saudi Opportunity Snapshot.

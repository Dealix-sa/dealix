# مصنع العروض (Proposal Factory)

## المبدأ

ينتج المصنع **ملخصات عروض (briefs)** وليست عروضاً ملزمة. كل ملخص يحمل:

- النطاق (scope).
- المخرجات (deliverables).
- **نطاق سعري** (pricing range) — لا سعر نهائي.
- خارج النطاق (out_of_scope).
- الجدول الزمني (timeline).
- معايير القبول (acceptance criteria).
- `final_price_allowed = false`
- `approval_required = true`

## الحزم القياسية (نطاقات للنقاش فقط)

| الحزمة | النطاق السعري (ر.س) | المدة |
|--------|---------------------|-------|
| سبرنت بطاقات النمو (7 أيام) | 5,000–12,000 | 7 أيام |
| سبرنت النظام التجاري (14 يوم) | 15,000–35,000 | 14 يوم |
| النظام التجاري المُدار (شهري) | 5,000–25,000 / شهر | شهري |

النطاقات قابلة للتهيئة عبر `data/commercial/pricing_guardrails.sample.json`.

## خارج النطاق دائماً

- واتساب بارد / رسائل بدون موافقة.
- التزامات إيرادات أو ROI مضمونة.
- تسعير نهائي أو خصومات أو شروط عقد بدون موافقة المؤسس.
- إرسال أي رسالة خارجية بدون موافقة صريحة.
- الجمع من مصادر محظورة.

## الإنهاء (Finalization) — A3

تحويل الملخص إلى عرض ملزم بسعر نهائي إجراء **مُقيَّد**. تتحقق
`safety.can_finalize_proposal(...)` من:

```
PROPOSAL_FINALIZATION_ENABLED=true
founder_approved=true
pricing_within_guardrails=true
```

ولا يُحدِّد النظام السعر بنفسه أبداً.

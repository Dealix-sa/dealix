## كيف تُعاد توليدها / How to regenerate (any day)

```bash

python3 scripts/export_outreach_ready.py               # outreach-ready (needs live API)
```

1. افتح الموجز اليومي / Open the daily brief.
2. املأ أعلى 10 خانات في الـ CSV بأسماء حقيقية من شبكتك / Fill the top-10 CSV slots with real names from your network.
3. راجع مسودات اللمسة الأولى، عدّلها، ثم أرسلها يدويًا (5 كحد أقصى/يوم) / Review the drafts, edit, send manually (≤5/day).

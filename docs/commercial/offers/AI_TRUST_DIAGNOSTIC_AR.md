# فحص جاهزية وأمان الذكاء الاصطناعي (AI Trust Diagnostic)

**لمن:** الجهات الحكومية، شركات التقنية، المؤسسات التي تطور أنظمة ذكاء اصطناعي محلياً.
**المشكلة:** مخاطر تتعلق بخصوصية البيانات (PDPL)، هلوسة النماذج، ثغرات حقن الأوامر (Prompt Injection)، وعدم التوافق مع معايير SDAIA و NIST.
**النتيجة:** فحص شامل وتقرير مخاطر محوكم يضمن الامتثال التنظيمي ويؤمّن أدوات ووكلاء الذكاء الاصطناعي.
**المدة:** 7-14 يوم عمل.
**السعر يبدأ من:** 5,000 - 20,000 ريال سعودي.

## التسليمات:
1. مراجعة وتقييم وكلاء الذكاء الاصطناعي (Agent Registry Review).
2. فحص أمان أدوات الربط (Tool Vulnerability Scan).
3. تقرير توافق مع سياسات حماية البيانات الشخصية (PDPL Compliance Report).
4. إعداد مصفوفة الموافقات البشرية والمخاطر.

## Proof Pack:
سيتم تسليم تقرير أمان مفصل يوثق الثغرات المكتشفة، مستوى خطورتها (L1-L4)، التوصيات العلاجية، وتوثيق عدم وجود تسريب لبيانات شخصية (PII).

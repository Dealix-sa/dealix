# Retainer مصغّر — بوابات Proof/Adoption (RETAINER_PILOT_MINI_AR)

> عرض retainer مصغّر لعميل سعودي واحد كبوابة دخول قبل الـ annual contract.
> Client-facing. تصميم: قيمة شهرية محدودة + خروج نظيف خلال 30 يوم بدون التزام سنوي.

## الغرض

إزالة الاحتكاك التجاري للعميل الأول: بدل التزام سنوي بـ 60–120 ألف ريال، يدفع
العميل retainer مصغّر شهري 8–15 ألف ريال لمدة 60 يوم كـ proof-of-value. بعدها
بوابة قرار: adopt (تحويل لعقد سنوي) أو exit (خروج بكل الأدوات والوثائق).

## التسعير

| البند | القيمة |
|---|---|
| Setup (مرة واحدة) | 12,000 SAR |
| Monthly retainer | 8,000 SAR |
| المدة الدنيا | 60 يوم |
| المدة القصوى قبل العقد | 90 يوم |
| Exit fee | 0 |

## بوابات Proof/Adoption

### بوابة 30 يوم — Proof-of-Value

- مؤشر L0 (system live): API + dashboard مفعّلان.
- مؤشر L1 (data flowing): 100 lead على الأقل في النظام.
- مؤشر L2 (workflow proven): outreach drafts + review queue مشغولان.
- مؤشر L3 (one metric moved): KPI واحد محسوب before/after على بيانات العميل.

### بوابة 60 يوم — Adoption Gate

- Founder + 1 champion من العميل يستخدمان النظام أسبوعيًا.
- 3 عمليات أسبوعية مكتملة (outreach → review → send).
- توثيق SOP داخلي للعميل.
- قرار: adopt / extend pilot / exit.

## النطاق

**يشمل:**
- Revenue OS lite (lead scoring + outreach drafts + weekly brief).
- Dashboard واحد بـ 5 KPIs محددة مع العميل.
- مكالمة أسبوعية 30 دقيقة مع founder.
- Audit trail + proof pack نهاية الـ 60 يوم.

**لا يشمل:**
- Custom integrations (تباع منفصلة).
- Trust Pack أو Enterprise compliance (تتطلب BOA4 — راجع
  [BU4_TRUST_ACTIVATION_GATE_AR.md](../enterprise_trust/BU4_TRUST_ACTIVATION_GATE_AR.md)).
- SLA للدعم — retainer مصغّر بدعم async فقط.

## مخرجات الـ Exit

إذا قرر العميل الخروج بعد 60 يوم:
- كل البيانات في صيغة JSON + CSV export.
- كل الـ SOPs كـ Markdown.
- توثيق architecture للنظام اللي شغّلناه.
- لا lock-in، لا cancellation fee.

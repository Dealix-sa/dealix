# Dealix Value-Loop Growth System

Dealix should grow by making useful, shareable artifacts, not by volume-based noise.

## Core loops

1. **Saudi Opportunity Snapshot**
   - A small proof-backed snapshot for a sector, company type, or market-entry case.
   - CTA: request a custom snapshot.

2. **Revenue Leak Scanner**
   - A checklist that helps Saudi companies see where leads and follow-ups are leaking.
   - CTA: run a 7-14 day sprint.

3. **B2G Readiness Checklist**
   - A readiness artifact for companies preparing for enterprise or public-sector opportunities.
   - CTA: request a readiness sprint.

4. **Proof Pack Content**
   - Weekly summary of what was found, what was drafted, what was approved, and what evidence supports the next action.
   - CTA: request a similar proof pack.

5. **Partner Referral Loop**
   - Legal, accounting, marketing, and technical partners receive short intro drafts and referral terms for review.
   - CTA: create a qualified intro.

## Distribution rules

- Low-volume, high-quality, high-signal only.
- Use source-backed claims.
- Keep every public artifact founder-approved.
- Use opt-out language for outbound email.
- No fabricated personalization.
- No fabricated proof.
- No hidden external automation.

# Dealix Success Architecture — The Complete System

---

## المقدمة — Introduction

Dealix ليس موقعاً إلكترونياً ولا خدمة استشارية عامة. هو نظام تنفيذ أعمال مبني على أدلة قابلة للقياس. كل قرار يمر بحوكمة، وكل مخرج يُوثَّق، وكل وكيل يعمل ضمن صلاحيات محددة.

Dealix is not a website or a generic advisory service. It is a business execution system built on measurable evidence. Every decision passes through governance, every output is documented, and every agent operates within defined permissions.

## 4 Scores رئيسية — 4 Core Scores

### Launch Score (0–100)
يقيس جاهزية الإطلاق الأولي. 35 بنداً مُرجَّحاً.  
التقرير: `reports/launch/LAUNCH_SCORECARD.md`

| النطاق | الحالة |
|--------|--------|
| 90–100 | جاهز للإطلاق — Launch Ready |
| 75–89 | إطلاق محدود — Soft Launch |
| 60–74 | تدريب جاف — Dry Run |
| أقل من 60 | غير جاهز — Not Ready |

### Scale Score (0–100)
يقيس الجاهزية للتوسع. يُحسب أسبوعياً.  
التقرير: `reports/scale/ULTIMATE_SCALE_SCORECARD.md`

### Delivery Health Score (0–100)
يقيس صحة خط التسليم النشط.  
التقرير: `reports/delivery/DELIVERY_HEALTH_SCORECARD.md`

### Agent Governance Score (0–100)
يقيس مستوى الامتثال التشغيلي للوكلاء.  
التقرير: `reports/agents/AGENT_PERMISSION_AUDIT.md`

## شكل القرار اليومي — Daily Decision Format

كل صباح يصدر قرار واحد من المؤسس. خمسة خيارات فقط:

| القرار | المعنى | الشرط |
|--------|--------|--------|
| **GO** | تنفيذ الخطة كما هي | Scores كلها أعلى من 70 |
| **HOLD** | إيقاف مؤقت ريثما تُحل مشكلة محددة | خطر محدد في غرفة واحدة |
| **FIX** | إصلاح مكون معطل قبل المتابعة | Delivery أو Agent Score < 60 |
| **SCALE** | توسيع النطاق أو دخول قطاع جديد | Scale Score ≥ 80 |
| **PAUSE** | إيقاف كامل للنظام — مراجعة جذرية | خطر أمني أو حوكمة |

القرار يُسجَّل في: `reports/founder/DAILY_SUPER_COMMAND.md`

## مبدأ Dealix الأساسي — Core Principle

**Dealix يعتمد على الأدلة لا على الثقة بالكلام.**  
**Dealix operates on evidence, not on verbal assurances.**

- كل ادعاء يحتاج مقياساً
- كل وكيل يحتاج سجلاً
- كل عميل يحتاج قبولاً موثقاً
- كل توسع يحتاج جاهزية مُثبَتة

لا ضمانات مبيعات. لا ROI مضمون. الفرص المُثبتة بأدلة هي ما نقدمه.

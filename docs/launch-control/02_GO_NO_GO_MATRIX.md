# Go / No-Go Matrix — مصفوفة المضي / التوقّف

> This matrix is binding. Anything in the NO-GO column is forbidden at launch regardless of pressure or convenience.
>
> هذه المصفوفة مُلزِمة. كل ما في عمود التوقّف ممنوع عند الإطلاق بغض النظر عن الضغط أو السهولة.

---

## EN — Matrix

| GO (allowed at launch) | NO-GO (forbidden at launch) |
|---|---|
| Website public launch | Automated email sending / SMTP |
| Publishing commercial docs and offer ladder | WhatsApp cold outreach or outbound automation |
| Generating 400+/day review-only drafts | LinkedIn automation (connect, message, scrape) |
| Founder review of drafts and top-50 list | Website contact-form auto-submit |
| Manual outreach by founder **after** approval | Bulk sending of any kind |
| Paid diagnostics (499 SAR Sprint) | Paid ads without tracking, conversion event, and compliance review |
| Discovery calls | Collecting sensitive personal data before a signed agreement |
| Proposal generation (manual send) | Sending anything from GitHub Actions |
| Content calendar generation (planning) | Auto-posting to any social platform |
| Manual social posting by a human | Storing email/phone in the CRM (forbidden fields) |

Every approved outreach is sent by the founder, by hand, one at a time. No system in this launch sends external messages.

---

## AR — المصفوفة

| المضي (مسموح عند الإطلاق) | التوقّف (ممنوع عند الإطلاق) |
|---|---|
| إطلاق الموقع العام | الإرسال الآلي للبريد / SMTP |
| نشر المستندات التجارية وسلّم العروض | تواصل واتساب البارد أو الأتمتة الصادرة |
| توليد أكثر من 400 مسودة للمراجعة يوميًا | أتمتة لينكدإن (إضافة، رسائل، كشط) |
| مراجعة المؤسس للمسودات وقائمة أعلى 50 | الإرسال التلقائي لنموذج الاتصال بالموقع |
| التواصل اليدوي من المؤسس **بعد** الموافقة | الإرسال بالجملة بأي شكل |
| التشخيص المدفوع (سبرنت 499 ريال) | الإعلانات المدفوعة دون تتبع وحدث تحويل ومراجعة امتثال |
| مكالمات الاستكشاف | جمع بيانات شخصية حساسة قبل اتفاق موقّع |
| توليد العروض (إرسال يدوي) | إرسال أي شيء من GitHub Actions |
| توليد تقويم المحتوى (تخطيط) | النشر التلقائي على أي منصة اجتماعية |
| النشر الاجتماعي اليدوي بواسطة إنسان | تخزين البريد/الهاتف في CRM (حقول ممنوعة) |

كل تواصل معتمد يُرسله المؤسس يدويًا، واحدًا تلو الآخر. لا يوجد نظام في هذا الإطلاق يرسل رسائل خارجية.

---

## Decision rule — قاعدة القرار

- **EN:** If an action would cause an external message to leave the system without a human pressing send for that specific message, it is NO-GO.
- **AR:** إذا كان أي إجراء سيؤدي إلى مغادرة رسالة خارجية للنظام دون أن يضغط إنسان على الإرسال لتلك الرسالة بعينها، فهو توقّف.

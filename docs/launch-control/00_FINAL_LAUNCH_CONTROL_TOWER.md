# Final Launch Control Tower — Dealix — برج التحكم النهائي للإطلاق

The single command surface for the Dealix commercial launch. This is the index; each numbered file is a working control. The founder runs the tower from Go/No-Go through the first 30 days.

## Governing rule — القاعدة الحاكمة

**EN:** AI drafts, ranks, and recommends. Founder reviews, approves, and sends manually. The system never sends externally.

**AR:** الذكاء الاصطناعي يصيغ ويرتّب ويوصي. المؤسس يراجع ويعتمد ويرسل يدويًا. النظام لا يرسل خارجيًا إطلاقًا.

This rule governs every GO decision below. No item that contradicts it can be approved.

## What launch means here — ما معنى الإطلاق هنا

Launch is the public website going live, commercial positioning being stated, and review-only draft generation being available to the founder. Launch is **not** any system sending external messages. Sending stays manual and founder-approved.

## How to operate the tower — كيفية التشغيل

1. Score readiness with `01_LAUNCH_SCORECARD.md`.
2. Decide with `02_GO_NO_GO_MATRIX.md`. Every GO item must be green; every NO-GO item must be confirmed off.
3. Attach evidence per `03_EVIDENCE_PACK.md`.
4. Run the first 30 days from `04_30_DAY_WAR_ROOM.md` and the daily loop from `05_DAILY_COMMAND_CENTER.md`.
5. Keep `06_FAILURE_RESPONSE_PLAYBOOK.md` open during launch week.

# Founder Daily Loop — الإيقاع اليومي للمؤسس

> A one-page lean daily sell cadence. Run it every working day until customer #1 is closed and delivered.

---

## القسم العربي

### الصباح (45–60 دقيقة)

- [ ] راجع `../ops/pipeline_tracker.csv`: من يستحق متابعة اليوم (D+2 / D+5 / D+9).
- [ ] أرسل 5 رسائل مُخصّصة كحد أقصى (إرسال يدوي فقط، لا أتمتة).
- [ ] سجّل كل رسالة فوراً في المتتبّع (`message_version`, `sent_at`, `next_followup`).

### منتصف اليوم (مستمر)

- [ ] رُد على أي وارد خلال 30 دقيقة — سرعة الرد أهم من سرعة الإرسال.
- [ ] أي طلب توقّف ← الحالة `opted_out` فوراً، لا مراسلة بعدها.
- [ ] احجز مكالمات التشخيص في فترات ثابتة لتجنّب تشتّت اليوم.

### وقت المكالمات

- [ ] أدِر المكالمات المحجوزة عبر [`discovery_and_close_script.md`](discovery_and_close_script.md).
- [ ] بعد كل مكالمة فوراً، املأ [`qualification_scorecard.md`](qualification_scorecard.md).
- [ ] إذا كانت النتيجة ≥ 85، أرسل العرض ورابط ميسر في نفس اليوم.

### المساء (15–20 دقيقة)

- [ ] حدّث `pipeline_tracker.csv`: حالات الرد، المواعيد، خطط المتابعة.
- [ ] سجّل أي احتكاك واجهته اليوم (اعتراض متكرر، خطوة بطيئة) لتحسينها غداً.
- [ ] حدّد رسائل ومكالمات الغد.

### حدود ثابتة

- 5 رسائل مُخصّصة في الساعة كحد أقصى. لا scraping، لا أتمتة LinkedIn أو واتساب.
- لا تواصل خارجي نيابةً عن أي عميل دون موافقة موثّقة.
- لا وعود بنتائج مضمونة في أي رسالة أو مكالمة.

## English Section

### Morning (45-60 min)

- [ ] Review `../ops/pipeline_tracker.csv`: who is due for follow-up today (D+2 / D+5 / D+9).
- [ ] Send up to 5 personalized messages (manual send only, no automation).
- [ ] Log every message immediately in the tracker (`message_version`, `sent_at`, `next_followup`).

### Midday (continuous)

- [ ] Reply to any inbound within 30 minutes — speed of reply beats speed of sending.
- [ ] Any stop request → set status `opted_out` immediately, no further messaging.
- [ ] Book diagnostic calls in fixed slots to keep the day from fragmenting.

### Call time

- [ ] Run booked calls with [`discovery_and_close_script.md`](discovery_and_close_script.md).
- [ ] Right after each call, fill [`qualification_scorecard.md`](qualification_scorecard.md).
- [ ] If the score is ≥ 85, send the offer and a Moyasar link the same day.

### Evening (15-20 min)

- [ ] Update `pipeline_tracker.csv`: reply statuses, bookings, follow-up plans.
- [ ] Log any friction met today (a recurring objection, a slow step) to improve tomorrow.
- [ ] Set tomorrow's messages and calls.

### Fixed limits

- Max 5 personalized messages per hour. No scraping, no LinkedIn or WhatsApp automation.
- No external outreach on any customer's behalf without logged approval.
- No promises of guaranteed outcomes in any message or call.

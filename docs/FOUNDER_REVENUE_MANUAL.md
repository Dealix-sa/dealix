## PART 4: BANT QUALIFICATION FRAMEWORK (20-Point System)

### What BANT Is

**Budget, Authority, Need, Timeline** — a sales qualification scoring system. Use it to decide who to spend time on.

### Scoring (20 points max)

| Factor | Indicator | Points |
|--------|-----------|--------|
| **Budget (5 pts)** | Has discretionary budget? | 5 = "Yes, approved"; 0 = "No" |
| **Authority (5 pts)** | Can they sign contracts? | 5 = "Yes, founder/CEO"; 3 = "Can influence"; 0 = "No" |
| **Need (5 pts)** | Real pain point? | 5 = "Major (10M+ SAR revenue impact)"; 3 = "Moderate"; 0 = "Nice-to-have" |
| **Timeline (5 pts)** | Will decide soon? | 5 = "This month"; 3 = "Q3"; 0 = "Q4 or later" |

### How to Score

**During diagnostic call:**
- "So عندك budget لـ CRM في الـ Q2?"
- "أنت اللي بتوقع الـ contract?"
- "كم عميل بتفقد كل شهر لأن الـ follow-up ما صار?"
- "متى بدك تبدأ الحل؟"

**AI agent scores** the call; you review on `/decisions.html`. Mark ✅ or ✏️ edit.

### Use Cases

- **16-20 pts:** MOVE FAST. Close this week. Use founder time.
- **12-15 pts:** Follow up 2x. If still hot, move forward.
- **8-11 pts:** Use AI agent to follow up. Flag if they heat up.
- **0-7 pts:** End of funnel. Don't follow up.

## PART 12: DECISION TREE FOR COMMON BLOCKERS

### Blocker: "Nobody's responding to my WhatsApps"

**Diagnosis:** Script is probably too long or too formal

**Fix:**
```
Change from: "السلام عليكم ورحمة الله وبركاته. أنا سامي من ديليكس..."
Change to: "السلام عليكم 👋 أنا سامي، نظام مبيعات سعودي. فيه 30 دقيقة مجانية؟"
```

**Action:** Send 3 more using shorter script; see if response rate improves

---

### Blocker: "Diagnostics are booked but people aren't showing up"

**Diagnosis:** Unclear expectations or wrong time zone

**Fix:**
- Ask during WhatsApp: "الساعة 10 الصبح الأربعاء مناسب?" (specific time, Arabic)
- Send reminder 2 hours before call
- Have backup time slots ready

**Action:** Send calendar link + 2-hour reminder for next 3 calls

---

### Blocker: "People say 'I'm interested but need to think about it' "

**Diagnosis:** You didn't handle the actual objection; just moved on

**Fix:**
- "عندك أسئلة؟" (Do you have questions?)
- Wait for REAL objection: budget, timing, competitor
- Address it on the call
- Send video proof for "thinking" types

**Action:** Log objection; improve demo script for next 3 prospects

---

### Blocker: "First pilot is taking longer than 40 hours"

**Diagnosis:** You're over-delivering; setting unsustainable expectations

**Fix:**
- Stick to 14-day timeline; no extensions
- Day 1-3: data load + setup
- Day 4-14: daily support
- Day 14: delivery complete
- If customer needs more, upsell to Managed Ops (monthly retainer)

**Action:** Deliver pilot 1 on time; lesson for pilots 2 and 3

---

### Blocker: "Pipeline feels stalled; no pilots in the last 5 days"

**Diagnosis:** You stopped sending WhatsApps or demo script isn't converting

**Fix:**
- Check: Did you send 5 WhatsApps this week? (if no, start today)
- Check: Last 3 diagnostic call recordings. (if low quality, fix script)

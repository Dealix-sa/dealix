# خريطة إشارات القطاعات | Vertical Signal Map

## الغرض | Purpose

**عربي:** خريطة منظّمة تربط كل قطاع مستهدف بالإشارات التي نرصدها، ونوع مصدرها (يدوي)، والإجراء المقترح. الهدف تحويل الملاحظات إلى قرارات، اعتمادًا على البحث والإدخال اليدوي فقط دون أي تجريف.

**English:** A structured map linking each target vertical to the signals we observe, the (manual) source type, and the suggested action. The goal is to turn observations into decisions — using manual research and input only, with no scraping.

---

## كيف نقرأ الخريطة | How to Read the Map

- **القطاع | Vertical:** السوق المستهدف.
- **الإشارة | Signal:** الملاحظة القابلة للرصد.
- **نوع المصدر (يدوي) | Source Type (manual):** كيف رُصدت يدويًا.
- **الإجراء | Action:** الخطوة اليدوية المقترحة بعد اعتماد المؤسّس.

---

## الخريطة | The Map

| القطاع / Vertical | الإشارة / Signal | نوع المصدر (يدوي) / Source Type (manual) | الإجراء / Action |
|---|---|---|---|
| التجزئة / Retail | شكاوى متكررة من بطء المعالجة | محادثة عميل موثّقة | إعداد موجز فرصة للمراجعة |
| العقار / Real Estate | طلب متزايد على المتابعة الآلية | ملاحظة ميدانية | اقتراح حالة استخدام |
| الرعاية الصحية / Healthcare | حساسية تجاه خصوصية البيانات | قراءة عامة + مقابلة | تحديث رسالة الامتثال (PDPL) |
| الخدمات المهنية / Prof. Services | اعتماد ضعيف على لوحات القياس | استبيان يدوي | تجهيز عرض قيمة |
| الخدمات اللوجستية / Logistics | ضغط على هوامش التشغيل | محادثة شريك | بحث أعمق في السجل |

> كل صف يتطلب مصدرًا موثّقًا واعتمادًا قبل أي إجراء — Every row requires a documented source and approval before action.

---

## قواعد التصنيف | Classification Rules

- صنّف الأهمية: عالية / متوسطة / منخفضة.
- اربط كل إشارة بمصدرها اليدوي.
- لا استنتاجات بلا دليل موثّق.

---

## قواعد السلامة | Safety Guardrails

- لا تجريف ويب ولا جمع بيانات آلي | No scraping, no automated collection.
- لا إرسال خارجي تلقائي | No automated external sending.
- لا أرقام مفبركة ولا ادعاءات غير مثبتة | No fabricated metrics or unproven claims.
- الذكاء الاصطناعي يجهّز، المؤسّس يعتمد، الإجراء يدوي.

# Launch Go / No-Go — The Formal Launch Gate — بوابة الإطلاق الرسمية

The formal gate between stages of Dealix. It holds three things: the Private-Launch GO checklist (what must exist before we sell privately), the Public-Launch NO-GO list (what keeps us off the public market), and the phase ladder from closing the base to a limited public launch. Nothing here invents readiness; a box is checked only when the artifact actually exists.

البوابة الرسمية بين مراحل Dealix. تحمل ثلاثة أشياء: قائمة GO للإطلاق الخاص (ما يجب أن يوجد قبل البيع خاصّاً)، قائمة NO-GO للإطلاق العام (ما يبقينا خارج السوق العام)، وسُلّم المراحل من إغلاق القاعدة إلى إطلاق عام محدود. لا شيء هنا يختلق جاهزية؛ يُؤشَّر البند فقط عند وجود المُخرَج فعلاً.

Read the current stage from the Launch Control Tower ([`../00_platform_truth/LAUNCH_CONTROL_TOWER.md`](../00_platform_truth/LAUNCH_CONTROL_TOWER.md)). Today the Tower reads **No-Go for public launch** — 0 paid customers, 0 Proof Packs. This doc explains why and what changes it.

اقرأ المرحلة الحالية من برج التحكم. اليوم يقرأ البرج **No-Go للإطلاق العام** — 0 عميل مدفوع، 0 حزمة إثبات. يشرح هذا المستند لماذا وما الذي يغيّره.

## 1. Private-Launch GO checklist — قائمة GO للإطلاق الخاص

Every item below must exist before any private selling begins. All ten are required; this is an AND list, not a menu.

كل بند أدناه يجب أن يوجد قبل بدء أي بيع خاص. العشرة كلها مطلوبة؛ هذه قائمة AND لا قائمة اختيار.

| # | GO criterion | معيار GO |
|---|---|---|
| 1 | Platform Source of Truth exists | مرجع حقيقة المنصّة موجود |
| 2 | Command Sprint Offer ready | عرض سبرنت القيادة جاهز |
| 3 | Delivery template ready | قالب التسليم جاهز |
| 4 | Proof Pack template ready | قالب حزمة الإثبات جاهز |
| 5 | Claims Register ready | سجلّ الادّعاءات جاهز |
| 6 | Human Approval Policy ready | سياسة الاعتماد البشري جاهزة |
| 7 | No-Spam Policy ready | سياسة منع الإزعاج جاهزة |
| 8 | First 30 prospects ready | أول 30 عميلاً محتملاً جاهزون |
| 9 | Payment path ready | مسار الدفع جاهز |
| 10 | Founder Daily Command ready | قيادة المؤسس اليومية جاهزة |

When all ten are true, private selling at the Command Sprint launch price (499–1,500 SAR, per [`../01_go_to_market/PRICE_AUTHORITY.md`](../01_go_to_market/PRICE_AUTHORITY.md)) is GO. The 499 is a proof gate, not the company's price. No monthly subscription is offered at this stage.

عند تحقّق العشرة، البيع الخاص بسعر إطلاق سبرنت القيادة (499–1,500 ريال، وفق مرجع التسعير) هو GO. الـ499 بوابة إثبات وليست سعر الشركة. لا يُعرض اشتراك شهري في هذه المرحلة.

## 2. Public-Launch NO-GO list — قائمة NO-GO للإطلاق العام

Public launch stays **blocked** while any one of these is true. This is an OR list — a single true item is enough to hold the gate shut.

يبقى الإطلاق العام **محجوباً** ما دام أيٌّ من هذه صحيحاً. هذه قائمة OR — بند واحد صحيح يكفي لإبقاء البوابة مغلقة.

- 0 paid customers. صفر عملاء مدفوعين.
- 0 Proof Packs. صفر حِزم إثبات.
- Privacy not ready. الخصوصية غير جاهزة.
- Claims exaggerated. ادّعاءات مبالَغ فيها.
- Pricing inconsistent with the Price Authority. تسعير يخالف مرجع التسعير.
- Agents can send externally without approval (non-negotiable #8). الوكلاء يمكنهم الإرسال خارجياً بلا اعتماد.
- Delivery undocumented. التسليم غير موثّق.
- Security smoke fails. فحص الأمان الدخاني يفشل.
- No case study. لا دراسة حالة.

Today, at minimum the first two are true (0 paid customers, 0 Proof Packs), so we are NO-GO for public launch by design. Public launch is not a date; it is the moment every line above is false.

اليوم، الأوّلان على الأقل صحيحان (0 عميل مدفوع، 0 حزمة إثبات)، فنحن NO-GO للإطلاق العام عن قصد. الإطلاق العام ليس تاريخاً؛ هو اللحظة التي يصبح فيها كل سطر أعلاه خطأً.

## 3. Phase ladder — سُلّم المراحل

Stages run in order. A phase is not entered until the prior phase's exit is real and sourced.

تتتابع المراحل بالترتيب. لا تُدخَل مرحلة حتى يكون مخرَج المرحلة السابقة حقيقياً وموثّقاً.

| Phase | Window | Goal | Exit condition |
|---|---|---|---|
| 0 — Close the base | 7 days | All ten Private-Launch GO items true | GO checklist fully checked |
| 1 — First 30 days | 30 days | First paid Sprints + Proof Packs | 3 paid Sprints, 3 Proof Packs |
| 2 — First MRR | 60 days | First monthly conversion | 1 monthly conversion, MRR 2,999+ SAR |
| 3 — Limited public launch | 90 days | Public market, controlled | full NO-GO list false + case study |

**Phase 0 — Close the base (7 days):** finish every GO item. No outreach until the base is closed; selling on an incomplete base creates claims you cannot back.

**المرحلة 0 — إغلاق القاعدة (7 أيام):** أنجِز كل بنود GO. لا تواصل حتى تُغلق القاعدة؛ البيع على قاعدة ناقصة يخلق ادّعاءات لا تستطيع دعمها.

**Phase 1 — First 30 days:** run the daily and weekly rhythm to land the first 3 paid Sprints and produce 3 Proof Packs. KPIs for this phase live in [`CEO_SCORECARD.md`](./CEO_SCORECARD.md): 800+ companies researched, 100+ qualified, 30 manual messages, 5–10 replies, 3–6 diagnostic calls.

**المرحلة 1 — أول 30 يوماً:** شغّل الإيقاع اليومي والأسبوعي للوصول إلى أول 3 سبرنتات مدفوعة وإنتاج 3 حِزم إثبات. مؤشرات هذه المرحلة في بطاقة الأداء.

**Phase 2 — First MRR (60 days):** convert one Sprint customer to a monthly retainer (Starter Command, 2,999 SAR/mo per the Price Authority), only after a Sprint is paid, a Command Pack delivered, and the weekly rhythm confirmed useful. No subscription is offered before that sequence.

**المرحلة 2 — أول إيراد شهري (60 يوماً):** حوّل عميل سبرنت واحداً إلى اشتراك شهري (Starter Command، 2,999 ريال/شهر وفق مرجع التسعير)، فقط بعد دفع سبرنت وتسليم Command Pack وتأكيد فائدة الإيقاع الأسبوعي. لا يُعرض اشتراك قبل ذلك التسلسل.

**Phase 3 — Limited public launch (90 days):** open to the public only when the entire NO-GO list reads false and at least one anonymized case study exists. Even then the launch is limited and controlled, not a broad campaign.

**المرحلة 3 — إطلاق عام محدود (90 يوماً):** الانفتاح للعام فقط حين تقرأ كامل قائمة NO-GO "خطأ" وتوجد دراسة حالة واحدة مجهّلة على الأقل. وحتى حينها يكون الإطلاق محدوداً ومضبوطاً لا حملة واسعة.

## 4. Gate discipline — انضباط البوابة

The gate only moves one way per evidence: a phase advances when its exit is sourced, and reverts if a NO-GO line becomes true again. No phase is skipped to chase a date. Module promises stay inside the Module Status Map ([`../00_platform_truth/MODULE_STATUS_MAP.md`](../00_platform_truth/MODULE_STATUS_MAP.md)) — only the Command Sprint wedge is sellable now.

تتحرّك البوابة باتجاه واحد لكل دليل: تتقدّم المرحلة حين يُوثَّق مخرجها، وترتدّ إن أصبح سطر NO-GO صحيحاً ثانيةً. لا تُتخطّى مرحلة لمطاردة تاريخ. وعود الأنظمة تبقى داخل خريطة حالة الأنظمة — فقط إسفين سبرنت القيادة قابل للبيع الآن.

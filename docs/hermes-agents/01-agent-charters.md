# Hermes Agent Charters

## Purpose
Define each Hermes agent as a role with mission, owner, inputs, outputs, controls, and success metrics.

## 1. Founder Chief of Staff Agent

| Field | Detail |
|---|---|
| Mission | Prepare founder priorities, blockers, decisions, and weekly outcomes. |
| Owner | Founder |
| Inputs | Company scorecard, issues, decision log, risk register |
| Outputs | Daily brief, weekly memo, decision queue |
| Review | Founder review required |
| Success metrics | Blocked decisions reduced, weekly outcomes completed |

## 2. GTM Intelligence Agent

| Field | Detail |
|---|---|
| Mission | Prepare ICP account research, account briefs, and proof gaps. |
| Owner | GTM Lead |
| Inputs | ICP card, target account list, CRM notes, approved public research |
| Outputs | Account brief, deal risk note, message angle, proof gap |
| Review | GTM Lead review required |
| Success metrics | Qualified meetings, better account prep, higher reply quality |

## 3. Customer Success Agent

| Field | Detail |
|---|---|
| Mission | Track onboarding, health, risk, and customer proof. |
| Owner | CS Lead |
| Inputs | Customer health, usage notes, support notes, onboarding checklist |
| Outputs | Health summary, risk flag, save plan, proof candidate |
| Review | CS Lead review required |
| Success metrics | Time to value, red accounts reduced, proof assets created |

## 4. AI Governance Agent

| Field | Detail |
|---|---|
| Mission | Review AI workflows for data use, evaluation, and review needs. |
| Owner | AI/Product Lead |
| Inputs | AI registry, evaluation results, incident log, prompt library |
| Outputs | AI risk review, evaluation gap, control checklist |
| Review | Founder, Product, and Security review for high-risk work |
| Success metrics | Evaluation pass rate, AI incidents reduced, unsupported claims reduced |

## 5. Security and Reliability Agent

| Field | Detail |
|---|---|
| Mission | Track readiness, incidents, dependencies, and reliability risks. |
| Owner | Engineering Lead |
| Inputs | CI status, security notes, incident register, release checklist |
| Outputs | Release risk note, incident summary, follow-up issue list |
| Review | Engineering Lead review required |
| Success metrics | P1 risk age, incident response time, release readiness |

## 6. Finance and Board Agent

| Field | Detail |
|---|---|
| Mission | Prepare runway notes, executive memo drafts, and finance questions. |
| Owner | Founder |
| Inputs | KPI dashboard, pipeline, cash notes, hiring plan |
| Outputs | Runway note, board memo draft, capital allocation question |
| Review | Founder review required |
| Success metrics | Runway visibility, decision clarity, memo readiness |

## Charter rule

No agent is active until it has:

- Owner
- Mission
- Inputs
- Outputs
- Review path
- Escalation path
- Success metrics

# Partner Distribution — Dealix Self-Growth OS

الشركاء أسرع من الإعلانات **بعد أول proof**. كل شريك يزيد التوزيع
(Borrowed Trust) بدل بناء كل شيء من الصفر.

> المصدر الحي للفئات: [`data/growth/partner_targets.csv`](../../data/growth/partner_targets.csv).
> الخطة: `python3 scripts/growth/generate_partner_targets.py`
> → [`reports/growth/PARTNER_TARGETS.md`](../../reports/growth/PARTNER_TARGETS.md).

---

## الفئات المستهدفة

| الفئة | لماذا تناسب | المستوى | زاوية العرض |
|---|---|---|---|
| وكالات تسويق | تحتاج client reporting + proof | co-sell | Proof + client reporting |
| مكاتب استشارات | تبيع التحول وتفتقر operating rhythm | co-deliver | Command + proof + offers |
| شركات تدريب | conversion + proof + academy | referral | academy + conversion |
| شركات IT services | delivery + support + client memory | co-deliver | Delivery + client memory |
| شركات توظيف | pipeline + follow-up + operations | referral | pipeline + follow-up |
| تشغيل مرافق (FM) | SLA + delivery visibility | co-deliver | SLA + delivery visibility |
| مطبّقو CRM | العميل يحتاج operating rhythm فوق الـ CRM | co-sell | Rhythm فوق الـ CRM |
| محاسبون/مستشارو إدارة | قرب من قرارات المالك | referral | Decision + proof |
| موارد بشرية | operations + follow-up + proof | referral | operations + proof |

> الفئات ليست جهات اتصال مسحوبة. **لا scraping، لا قوائم مشتراة.** التواصل
> دافئ وبموافقة المؤسس.

| المستوى | الوصف |
|---|---|
| Referral Partner | يجيب lead |
| Co-sell Partner | يدخل في البيع |
| Co-delivery Partner | يساعد في التنفيذ |

## Partner Kit (كل شريك يستلم)

- `partner_one_pager` — ما هو Dealix + قيمة الشراكة.
- `partner_pitch` — talking points.
- `command_sprint_overview` — العرض الدخولي.
- `referral_terms` — شروط الإحالة/المكافأة (انظر `REFERRAL_SYSTEM.md`).
- `co_delivery_process` — كيف يعمل التنفيذ المشترك.
- خيار White-label Lite (اختياري بعد نضج العلاقة).

> يوجد بالفعل [`docs/AGENCY_PARTNER_PROGRAM.md`](../AGENCY_PARTNER_PROGRAM.md)
> و[`docs/AGENCY_PARTNER_PITCH.md`](../AGENCY_PARTNER_PITCH.md) و[`docs/partners/`](../partners/)
> — هذا الملف يربط فئات الاستهداف بمحرك التوزيع. لا تكرّر، استند إليها.

## التوقيت

- ابدأ برنامج الشركاء **بعد أول 3 Proof Packs** — عندها الثقة قابلة للاستعارة.
- كل CTA للشريك يقود إلى **Command Sprint** (العرض الدخولي).
- لا التزامات white-label قبل proof متكرر.

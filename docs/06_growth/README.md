## أول 10 أشياء للتنفيذ

1. أطلق **Business OS Score** كأداة أولى (انظر FREE_TOOLS_STRATEGY).
2. ابنِ **Command Sprint Landing Page** بمخرجاتها الأربعة.
3. فعّل **تدفق الدفع الذاتي** (Choose → Pay → Intake → Delivery).
4. انشر **Sample Command Pack** مُعقّمًا.
5. أطلق **3 صفحات سؤال/جواب** (Business OS، CRM vs OS، Command Sprint).
6. ابدأ **التقويم الأسبوعي** للمحتوى (الأحد Pain → السبت Review).
7. فعّل **تسلسل 7 أيام** بعد الأداة (opt-in).
8. أعدّ **Partner One-pager** وادعُ 3 شركاء.
9. ركّب **أحداث التحليلات** ولوحة القيادة اليومية.
10. شغّل **أول 10 تجارب** بفرضية ومقياس وقاعدة قرار.

## الحواجز الـ11 (لا تفاوض)

1. لا spam · لا واتساب بارد مؤتمت · لا scraping.
2. لا proof مزيّف · لا scarcity مزيّفة · لا وعد إيراد مضمون.
3. لا نشر خارجي بدون موافقة المؤسس.
4. كل ادعاء يُربط بدليل أو يُعاد صياغته: **تقدير تشغيلي وليس وعدًا بنتيجة**.
5. Approval-first AI: لا مسودة AI تخرج بدون مراجعة بشرية.
6. PDPL: لا تدريب على بيانات العميل · لا نشر اسم بدون موافقة.
7. كل أصل نمو يقود إلى Score أو Diagnostic أو Sprint فقط.

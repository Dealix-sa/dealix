# Conversion Playbook — كتاب التحويل

التحويل ليس ضغطًا، بل إزالة الغموض والمخاطرة. هنا سيكولوجيا التحويل الصادقة، الخطط العالمية الـ12 النظيفة، ومتى (وكيف) ندخل الإعلانات المدفوعة.

> القاعدة: نوضّح المخرج، نقلّل المخاطرة، نصغّر القرار. لا scarcity مزيّفة، لا وعد إيراد، لا "أفضل AI للشركات" كحملة عامة.

---

## 1) سيكولوجيا التحويل الصادقة

| المبدأ | التطبيق |
| --- | --- |
| وضّح المخرج | اذكر تسليمات Sprint الأربعة الملموسة خلال 7 أيام |
| قلّل المخاطرة | "لا نراسل عملاءك · لا ندرّب على بياناتك · لا ننشر اسمك" |
| صغّر القرار | ابدأ Sprint لا عقدًا سنويًا |
| تسعير المؤسس | سعر مؤسس لأول 3–5 لبناء Proof مباشر |
| scarcity صادقة | محدود لأن التسليم يدوي — لا عدّاد وهمي |

كل سطر يخفّض الاحتكاك بحقيقة لا بحيلة.

## 3) الإعلانات المدفوعة (لاحقًا فقط)

**لا نبدأ الإعلانات قبل توفّر كل:**

1. 3 Proof Packs على الأقل.
2. قصة case-safe واحدة (بموافقة).
3. ICP واضح + CTA واحد + landing page محدّدة.

### حملات صغيرة مسموح بها (عند الجاهزية)

- حملة لقطاع واحد محدّد → Sector Diagnostic.
- حملة لأداة واحدة → Business OS Score.
- حملة re-targeting لمن بدأ أداة ولم يكمل.

> **ممنوع:** حملة عامة من نوع "أفضل AI للشركات". غامضة، بلا ICP، تحرق الميزانية وتجلب زوارًا غير مؤهلين.

---

## الحواجز في هذه الصفحة

- لا وعد إيراد أو معدّل تحويل كحقيقة.
- لا scarcity مزيّفة.
- لا إعلان قبل 3 Proof Packs + قصة + ICP/CTA/landing.

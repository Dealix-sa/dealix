## Purpose
Monitor Dealix workers, queues, approvals, costs, and commercial conversion.

## Runtime Metrics
- worker runs
- worker failures
- queue backlog
- report generated
- approval queue generated
- no external send without approval
- suppression check pass

## Commercial Metrics
- leads discovered
- leads enriched
- A leads
- drafts created
- approvals pending
- sends ready
- follow-ups due
- replies
- samples
- proposals
- payment follow-ups
- cash

## SLOs
- Daily CEO report generated by 8:05.
- Approval center generated daily.
- Follow-up queue updated every 4h.
- No external send without approval: 100%.
- Suppression check before sending: 100%.
- Proposal draft within 24h of qualified trigger.

## Alerts
- daily report missing
- worker failed 3 times
- approval backlog > 50
- payment follow-up overdue
- external send attempted without approval

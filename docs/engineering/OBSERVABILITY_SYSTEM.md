# Observability System

## Purpose
Make Dealix automation traceable and debuggable.

## Current Observability
- script output
- generated reports
- audit files
- GitHub Actions logs
- private evidence ledger

## Required Logs
- automation run
- input source
- output file
- errors
- approval requirement
- next action

## Future Logs
- API logs
- worker logs
- agent logs
- user action logs
- audit trail

## Rule
No automation without traceable output.

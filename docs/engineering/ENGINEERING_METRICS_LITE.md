# Engineering Metrics Lite

## Current Metrics
- number of passing GitHub checks
- failed workflow count
- verifier coverage
- compile failures
- data boundary failures
- unsafe automation patterns
- time saved by scripts
- repeated workflows automated safely

## Later DORA Metrics
Use after real deployment exists:
- deployment frequency
- lead time for changes
- change failure rate
- failed deployment recovery time

## Rule
If engineering metrics improve but revenue does not move, engineering is not the bottleneck.

# Client Onboarding Runbook

## Day 0

- Confirm client objective.
- Collect basic files and access list.
- Define sensitive data boundaries.
- Agree acceptance criteria.

## Day 1

- Map current workflow.
- Identify top bottleneck.
- Build initial ledger.
- Produce first command report.

## Days 2–5

- Generate reviewed drafts.
- Prepare proposal/follow-up queue.
- Update proof notes.

## Day 7

- Deliver proof pack.
- Recommend next 30 days.
- Offer retainer or next sprint.

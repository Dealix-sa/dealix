# Quality Bar System

## Purpose
Define the minimum quality bar for Dealix outputs.

## Revenue Quality Bar
- lead matches ICP
- next_action clear
- outreach specific
- no overclaim
- action logged

## Sample Quality Bar
- 5 relevant opportunities
- clear why relevant
- evidence included
- suggested message
- trust note included

## Proposal Quality Bar
- objective clear
- scope clear
- price clear
- start condition clear
- no guarantee
- follow-up date

## Delivery Quality Bar
- intake complete
- lead table complete
- QA passed
- executive summary clear
- next action clear

## CEO Quality Bar
- decision has evidence
- next action clear
- risk considered
- review date set

## Rule
If output fails quality bar, fix before sending.

# Commercial KPI Tree

## Input metrics

- Companies researched.
- Companies verified.
- Drafts created.
- Drafts approved.
- Human-approved outreach actions.

## Conversion metrics

- Reply rate.
- Meeting rate.
- Proposal rate.
- Close rate.

## Delivery metrics

- Time to first report.
- Follow-up SLA.
- Proposal turnaround time.
- Proof pack completion.

## Financial metrics

- Sprint revenue.
- Retainer conversion.
- Monthly recurring revenue.
- Cash collected.

## Trust metrics

- Drafts rejected.
- Claims corrected.
- Opt-outs honored.
- Sensitive data incidents.

# موجز القيادة التشغيلية اليومي للمؤسس (SAMI DAILY COMMAND BRIEF)

مستند التوجيه والتركيز اليومي لـ سامي لإدارة شركة ديلكس والتحكم بالعمليات التجارية والتشغيلية في غضون **10-15 دقيقة** يومياً دون استهلاك طاقته الإبداعية.

---

## 1. طقوس البدء اليومي الصباحي (Sami's 5-min Morning Routine)
أول 5 دقائق من يوم العمل التشغيلي تبدأ بفتح الـ CLI وتشغيل الأوامر التالية:

```powershell
# 1. فحص سلامة النظام وجاهزية المكونات وقاعدة البيانات التراكمية
py -3 dealix.py status

# 2. استعراض موجز العمليات اليومي (معرفة الموافقات المعلقة والمبيعات الساخنة)
py -3 dealix.py command-brief
```

### قائمة مراجعة الفحص السريع (The Command Checklist)
* **هل توجد موافقات بشرية معلقة (Pending approvals)؟**
  * إذا نعم: قم باعتمادها أو رفضها فوراً عبر الـ CLI لضمان عدم تعطل الوكلاء ومعدلات الـ SLA الملتزم بها مع العملاء.
* **هل توجد إشارات ساخنة مكتشفة حديثاً تحتاج لتحويلها لفرصة؟**
  * إذا نعم: مررها لمحرك الترشيح لمعرفة العرض المناسب لها.
* **هل يوجد عميل تخطى مرحلة الـ Sprint ويحتاج إلى إصدار حزمة الإثبات (Proof Pack)؟**
  * إذا نعم: ولد التقرير بضغطة زر وأرسله له لتأمين الدفعة المالية الثانية والـ Retainer.

---

## 2. طقوس الإغلاق والتقييم المسائي (Sami's Evening Compounding Review)
قبل إغلاق جهازك بنهاية يوم العمل، خذ **5 دقائق** لإدخال النتائج التراكمية في قاعدة البيانات لضمان نمو أصول الشركة المستمر:

* **تسجيل الاعتراضات الصعبة**: دون أي اعتراض طرحه عميل اليوم لتغذية قاعدة معالجة الاعتراضات الفورية وتحسين أسلوب الإقناع.
* **تحديث سجل الحركات والوفر المالي**: تسجيل أي قيمة مالية موفرة للعملاء أو مبيعات جديدة لتقوية حزم الإثبات المستقبلية وجعلها أكثر مصداقية.
* **تشغيل المراجعة الأسبوعية التراكمية (مساء كل خميس)**:
  ```powershell
  py -3 dealix.py weekly-review
  ```

---

## 3. فلتر تركيز المؤسس الفوري (Founder Focus Filter)
قبل التزامك أو إطلاقك لأي مبادرة أو فكرة أو مشروع جديد، مررها على **الفلتر السري الفوري** التالي. 

> **لا تلتزم بالفكرة ما لم تجب بنعم على 4 على الأقل من الأسئلة السبعة التالية:**

1. هل تجلب هذه المبادرة دخلاً سريعاً (Cash Now) للشركة خلال 30 يوماً؟ `[نعم / لا]`
2. هل تبني أصلاً تقنياً أو تشغيلياً مستداماً للشركة يمكن تكراره؟ `[نعم / لا]`
3. هل تفتح قناة شراكة وتوزيع سريعة ومجدية (مثل White-label)؟ `[نعم / لا]`
4. هل يقل تدخل سامي البشري في تشغيلها يوماً بعد يوم بفضل الأتمتة؟ `[نعم / لا]`
5. هل تزيد من ثقة العملاء حوكمياً وأمنياً (NIST/SDAIA alignment)؟ `[نعم / لا]`
6. هل تولد بيانات خاصة أو دراسات حالة مشفرة ومقنعة؟ `[نعم / لا]`
7. هل تقود العميل بشكل طبيعي إلى عقد احتفاظ شهري (Retainer)؟ `[نعم / لا]`

* *حالة القرار المعتمد*: **أقل من 4 نعم = تجميد وتأجيل فوري للفكرة لعدم تشتيت التركيز التشغيلي.**

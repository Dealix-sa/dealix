# Template Approval Policy

## Purpose
Define how reusable outbound templates are reviewed before operational use.

## Rules
- Every template must have a clear business purpose
- Marketing language must avoid exaggerated or unprovable claims
- The team should prefer neutral, professional, founder-led language
- Template status should remain visible in the system

## Review Criteria
- accuracy
- tone
- legal/compliance sensitivity
- suitability for the target segment
- explicit reviewer ownership

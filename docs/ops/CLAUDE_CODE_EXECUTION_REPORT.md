## 5. Manual follow-up

1. **Set the internal token in production**

   ```bash
   export DEALIX_INTERNAL_TOKEN="<strong-random-token>"
   ```

   Without this, the internal API runs in `dev_unprotected` mode.
   Production deploys MUST set this (see
   `docs/security/INTERNAL_API_AUTH_GATE.md`).

2. **Bootstrap the private runtime once per host**

   ```bash
   make bootstrap-runtime PRIVATE_OPS=/opt/dealix-ops-private
   ```

3. **Wire the internal router** — handled automatically when
   `api/routers/internal/founder_console.py` is importable; if your
   environment uses an older `api/main.py`, follow the include guard at
   the end of that file.

4. **Configure GitHub required checks** — see
   `docs/security/BRANCH_PROTECTION_REQUIRED_CHECKS.md`.

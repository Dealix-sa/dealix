# Weekly Playbook Update Rule

## Purpose
Ensure Dealix improves weekly from real operating evidence.

## Owner
Sami / Founder CEO.

## Review Cadence
Weekly.

## Inputs
- Weekly Intelligence Review.
- Win/loss review.
- Delivery QA.
- Trust escalations.
- Customer feedback.
- Decision queue.

## Outputs
- Updated playbook.
- Updated checklist.
- Updated template.
- Updated pricing, ICP, or messaging.

## Rules
Every week must update at least one of:
- ICP strategy
- message template
- delivery playbook
- QA checklist
- pricing strategy
- trust policy
- productization candidate list

If nothing is updated, explain why.

## Metrics
- Playbook updates per month.
- Repeated issues reduced.
- Conversion improvement.
- Delivery rework reduction.

## Evidence
- git commit
- weekly review
- updated file
- decision log

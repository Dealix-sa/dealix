
## How to extend
1. Edit `dealix_manifest.yaml` and add a new layer with `required_files` + `required_keywords` + (optionally) `verifier`.
2. Re-run `make everything` — the new layer must PASS before merge.
3. Wire the verifier into `.github/workflows/dealix-everything.yml` if it's a hard gate.

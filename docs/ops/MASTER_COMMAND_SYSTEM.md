# Master Command System

## Purpose
Define the exact commands used to operate Dealix.

## Daily Commands
```bash
make mission-control
make company-check
make stage
```

## End of Day
```bash
make advance
make close-day
```

## Weekly Commands
```bash
make ceo-weekly
make weekly-close
make trust-full
make finance-full
make strategic-decision
make assurance
```

## Monthly Commands
```bash
make board
make monthly
make productization
make pricing
make audit
```

## Commercial Commands
```bash
make revenue-ops
make delivery
make content
```

## Founder Leverage Commands
```bash
make people
make partners
```

## Data Commands
```bash
make data-quality
make snapshot
make company-check
```

## CEO Full Command
```bash
make ceo-full
```

## Rule
If a command fails, fix the failed system before adding a new one.

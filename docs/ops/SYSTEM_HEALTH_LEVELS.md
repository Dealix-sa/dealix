# System Health Levels

## Purpose
Classify the maturity of every Dealix company system.

## Levels

| Level | Meaning |
|---:|---|
| 0 | Missing |
| 1 | Documented |
| 2 | Has template/checklist |
| 3 | Used in private ops |
| 4 | Produces measurable evidence |
| 5 | Improves through weekly learning |

## Systems

| System | Level | Evidence | Next Upgrade |
|---|---:|---|---|
| CEO Command | 1 | founder docs | use daily |
| Revenue | 1 | revenue docs | 25 leads |
| Delivery | 1 | kit | 3 samples |
| Trust | 1 | approval policy | approval log |
| Finance | 1 | model | expenses/cash |
| Learning | 1 | weekly review | first review |
| Productization | 1 | policy | repeated workflow |

## Rule
No system is Level 4 without real evidence.
No system is Level 5 without a learning update.

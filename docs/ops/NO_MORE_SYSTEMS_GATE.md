# No More Systems Gate

## Purpose
Prevent Dealix from becoming a documentation project instead of a company.

## Rule
Do not add new operating systems until market execution evidence exists.

## Required Evidence
- 25 leads.
- 25 DMs.
- 3 samples.
- 1 proposal.
- 1 payment / PO follow-up.
- 1 weekly learning review.

## Allowed Before Evidence
- bug fixes.
- verifier fixes.
- missing file fixes.
- security fixes.
- data schema fixes.
- revenue execution work.

## Not Allowed Before Evidence
- new SaaS architecture.
- new agent system.
- new dashboard version.
- new partner program.
- new hiring plan.
- investor deck.
- mobile app.

## CEO Question
Did this work move cash, proof, retention, trust, learning, or founder leverage?
If no, defer.

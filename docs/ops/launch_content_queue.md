## Execution rules

1. **Do NOT bulk-blast.** Outreach is founder-led and personalized — never automated.
2. **Log every message in `docs/ops/pipeline_tracker.csv`** (or Google Sheets copy).
3. **Respect opt-outs** immediately — add `opted_out` status, never message again.
4. **Target compliant segments only:** founders + business accounts + partner prospects. NO B2C mass DMs.
5. **No promises.** Never state a sales number, conversion rate, or ROI as fact. Use "estimated" or
   "case-safe pattern". Dealix never sends an external message without explicit per-send approval.

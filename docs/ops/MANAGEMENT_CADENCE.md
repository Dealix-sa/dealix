# Management Cadence

## Purpose
Define the rhythm that keeps Dealix executing, learning, and improving.

## Owner
Sami / Founder CEO.

## Daily
- Run stage command.
- Run daily command.
- Execute one revenue action.
- Update pipeline.
- Review approvals.
- Close day.

## Weekly
- Run weekly-close.
- Review revenue metrics.
- Review trust risks.
- Review delivery quality.
- Produce learning decision.
- Update one playbook.
- Commit system improvement.

## Monthly
- Finance review.
- Strategy review.
- Moat review.
- Productization review.
- Capital allocation review.
- 30-day scorecard.

## Quarterly
- Offer strategy.
- Pricing.
- Hiring triggers.
- Partner strategy.
- SaaS readiness.

## Rules
- No day closes without revenue action.
- No week closes without learning decision.
- No month closes without financial review.
- No quarter closes without strategy reset.

## Evidence
- daily brief
- weekly review
- monthly finance review
- strategy review
- Git commits

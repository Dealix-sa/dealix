# Logistics & Industrial Acquisition Playbook — Dealix

## Target buyer

- COO / Plant Manager / Head of Operations لشركات لوجستية أو تصنيعية سعودية بحجم 100–1000.

## Pain hypotheses

- توقف خطوط الإنتاج بدون توقع.
- تأخير شحن متكرر.
- ERP يعمل لكن لا يحوّل البيانات لقرار يومي.

## Signals

- expansion announcement حديث.
- توظيف "Data Analyst" أو "Continuous Improvement Manager".
- استخدام SAP/Oracle + Power BI لكن بدون use case واضح لـ AI.

## Offer

- Data Pack 1,500 SAR كبداية.
- Diagnostic 4,999 SAR للـ predictive maintenance أو route optimization.

## Proof needed

- Sample report من قطاع مشابه (anonymized).
- ZATCA invoice.

## Trust risks

- AIR-001 (hallucination) — توقعات تشغيلية تحتاج evidence قوي.
- REV-001 (single customer dependency) — تنويع pipeline.

## Approval class

- `medium`.

## Sample / Proposal / Follow-up

- نفس النموذج. Follow-up: D+5, D+14, D+30 (B2B ثقيل دورة شراء أطول).

## Objections

| اعتراض | الرد |
| --- | --- |
| "نحن نستخدم ERP" | "Data Pack يحوّل بيانات ERP لقرار، ليس بديل" |
| "نحتاج on-prem" | "Sample فقط offline، deployment يُحدَّد لاحقاً" |

## Disqualification

- شركات بدون ERP/MES.
- شركات بدون قرار شراء فوق 50K SAR.

## الملكية

- Owner: Sales lead.

# SaaS / Software Acquisition Playbook — Dealix

## Target buyer

- Founder / CTO / VP Product لشركات SaaS B2B سعودية وإقليمية بحجم 20–200.

## Pain hypotheses

- منتجهم لا يحتوي على AI features.
- ضغط من المنافسين لإضافة "AI copilot".
- ليس عندهم فريق ML داخلي.

## Signals

- press release عن AI roadmap.
- توظيف "Head of AI" حديثاً.
- تكامل مع OpenAI/Anthropic بشكل سطحي.

## Offer

- Sample integration 1,500 SAR.
- Managed Ops 2,999–4,999 SAR/شهر.
- Custom AI ≥25K.

## Proof needed

- Demo integration على بيئة sandbox.
- Audit log + observability.
- شهادة "Dealix لا ينافسكم".

## Trust risks

- AIR-005 (scope creep).
- AIR-008 (LLM cost) — budget caps صريحة.

## Approval class

- `medium`.

## Sample / Proposal / Follow-up

- Discovery technical.
- Sample = sandbox integration.
- Follow-up: D+3, D+10, D+21.

## Objections

| اعتراض | الرد |
| --- | --- |
| "نبنيه داخلياً" | "نختصر التجربة الأولى من 3 شهور لـ 2 أسابيع" |
| "نستخدم OpenAI مباشرة" | "Dealix يضيف layer التشغيل (eval/audit/trust)، ليس بديل LLM" |

## Disqualification

- شركات بدون MRR > 50K SAR.
- شركات بدون API stable.

## الملكية

- Owner: Sales lead.

# Cybersecurity Acquisition Playbook — Dealix

## Target buyer

- CISO / Head of Security / IT Director في شركات سعودية بحجم 100–2000 موظف.
- NCA-compliant عقلية.

## Pain hypotheses

- تقارير الـ audit تستغرق أسابيع.
- الـ alerts كثيرة، الـ signal منخفض.
- قرارات تقنية بدون evidence قابل للمراجعة.

## Signals

- recent NCA framework adoption.
- hiring SOC analysts.
- مشاركة في Black Hat MEA أو CISO Forum.

## Offer

- **Diagnostic** 4,999 SAR — تحليل بيانات alerts/audit + ROI map.
- Data Pack 1,500 SAR كمدخل.

## Proof needed

- Sample report مع PII redacted.
- شهادة عدم تخزين أي raw security data.

## Draft angles

- "كيف نختصر MTTR من ساعات إلى دقائق؟"
- "Diagnostic لاستخدام AI داخل SOC السعودي".
- "PDPL + NCA compliance بدون تعطيل العمليات".

## Trust risks

- AIR-003 (data leakage) — لا تخزين بيانات raw عند Dealix.
- AIR-005 (scope creep) — Diagnostic فقط، لا اتصال مباشر بـ SIEM.

## Approval class

- `high` — يحتاج founder + legal review.

## Sample / Proposal / Follow-up

- نفس النموذج العام (Discovery → Sample → ZATCA proposal → 50% advance).
- Follow-up أبطأ: D+5, D+14, D+30.

## Objections

| اعتراض | الرد |
| --- | --- |
| "Compliance" | "Dealix لا يخزن raw — كل تحليل offline على عينة معتمدة منكم" |
| "نستخدم vendor عالمي" | "نكمل، لا نحل محل — نحوّل reports إلى قرار" |

## Disqualification rules

- مؤسسة حكومية بدون tender.
- لا budget معتمد (تأخير >6 شهور).

## الملكية

- Owner: Sales lead.
- Reviewer: Founder + legal.

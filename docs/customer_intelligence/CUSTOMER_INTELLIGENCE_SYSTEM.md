# Customer Intelligence System
## Purpose
Turn every lead, reply, objection, call, and delivery into reusable intelligence.
## Inputs
- DMs
- replies
- call notes
- objections
- samples
- proposals
- win/loss reasons
- feedback
## Outputs
- buyer patterns
- objection patterns
- sector insights
- ICP updates
- message updates
- pricing updates
## Rules
- Every rejection teaches something.
- Every objection gets categorized.
- Every win gets decomposed.
- Every sector with weak signal gets killed or deferred.

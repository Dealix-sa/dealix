# حواجز استخدام AI في Dealix

## مسموح
- توليد مسودات.
- تلخيص بيانات غير حساسة.
- اقتراح next actions.
- بناء تقارير داخلية.

## ممنوع بدون موافقة صريحة
- إرسال رسائل للعملاء تلقائيًا.
- اتخاذ قرار مالي/قانوني نهائي.
- معالجة بيانات حساسة خارج الأنظمة المعتمدة.
- تشغيل agent بصلاحيات كتابة كاملة دون حدود.

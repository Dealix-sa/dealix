# SaaS Audit Logging Policy

## يسجل دائمًا
- login/logout
- invite user
- change role
- create/delete/export data
- run agent
- approve external draft
- billing plan change
- integration connected/disconnected

## لا يسجل
- secrets raw
- full message bodies الحساسة
- بيانات شخصية غير لازمة

## مدة الاحتفاظ
MVP: 90 يوم. Enterprise: حسب العقد.

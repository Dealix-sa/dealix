# سكريبت Intake عبر واتساب — دليل محادثة كامل

**Version**: 2.0 | **Last updated**: 2026-06-10  
**الاستخدام**: المؤسس يستخدم هذا السكريبت يدوياً في محادثة واتساب — لا أتمتة، لا بوت.

---

## القاعدة الذهبية

- لا تسأل أكثر من سؤالين في رسالة واحدة
- لا تتابع جواباً بسؤال فوري — اعطِ الشخص لحظة
- إذا كان الجواب ضعيفاً أو غير واضح، استفسر برفق قبل الانتقال
- دوّن كل إجابة في `CLIENT_INTAKE_TEMPLATE.csv` فوراً أو بعد المحادثة

---

## الرسالة الافتتاحية (Opening)

```
السلام عليكم،

شكراً على اهتمامكم بـ Dealix.

حتى نفهم وضعكم بشكل صحيح ونوضح لكم هل نقدر نخدمكم — نحتاج نأخذ بيانات بسيطة عن شركتكم وطريقة عملكم الحالية.

المحادثة قصيرة (5-10 دقائق)، وبعدها نطلع لكم:
• تشخيص مبدئي سريع
• أول نظام مناسب لوضعكم
• هل Diagnostic Sprint مناسب أم لا

نبدأ؟
```

---

## الجزء 1 — بيانات الشركة

**رسالة 1**:
```
أولاً — شوية معلومات عن الشركة:

1. ما اسم الشركة؟
2. ما القطاع؟ (عيادات، تدريب، عقار، مطاعم، خدمات، غيره)
```

**رسالة 2** (بعد الجواب):
```
3. في أي مدينة تتواجدون؟
4. كم موظف تقريباً؟ وكم فرع لو متعددين؟
```

---

## الجزء 2 — طريقة العمل الحالية

**رسالة 3**:
```
الآن — حول طريقة عملكم الحالية:

5. من أين تأتي أغلب الاستفسارات؟
   (واتساب / إنستغرام / موقع / مكالمات / غيره)

6. كم استفسار تقريباً يدخل في الأسبوع؟
```

**رسالة 4**:
```
7. خلال كم ساعة ترد عادة على الاستفسارات؟

8. هل تستخدمون CRM أو نظام لتتبع العملاء؟
   (نعم — وما اسمه / أو لا)
```

**رسالة 5**:
```
9. هل الإدارة تستلم تقرير يومي أو أسبوعي عن الأداء؟
   (نعم / لا)

10. هل عندكم تقييمات على Google أو Maps؟ ما متوسط التقييم تقريباً؟
```

**رسالة 6**:
```
11. هل ترسلون عروض أو proposals للعملاء المهتمين؟
    لو نعم — كم نسبة الإغلاق تقريباً؟
```

---

## الجزء 3 — المشكلة الأساسية

**رسالة 7**:
```
الآن السؤال الأهم:

12. ما أكثر شيء يضيع عليكم حالياً؟
    (استفسارات واتساب / المتابعة / التقييمات / العروض / التقارير / العمليات / غيره)

13. وما الشيء الثاني؟
```

---

## الجزء 4 — الهدف

**رسالة 8**:
```
14. ما الهدف الأساسي اللي تبونه تحققوه خلال الشهر القادم؟

15. وخلال 3 أشهر؟
```

---

## الجزء 5 — الاستعداد

**رسالة 9**:
```
أسئلة أخيرة:

16. الميزانية التقريبية لو قررتم تبدأون؟
    ◻️ أقل من 10 آلاف ريال
    ◻️ 10-25 ألف
    ◻️ 25-75 ألف
    ◻️ 75 ألف فأكثر

17. هل أنت صاحب القرار النهائي؟ أو هناك طرف ثانٍ يشارك في القرار؟
```

**رسالة 10**:
```
18. متى تريدون تبدأون تقريباً؟
    ◻️ هذا الشهر
    ◻️ الربع القادم
    ◻️ لا يوجد جدول محدد
```

## الرسالة الختامية

### إذا كان مؤهلاً (Tier A أو B):

```
ممتاز، شكراً على وقتكم.

بناءً على ما شاركتموه، عندي توصية أولية تحتاج 5 دقائق أشرحها لكم.

توصيتي هي: [اسم النظام أو Diagnostic Sprint]

وهذا مناسب لأن: [سبب مبسط مرتبط بمشكلتهم]

نقدر نحجز Zoom قصير (20 دقيقة) هذا الأسبوع أو الأسبوع القادم أشرح لكم الفكرة الكاملة؟
```

### إذا كان يحتاج تأهيل (Tier C):

```
شكراً، وضعكم واضح الآن.

أرسل لكم ملخص قصير يوضح:
• أين الفرصة الأكبر في وضعكم الحالي
• كيف تبدأون حتى لو الميزانية محدودة الآن

هل تمانعون أرسله على هذا الرقم؟
```

### إذا كان خارج ICP (Tier D):

```
شكراً على الوقت.

وضعكم الحالي يستفيد أكثر لاحقاً لما يكون حجم الشركة أكبر.

سأبقى على تواصل وأرسل لكم محتوى مفيد بين فترة وأخرى.
```

## بيانات تُدوَّن بعد المحادثة

افتح `company/runtime/intake/YYYY-MM-DD/CLIENT_INTAKE_TEMPLATE.csv` وأضف صف:

```
اسم الشركة, القطاع, المدينة, الموظفون, الفروع,
اسم المسؤول, المنصب, واتساب, إيميل, موقع,
الاستفسارات أسبوعياً, القناة الرئيسية, وقت الاستجابة (ساعات),
هل CRM؟, اسم CRM, هل تقارير؟, تكرار التقارير,
هل تقييمات؟, متوسط التقييم, هل عروض؟, نسبة الإغلاق,
المشكلة الرئيسية, المشكلة الثانية, هدف 30 يوم, هدف 90 يوم,
الميزانية, صانع القرار, الجدول الزمني,
[باقي الحقول تلقائية]
```

ثم شغّل:
```bash
python company/intake/intake_engine.py
```

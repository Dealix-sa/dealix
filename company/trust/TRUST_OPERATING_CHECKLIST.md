# Dealix Trust Operating Checklist

## Daily
- Production health passes
- No secret values in logs
- No failed deploy ignored
- No customer data committed
- No outbound automation without approval

## Weekly
- Check GitHub Actions failures
- Check Railway latest deploy
- Check dependency warnings
- Check Gitleaks status
- Check API auth warnings
- Check backup/export readiness

## Monthly
- Rotate exposed or old secrets if needed
- Review data handling
- Review customer access
- Review incident runbook
- Review privacy/service agreement templates

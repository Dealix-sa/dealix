# Daily Execution Map

## Morning
1. Run Master Company Day.
2. Open CEO report.
3. Open approval queue.
4. Pick top 20 prospects.
5. Send reviewed messages manually.

## Midday
1. Reply to warm leads.
2. Book discovery calls.
3. Push Diagnostic Sprint.
4. Update CRM.

## Evening
1. Update statuses.
2. Record replies and objections.
3. Generate proposal stubs for warm leads.
4. Review tomorrow’s sector.

# CEO Decision Rules

## Rule 1: No custom work without paid diagnostic or deposit
Every custom request starts with:
- Diagnostic Sprint, or
- Setup deposit, or
- Signed scope

## Rule 2: Sell transformation, not tools
Never lead with "dashboard", "bot", or "CRM".
Lead with:
- lost revenue
- weak follow-up
- unclear reporting
- scattered WhatsApp
- unmanaged reviews
- inconsistent brand
- operational leakage

## Rule 3: One first workflow only
Every client starts with one measurable workflow:
Lead → qualification → follow-up → booking/proposal → weekly report.

## Rule 4: Founder approves external actions
The system drafts. Founder approves:
- message sending
- proposal sending
- invoice/payment
- client commitments

## Rule 5: Every delivery becomes reusable
Every client output becomes:
- template
- SOP
- case study
- dashboard card
- report format
- objection intelligence

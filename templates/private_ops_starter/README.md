# Private Ops Starter

Copy the contents of this directory into a separate, private repository
(e.g. `dealix-ops-private`). These files contain CSV ledgers and review
templates the CEO operating model expects. Nothing here is meant to be
committed to the public Dealix repo — keep cash, pipeline, MRR, and
expenses in the private repo only.

Expected target layout (after copying):

```
dealix-ops-private/
├── finance/
│   ├── expenses.csv
│   ├── runway_estimate.md
│   └── monthly_finance_review.md   # generated by `make finance`
├── revenue/
│   ├── cash_collected.csv
│   ├── pipeline_value.csv
│   └── mrr_tracker.csv
└── founder/
    └── weekly_ceo_review_template.md
```

Then run from the public repo:

```
make finance PRIVATE_OPS=/path/to/dealix-ops-private
make ceo-verify
```
